Bretonian Bulldog

Tue Apr 06, 2004 5:44 pm by Bretonian Bulldog

Hmm. It seems to me that you've defined the room properly in your system's BASES folder ini., and so I'd guess that the problem is somewhere in the corresponding room ini. file in the ROOMS folder. My reasoning is that Freelancer knows that your base has a Shipdealer because it shows the icon on the top bar (so it must be defined properly in the base's BASES ini) but doesn't know how to create it (so a problem somewhere in the ROOMS ini seems likely.)

This is just a guess really though, as I haven't had that exact problem myself before.

EDIT: Actually, now that I think about it there could be a problem in the BASES ini too. It may be telling the game to look in the wrong place for the corresponding rooms. My advice would be to rigourously check and double-check both files.
