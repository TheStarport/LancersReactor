The use of signatures and Pictures!

Signatures... almost everybody uses them (and should). Most of the time a signature will make a quick reader know that you're participating in the discussion. Some use text signatures, other use pictures or a combination of those two.

There aren't any written rules to follow when you use a picture in your signature. There are some size and dimension restrictions but that's it. You're free in what picture to choose, just as long as it's not violent of sexually oriented.

You can put your signature in your post by ticking the "Check here to include your profile signature" box. This brings me to my main point of this thread. This has been brought up before but I want everyone to know.

Please use your signature, if it's a picture, only once per thread (like in your first post). So everybody will know that you're participating in the discussion.

Text-Based signatures are welcome to be used, but NOT in every post. Please refrain from using the text based sig more than once per page of a thread.

You're absolutly free to decide wether or not you will follow this rule. It's not official but I think it will makes the forum look smoother.

Regarding Pictures on this, and other, forums

Please follow these guidelines:

1. once you have taken your screenshot - use PAINT (note - NOT photoshop) and resize it to around 800 x 600 and then SAVE it as a JPEG. This will make it around 50Kb per screenshot. USE paint, or if you HAVE to use photoshop - use a POOR quality setting, YOU CANNOT TELL THE DIFFERENCE FOR THIS IDEA, as this will keep them SMALL in size (around 50kb instead of 200kb).

2. Once you have saved it, now RESIZE the picture AGAIN to around 250x175 or so. Then SAVE THIS AS A JPEG FILE AND CALL IT THUMB!!!!!

So then you have two files:
1. Desktop.jpg
2. Thumb.jpg

Now - use some nice coding from the forums to create the IMAGE of the thumb on the forum which is an ACTIVE LINK to the picture size. This will ensure that loading this thread will take seconds, EVEN for a 56k modem user. So they see a small version (only 16kb or so) and click if they want to see more. Then they can see the full sized version (which can be any size you like) in a new window.

See my example below



The code for this is (*subsitute square brackets instead of the curly ones)
{url=http://www.pathfinderstudios.com/PICTURES/29.JPG}{img=http://www.pathfinderstudios.com/PICTURES/29thumb.jpg}{/url}

There we go, obviously change the links to YOUR host of pictures
The image above is only about 100x75 - so yours will be alot bigger of course - which is nice to . In effect you could do this in a thread containing one picture or around 300 x 225 or so - which is linked to the full size version
