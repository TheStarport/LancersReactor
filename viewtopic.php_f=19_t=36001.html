Multiplayer Questions

Hello, I was wondering if any of you can answer my questions regarding multiplayer.

1. When you are a pirate and destroy a ship coming through will it drop what it had in its cargo hold? and if it doesnt are there any mods that enable that?

2. When starting up a server there is a checkbox that says "allow new players" if you uncheck that box can players join? Do they use a character from another server?

3. Is there any way to join or create another faction besides "navy" and if so how?

4. is there any way to make factions all the way love you or hate you? my bar is always one square away from becoming all one way or the other.

Please help me with any of these questions, any answers are accepted, thanks.

I'll try.

1. When you destroy an AI ship (called NPC or Non-Player Character) then usually yes, it's cargo drops and some of it's equipment. Not all the time but sometimes depending on many factors but usually an internal "roll of the dice". If you mean does a player character drop cargo, no, unless he does it himself. There are mods that enable this though as if the character is like an NPC. EVO is one I know of.

2. If you uncheck new players then anyone logging on the server cannot create a new character! They can play one that has already been created on that server. Characters on one server are NEVER found on another server unless the other server's sysadmin got a copy of the FL MP saved game file and copied it into your account. Rarely, if ever, done.

3. You start with the Liberty Navy rep at +0.9. This makes your visible faction tag say Navy. If another rep is at +0.9 then you'd see that faction tag instead. This rarely happens even when you do everything to maximize a rep. It's more easily done from the server where an FL admin program (like IFSO) can set any reps to anything (+1 to -1) you want.

4. As much as you might attempt to maximize a faction rep, I have never seen it move above +/-0.91. You can manually set it to the maximum of +/-1.

Hope that covers that!
