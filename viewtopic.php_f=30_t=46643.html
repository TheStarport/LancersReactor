FL System Calculator

<span style="text-decoration: underline">FL System Calculator © by RimShot</span>

Link

<span style="text-decoration: underline">Description:</span>
This tool allows to scale/move the content of entire systems.

<span style="text-decoration: underline">Example:</span>
New York at scale 200% with settings for normal planet size.
Link

<span style="text-decoration: underline">Instructions:</span>
1. Upload a system .ini file
2. Adjust settings
3. Click Calculate
4. Focus on textarea, Ctrl+A, Ctrl+C to clipboard
or Save Target As... to HDD

<span style="text-decoration: underline">Options:</span>
Scale LightSource range
- self explaining.
Recognize Object parent
- this will fix the child <-> parent distance
Fix dockable Object to planet
- this will fix the ring/moor <-> planet distance
Scale Zone size
- self explaining, the following options are sub
Scale death/atmosphere Zone size
- death zones are found by damage (greater or equal 1000000, which can be lowered if necessary), atmosphere zones by taking the same place (position). not perfect, but should do for now.
Scale vignette_type Zone size
- yet another option, self explaining.
Scale cylindrical Zone radius
- you might not want pathes to get thicker.

<span style="text-decoration: underline">Attention:</span>
You have to edit NavMapScale manually in universe.ini.
The SP Storyline will be screwed, as positions wont match anymore.
While its possible to make a tool for editing mission files, im not planning to, so i recommend the usage for MP and/or OpenSP only.

<span style="text-decoration: underline">Notes:</span>
Dont blame me for errors, report them and i will see what i can do.
If you are using this tool for your mod, please give a little credit -thanks, apprechiated.
