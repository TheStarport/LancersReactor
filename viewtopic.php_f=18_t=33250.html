Riely

Mon Jul 26, 2004 2:39 pm by Riely

It is in /Freelancer/DATA/Characters/New Character.ini
You need to modify the [package section.

[package
nickname = ge_fighter <--Change this to the longsword reference in Shiparch.ini
strid_name = 11051
strid_desc = 11551
ship = ge_fighter <--Same as above
loadout = msn_playerloadout <-- Change to the longswords loadout in loadouts.ini
money = 5000 <--- Can always change how much money you have and buy the ship as well :-p

Keep in mind, if you do this, it will only affect your ship (I believe), so you will need to make the mod downloadable for others, unless you don't want them to start with it.

Hope that helps.

-Riely
