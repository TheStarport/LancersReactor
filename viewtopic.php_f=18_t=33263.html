Chips

Sun Jul 25, 2004 3:28 pm by Chips

Make the weapon have NO weapon_goods.ini entry - that way they cannot see it. Now mount it on the loadout in the goods.ini file on one of the hardpoints for that ship (like HpWeapon01).

Now go to the shiparch.ini file and remove the HpWeapon01 reference for that ship (so that players cannot see the mount etc).

Now you have a non transferable, unmountable weapon. By the way, if you have the power usage set to max it should take the power of the plant down. SOmethign is wrong if it doesn't.
