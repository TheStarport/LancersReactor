Thu Jan 06, 2005 11:47 am by Boscoe

With computers, often the best cure is to kill the patient. Before you do though, if it's not too late, try redownloading Direct X from MS's site (not windows update). Open "run" the type "dxdiag" (without quotes) and see if you have errors there. Just sounds suspiciously like a corrupted DX file.

Edit: Just had the same thing happen. Ran SDK 1.3 and it stopped. Download SDK 1.3 (in downloads section) install and run it. Then try again. Worked for me, and I am using mods.
