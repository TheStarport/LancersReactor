MUUUAAAHHHHAAAA!!!

I did it! With the 1 GB DDR memory rated and running at 400 MHz (FSB 200 MHz) I got the bloody Athlon XP 2500 Barton to run fast! Since it is multiplier locked it runs at 11.5x200 MHz and now it looks like an Athlon XP 3200! The trick? Upped the CPU voltage from 1.675 Vdc to 1.70 Vdc. (To novices out there this negates all warranties! You have been warned!) I put on a big honking copper heatsink and it runs coooool! Been running for 3 days with no problems.

So Joe? AMD rules again, this server screams and I need no other CPU, thank you.
