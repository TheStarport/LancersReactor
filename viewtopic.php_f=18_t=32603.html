capitalmerc

Sat Jul 24, 2004 5:21 am by capitalmerc

add a large_moor to your base you see the format for it in solararch.ini

so open up solararch.ini and find your base. then replace one of the docking mounts(A/B/C/D etc..) with a jump or large_moor depending on what effect you want. the large moor all cap ship to dock quickly, and i have extended the distance from the base that you need to be to activate it so that capships dont get stuck. the jump will allow all ships to dock as described above.

docking_sphere=jump, HpDockMountA, 600 ;add this line to allow all ships to dock quickly when within 300m of the base.

docking_sphere=moor_large, HpDockMountA, 600 ;add this line to allow all ships that dock with large moors to dock quickly when within 300m of the base.



other examples

docking_sphere=moor_medium, HpDockMountA, 600 ;add this line to allow all ships that dock with medium moors to dock quickly when within 300m of the base.

docking_sphere=berth, HpDockMountA, 300 ;add this line to allow all ships that dock with berths (this is fighters) to dock quickly when within 150m of the base.



the problem you are having probably come from the docking range, if the moor for the big ships is set so that you only dock when 4m away (default setting i think) then the large ship (because of poor handling, and huge size) will need to spend a long time manouvering to get close to the base, by increasing/playying with the docking sphere range you can create the effect that you want.
