In my experience, no you can't run both the client and the server on 1 machine. I've done it with a P4 3Gig and a Gig of DDR3200 RAM and i constantly crashed the server and the lag was huge. Others may of had a different exprience but thats mine.

In answer to question 2 - I also have an athlon XP1800 with 512mb PC133 RAM which i use to host my LAN games. this takes 10 users (just). I think it would do more but the memory bandwidth is useless due to KLE133 chipset.

Qustion 3.... I run a 24/7 server (see below) it's unmodded and rarely crashes. when it does crash, the server launcher starts it up again so down time is about 30 seconds! Catch is it's pass worded. But CheatersSuck is the pass!

as for the 56k dial up, the numbers say it should be fine but again in my experience, i have a much better time with ADSL!

Phew... I think thats covered it!

FWIW, I run Freelancer in the UK (imported) on a dual Celeron 500MHz machine on an aging Abit BP6 mobo. (vader.fragLANd.com)

I've capped it at 6 players since it's only on a 512/256 ADSL connection - but I've played with 4 net players + 1 (myself) on LAN with no lag issues.

One of the regulars even connects via modem from Australia!
He does fine in dogfights etc, but doesn't join formations as the ping differences between LAN and Oz dial up means he crashes into me when changing direction

I don't know for sure, but I doubt that FL is multi-threaded - so ppl should experience similar performance on a single CPU 500MHz?
