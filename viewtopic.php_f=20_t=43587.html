Fraid I have no idea, is your disk scratched? I'd suggest a house clean by using disk defrag, followed by cCleaner to remove any excess registery entries etc. Sure, it most likely won't sort it out - but it can't hurt can it.

After all that, i'd run other software in the disk drive (for fluff n dust) first, and then try the reinstall once more.

To be honest, I haven't seen any single one reason for errors in installation at all, they are tough ones - possibly corruption (how old is the disk?). You can buy freelancer for a couple of quid these days, so it maybe worth while investing in another disk if it still doesn't work.
