If you don't want a real mod, get the IONCROSS character editor. Then you can start the game with any ship, any weapon, and start from any location. You can also use it to open the gates and holes from the start of the game and show you the whole map from the beginning. Also, you can adjust youreps with all factions to start a new game affilliated with a different faction.
I've played the storyline with every fighter in the game. Only times I was unable to complete it was when I was flying the Startracker, Pirhana, Patriot, and Starflier. Have fun.
