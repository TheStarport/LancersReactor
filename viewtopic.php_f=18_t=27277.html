For Mod Makers
FLScan Version 1.3 Freelancer Mod Scanner by Accushot
FLScan is a program that will scan your modified Freelancer ini files looking for errors, saving you the time and headaches that this usually entails. You can direct the program to scan certain areas of your project or to simply scan the whole lot. An error report will be created which details the problems, common causes and possible ways to fix the problem.
Another new feature introduced since version 1.2 is FLScan's ability to repair THN files that have been corrupted by the Fled-Thorn decompilation process. When the files are decompiled certain fields are incorrectly decoded and as a result the resulting scene is often completely unrecognisable.
The PathBuilder is a new feature introduced in version 1.3 that can generate pathfinding information for a mod automatically. As well as saving you time this should help eliminate errors caused by incorrect pathfinding files, such as AI and navmap errors that can be serious enough to crash the game. In order to do this FLScan will inspect your universe files and attempt to plot paths around the universe, travelling as a player or NPC might do.
Download FLScan v1.3 here.

FL Rotation Calculator (FLRotCalc) by Erik McClure
(FLRotCalc) is a command line utility used for determining the rotation necessary for one object to point to another object.
You enter in a first set of coordinates, and then enter in the second set of coordinates.
The program will return rotational values that make the first object point to the second set of coordinates.
The most practical use of this is Tradelanes.
Put the value of the first tradelane in, then the value of the second tradelane in,
and then assign the resulting rotational value to all of your tradelanes.
They should match up in a perfect line! Useful if you need to calculate the rotations for tradelanes that go up or down.
3D patrol paths are also an application of this.
As long as you copy the position values into the program as you see them in the INI (minus the spaces), of course it should work like a charm.
If it crashes on you, check the format of the coordinates your feeding it, and make sure there are no spaces, no weird characters, only the coordinates, a comma separating each of the 3 coordinates, and negative signs where necessary.

Download the FL Rotation Calculator (FLRotCalc) here.

Freelancer Server Manager - server version by Buck Rogers
Here is a brief overview of what it does.
The package incorporates excellent cheat prevention but needs the player to install a small client for it to work. (Microsoft .Net Framework v2.0 is needed) The cheat prevention uses a CRC checksum on crucial ini files and checks with the server before allowing the player to connect.
It has built in banning, kicking and kick to bases. There is a nifty little auto unban function that will allow you to enter a time and date for banned players to be unbanned automatically and it will announce the unbans to the freelancer universe.
It has built in Server Monitoring so if your server crashes it will shut down that awful error reporting window and restart your server. It has a built in timed restart
option to restart your server every X hours.
It maintains a list of online players which displays their location, their ship and how many credits they have.
It has a very comprehensive ship editor for changing ships and equipment and will even detect if a player has multi mounted during heavy lag and give you the option to remove the excess.
There is a similar cargo/equipment editor.
There is a rep editor that will allow your players to have those faction tags at the front of their names.
A banners section that allows up to 34 individually controlled announcements.
View Accounts by character name. A nice touch here is that the cheat prevention system uses a players MPAccountID as it's username and so stores their Account key server side so you will never get players saying that they have reinstalled and lost their characters. You can now retrieve this server side.
It has a built in stat generator that allows you to create 5 different pages filtered by Kills, Missions, Money, Time Online and by Account. It also allows you customize how the page looks.
