redeye

Tue Aug 05, 2003 2:37 am by redeye

At the time if i had removed the 3 intro movies it would have made the mod incompatible with any major mod.

However as FLMM 1.2 will allow mods that alter the same .ini files to coexist i should probably upload a new version.

Anyway to remove the intro movies you must find the following lines in 'freealncer.ini' located in your .../Freelancer/EXE/ directory :

{Startup}
movie_file = movies\MGS_Logo_Final.wmv
movie_file = movies\DA_Logo_Final.wmv
movie_file = movies\FL_Intro.wmv

(i have replaced square brackets ( [ ) with curley brackets ( { ) as the forums do not correctly display square brackets)

Delete all the 'movie_file' lines and the intro movies will be removed. Some versions of BiniQDU do not correctly save Freelancer.ini so you might have to save it as text, edit then save as Freelancer.ini to get it to work.

Chips

Tue Aug 05, 2003 5:52 am by Chips

yeah - open up the exe folder, and then open the freelancer.ini file, scroll down a little until you see the following lines:

[Startup
movie_file = movies\MGS_Logo_Final.wmv
movie_file = movies\DA_Logo_Final.wmv
movie_file = movies\FL_Intro.wmv

Then simply put a couple of semi-colons in front to deactivate them - like so:

[Startup
;;;movie_file = movies\MGS_Logo_Final.wmv
;;;movie_file = movies\DA_Logo_Final.wmv
;;;movie_file = movies\FL_Intro.wmv
Save and quick starts from now on
