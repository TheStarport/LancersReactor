GMAX\3DS Help Request - Jenova Class Ships Project

Hey guys... I got a small problem. Let me explain:

First off, my project idea is to create a whole set of Jenova Class ships, from Light Fghters to Capital Class, all bearing an identical structure peice, giving the name, Jenova, a more realistic class. I see tons of new ships, out on the net each having their own class but they have no other types of the same class, making them look like an isolated lost puppy. I plan to change that with the new Jenova Class series of Ships.

What I need is either someone to take the DXF Exports, make a few modifications in regards to my problem below, and do all the complicated stuff such as texturing, infocard creation, etc...

My creations won't stop here. I'll be using VHE(Valve Hammer Editor) to create even more Class Sets of ships, bring a whole new meaning to the word, "Ship Mod"...

Now on to my problem.

I use ValveHammerEditor to create very detailed ships, but since VHE is NOT a Modeling tool, I've encountered a problem that when I export from VHE to DXF(Autocad) format, and import that into GMAX, doing some required shrinkwrapping and scaling down, then, here is where the problem is...

I import the GMAX 3DS I exported into MS3D... When I go to create the materials for the skins, and applythem to the model, the texture is scaled to fit the size of the face on each face of each polygon instead of treating all faces together as a single face and scaling the texture to fit.

So, I am asking if anyone can use the DXF Exports from VHE I have made and get a basic texture to properly mesh on the model.
____________________-

Another problem I run into is when I do the transparency effect for the cockpit glass, since VHE does not have the ability to do single polygon spheres, I instead had to use multiple polygons to get the rounded effect, as such, when made to be partialy transparent in 3DS, you see all the faces of each of those polygons inside the cockpit and literaly ruins the effect of glass, and ruins the point of even making it partialy transparent to begin with.

If anyone could somehow create an exact replica of the cockpit glass using only 1 polygon, this would surely solve my problem.



CONTACT ME VIA EMAIL to get my DXF Exports so you can see what you can do. And if by chance you do figure out solutions to both or either of my problems, please let me know how you did it.
