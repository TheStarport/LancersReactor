Server "Handshaking" & Mod Sharing

I had began to wonder what else could be done in the Multiplayer Universe that could add more depth and scope to this aging but tirelessly popular game.
Now, besides adding height to an already 3 dimensional universe and some of the most facinating creative ideas I've ever seen, this one idea came to mind.

I think a global server is not made up of just one server, but several servers that are working synchronously. So, here is what I was thinking...

A Server Network

A group of servers running complimentary mods that handle different aspects of the game. Instead of possibly "overloading" one server, several servers share the load in a term called, "Handshaking" and "Handoff".

This probably involves the risky operation called "file sharing".
But, I wonder if this is possible to do.
I'd like to know all the pros and cons on this.
