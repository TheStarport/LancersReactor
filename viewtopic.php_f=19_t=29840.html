Pinger

Fri Apr 02, 2004 3:09 pm by Pinger

your describing hardware lag, ie poorly setup windows and pc.

1. Defrag active HD's once every 2 weeks
2. Set the Page/Virtual memory file to a Fixed Miimum and Maximum size, dont let windows control it. Rough guide is 2-2.5 times your Mobo's physical Ram but anything over 768 isn't needed, so if your mobo has over a 512 ram just set min/max both to 512.
3. Don't have msn/icq/aim etc running while gaming, system hogs and prone to memory leaks.
4. Dont have any Office utils running from the startup folder, same reason.
