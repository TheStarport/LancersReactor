The archetype will need to match the name of the ship in the shiparch.ini file(rs_viper), and then the loadout will need to match the ships hardpoints (either look for an existing loadout for this ship using the search function on the loadouts.ini file and search by the ships nickname for anohter loaodut, or us the UTF editor to find out the names of the hardpoints.
Once you have done that, you can make sure it has the hardpoints, IF in doubt do this:

nickname = MP_ge_fighter
archetype = rs_viper
equip = ge_gf1_engine_01
equip = ge_fighter_power01
equip = ge_s_scanner_01
equip = ge_s_tractor_01

Because this means nothing is mounted. However, it does have a crap power supply (starflier) so you might be best off finding another one. Same with engine if this is for a custom mod game etc. You will start with a bare ship and nothing else, not even weapons - but this way you have nothing "mounted" on mounts, which means its "safe". You can give them cargo for the rest of the stuff, and let them mount it instead.

cargo = ge_s_battery_01, 3
cargo = ge_s_repair_01, 3
cargo = li_gun01_mark01
cargo = li_gun01_mark01

Now they start with 3 shield bats and nano bots, and two crap guns that they have to mount.
