I am a n00b so don't listen to me but what you do is:

1. make sure you use bini and unencrypt the \data\equipment\market_ships.ini file

2. find the code for the new berlin base under a "BaseGood" tag (god knows I have no clue what it is, i only know manhattan is Li01_01_base)
edit: here are the base codes

3. find the package name for the banshee (there is a list on here, you might have to search for it)
edit: ship package codes list


4. There is a string of 6 or 7 comma seperated numbers following the package name. The first number will be 20. Change it to 16. all done.
