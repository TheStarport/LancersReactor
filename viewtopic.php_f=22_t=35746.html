JP4whlr, try JoBoomz' IFSO, Ioncross FL Server Operator. It's admin software for flserver and does an excellent job in many areas. I have used it ever since it came out and it is matured. I also use FLAC, FL Anti-Cheat. While there is some overlap between the two programs, FLAC makes a thorough list of everyone that logs on along with account directory name and IP address. Excellent for tracking possible account-hopping modders.

ditto to that, but would add:

IFSO for admins you trust with remote access, as it has no telnet capabilities, so anyone not physically at the server pc or on its lan, then they would have to use a Remote desktop utility to access it, so has to be someone you trust with 100% access to the server.

FLAC for anyone else, as you can limit the access.

or finally the Old favorite and backup I use FLadmin 1.29, with it you can give people 3 levels of access via any telnet utility.

lvl 1. Kick players only
lvl 2. Kick and ban
lvl 3. all above plus server stop/restart
