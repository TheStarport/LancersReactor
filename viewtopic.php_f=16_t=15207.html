IF YOU ARE GOING TO POST ABOUT THE RACE READ THIS FIRST

I don't know why so many people have problems with the race, but it seems like people post about it all the time.

Ok, this is what you guys do to win the race.

1) Turn on the Cruise Engines
2) Don't use "Go To," it only slows you down
3) Use mouse flight so you have complete control over your ship
4) Try to anticipate the next turn as much as possible
5) Hug the turns if you can
6) Get a light fighter if you are having problems

Thats all I can think of for now. If anybody would like to add to the list, go ahead.

Also, the guy you're racing stops or wrecks or something and turns on defenses. While he's doing that, he is NOT moving. So you pass him and keep on going.
