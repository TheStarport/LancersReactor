pci

Fri Mar 07, 2003 10:53 pm by pci

Titan that codec labelled "MPEG Layer-3 Professional Audio Codec" looks like the (professional) version of the Fraunhofer codec. Or something similar that is preventing Freelancer from installing correctly.

Select that line and click the [Remove button.
Yon shouldn't need to Restart Windows
Then run the Windows Media redist (wmfdist.exe comes with game)
This will install the (advanced) version of the codec.
Then Freelancer should work fine.

pci

Sat Mar 08, 2003 12:52 am by pci

Registry Solution

You can make a text file named "L3clean.reg" containing the following lines... then just double-click on that file to let Windows remove the registry keys for you.

Windows Registry Editor Version 5.00
[HKEY_LOCAL_MACHINE\SOFTWARE\Microsoft\Windows NT\CurrentVersion\Drivers32
"msacm.l3codec"=-
"msacm.l3acm"=-

Note: add the right bracket to the end of that line since this forum strips it?

Or you can manually run RegEdit and remove the keys listed above.

Then run WMFDIST.exe which will now successfully install the Fraunhofer (advanced) codec.

Freelancer should now play sound correctly.
