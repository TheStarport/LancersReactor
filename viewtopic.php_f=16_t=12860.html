For those who would like to replay SP with map/money/faction

I can't really reproduce the effect exactly, but I know it can be done (I did it myself, and was able to play the SP with a Titan without cheating, at least, not exactly.)

Under the My Documents\My Games\Freelancer\Accts\SinglePlayer folder, there should be a file called restart.fl. Apparently it's a replica of the latest save you made but in a "restart" format. If you juggle your saves correctly, e.g. remove all other saves, rename this file as autosave.fl, then load it, there is a chance that you will be able to load this "restart.fl" file and the whole SP will restart with your money, faction relations and universe map intact. The only bad thing is that all the wrecks you found will not be respawned, and your entire ship with all equipment/cargo disappears. Your money is intact though.

If you combine this with the opengate MOD, after missions 1a and 1b you can fly off to either the outcasts/corsairs/hessians/whatever uber-faction and get a lvl 10 ship fast and replay the rest of the SP campaign. Only thing is that most likely the only lvl 10 guns you will get is the nomad cannons, unless you did not clear all the wrecks in the latest save you loaded in Freelancer.

I know this sounds a lot like a hoax, but if you're really bored... try it out. One way that seems to work is that you make sure freelancer is running, load your latest save, back to main menu, * without exiting the game * remove all other saves except restart.fl and rename it autosave.fl, then load the autosave.

If anyone comes out with an exact procedure, I'll be interested
