Final list of ideas:

Reading all the posts, i'd like to present a final list of ideas. Then we can change that, rather than everyone posting new stuff.

LIST

Abolish VHFs (Then we can make All LFs equal, and all HFs equal.) Yes, Defenders do equal Centurions now.


All choose factions at start. More can join in later, but we should start with at least 5 or so.

No new tech advancement

No new systems

Turn (no money, it will get too complicated, methinks) system (i can design an Excel sheet for this if we work out all the details) so far we have something like this:

5 Producing Bases per Faction maximum

Each bases gets 1 turn a day, so maximum is 5 turns.

Base Classes: (and examples)

Capital Shipyard: Norfolk Shipyard, Alster Shipyard, Yukawa Shipyard (any large shipyard) Can make weapons platforms


Fighter Shipyard: Planets, Main Base for each pirate faction (eg Vogtland, Buffalo, Kyoto). Main base for every faction. (eg GMG=Aomori Station, others will be worked out)

Other Bases: Anything really. Simply generate money, can't be used to build anything.

Weapons Platforms: 5 Turns (worth 100 points, but cant move. can only be placed once. maximum of 5 next to a base.)
Minefield: 50 Turns. Placed around 1 base, makes it impervious to fighters. No effect on capital ships. Battleships. (a force using fighters against a defending minefielded base cant use the fighter points)

Battleship: 10 turns (worth 100 points in battle)
Crusier/Dreadnought: 5 Turns (worth 40 points in battle)
Gunboat: 1 Turn: (worth 5 points in battle)
HF: 0.5 Turns: (worth 2 points in battle)
LF: 0.25 Turns: (worth 1 point in battle)

Battle System:

Unless there has been a formal arrangement to meet in a neutral system (u gotta come up with a good reason for this) then one side must attack anothers system/base.

All the points are added up. A defending bonus of x1.5 is given to the defending side.

Whoever has the highest number of points wins, and the double the difference is taken off the losing side. if the attackers win, they get the system. If they lose, they retreat to the nearest neutral/allied system.
