bakedpotato

Thu Jul 05, 2007 2:34 am by bakedpotato

You have some strange things happening.
First I have to ask do you have a network card installed, either on one of your motherboard slots or built into the motherboard ?

Second why are you using the 1.1 patch ? You don't need to use the 1.1 patch. For me it caused problems instead of doing anything good.

Third with an IP of 192.168.1.104 you must be using a router, correct ?
Have you configured your router ?

Your game version is 2.7 and you say no Mod's... To have a game version of 2.7 you had to of had a Mod change the version number...

I suggest you check out The Complete Guide to Setting Up an FL Server

And http://www.dnsstuff.com/

Then I suggest you do a clean install of Freelancer
delete your Freelancer folder located at
C:\Program Files\Microsoft Games\Freelancer
- don't uninstall Freelancer, just delete your Freelancer folder so that you can be sure that no files get left behind by doing an uninstall.

And your saved games folder
C:\Documents and Settings\your name\My Documents\My Games\Freelancer\Accts
Both the Singleplayer and Multiplayer folders are sub folders in the Accts folder.

then install Freelancer again
don't install the 1.1 patch
but instead install the Freelancer QuickFix-v1.0c patch

Everyone connecting to the LAN must have the same files so they need to do the same thing... A clean install of Freelancer - and no saved games
and then install the Quickfix patch.

If you want to use a Mod activate the Mod after you do everything listed above.
