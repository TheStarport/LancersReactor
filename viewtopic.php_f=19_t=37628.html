First question:

- You should create a new MP character with the name you want.
- Then rename your SP saved game file.
- Do not delete the new character file yet.
- Next you should use the FL Save Editor. It's simple and quirky (you have to open the FL saved game file twice for a saved game file to appear) but it works. Open both the SP and new character saved game files (yes, FL Save Editor should be open in two different windows).
- In the new character file, copy the hexidecimal string in the box on the line beginning with "name =", copy no spaces.
- Paste this string into the SP character file, replacing the hexidecimal string after "name =". Keep the spaces as they were! Don't delete or add more space(s).
- SAVE the SP character then encode.
- Delete the new character and place the newly modified SP character into the same directory where the new character used to be.
- Done! This way will be the most complete and errorless way of renaming a character. Happy flying!

Second question:

- After say you invite your son, he should see the guy holding up his hand in the little menu in the top, right of his screen. It's actually the left most icon and it should be red.
- Click on that and it will prompt you to accept the invite.
- If the grouping is successful you should see both players in the chat screen with the same group number.
- Selecting Group Chat in the chat screen will result in magenta-colored chat messages between everyone in the group regardless of location. Up to eight players can be in a group. Have fun!
