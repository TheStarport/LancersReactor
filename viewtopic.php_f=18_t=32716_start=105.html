Furious

Tue Sep 28, 2004 7:26 pm by Furious

Sounds like someone has re-discovered how the ADMIN ships teleport on the IONCROSS server... not that complicated really.
Our solution does require some server-side software, but it works.

Basically you purchase a commodity that indicates where you want to teleport to. When you launch, the server software detects that you have this item in your hold. It then kicks you from the server and edits your position to the location you wanted to go to. When you reconnect, voila! You've travelled half-way across the galaxy.

We generally use it to move people to bases, but in theory you could have it place you out in space. You would need to pre-specify the destination locations. The fun part of this is that it could teleport you to a hostile base.....

Really want to be mean? Put some of these commodities on NPC ships as loot and make all the ids_names the same so you cant tell where the go.

Warning: there is enormous potential for abuse of this system. In theory you could teleport back and forth between bases without actually flying anywhere. That's one reason we only use it for the ADMINS.

This isn't exactly a hyperdrive, but it works.

This sytem was devised by Victor and JoeBoomz
