To change the starting money either use Chips server opp mod or edit the mpnewcharacter.fl file found in the exe folder of the freelancer install folder.

You don't stop being a LAN server. If you have a LAN between your 'puter and your modem then your server is ALSO a LAN server.

If you are talking about not wanting to see your server as a LAN server in the global server list, why? You actually WANT it to be a LAN server because when you play you will play through the LAN which is far faster than most Internet connections are. If your own server says Internet then you must go twice through your Internet connection to get to your own server! I could be wrong but why worry about it when a LAN connection is almost always faster? IMHO
