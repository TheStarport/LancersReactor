In a word: yes. As to how... basically, you can program a zone to have an Encounter that has a low probability of of occuring, causes the spawned objects to target selectively (say, Transports first, Freighters second, Fighters third, Stations fourth, etc.) and spawns a large number of the ships, using custom Formations that cause the "swarms" to use interesting swarm formations.

So, yes, it can be done. Easy? If you're using Toolkit as a base, yes. I could probably code up something like this in Toolkit in an hour or two.
