Hmm, not sure here. Is it when you first try to start FL, as in you click on the FL icon, it presents you with a screen (saying Freelancer) and then it crashes at that point? If so, use FLMM to do this. Go into FLMM (activate it) and then click on the tools tab. Now click on the "Go to save game folder" - which will open up a screen with some folders and files in ti - which are your freelancer Save Game files. Delete all these files/folders from the folder, and then close it all. Reinstall the game to be sure, and then play .

If that is not the problem, PLEASE be as specific as possible - exactly WHERE does it crash? Is it when you try to play SP? Do you get to the menu screen? Or does it crash when loading a save game etc?

We need more details to be able to help

Also - are you trying to run anything else with it? Cause both these mods change many files, and if your trying to run a beam weapon mod, or something like that, you might have some trouble in the fact that it might well be removing or replacing necessary files instead!

Erik - Errors with "content.dll" usually occur when there are problems with the Mods that you are running. Remove the mods you are running (if any), and then use the "Restore Backups" option in FLMM. If that doesn't work, try reinstalling FL.

BB - Chips is right; the zip archive has most likely become corrupted during the download process; it happens (believe me I know ). Either download it again, or you can try a command line utility called "pkzipfix" which may fix the archive. yuo should be able to find it through a search engine.
