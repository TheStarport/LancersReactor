AFAIK you can't connect directly by IP, but you can set FL to show specific server(s) in the list without reading them from the global server. The command is
freelancer.exe -s{IP/DNS}:port [..
e.g.: freelancer.exe -s131.107.83.10:2302 -swww.flserver.com:2302
Those servers have to be on-line and match the filters you've set to show up in the list though.
