ok

to edit the ini files you need bini , this is the command line version just drop your ini file on top of it and it prints out a text file. notepad wont do to read it though - use wordpad or any other hex editor you like.its important to remember to remove the ".txt"from the end of the file once your finished though - no need to recompress it and not all ini files are compressed to start with.its trial and error mostly.you will also need these editors to read and make xml scripts if you decide to make your changes into a mod compatible with flmm (i strongly recomend it) the beta version of which can be found here

utf edit can be found here and can be used to modify ships and icons and things.

these progs ARE actually hosted here on the site but im giving you links to mine for speed not out of cheek
as for tutorials its probably better to go for the specific ones relating to your desires but the best general ship modding ones (i know your not that interested but you might be one day) are the killermatrix one for actually modeling the ship and the zen one for getting it into the game with all the flashy bits.a great deal of modding is done with the ini files but sometimes you have to think out of the box to find a solution to your needs.

to modify the dll files youll need res hack

the information IS here but you might have to word it several different ways in the search function before you find exactly what your looking for.id never modded anything till a month or two ago and i made it so you should have no problems

hope this helps you get started
