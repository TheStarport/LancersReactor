Good, model looks much closer. As for your other questions, try looking in the tutorial forum, specifially hardpoints, or go to the download section and grab the Kasidias texture guide. As for packaging the whole thing together, you'll need the CMP exporter version 0.2.1 and the MAT exporter. Then export the ship using the CMP exporter, then create the MAT file by exporting using the MAT exporter. Then all you have to do is add the ship into the ini files, which you can find instructions for in the tutorial forum.
