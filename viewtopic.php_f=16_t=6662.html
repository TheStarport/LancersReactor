Well, figures, minutes after i post a msg for help i find a way to finish the mission. I guess i'll answer my own plea. Up until this point i have used my thrusters rarely. It was all pretty easy once i started using the thrusters. So if anyone has been neglecting the use of thrusters like me, start using them in battle, they are very usefull, although i would still trade my thrusters in for a second shield if i could.
