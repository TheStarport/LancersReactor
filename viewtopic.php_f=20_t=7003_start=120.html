Sun Jun 15, 2003 10:32 pm by the_flames

If you have installed the Fraunhoffer MP3 codec that comes with the Nimo Codec pack then you will have problems with the sound. The way that I got around it was to delete l3codeca.acm, and all similar named files (ie. l3codecx.ax etc.) from Windows\System32 and then run the file 'WMFDIST.EXE' from the REDIST folder on the CD.
