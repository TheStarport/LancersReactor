I Solved the problem!!!!!!
While I was using Milkshape to create my surface hardpoints I was not using it to make the hardpoints not directly attached to the ship. I was using the HardCMP editor to create such points as HpContrail, HpHeadlight, and HpMount. It seems that if you don't create the HpMount in milkshape then you will get the problem I had. Using the HardCMP editor does seems to work fine for all the other points. Thanks to those who took the time to reply and try to help me solve the problem. I also noticed that if you apply the same texture to more than one group it doesn't always export the .MAT correctly. To fix this I did a few things. I took all the groups that were sharing the same material (Texture) and regrouped them into one. Then I made sure that the group name was the same as the material name. Most of the time the .MAT exporter for milkshape works fine but I had a few models that it was inconsistant with. There was one that just would not include one texture at all. In that case I exported the .MAT anyway and then used the UTF edit utility to import the missing texture. The .MAT had nothing to do with the problem I posted but I thought I'd throw it in here in case someone else is having some of the same troubles I had. Special thanks to Drizzt4.0 for his outstanding modeling tutorial. ---PEACE---
