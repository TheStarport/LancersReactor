Frontierspace v1.75


Its that time again and the new version of Frontierspace is being released.

Current Bug list
All problems have download resources, here
01. A patrol in Ross-154 has been causing a crash, the update removes all of the patrols form the system. If a server runs this file all clients receive the benefits.
02. The German No-cd exe has the wrong version number.
03. The English No-cd exe has the wrong version number.
04. The Ioncross Server Op datafiles have a "subscript-9" entry when used.
05. During the quest (near the end) one of the text entries are duplicated.

Download Link
Download at TLR

Other links:
Download Frontierspace v1.75 at FileFront

Developers other forum board
You will need to register to view the board:
http://frontierspace.co.uk/siteforum/YaBB.pl?board=FS175

Universe map for FS v1.75
The universe map can be seen here (right click and choose "save target as" to download)

The release includes:
Frontierspace v1.75
FS v1.75 server op
(OpenSP is built into the normal version instead, thanks to Bejaymac for showing me how)
Universe Map for 1.75
IFSO data files
FLShell
FLShell datafiles for FS v175
Installation guide txt
The No-cd exe's are in the mods exe folder (both German and english)

Some of the changes
Mostly a bug hunt again, including:
Wolf-359 patrols
Nomad Prime Patrols
Manchester trade patrol
Courier goods prices
New goods for long distance/between major bases. In both cargopods and normal goods.
Cargopods show trade in value at sold price. Higher value too.
Freighters and other trade vessels have increased cargo capacity
New base traffic at 25 other planets.
Added to Tohoku, a GMG base and Kusari Navy
Added a few extra multiple loadouts at overlapping difficulty ranges.
Added FS specific ships to FLShell's scaning ability for custom freighters and larger ships.
Added Universal and Synth Foods FlShell player starts
Added some extra player starts to the single player.
Integrated OpenSp with the Client version to remove players being able to play single player without OpenSp (and avoid the problems)
Added Molly base in Ross-128
Modified the capship handling
Modified Capship weapon ranges. B'ship are 2.5K, Cruiser 2K and Gunboats 1.5K as a general rule
Modified Larger Trade Vessel weapon ranges. Transports 1.5K, Trains 1.75K and Repair/Utility ships 1.25K. Groups like GMG and IMG have modified ranges.
Added the Mancer 5.10 ship pack to the game as buyable and to factions.
Modified the power usage of all big ships, reducing the power draw.
New system "Orphelia", a Nomad vs Jetan warzone reachable by Lucifer, Proxima Majoris and BD-946.
There is more, but that will do for now

Frontierspace Website
