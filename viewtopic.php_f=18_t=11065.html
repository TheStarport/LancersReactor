the lines that u should modify are:
marketgood = rf_package, 20, -1, 0, 0, 1, 1, 1 - this removes the Banshee
marketgood = coe2_package, 32, -1, 1, 1, 0, 1, 1 - this adds the TItan
note: The Bashee is available at level 20, while the Titan will be only at level 32, so if u want to make it available at a lower level, 18 for ex, put this instead:
marketgood = coe2_package, 18, -1, 1, 1, 0, 1, 1
