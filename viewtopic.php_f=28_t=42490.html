UFO Ship Mod (for mod developers)



Sorry for the bad screenshots, Im working only with what I recovered from my old hard drive after an accidental format. The top is from hard cmp.

The zip file at the bottom contains the ufo files (I cannot remember if i finished with the icon or not) as mentioned above, these have been recovered from an older hard drive and I no longer have freelancer installed to finish making and testing it.
So I have submitted the best original work from a mod I was working on, this ship. The 3 ini files contain the entries that will be needed to be added to the mod ini files with a few obvious changes (invisible_gun will need changing for a different kind of gun, it just looked better with a gun with the barrels hidden, this is not hard to do (as mentioned in the subject this is for mod developers and I did not want to complicate the integration of this ship by having a lot of things that needed adding)) merely substituting the gun names for something else in your mod will do the job Dont forget the IDS numbers too.

the rest of the equipment and lights are generic, already in freelancer equipment (i think)

So yea, I thought Id make this available to anyone who wanted it, its an original ship, made originally well over six months ago. It is public domain as far as I'm concerned, use it as you will, i dont even ask for credit or permission. Unfortunately, I couldn't find the source milkshape files, though the hardpoints were added with hard cmp so they are editable as required.
