Quick-FAQ - 1/2/2007

I have a problem!
Please update to the latest version of SUR Exporter Plugin to see if that solves your problem.

It still doesn't work!
The SUR Exporter is very finnicky and requires careful planning in your model to even come close to making a proper file. You need to make sure that you use a low-poly version of your model to create the SUR file from or you will run into many complications. Also, check out the tutorials listed above for some of the latest information on creating working SUR files.
