Send To Notepad

If you need to open .xml files and .fl files with notepad do the following.

1. Open the Start Menu and press Run...

2. Type sendto into the field marked Open: and a folder called SendTo should open.

3. Right hand click in the folder and select New and then Shortcut.

4. Type notepad into the field marked Type the location of the item: and press Next.

5. Type Notepad into the field marked Type a name for this shortcut: and press Finished.

Now when you wish to open a file with Notepad that it does not recognise right hand click on the file and select Send To and then Notepad. So long as Notepad can read the file it will open it.

The SendTo feature is a standard feature of Windows XP. Righ click on any file and it is in the menu that appears.

The SendTo folder is a hidden folder that can be found at C:\Documents and Settings\Tim\SendTo. Each user of a XP system can have their own custom SendTo list available at the click of a mouse.

Place a shortcut to the folder C:\Program Files\Freelancer Mod Manager\mods in the SendTo and when you unzip a mod you can SendTo your new mod to the FLMM mod folder with two clicks of a mouse.

Another example of its use; for people who use AlbumWrap Extractor with their mp3's. Place a shortcut to AlbumWrap Extractor in your SendTo folder. When you download a wraped album just SendTo AlbumWrap Extractor to unwrap it. Two clicks of a mouse - done and dusted.
