IT WORKS!!!

Here's what you need to do in order to host a Freelancer LAN party off of one copy of Freelancer:

1. Make a server only installation on the dedicated server.
2. Install FL on all client machines using the original copy.
3. Burn a copy of FL for each client machine.
4. Start the server.
5. Start the game on a client machine with the master CD in the drive.
6. At the main menu, swap the master CD out for a burned copy.
7. Go to the multiplayer menu and connect through the LAN server list.
8. Repeat steps 5 thru 7 for each game client.

** The nice thing is that the account ID's appear to be randomly generated whenever the mulitplayer game is run for the first time. So all the game sessions are seen as unique even though they're from the same copy!

This was tested on my home network using 3 computers. 2 game clients and one server installation.

Enjoy! %D
