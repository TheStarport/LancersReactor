click REDIST folder. Then double click MSXML3.msi file. Window will come up
for you to INSTALL. After install RESTART. Now you can run freelancer on
Vista. Next I went to Lancers Reactor and in downloads on page 4 I loaded
the Quick Fix Patch by Louva-Dues which installs all fixes for freelancer and
installs SDK 1.3. Now restart again. Next download Freelancer Mod Manager
1.31. Restart again. Now pick one good Mod addon and download it. I used
Discovery 4.80 because it works with FLMM 1.3 and installs very clean with
Vista. From FLMM on desktop Launch Freelancer. And it works thanks to
everyone who gave me feedback at Lancers Reactor i was able to figure this
all out. Thanks Guys

Sun Apr 15, 2007 7:54 pm by Eradicator

You're absolutely right - I was misinformed by Microsoft.

I use AVG, and was trying to uninstall NIS. I could never uninstall it until I used the tool below. Anyway, the problem is that your running AV program was registered as a service, and is running under a Full System Administrator priviledged profile (even if you aren't logged on as an Admin). It's a background program. The apps aren't problematic with Vista's architecture - it's a problem with Vista's security and layering. As of right now, the only solution is to uninstall your AV program.

AVG is a very good program to use, it catches everything. If you are worried because it didn't block FL's access, don't - because AVG can auto-detect multiplayer gaming requests, as it says on the website.

Here is an easy way to un-install Norton products:

Get this:
http://service1.symantec.com/SUPPORT/ts ... 3108162039

It's the Norton Removal Tool. It will remove any trace of any Norton AV/FW product detected. That was the problem I was having.

Try it on your systems if you have NIS installed or if it appears in Vista's Uninstall Programs list.

Sorry for the confusion.
