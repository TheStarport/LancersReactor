Well it's not a tutorial per se, but go to
/freelancer/data/equipment
edit goods.ini then search for the package that refers to your ship (whether original or a cloned new version like the Rhineland gunship)

IE for the Eagle, it's Gf6_package

then change the engine to the type you want.
default for the Eagle is
addon=ge_gf6_engine_01, internal, 1
change the "ge_gf6_engine_01"
to
"ge_oe_engine_01"
for the Order engine flames
The Patriot looks better with Order engine flames in my opinion.

Using the Fighter engines instead of the large engine on a gunboat will give you the big flame when in Cruise.
