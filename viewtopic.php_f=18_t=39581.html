Alcander

Fri Jun 24, 2005 1:46 am by Alcander

I'm terribly sorry I didn't respond to you earlier. I've been away for a few days. At any rate, the NavMap background can be changed by changing the TGA texture in pretty_map.3db in the Interface directory. The names can be changed and the positions can be moved if you modify the EXE. You can also change the IDS names in resources.dll to change the NavMap names but the locations still can only be modified by changing the EXE.

And while <technically> it's possible to probably change the NavMap to 250x250 - it would be ridicously hard because the Interface components are made up of many subcomponents and that would mean spacing each side of the NavMap farther out, getting the background to render more, and having to add code for Nav positions beyond A through F. So in the end you'd probably end up having a really crappy mockup if you tried to change the NavMap since everything else has to be changed accordingly.
