NecroFlavor:
hi i'm a newb at all this can you tell me if i can stop the Shields regen on the NPC's and all i want is the new ship's /wep.s/trade items will all but the way the enemy does when they attack. can't stand that and is there something i can do to stop them from using mines


To stop the fast shield regeneration: Delete the ST_Equip.ini file in the Privateer's Equipment folder.
To change the NPC's AI: Delete the Pilots_Story.ini file in the Privateer's Missions folder.
To stop the NPC's from using mines: edit the Loadouts.ini file in the Privateer's Ship folder.
Change "cargo = mine02_mark05_ammo, 10" to "cargo = mine02_mark05_ammo, 0"
