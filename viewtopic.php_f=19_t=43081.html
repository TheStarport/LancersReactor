Hi azza,

I think my problem is different than yours. My game does say it's connected to the global server, it just doesn't display any internet games. I fear that the DOS command might not work for you, but you can try:

Write a bat file or open MS-DOS, and go to the folder where Freelancer.exe is. Then type:

FREELANCER.EXE -s<hostname>:<port>

where <hostname> is the namer or ip adress of the server and <port> is the port number where the server is running. You can add multiple servers this way, by typing:

FREELANCER.EXE -s<hostname>:<port> -s<hostname>:<port>

etc etc until you added all servers.

I hope this works for you, it has worked for me in the past. I made a thread like this before in another forum, but I can't remember where it was...
