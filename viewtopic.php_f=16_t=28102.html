FL data browser

FLStat_beta

This is a program that displays part of the information in Freelancer ini files.
All guns, ships, shields, commodities, their prices and location. For commodities you have the best buy/sell prices, therefore you have the best trade routes. For the equipment or ships you have the parameters so you can compare them if you don't know them already.
It has no graphics, just the information, but it loads reasonably fast.
It does NOT touch your ini files in any way, the only file written is flsini.tmp in windows temp folder, used to unpack binary ini.

I'm glad to see that my post has been noticed
By the time I started to work at my tool I hadn't known about FLDataStorm, which is a good program, if you need to get a detailed map of Sirius or information about ships or equipment then get it. It works with mods, since it reads the ini files too.
One drawback to FLDataStorm, however: it seems to take for ages to load the ini files and if you happen to like a mod but you still play vanilla FL then you have a problem switching between them. And actually this was the very reason I started to work at such a tool, I already knew vanilla FL but i realized that I know nothing about Victor's but I play vanilla alot.
One reason why you may whant to check my tool: if you don't know ship or equipment parameters you can easily compare them and see what they can do. When checking a new mod you ask yourself what gun is the best? What ship is more agile, what weapons can it mount? (well for that thanx again Dev )
FLStat is still a beta version that means there still may be bugs, but DataStorm has bugs too E.g. still no planet is listed as rep hack seller, but all these are inherent and will be fixed over time.
Again, DataStorm is a good tool dont get me wrong

Edit: I named this tool FLStat, after that I found there already is a FLStats that lists players and servers ranks, oh well, I hope that won't trouble any1
