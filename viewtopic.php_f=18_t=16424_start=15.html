ACK!!!!! Here's what I discovered. The two mods are incompatible for some reason. Porsche: I hope that as you make new ships, you make a combo mod with all of them in the same package. They are all so beautiful!

I was using the Slave/A-Wing Combo Mod. I unloaded that and one other mod I was using with Mod Manager. Then I loaded the B-Wing Mod. Freelancer would not start. I tried using Mod Magager to delete the AutoSave file. No Go. I used the Reinstall feature of Freelancer. No Go. I uninstalled Freelancer without uninstalling my saved game files AND I uninstalled Mod Manager. Then I reinstalled both. No go. So then I had the idea of manally deleting the autosave file and saved game files by date, the newest one first, one at a time, without the benefit of using Freelancer to do so. The first (newest) file was the only saved game involving the Slave ship. Freelancer started. Then I loaded the B-Wing Mod and a couple of others and Freelancer ran fine.

Moral of the story:
If you are running the Slave/A-Wing Mod and plan to switch to the B-Wing Mod, save your game right then so it is the newest saved game file. Delete any others involving those ships. I think (but I don't know yet) that you can just move that saved game file to another folder for later use if you want to switch back. Just get it out of the saved game folder and delete the autosave file.

Then uninstall the Slave/A-Wing Mod and install the B-Wing Mod. You should be good to go.

I think that if you handle it this way you can avoid what I had to do. I am assuming that if you drop a saved Slave/A-Wing game or a saved B-Wing game into the saved game directory after unloading/loading the proper mod, that saved game will work. But that is just an assumption.

It's not going to be a problem for me because The B-Wing ROCKS and I am going to stick with it until Porsche come out with something better! And that IS possible!

There are some graphics problems in the B-Wing scripts involving landing and moving into the equipment bay after docking, but no graphics problems in flight, however I do not use Turret View, so I don't know about that. This thing is as agile as an Eagle Civilian and faster. Looks better from behind than the Slave too. The cargo room allows you to make actual money transporting comodities. The gun mounts allow you to power up! I made Simple Mod V1a and 3 (available elsewhere on this site) and I am using Version 2 of that (a personal version not quite as tweaked and never uploaded to this site) with the B-Wing. But V3 has more powerful weapons anyway. Those weapons and this ship are just about invincible.

Thanks to Porsche! I can't imagine how much work this must have been.
