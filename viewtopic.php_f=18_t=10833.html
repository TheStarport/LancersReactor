*** MP/SP MOD - fiskMOD v.0.3a

fisk Combined server/client-mod (0.3a) for Freelancer 1.0
""""""""""""""""""""""""""""""""""""""""""""""""""

TO BE USED BOTH AT THE CLIENT AND THE SERVER (otherwise Freelancer will crash)
ALSO -PATCHED SERVER/CLIENT- will not work properly with non-patched servers.
Non-patched clients will experience strange behaviour on patched servers.

Use on own risk, I'm not guaranteeing anything; make proper backups before installing.

For this patch to work best, the server should remove all old saved characters in
%WINDIR%\Documents and Settings\%USERNAME%\My Documents\My Games\Freelancer\Accts\MultiPlayer\[server-data (ie. 23-0670c13e)

1. Extract into your freelancer directory (ie. d:\games\freelancer\ ) (replacing all files)
2. Run the game, connect to the server - PLAY!

------

This is a combination of many features currently available here, albeit modified,
I should still give credit to all the people here on lancersreactor who "invented"
modding. Special thanks ofcourse to: Wanderer and HCl (HydrogenCloride hurts).

If you're used to playing the regular game, this will probably be the most frustrating experience you've ever had so far. It's a pain to evolve in this hostile universe, you need money and friends. Flying alone in this version will make you feel small, whether you're in a Titan or the tiniest of fighters.

-------

For those of you who are against this mod, just don't use it; whining won't get you anywhere.



------

What it does:

0.3a

* Once again tweaked the NPC-AI
* Tweaked the freighter shields slightly.
* Cruise-engine disruptors tweaked.

0.2

* Don't fly into asteroids.


0.2b

* I've made a mess; tweaking values, this edition should be more balanced.
* Fixed the dreadnaught so it wouldn't move sideways at 300000000000 kp/s (!)


0.2a

* Tweaked the thrusters once more, they ran out of energy too fast.
* Modified player-shields (TESTING!)


0.1

* Balanced the AI-regenerationrate, they're still tough, but now slightly tweaked so not to be as "impossible" as previously.
* Speed of ships are changed, so that each ship has an individual spec. as to speed, also each thruster now have an own parameter as to speed,
* Added power to the missiles and mines (mines that were next to useless before, now have increased capability)

0.1b

* Greatly improved dreadnaught ( Yes, I know most of you haven't even seen it yet)
* Equipment for dreadnaught (engines, thrusters, shields, etc.) purchaseable, increased prizes for it.
- Working on a possibility to allow ships to dock in a dreadnaught, so far; nothing - though.

0.1a

* Improves AI, and all NPC's shields regenerate
* The player is now Xenos to start with, and IS NOT an ally of Liberty, it will be a quite different challenge.
* There is a purchasable Dreadnaught in the universe, it's price is huge, but it's firepower exceeds any others.
+ secrets


REMEMBER, this is the testversion only, use at own risk.
