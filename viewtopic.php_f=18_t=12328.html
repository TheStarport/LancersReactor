Wed Apr 02, 2003 1:13 pm by Xocyll

All the NPC team mate ships are normal ships with slightly different loadouts.

Juni always flies a Defender, only since it's defined and equiped differently for each mission, it sometimes has weapons on it that are above what you would normally be able to equip.

If you scan the ship you'll see the normal info card for it, IE class 4 max, class 2 optimum, even when she has class 8 weapons on it.

Her ship has infinite_power instead of normal fighter level power, and it's armor increases for each mission (there seems to be an armor mulitplier in effect for those special npc ships that alters the normal armor level)

All the special ships are defined in data/ships/loadouts.ini

re: the CSV, all the ship types that you can't normally pilot seem to have a 1 million cargo size, presumably so they didn't have to worry about keeping track of each ship's cargo size when doing special loadouts.

As long as you're willing to read some of the tutorials and prowl around in the .inis you can fly pretty much anything you can see in the game, since you can define your own versions.

If you can see it, then it has a defined arcitecture, and you can link to that.

Some of them are just plain not worth flying though, like the heavy lifter, since it's freaking HUGE and only has 5 turrets.

Big ships are seriously NOT fun to try to get through asteroid fields.
