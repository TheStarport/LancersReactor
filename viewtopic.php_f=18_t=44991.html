Starting from a clean Universe?

I am wanting to create my mod using no systems or bases from the original map. Using Freelancer Explorer, I am able to remove every system and am then looking ata clean slate. I can add a system and then add a base in that system. I change the starting character so they start on that base, but when I load the mod, the server launches fine, but the game will crash to the desk top.


How would I accomplish removing all systems and starting from scratch? Any suggestions would be very helpful.

Thu Nov 23, 2006 9:20 pm by blackhole

Freelancer uses newplayer.fl and mpnewplayer.fl to determine what system a player starts in. You must edit the system values in those files, or name one of your new systems li01 and the corresponding base li01_01_base
