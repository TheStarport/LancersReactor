My server keeps dropping people

Hopefully someone here can help me with this!

I run a dedicated server for me and my friends to play on, but recently they have been having trouble staying connected. Here's a quick and dirty timeline/explination of the history of the server:

The server was originally running on a Win98 box. The server is behind a Linksys router, but I have it set up as the DMZ host. With the DMZ pointing to that box's IP no one had any problems playing whatsoever. I did not make any additional settings to the router such as port forwarding, etc.

I recently upgraded the mobo and CPU of that server and upgraded the OS to XP Pro. The server has the same IP and is still set up to be the DMZ. However now when my friends connect they get disconnected all the time. The only thing that has changed on the server at this point is the OS.

I should note that the server has two NICs in it, one that connects it to the router and one that connects it to my LAN. (I run dual NICs in all of my machines, so in a sense I have two networks, a local 10.0.0.0 one that is isolated from the 'net, and a second one that connects to the routed network.) I can connect to my server with no problem, but I'm connecting over the LAN.

The server broadcasts to the 'net. People can see it and join it just fine, but they just get dropped at random.

If anyone has any thoughts or suggestions, please let me know.

Thanks,
