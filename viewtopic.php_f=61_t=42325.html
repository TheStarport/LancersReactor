The aforementioned error, which I've committed, is caused by zipping the file, but not the complete directory to it.
IE:
I released a mod of my Hovis Race Bypass.
When I first released it, I just zipped it up, add the .flmod extension, and sent it in.
Turns out, the only thing in it were the file, and the directories; Missions\M06
I ended up fixing it by adding the folder DATA.
So that it now appears as Data\Missions\M06\M06.ini
If you get that error, simply create a "Data" folder under the mod's main folder, and then dump everything into it.
That'll usually get it working.
It's a common mistake that we need to keep an eye out for.

Watch your 6!
