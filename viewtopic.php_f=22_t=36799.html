Most FL admins are actually system admins (sysadmins) because they admin right on the server itself.

If you wish to remote admin you can through FLAdmin by Fiik. Although Remote Desktop and other VNCs will work as well, this is just being a sysadmin from a distance.

To admin in the first place you should have one of the few FL admin programs out there. I suggest IFSO (Ioncross FL Server Operator) for sure and FLAC (FL Anti-Cheat, at the bottom of the forum) for more network admin if you qualify (ask Eagle) to use it.
