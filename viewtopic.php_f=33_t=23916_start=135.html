Michael Dan

Fri Oct 17, 2003 7:37 am by Michael Dan

The jumpholes to EFA are in all great houses main systems, including new london. The jumpholes to Narn and minbari empires are in dublin, look for them north of the 2 northern gold fields. centauri empire jhole is in Edinburgh, S-E.

Reddawg, i'm glad that u like my mod the most. About that MP problem, it is not certainly cause of these new character faces, since only a characters.ini file is modified and after re-testing yesterday i discovered that is NOT working in MP, only in SP and OpenSP modes. U must ensure that u're using the same version and have the exact file that the server has.
For this u have to re-instal the mod and check with the server admin for its curent version. its have to be the fiileplanet 3.12 one. Be careful though, sometimes at instalation of the mod with FLMM, all files in FL folder become read-only, so then no BAT files will work anymore. so please remove the atribute from whole folder...from properties, uncheck the readonly checkbox. Then either reinstall the mod with flmm or copy all file from the mod ZIP's archive to FL folder directly(to make sure u overwrite the old ones).
Also leave it with default settings, don't run level3.bat or any other bat files. and make sure u don't have the 1.1 patch instaled...
