Gibbon

Mon Sep 03, 2007 10:39 pm by Gibbon

Ok, so let's see if i can make this clear.

You have a mod already with new systems for the client. Add in the new admin stuff, system, weapons, bases, whatever. Then stick this version on the server. At this point, only the server has the admin system. Then give your admins the new client version of your mod with the admin systems added to it. The point is, if the server has it then the admins can use it. Means they can utilise the system but your players cant, as they don't have the files. This also means you don't have to close off any jh/jg as the players dont have it but the admins do.

Example:

You could make a jg and stick it outside manhattan. Only the admins would see it and be able to use it as they have the files. As long as its on the server as well then the admins can use it.
