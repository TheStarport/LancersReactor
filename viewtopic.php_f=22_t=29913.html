Fri Apr 02, 2004 9:31 pm by Sanic

Also, the server running the game should be connected to the internet and all the computers connected to your LAN can play there AND the server can also be accessable to players online if you allow them too. The server setup screen asks you if you want to allow new players and if you want the server accessable on the internet, as well as if you want PVP turned on. So there is no reason that your family and friends can't play over a LAN and have internet players on at the same time, though your computer may have trouble handling too many players at once.
