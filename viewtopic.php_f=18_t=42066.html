Mod Loader

From looking around topics lately, anyone with a mod loader seems to attract attention from modders. It gave me an idea; how many people would be interested in a Freelancer mod loader?

The concept:
Server administrators will release a customized mod loader to the clients which will in turn apply the modification. Each modification's loader will have a unique key with which all the client app's configuration is encrypted. This will prevent tampering with the client app and the modification's configuration. In other words, all the modification configuration will be encrypted with an individual key for security purposes.

The loader itself:
The loader is an automated update system where the client app downloads any new updates. After downloading the updates, if any, the loader would verify the Freelancer data files, start Freelancer, and then monitor the data files to prevent in-game tampering.

There is much more detail, however, this is just a very general overview to see if there is interest in something like this.
