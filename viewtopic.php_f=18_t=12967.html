you may be saving wrong: you must go to bini and open the file, then click "save as" and save it as a "ini" NOT "ini and txt"..

the other thing that could be wrong is (correct me if i am wrong someone) but to my knowledge the server must have the same starting character as you. try hosting your own server (Start -> Programs -> Microsoft Games -> SLServer) and see if you can start a game with your modded character file. this will tell you aht the problem is - if it starts fine then its because you have a server-side mod, if it crashes, then you are doing something wrong with bini, or with what you type in.
