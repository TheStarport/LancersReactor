You could try using a mod i released recently "Broken Oath" that ones a server side mod with quite a few options for starting location, ship and credits.
Other than that, there was a post recently that had this:

This is from Data\Characters\NewCharacter.ini in SDK

[Package
nickname = ge_fighter<--The package from below
strid_name = 11051
strid_desc = 11551
ship = ge_fighter<--Starting ship Starflier
loadout = MP_ge_fighter<--From Shiparch.ini
money = 10000<--starting money

[Faction
nickname = new_player
rep_group = co_ic_grp<--Only works if you edit mpnewcharacter.fl to allow factions
base = Li01_01_Base<--starting location Manhattan
Package = ge_fighter<--This is the package. not the ship (its got the same nickname)
Pilot = trent

For starting location bases, you're going to need the database of the nicknames and bases. Or, you could just ask for the odd one or two if you like. I'm feeling helpful. Feel free to post up if you need any other input.
