It is easy to add a ship to be sold. Go into the uncompressed market_ships.ini and change some of the data around.
To add the titan to manhattan here is how.
change: marketgood = coe2_package, 32, -1, 0, 0, 1, 1, 1
to this: marketgood=coe2_package, 0, -1, 1, 1, 0, 1, 1
now you must remove one of the other ships i recommend the rhino
change: marketgood = lfr_package, 1, -1, 1, 1, 0, 1, 1
to: marketgood = lfr_package, 1, -1, 0, 0, 1, 1, 1
this will add the titan to manhattans ship dealer but i dont know how to mud the price yet or your credits without messing the game up.
