oh dear oh dear. lsass is the Local Security Authority Subsystem Service, this basically is what allows you to logon to Windoze.

if this was me, I'd try to get XP (presumably) into safe mode and see if lsass is disabled as a service, and re-enable it and reboot. If that doesn't work, I'd run the repair console or Automated System Recovery and repair it, if that fails, then reinstall Windows (as a repair) over the top of your existing installation, then run a registry repair utility. You could also try Sytem Restore, if you have a restore point prior to this going wrong. Hope that helps, tried to keep it short n sweet.
