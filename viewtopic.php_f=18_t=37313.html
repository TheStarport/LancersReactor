Wed Feb 02, 2005 2:07 pm by Zack Lyons

You want to edit the map itself.

Get yourself a UTF editor and a .tga plugin for photoshop (yes you need Photoshop as well) and go to DATA\INTERFACE\NEURONET\NAVMAP\NEWNAVMAP
You want nav_prettymap.3db. Open it in the utf editor and go to Texture library>fancymap.tga. Export MIP0 and save it in .tga format. Open it in Photoshop and edit to your heart's content. Save it, and Import it back into the utf editor. MAKE SURE YOU HAVE A BACKUP FILE.
