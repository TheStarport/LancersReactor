Mods are either "on" or "off". You cannot make them optional, so far as I am aware- if the server's running it, then it's running it... client settings don't matter, for things like Systems.

Now, if you're wanting a script that offers options as to what is on or off... on the server side... yes, that can be done. But... and here I'll be frank... if the server has things running on it that don't exist on the client side, then the clients are likely to crash
