Actually, i don't think primus has a commodity trader. You might need to add it to the bases rooms too.
The base layout is the same for Primus and Malta. So take Hi01_01_base.ini and add its trader entry. Ship too if you want to sell the Anubis there or something.
This is what you need in ew06_01_base.ini:
[BaseInfo
nickname = Ew06_01_Base
start_room = Cityscape

[Room
nickname = Bar
file = Universe\Systems\Ew06\Bases\Rooms\Ew06_01_bar.ini

[Room
nickname = Cityscape
file = Universe\Systems\Ew06\Bases\Rooms\Ew06_01_cityscape.ini

[Room
nickname = Trader
file = Universe\Systems\Ew06\Bases\Rooms\Ew06_01_trader.ini <--added

[Room
nickname = Equipment
file = Universe\Systems\Ew06\Bases\Rooms\Ew06_01_equipment.ini

[Room
nickname = ShipDealer
file = Universe\Systems\Ew06\Bases\Rooms\Ew06_01_shipdealer.ini <--added

Then copy Hi01_01_shipdealer.ini and Hi01_01_trader.ini into ew06 and rename them. Replace the entries already in ew06_01 rooms with their hi01_01 counterparts and thats all done for the base side of things.

You will need to add entries to mbase for ew06_01 as now they have traders and a shipdealer.

Open up missions/mbase.ini and find ew06_01. Then add:
[GF_NPC
nickname = ew0601_fix_trader
body = li_bartender_body
head = sh_male6_head
lefthand = male_hand_left_black
righthand = male_hand_right_black
individual_name = 245180<--replace with your own name, i just use "Trader"
affiliation = fc_or_grp
voice = rvp111

[GF_NPC
nickname = ew0601_fix_ship
body = li_bartender_body
head = sh_male5_head
lefthand = male_hand_left_black
righthand = male_hand_right_black
individual_name = 245144<--replace with your own name, i just use "Shipdealer"
affiliation = fc_or_grp
voice = rvp106

and at the bottom add:

[MRoom
nickname = trader
character_density = 2
fixture = ew0601_fix_trader, Zs/NPC/Trader/01/A/Stand, scripts\vendors\li_commtrader_fidget.thn, trader

[MRoom
nickname = ShipDealer
character_density = 2
fixture = ew0601hi0101_fix_ship, Zs/NPC/Shipdealer/01/A/Stand, scripts\vendors\li_shipdealer_fidget.thn, ShipDealer

I'm guessing you an add ships to the base ok, with market_ships.ini.
