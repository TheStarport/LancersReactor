Manhattan_Guy,

If you can start the server.exe and run the mod in MP and use a lan connection to freelancer.. then when you initialise the game, error messages appear on the console.. and by watching the console it may give you a hint as to what is wrong.. It's a method I use a lot when testing as errors appear in red and warnings in yellow..it also shows system entries and exits which can help pinpoint problems in system files.. The text that appears can be highlighted and copied to notepad or wordpad. If/when the mod crashes, it sends an error report to a file called fl-server-errors.log located on the root directory of C:\ drive.. typical reports are like this:

1: E:\FL\Scratch\Source\Common\Archetype.cpp(60) : *** ERROR: ArchDB::Get(2425843851) failed
or a very common one:
1: E:\FL\Scratch\Source\Common\GoodList.cpp(450) : *** ERROR: Cannot open goods file ..\data\equipment\engine_good.ini

Now if i'm preaching to the converted forgive me but I have read the post and you dont mention it.

One other thing.., if FLscan is giving you an error on engine_sounds.ini it may mean that the problem is there.. sounds to me like good place to start looking..

Dunno if any of the above will help but it's worth a shot..

Chips

Fri Sep 09, 2005 1:36 pm by Chips

Those errors aren't causing the crash... but, to fix them:

Open up the engine_goods.ini file - hit space bar, and then save it. This will get rid of the "cannot open file" stuff.

To eradicate that engine problem, go into the your system file for:

Li01
Li02
Li03
Li04
Br01
Br03
Ku02
Rh02

I cannot remember other systems with battleships in - anyhew - search for the battleships stations, and then find their loadouts. Add a suffix of _station to the loadouts (so its like rh_battleship_station as the loadouts nickname).

Now go into the loadouts_special.ini file - and search for the names of the original loadouts for the battleships, copy these and paste to the bottom of the file (so thats the Bretonian battleship, rheinland, kusari and lib!) and add the suffix to the nickname - so they are now called rh_battleship_station etc.

Now remove the flippin engine from the loadout of said station
I think there was something else too - like a tractor beam or something that you should remove from the station loadouts as well. The reason for renaming them loadouts? Because if I remember rightly, its also the loadout used in the single player file i think.
