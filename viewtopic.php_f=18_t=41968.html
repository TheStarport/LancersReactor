Alcander

Tue Dec 06, 2005 10:00 am by Alcander

While it's technically impossible to create NEW hardpoints. It is possible to repurpose the existing hardpoints. With EXE/DLL modification you can create 10+ classes if you want, but you would need to have a reservoir to draw the new hardpoints from. The only idea I've had for this is to repurpose all of the shield hardpoints (freighter, fighter, etc) and create one shield hardpoint type for all ships - then just limit which ships can equip to a certain class (like fighters can have 1-10 but freighters can only have 1-6 and then have shield boosters or whatever). Then with the extra shield hardpoints you could create new classes.
