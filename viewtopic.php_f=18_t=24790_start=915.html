Because your using everything BUT the planetscape for the base, you wont have the launch pad area on it. besicly u need to get another base template, such as Buffalo or West point to implement into your base. i think there is also a way of changing the ini file of your base in the BASES folder of that particular system, heres mine

The location of it (this applies to other bases aswell )
My mod/DATA/UNIVERSE/SYSTEMS/LI06/BASES/LI01_01_Base.ini

The data within that file

[BaseInfo
nickname = LI06_01_Base
start_room = Deck

[Room
nickname = Bar
file = Universe\Systems\LI06\Bases\Rooms\LI06_01_bar.ini

[Room
nickname = Deck
file = Universe\Systems\LI06\Bases\Rooms\LI06_01_deck.ini

[Room
nickname = Deck2
file = Universe\Systems\LI06\Bases\Rooms\LI06_01_deck2.ini

Note the last line, that room is the room from which you take off in - in my case atleast

Hope this helps!
