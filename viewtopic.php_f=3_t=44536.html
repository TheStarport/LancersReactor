Okay heres my two cents:

First of all connect ALL the PC's you want to play FL on as a LAN
Switch all these PC's on
When thats done run the Home Network setup wizard
if a screen shows up stating you have disconnected hardware (and if that hardware IS your new card) you have set something up wrong. My recommendation is to get the latest drivers from your manufacturers website.
If the disconnected hardware is NOT your LAN and something else you dont wish to use, click ignore disconnected hardware at the bottom.
If the machine you are on now has the internet installed on it, you MUST click this machine is the one with internet all machines connect through me.
set your workgroup name as anything you want it to be, but it MUST be rembereable. for the purposes of this it will be MSHOME.
Turn file and print sharing on if you have a firewall (other than MS firewall which personally wouldnt even protect you from a n00b with knowledge of the ping command) if you dont have a firewall turn file sharing off to improve security.
then wait for the wizard to do its magic, then when it is done click the last option (do not run wizard on other machines).

Now repeat this process on all the other LAN machines (with the exception of instead of clicking all machines connect to the internet through me, click the one that says i connect to the internet through a residential gateway)

Restart everything and then hopefully that should set you on a winner. If not im sorry i couldnt help.
