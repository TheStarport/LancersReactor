The Sith War: Need Modders of all talents!

I'm in need of some help for a Star Wars MOD I'm undertaking which takes place around the time of the KotOR period. If you're at all interested in the KotOR games or comics then this mod is for you!

Time Period:
The exact time period I'd like to place the mod in is just prior to the destruction of the Star Forge. That way all of the locations within both KotOR games are open to be used.
The general Idea is that with the StarForge operating, the Sith are at their peak power and ready to invade while the Republic must fight them off (and perhaps destroy the Starforge).
The Starforge itself could be coded as a large destructable object with an astronomical number of hitpoints so that on a server the the Republic's goal would be to destroy it. The Sith would have a similar goal, say to destroy Citadel Station on Telos. These things could become server events which don't have to be organised. It also makes playing it A LOT less boring as there's more to do than shoot NPCs or trade if there's very few people online.

General Info:
Since there are very few ships I intend on creating several "sub-classes" of ships which allows for several variations of the ships (essentially clones with different stats). e.g. Interceptors, Bombers, Space Superiority etc.

To add to the variations I'm thinking of an EVE Online type system where each ship has a set number of "Active" and "Passive" modules/mounts which can carry different types of shield boosters, energy boosters etc to also add to the "variety". This also draws similarities with the games where several weapons were upgradable.

Help Needed:
Modelers/Skinners - I have several models lined up which I can use once I've secured author permissions. However I'm still in need of models for:
-Sith Star Destroyer/Leviathan Star Destroyer
-L-Wing Assault Fighter (bomber)
-Ravager Star Destroyer
-And any other original designed craft which could expand the ship range.
Could also include modelling some stations (such as the Star Forge, Citadel Station etc.)

Although I can manage most aspects of FL coding, I'll need help as there is alot to do. So anybody who can code systems (especially) is very welcome!
Anybody particularly good at Factions and Economy are also needed

I am completely open to any suggestions for the mod and will take every one of them onboard (as long as they don't radically change the mod). I'm not really into the whole "Mod Team Leader is a God" or anything.
