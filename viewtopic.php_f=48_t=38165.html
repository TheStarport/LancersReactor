Problem: Server crashes after 10 seconds running

Error: Explorer wrote wrong information into the inis or couldn't write into the inis

Solution 1: try to deactivate writing protection for the whole freelancer directorie (it worked great for me). Some DLLs are writeprotected, thats why the explorer won't change the files.

The same with FLED-ids.

Before using any program, deactivate the write protection, everytime you started freelancer.

Solution 2: Get crashinformation
When the server is running turn on console quickly

menu>>view>>console

there is often some information what kind of information is missing often something like

head1 not found on blablabla.ccp
or
mission linked to astoridfield not found

copy the errormessage and post it here, that would give us some information
