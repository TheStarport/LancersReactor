I don't know if you lot realise it or not but there are 3 versions of Freelancer.exe doing the rounds.

1} The standard exe that was released with M$'s version of FL.
2} The hacked NoCD version of "1"
3} The official NoCD version that everybody that has a budget released version of FL has.

Number 2 is still illegal, but number 3 was authorised by M$ to prevent anyone else getting a look at thier CD piracy protection software.

Every mod that contains an exe should have either version 2 or 3 in it (preferably 3) as none of the budget owners will be able to use a mod with the old exe because of the CD protection.

@ opwnageo, if you have problems with the legality of even changing the version number on the exe then why are you even modding FL. In reality all modding is illegal as we're altering someone elses property (check the EULA), however the industry allows us some freedom because we extend the life of thier games (giving them more sales).
