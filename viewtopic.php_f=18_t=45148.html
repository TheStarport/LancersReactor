Thames doesn''t like CapShips

I'm making a capships mod for my server. I am able to dock at and launch from every base/planet in the game but one. Outpost Thames in New London.

I can dock there fine but when I attempt to launch the server crashes.

Outpost thames uses the trading_outpost_lod.cmp which is used by numerous other bases so it's not the cmp

This happens with the Kusari BS and the Liberty DN so far. (Only two ships I've tried) so it's not isolated to one particular ship. Both ships can dock at and launch from every other base.

medium_moors have been set to jumps and the range of the docking points have been increased just like all other bases so that's not it either.

Rep has been set to .90 everyone so that I can test all bases without worrying about whether or not I'm friendly so it can't be a reputation thing.

When in Console view on FLServer no error message is displayed before the server crashes so I have little to go on.

What would cause this to happen at only one base in the game?
