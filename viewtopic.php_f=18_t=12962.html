Vertical Slide

OK, dont know if anyone has found this but here is the ability to slide vertically

Add this to keylist.ini

[key
id = USER_MANEUVER_SLIDE_EVADE_UP

[key
id = USER_MANEUVER_SLIDE_EVADE_DOWN

and add this to keymap.ini

[KeyCmd
nickname = USER_MANEUVER_SLIDE_EVADE_UP
ids_name = 1958
ids_info = 3228
state = keydown, keyup
key = "K"

[KeyCmd
nickname = USER_MANEUVER_SLIDE_EVADE_DOWN
ids_name = 1959
ids_info = 3229
state = keydown, keyup
key = "L"

You can choose your own keys, or map them in the options menu of the game.
