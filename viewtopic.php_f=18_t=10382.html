Where can i park my TigerShark?

Ok, just a progress report on my work with 3D format research: i finally managed to import a completely new model into FreeLancer, the Tigershark, from Wing Commander Prophecy. Pics here and here (PNG format, ~750 kb each).

This test is made of only one mesh, there are still issues in importing multiple meshes, but i'm confident i can fix this soon.

I realize i've been posting many progress reports lately and not much else. In my defense, i can tell you that real life is succeeding in keeping me busy, especially now with University-related work load increasing steadily . In any case, i hope to change this soon. I'll be releasing my little UTF editor along with documentation and source code in a few days, if all goes well. The editor will allow you to get familiar with UTF files and manipulate them a bit, but you can always build your own editor based on the documentation, or expand my own, if you are so inclined. After that, i'll be releasing a model exporter for Milkshape 3D (it will export either to CMP format, or VMS, i haven't decided yet.. more on this soon). As usual, file format documentation and source code will be available.

I'll keep you all informed

Mario

Well, i wasn't planning on uploading it: it was just a test to see if the data i gathered together holds water, and there was no concern on my part with stats, hardpoints, engine locations, etc (the engine is particularly bad, since it's on the middle of the ship). Still, if you want to take it for a spin, i'll be glad to upload it to a temp location as soon as i get home It replaces the Liberty Elite files (namely, li_elite.cmp and li_playerships.mat) and requires a bit of an editing of shiparch.ini.

As for the conversion process... I understand that not everyone has access to Milkshape 3D, but that's the only 3D modelling program i have at the moment, so in terms of plug-ins that's all i can do. I suppose a specific format converters could also be worked on (dxf, 3ds, etc)... but given the interest shown by other people, on other threads, to work on converters / plug-ins of their own, i'll probably just release the Milkshape plugin (+doc and source) and let people implement converters and other plug-ins based on that.

Also the Milkshape 3D plugin will most likly convert to CMP format directly... I'm looking into the possibility of placing hardpoints directly in Milkshape, by using specially named 1-vertice meshes. This way there'd be no need of creating a tool especially for placing hardpoints and such. More news as work proceeds
