The EvoChron community welcomes TLR

I think I speak on behalf of the entire SW3D community as I welcome you into the universe of EvoChron: Alliance.

I could write a little recommendation for the game here, but I won't. If you're interested what this game has to offer, I'd say: check out the website @ www.evochron.com.

We'll try to establish announcements for multiplayer games as well. If you've got any problems connecting, try forwarding the DirectPlay ports on your router - usually helps. There is a 24/7 server available and you're free to host multiplayer games. With hosting, watch that you enter your EXTERNAL IP adress when hosting. Games hosted on LAN IP's will not be available online.

If you've got any questions concerning the game, feel free to ask them here or for technical problems on the SW3D forums.
