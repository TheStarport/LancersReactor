Sun Mar 09, 2003 7:54 am by Grahf X

I don't know what you mean by NEW ship but here is my running example
to make an Anubis available at the first base (Manhattan)
1. Go to the market_ships.ini file
2. Find the base number Li01_01
3. Now change this
marketgood = oe_package, 16, -1, 0, 0, 1, 1, 1
to this
marketgood = oe_package, 0, -1, 1, 1, 0, 1, 1
Before you start to play though you need to remove one of the other ships for sale there by changing the number like you did for the Anubis. And I dont think there is a way to skip the story in SP, sorry. If this doesnt help you post again and I will break it down some more or maybe someone else will beat me to it.
