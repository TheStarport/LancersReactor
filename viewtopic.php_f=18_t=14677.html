Daekar

Mon Apr 21, 2003 8:37 pm by Daekar

Ok... i'm not sure i'm clear on what you're asking. If you're simply modifying every dagger in the game then you should keep the same ship id and everything (in the ini files)... if you're going to clone the dagger and alter it then you should still keep the id numbers the same *UNLESS* you've also altered the name strings in the dll files... in which case you should use the id numbers that you generate with the utility for dlls available in the downloads section. There's a good tutorial at another forum which is available through a link in the tutorial forum if you would like to try your hand at modding name strings. If you do alter dlls, follow the FLMM directions for file placement... otherwise you might crash the game or simply have unchanged names. Anymore questions i'll be happy to point you in the right direction!
