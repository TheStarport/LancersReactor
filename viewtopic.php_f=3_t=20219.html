Check which faction owns that base by talking to the characters in the bar. It might be IMG or Republican Shipping.Your rep might not be too good with those factions. You'll probably need to be about 3-4 blocks from hostile red (or more) to purchase their goodies.

You can also check who controls a base by bringing up your NAV map and clicking on the base and then clicking the question mark icon when it comes up. It will give you all kinds of useful info about the base, even the type of commodities sold there as well as which faction controls it.
