In the Freelancer folder in My Documents, there should be a file called "PerfOptions.ini". Open it in Notepad and down near the bottom is the resolution.

As for the movies, I don't think you can run those in anything other than their display resolution - usually 800x600. You could try renaming the movie files or the folder they're in, and the game may skip them altogether on load. I did this to BF1942 and it worked fine, don't know about this one though.

Chips

Mon Aug 23, 2004 3:11 pm by Chips

There is a mod to remove intro videos. Otherwise - open up Freelancer.ini file inside the Microsoft Games/Freelancer/Exe folder - and then you will see


[Startup
movie_file = movies\mgs_logo_final.wmv
movie_file = movies\DA_Logo_Final.wmv
movie_file = movies\FL_Intro.wmv


Put semi colons at the beginning of each line to comment out the vid:

[Startup
;movie_file = movies\mgs_logo_final.wmv
;movie_file = movies\DA_Logo_Final.wmv
;movie_file = movies\FL_Intro.wmv


That will stop em playing for you.
Save file and exit (be careful to ensure its saved correctly as an ini file and not a txt file!!

If you open it and see gobbledygook - saying BINI at some point in it, then download BINI and drag the file onto the bini.exe file. This will "decode" it and make an text file of it called Freelancer.ini.txt

You can now delete the original ini file, open that Freelancer.ini.txt file, make the above changes and click "save as"

Now ensure that it saves as ALL FILE TYPES (*.*) and rename it to Freelancer.ini

Click on save - and the ini file should reappear for you
Delete that old text file of it.
