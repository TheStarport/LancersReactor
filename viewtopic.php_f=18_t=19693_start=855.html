Lethal Dosage

Sun Jan 30, 2005 8:19 am by Lethal Dosage

Hallc, If I knew the answer to your question, I'd answer it.

Draven, Open ANY folder. in the taskbar under Tools, goto Folder Options. Under the 'View' Tab in Folder Options there is a scroll down list.
About half way down (just below 'Hidden Files and Folders') there is a check-box saying 'Hide known files types'. Make sure this is NOT checked, then apply changes and find your way to wherever you mod .zip files are.

Find the file, and right click on it and select 'rename'. At the end of the mods name the should be .zip.flmod . If there is not, write it in (must be spelt right).
So, any mods would be shown as:

Yourmod.zip.flmod
TNG.zip.flmod
Evoloutions.zip.flmod

etc. etc. Once done, they should show up as Freelancer Mod Archives, and on double click should install to you FLMM mod directory - and into FLMM ready for use.

If that doesn't explain it, then sorry, I don't know how to put it any other way.
