Chips

Wed Apr 13, 2005 2:06 pm by Chips

I do believe that if you use the thread saying it holds all the answers in the newbie modders forum, that it contains a link on how to make hardpoints.

Otherwise, search for hardpoint editor or creator - download, install, and then double click on the CMP file of your ship.


As for FL:Explorer - that is a system creation tool - its not really a mod tool strictly speaking, and especially if in the mod your trying to edit uses XML scripts (FLMM script) - then you won't see anything.

However, there are plenty of tutorials on how to make guns, bases, set items to sell, make ships fly etc etc. Check the newbie modders forum, and read the post saying it has all the information. Its like help file
