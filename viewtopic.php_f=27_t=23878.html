Strider - oTO

Sat Sep 20, 2003 11:25 pm by Strider - oTO

Found the problem with Mission 3 (Willard Station) You need to open up loadouts.ini which is located in the freelancer\data\ships directory. Find the entry for MSN03_Nomad_Rheinland_Bomber

these two lines must be deleted:

equip = torpedo01_mark01, HpTorpedo01
cargo = torpedo01_mark01_ammo, 2

for some reason, it did not like the fact the bombers had torpedos. Anyways, hope this works for everyone else with the problem.

I found the game crashes again when you fight rhineland fighters in later missions. To fix all related crashes, all torpedo loadouts for rhineland fighters must be deleted from loadouts.ini
this means

equip = torpedo01_mark01, HpTorpedo01
cargo = torpedo01_mark01_ammo, 2

must be deleted from any rhineland entry within loadouts.ini
Look for entries that have "rh_ " in them. Those will be rhineland ships
