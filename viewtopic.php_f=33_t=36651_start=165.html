@ elreicht:
A loadout is the set of equipment, weapns, lights, and cargo a ship has. For example, you can talk about the default loadout that a ship comes with when you buy it, which includes all of the powerplant, engine, scanner, trctor, lights, contrails, and sometimes shields/weapons. And you can talk about loadouts with regards to a player's set of custom weapons and shields, the stuff they personaly prefer to fight with.
Make sense?

@ Shawn:
When you say "scripts," do you mean the scipt in .xml format? These are actually all a basic mod needs - physical ship models/materials and new bases, sounds and icons do require teh actual file itself. FLMM and xml make it easy to save a LOT of time and space by adding only the things a mod adds to a file instead of having to append the whole ini file to the mod. Rebalnace is actually a very sloppy and messy way to make a mod file - we could use scripting pretty much for everything we did, but there is so much changed that it's easier to just throw the whole file in there -- don't forget anything that way
