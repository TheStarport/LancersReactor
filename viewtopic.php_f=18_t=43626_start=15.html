A) The same program used to build ships and stations in the mod anyway... i.e.

Gmax (or any other 3D construction program that allows export of file formats to Milkshape).

Milkshape with the CMP and Mat Exporters.

Model Tool (as this is essential for adding the model radius on large models and adjusting Sur files)

A good image editing program for textures (most use Photoshop, I use Ulead PhotoExpress though and Paint).

A DDS converter to get your TGA files smaller.

UTF-Editor for any changes within the UTF you need to do.

The SDK for decompressed ini files.

Notepad to edit the ini files.

Freelancer Mod Manager for building the mod within and activation and deactivation of the mod.


CMP and MAT files will work fine using either TGA or the much better compressed DDS formats for the textures. Either this, or you can build it as a 3db which essentially is a CMP and MAT file combined (in the simpliest way to explain it).
