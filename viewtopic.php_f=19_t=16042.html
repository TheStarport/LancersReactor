The server admin can change the ini file so new characters start in a different faction, although from my own experience it seems that most crash the server when the player joins.

Your faction never changes from that point on. If your reputation with your "home" faction, Liberty Navy by default, becomes hostile you are no longer a member of that faction but you can't re-join it or join others. AFAIK the only effect it has is to stick the faction at the end of your name anyway - would of been a nice if they'd done a bit more with it.

So, chances ar, if you're seeing people flying around with different factions it means the server admin has changed it. If most players are Navy and only a few are other factions (or vice versa) they are probably admin players.
