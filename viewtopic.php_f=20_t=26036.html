i run virtually the same machine as poor old Borak, but with just more RAM and a faster processor. I have the same OS and graphics cards as Borak, and I had similar problems.

1) First things first, have you got any mods running?
2) Do you have any unofficial updates, patches, etc. running?
3) Do you have FLServer running?
4) When was the last time you defragmented your drive?

I find that if any of the first three are happening, then the game tends to freeze often, i Alt+Tab or Ctrl+Esc and then re-open Freelancer. If the sound continues on mine when a game freezes, you can tell if the game itself has locked up or if it's just the graphics card.

If you haven't defraggd your drive lately, then do it! Win2K doesn't like running on a fragmented drive. Dunno why, just doesn't lol.

Hope this helps
