Mon Mar 10, 2003 7:12 am by Hardeyes

Well releasing an editor to the public and using their internal tool chain are two completely different things. It costs a lot of money to cleanup/stabilize, seperate out (i.e. modularize and seperate from code they don't want to release) and more importantly document any internal tools for public release. And more likely than not, the internal editing tools are tied into professional tools (i.e. plugins for Maya or Max) which they won't expect Joe-Modder to own.

I hope they do release some mod tools but I wouldn't hold my breath. Dungeon Siege was built from the ground up knowing full well that they'd release the tools to the public (and it still took them a few months to polish them up after the game was released).
