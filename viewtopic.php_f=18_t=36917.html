Your asking for a couple of things here,

1) The Nomads are a "proper faction" already just one that does not list in game faction listings. To change that you should do a serach for "show all factions" or "see all factions" or something like that.

2) The Nomad Homeworld "Base" was never ment to BE a base so I dont believe it has any docking points. To change that I believe you would have to get a program like UTF editer and add docking points. You would most likely have to also make some changes to its entry in the solar/solararch.ini file and create a base for it just like you would if you were adding any other new base to the game.
