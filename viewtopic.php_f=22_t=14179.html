gh_nukecore

Wed Apr 16, 2003 4:32 pm by gh_nukecore

Just thought I'd chime in here, as I'm the server operator. Speaking to the banning issue, if we receive 5 or more complaints of cheating, we generally will ban the account if we substantiate the claim with a few clans that we have contact with that use the server 24/7. If you think you were banned, please shoot me an email with your ingame name.

Speaking to the server not being good and crashing, let me give you some insight into how that server is running.

Server Specs: Dual AMD Athlon MP 1900, 2 gigs Memory
Hosting Center: Internap Atlanta
Hosting Company: GamingHost.Com, of which I am the co-founder.

The server itself, as is our other gaming servers, is of top quality. We basically bought this machine just for freelancer, to give back to the community for free, as we have done so for NWN. Kind of our way of saying thanks to the few hundred gaming customers that have chosen our services for other paid hosting games.

The Game server executable itself (Freelancer) currently crashes itself once an hour. We have a restarter that watches for crashes and reboots the server when this happens. We have tested it running FreeLancer at 128 players and while the server itself is fine, the game is not happy once it hits ~35 players or so. This is no fault of our operation, rather, it is an issue, as well as other issues that apparently are being worked on in the server patch that was announced. Our hope is that the publisher fixes the problem so that the server player limit can be raised and enjoyed by more folks without the gamelag/crashing that currently happens.

You'll notice that other large servers (over 25 players logged in constantly) have the exact same issue. We hope that players that are regulars on the server, as well as other server operators have contacted the publisher expressing hope that the patch arrives soon as we have done, as the game is quite awesome and the server requirements themselves are very low bandwidth wise.

We are committed to keeping at least one FL server up for the lifetime of the game, and hope to provide larger universes for the players as the publisher releases patches.

If anyone has constructive criticism or suggestions on how to make the server more smooth, we are more than happy to listen and try suggestions (other than lowering the player limit, 25 seems to be the best middle ground at the moment).

Sorry for the long message, I just wanted to make sure that the issues were better explained from our perspective. Our interest is in maintaining and operating a fun Universe where role-play can be enjoyed for as long as the players choose to use our Alpha Servers.

You can reach me directly at matt@gaminghost.com

Happy Star Blazing!

Matt
