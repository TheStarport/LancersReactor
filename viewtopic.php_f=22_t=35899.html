How have you connected there two computers?
Don't quite understand this statement but I will assume you are asking how are two 'puters connected together. Vast majority of time people run cat 5 cable or wireless for 10 or 100 Mbps Ethernet. A DHCP server must be present to administer LAN IP addresses and that server is one of the many things a gateway does. Most gateways can have up to one subnet of LAN IP addresses, like x.x.x.2-254. The x's are chosen for your own LAN but should be one of the three private IP addresses never found on the public Internet (see here under #3, the 255 number effectively means any number from 0 to 254). The 2-254 is the subnet I mentioned and thus you can see you can have up to 253 Ethernet devices on your LAN. The number x.x.x.1 is always reserved for the LAN gateway IP address. So you can see as long as a LAN IP address is available you can connect a large amount of 'puters on a LAN and they all share the FL server plus any connection with the Internet.

An exception would be those 'puters that use another means to connect to a gateway like Firewire or USB. Pretty sure those gateways only accept single connections from other 'puters.
