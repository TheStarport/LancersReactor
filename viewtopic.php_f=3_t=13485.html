Improved Communication Display

Yes i know this was written as a reply, i was writing it out and figured it made a good thread in and of its own, so i decided to post it:

Better Communications System. EXACTLY like STARLANCER but improved, with various things to ask it using numbered button commands

Request Backup: The base sends a wing of fight who form with you and help you within the same system for a set amount of credits

Request Escort: Pay a base, and click a BASE in the BASE LIST where you want to be escorted to, it automatically sets some player waypoints, if you stray one system far from the set path your escorts leave

Register: Allows you to make a checkpoint at the base without docking (when u die u respawn there)

Trade: allows a player to ask what commodities sell best and worst on that base, and how much commodities in his hold would sell there

and various In-System commands, DISTRESS CALL which activates a radar blip for everyone on the system which indicates a stranded player. also, when selecting a ship, you click HAIL button which opens another screen where you can choose the following:

Destination: where their going

Home: Asks them where their trip started at

Waypoints (patrols only): How many checkpoints are left on their patrol path

Designation: Asks a ship who he works for, what he is and what hes doing

Drop Cargo: Opens the ships Cargo Scan screan, where the recycle bin (jettison button) is replaced by a HAIL icon, which you click, then asks how much you want from the ship

Taunt: Select various taunts to send to a ship, each calling for a different and random return such as "Im looking at a dead man" and the reply being "you must be looking in the mirror"
