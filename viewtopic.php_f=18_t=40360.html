Hovis Solution

Playtesting my mod the other day and came up with a few ideas.
Basically, i've inceased the cruise speed in my mod to 450 and was finding it a bit tricky doing the race course. I've come up with a solution that can be used no matter what the cruise speed the modder has set.
Hovis Racers, an Intercepter class light fighter, 2/1, 4opt2, 2xthrusters and is using the dagger archetype. Whats different, is that i've created a racing powerplant which is only available when buying a racer. Its a basic borderworld fighter plant with 5000 thrust capacity, regen 450, a hardwired rally thruster that is twice the output and power of the advanced. The only way to get the plant and the thruster is from the goods package. The thruster has not been added to the buyable goods list so once the ship is sold people can't pop it onto their eagle.. Its mentioned instead in the ships infocard.

*edit, actually the thruster is different to 2xAdvanced thruster, i had changed it before hand. The stats are max force 160000, power usage 250

The ship itself comes in 5 colours, just to keep things interesting.
You can see these here:
Hovis Racer Blue
Hovis Racer Red
Hovis Racer Green
Hovis Racer Purple
Hovis Racer Yellow
So, if you mount the heavy thruster as well you end up being able to thrust at over 450 and should be able to leave Hovis for dust.

It would be good to be able to create a racing minisystem for multiplayer. I know that the mssion in the story is done by triggers so no go, but maybe using the dense asteroid field with zones and lanes excluded it could create a racing track. You could have several tracks in one system and use tradelanes to take you from one to another..
