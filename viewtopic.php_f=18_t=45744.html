Hi Richard, you can set it as the starting ship and loadout in Characters\NewCharacter.ini

[Package
nickname = ge_fighter<--the package not the ship
strid_name = 11051
strid_desc = 11551
ship = ge_fighter<--
loadout = MP_ge_fighter<--you can add loadouts and they will work server-side
money = 2000

[Faction
nickname = new_player
rep_group = co_ic_grp
base = Li01_01_Base
Package = ge_fighter<--linked to the above entry
Pilot = trent

Another way would be to use FlShell and add a Order based DOt file. If you want to go this way then let me know and i'll go through about making up custom Dots if you like.
