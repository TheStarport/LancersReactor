Well im no expert at textures and i've not done it myself before but it should work like this.

Say you have a website and the background to it is called background.jpg. If you went to your server and deleted background.jpg then saved a different file as background.jpg and uploaded then the new file would be your new background. Now think of Trents clothes as a picture. If you replace the old picture with a new one with the same file name then when Freelancer looks for the picture of Trents clothing it will find your one not the default one and display that. All you have to do is replace the old one with yours.
