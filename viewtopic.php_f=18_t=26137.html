KYJellyFish

Sun Nov 16, 2003 6:06 pm by KYJellyFish

To Edit trent like I did is very easy.

You'll need to use your UTF editor
Pic a character file you want to edit
either from the heads or bodies directory.
Open that in the utf editor and and open up texture library an select the tga you want to extract. But Select MIP0 only.

Now that you have the texture do whatever you want to it save it. Now shrink it 50% of the original pic ture size save it Iike to save mine as "whatever"0.tga, "whatver"1.tga etc.... untill you last pic is 1 pixel by one pixel.

go back to utf editor and import you new texture to each MIP you have to have all those little files or your texture won't work in the game and also you have to keep the texture to the model you took it from. Example you extracted pl_male2_head.dfm textures but imported them into pi_pirate5_head.dfm you texture will look distorted as male2 is is bald and trent has hair.

You can choose the DDS option for your files but I found them to be "stretched" over the model but DDS eliminates the hole joy of resizing images over and over.

But that pretty much covers it... Have fun!
