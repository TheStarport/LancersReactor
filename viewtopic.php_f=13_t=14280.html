Tarrax

Fri May 16, 2003 9:46 pm by Tarrax

Hey there,

The SERVER Mod is not only for running a B5 Modded server but is also required to play on one. The name is actually a little confusing in this aspect. But to make it simple, if you download the B5 Server Mod and install it to your Freelancer directory it will allow you to both host a Server and also play on a remote one.

BTW, unlike most other Mods, this one overwrites more than just INI files. This being the case if you try to use FLMM to install/uninstall it you will run into problems resulting in a re-install of the game being required. Best bet is to backup the entire game folder prior to installing the mod, then overwriting back the original files when you want to play the original game or another Mod.

Hope this info helps and have fun.


Tarrax
