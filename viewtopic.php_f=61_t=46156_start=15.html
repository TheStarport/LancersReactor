As Chris said: "When I get home I go through your checks; I know for one that I had put my 3db file in:equipment\models\commodities\nn_icons\cylonraider.3db"

The 3db file can remain in the same folder as your ship cmp & mat file. The main thing is to make sure the shiparch.ini references where that file is.
Here is an example from my own files:
Under: DATA\EQUIPMENT\goods.ini

[Good
nickname = D1954X_hull
category = shiphull
ship = D1954X
price = 2500000
ids_name = 237095
item_icon = SHIPS\CUSTOM\D1954X\D1954X.3db

My 3db file is in the same folder as the cmp & mat files for ease of referencing since the ship itself isn't really considered a commodity, but rather "equipment" .

As for LOD ranges on custom ships...
I have them set at : 0, 999999

Ogu

Fri Jul 27, 2007 5:06 am by Ogu

As long as the LOD ranges are at least around 1500, you should be able to see the ship.

I'm not exactly sure what you mean by "it won't show up" in the dealer. This could mean a few things:
1. There is an icon for your ship when you go to buy it, but you can't see your ship in the little rotateable preview window.

2. You can't see the ship in your icon (ie, the icon is black or clear), but the ship actually shows up in the preview window.

3. There isn't an icon for your ship showing up, so you can't buy it at all. If this is the case, there should be an empty ship sale location - by this I mean that if you made it for sale at Manhattan, there would only be two ships for sale instead of 3.

Remember, you can only have 3 ships for sale at a ship dealer. Any more won't show up and you'd crash when you went to the ship dealer.
