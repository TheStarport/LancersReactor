FD, it is shared ram, all laptops are now. If you check the memory count at boot then compare it with the amount the system applet properties tell you Windows is using, the difference is the amount of RAM used for video. You can change that amount in the BIOS (usually in integrated peripherals) which is handy if you never intend to play any games on it, cos you can knock it down to 64mb or 32mb, which is all you need if you ain't gaming.
