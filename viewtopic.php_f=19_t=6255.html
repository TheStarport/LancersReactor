Stinger

Thu Feb 13, 2003 6:42 am by Stinger

Not too familiar with that particular firewall, but i'll tell you what I do know about firewalls in general.

If you can tweak the configuration within the firewall itself to make specific pieces of software "trusted internet server' rights, that is golden. If not then keep in mind Freelancer uses the standard DirectPlay ports, so you'r firewall needs to pass UDP ports 2302-2304 to the server. Here's some more thoughts...

Internet Explorer 5.0 or Newer - To play Freelancer multiplayer online, you must first install Internet Explorer version 5 or newer.

Routers - Ensure that you have the latest firmware updates from the manufacturer.

Setting up and Disabling Internet Connection Sharing - You may want to experiment with turning this function off and attempting to connect again.
