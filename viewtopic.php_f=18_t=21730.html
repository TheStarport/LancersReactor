I hope this info is important to modders

The resource numbers for the .dll's.

Resources.dll -> ????
InfoCards.dll ->Contains all infocard text resources
MiscText.dll ->Contains other bulk text resource: rumor, offer, and mission text
NameResources.dll ->Contains names of things, primarily.
EquipResources.dll ->Contains names of equipment (in string table) and infocards for equipment (in HTML resources)
OfferBribeResources.dll ->Contains random mission offers and bribes
MiscTextInfo2.dll ->Contains MP UI elements as well as other misc. text



0 - 65535 = Resources.dll
65536 - 131071 = InfoCards.dll
131072 - 196607 = MiscText.dll
196608 - 262147 = NameResources.dll
262144 - 327679 = EquipResources.dll
327680 - 393215 = OfferBribeResources.dll
393216 - 458751 = MiscTextInfo2.dll


I dont know if thats to important but I think it is if you want to have an organized Freelancer mod. Hope this helps some people if they're looking for this information.
