Ship Ideas and cloaking opinion

First off I would like to talk about ship ideas.

I think it would be cool if and when modeling\mesh editing becomes available, to have these ships modelled into the game :

Stargate SG-1 Ships

Goa'uld Death Glider - Light Fighter Class
X-302 - Heavy Fighter Class
Prometheus - Battleship\Cruiser Class
Goa'uld Cargo Ship - Freighter class

I think these would look really cool in freelancer especially the X-302 which is my fave ship in the SG-1 series.

Anyway that's all on the ships, Now onto the cloak Mod :

Cloaking is a very delicate matter in FL Servers because it can give unfair advantages to players. Here are what rules I think should be implemented if this mod is ever created (if possible at all)

1. Cloaks should disable all weaponry, docking, trading, scanning, cruise.
2. I have seen threads on upgradable scanners, and I think there should be a scanner to detect cloaked ships.
3. They should use power consumption the same as weapons.
4. Should use cruise power up function so that u can't just cloak stright away, the same for de-cloaking
Explainations :

To rule 1. : I have chose all disabilities because
1. Players can't use there cloak to sneak up on other players and decloak to attack as they will appear on scanners half-way trhru de-cloaking giving players a chance to manouver out of the way, no docking so players cant cloak and dock to retreat from enemies easily, trading, well you can trade with someone you can't see, scanning is optional.

To rule 2. : There should be a way of detecting cloaked ships as this would definitly make cloaking have an unfair adavantage.

To rule 3. : Cloaks should use power consumption so as not to have infinite cloak and so have to recharge after a period of time, giving the option of upgradable cloaks.

To rule 4. : Cloaks should use cruise power-up function as default because like cruise we don't want it to be too easy to cloak straight away, and in addition to cruise disrupters, there should be cloak disrupters also.

Anyway thats just my opinion and suggested ruleset if cloaks were ever implemented server-side at least.
I may have been to harsh on some of the rules but that is how they should be modded to keep the game fair for everyone.
