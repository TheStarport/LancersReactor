Cash, heh. Extract flcodec.exe from Freelancer Simple Save Editor 1.0 by TheJkWhoSaysNi (downloads section). You can do this while Freelancer is running, although the save file summary box in the lower left won't update until you re-load.

1. Open a dos box and go to the freelancer SP/MP saves folder.
2. Choose a save file (usually the most recent one, look at timestamp).
2. "flcodec -d <savefile.fl> char.txt"
3. "edit char.txt"
4. Using edit which you opened in (3), scroll down a little bit to where it says credits.
5. Replace piddling amount with godly amount.
6. Some creatures may not have access to a treasury. ^H^H^H^Htrack_1_4hd.sdt
7. file -> save, then exit
8. "flcodec -e char.txt <savefile.fl>" Yes this WILL overwrite the original save file, so backup before doing this if you want.
9. Go back to Freelancer, re-load.

The default SaveEditor should do all that behind the scenes, but with some mods e.g. Rebalance, there is way too much new system info and it quits with an overflow error. Oh, and Ignore the random Dungeon Keeper ramblings.
