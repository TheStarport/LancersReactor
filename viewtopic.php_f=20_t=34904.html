*FIX* MP Internet Connection Issues - FIREWALLS

Does exactly what you expect - simple instructions on how to grant your Freelancer.exe access to the interent through the new Windows XP firewall. Be warned, if you are crashing when attempting to connect to the MP games - then PLEASE try turning off 3D sound in your games options menu. Select options, and disable 3D sound. Then restart the game and try again.

Do the same for FLServer.exe if needed too.

Warning - you may have to RE-Grant access IF you use a mod which changes the version number. This is a common occurance for not being able to connect to the net when using mods.

Click on the Start button
Click on CONTROL PANEL
DOUBLE Click on WINDOWS FIREWALL
Click on the EXCEPTIONS tab
Click on ADD PROGRAM
Click on BROWSE
Navigate to Freelancer EXE folder which contains the Freelancer.exe
Select the Freelancer.exe file and click on OKAY.
See its been added to the allowed list
Apply changes and exit.

McAfee Personal Firewalls

Start your firewall by double clicking on the icon in your "tray" (bottom left of screen).

Click on the internet applications tab.
Click on small icon saying "New Allowed Application"
Navigate to your Freelancer.exe contained in the EXE folder of Freelancer
Default location is C:\Program files\Microsoft games\Freelancer\Exe
Double click on the Freelancer.exe file
It should now appear in your listings as ALLOW FULL ACCESS in green.
Close application

You will need to repeat this EVERY time the Freelancer.exe file is modified by mods. If you play Evolutions/TNG/WTS/Epsilon you will need to repeat this EVERY time you activate/deactivate a mod to allow the new Freelancer file version access to the net.

Zone Alarm

Open your Zone alarm
Click on Program Control Tab (left side)
Click on Add Program
Navigate to your Freelancer's EXE folder
default is C:\Program Files\Microsoft games\Freelancer\EXE\
Click on Freelancer.exe and click ADD
Close Zone Alarm.

Remember to do the same for the FLServer.exe if you are running that!

Norton Personal Firewall (2004)

Right-click on the Norton Personal Firewall icon in your system tray and select the top menu option which should be Norton Personal Firewall
Click on Personal Firewall in the Status & Settings tab
Click on the Configure button in the blue box on the right
Click on Programs tab
Click Add button
Browse to your freelancer.exe
In Program Control dialog select Permit in the dropdown labeled 'What do you want to do?' and press OK
Repeat for flserver.exe

Kerio Personal Firewall

Right-click Kerio Personal Firewall icon in system tray and select Administration
Click on Advanced button
Click Add button
Enter information:
- description: Freelancer Client(or Server if flserver.exe)
- protocol: TCP and UDP
- direction: both directions
- port type: Any port
- application: Only selected below
- remote endpoint: Any address
- remote port: Any port
- rule valid: always
- action: permit
Click Browse button and navigate to freelancer.exe
Repeat for flserver.exe
Apply then hit Ok

If you use mods that change the freelancer.exe or flserver.exe you will get a popup notification that the file has changed and you will need to press OK for it to allow the rule to apply to it.
