Wed Jun 21, 2006 11:47 am by bakedpotato

The problem, as I read what you say you did, is you didn't follow the instructions exactly as they are written...

It says delete your Freelancer folder...not uninstall Freelancer.
You see the problem is that if you uninstall Freelancer a lot of the time files from MOD's get left in the Freelancer folder.
You can uninstall Freelancer if you check the folders too make sure there are no files left behind... but it's easier to just delete the Freelancer folder.
That way you make sure that you don't have any files left over after an uninstall.

Then install the SDK files after you install Freelancer again.
Then activate the MOD that you want to use.

Also you should move or delete all your saved game files because they will make Freelancer crash. That's guaranteed.
