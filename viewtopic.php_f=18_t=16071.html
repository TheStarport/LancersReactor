Tue May 06, 2003 3:13 pm by bakedpotato

that mod was made by killermatrix and he would be the one to answer this question. I am not going to get into the technical side of this but I'll give a general idea of how to install mods. Some may have differences but heres the general idea. Download the mod manager, I use flmm,
install the mod manager, download all the mods that you think look interesting to you, right click on the mod file, you should get a pop up box with the options
open
open with
run as
rename
properties
scan with norton antivirus
clean sweep
send to
pin to start menu
create shortcut
cut
copy
delete
theses are just some of the options you will see some people will have more some will have less depending on the programs installed on your computer.
unfortunately there is not a stantard install program for mods yet.
the babalon 5 mod has the open option, left click on open it will take you to the install screen with the option
run as
which user account
it will give you the option of which directory to install the files to.
now you must decide where to put the files if you select freelancer you overwrite any files with the same name and extension so you might want to
create a folder before you begin so you can install the files to the folder you just created, then you manually put them in there respective folder. use backups or move any original files with the same name incase you decide later you do not want this mod installed. you can install it in the mod manager directory this is where it should go.
other mods have the option open with and another pop up asks run with the mod manager, select this option you dont even have to unzip them first and it will install the mod in the mod manager for you.
now start the mod manager and select which mod or mods you want to activate then tell the mod manager to start the game and your good to go.
always use the mod manager to start the game up it activates and deactivates the mods you want to use for you.
I hope I did'nt confuse you, any posts on this will be answered if I can.
