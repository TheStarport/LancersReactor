Tempered Enjoyment

Having used earlier versions and mostly enjoying them decided to use the latest & do SP again. Other than a few minor gripes the enjoyment has been tempered by the same fault which has plagued it from the beginning. {& this is NOT the only MOD with this problem IMMHWO}.

That is the ship prices have been made so high as to be obscene and ridiculous. {Come on a Patriot for over 100K?} I like to try diff ships when available, even in SP. But needing 500,000 to 1,000,000 credits minimum is really a bummer. I can partially understand the prices but there needs to be a balance. If prices are increased so drastically then random & game misson payouts need to be increased.

So, get a freighter and trade for awhile; I have no problem with that. However, any decent freighter needs 2Mil or more. Some might say, "Not so" but considering additional/better weapons & equipment it does.

Thanks for listening.

It is hard to soar with eagles when you fly with turkeys.
