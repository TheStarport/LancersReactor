Hi,

I know it's kinda weird for jumphole. So double click on a system, in the universe view. You enter the "System view" mode not the editing one.
Right click on a point not used by any object and it will bring a menu proposing to add a jumphole.
You need to set the start and end (preferably into another system) , Double clicking exit the system view mode.
Once you have selected the start and end , a Dialog box will appear and ask you more details.

And that's all.

I am planning to make it able to load text ini files today so it should be compatible with the mods.
However the script in flmod manager does not support suppression at the moment, so it might take longer to do a full support for it.
