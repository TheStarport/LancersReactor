"Does anyone know how to allow all new players in MP to have altered start money?"

Modify the mpnewcharacter.fl (in the Exe folder) by opening it in notepad (no conversion needed).
You can change start loc, money (level), faction.

"How to change ship values as a server ie weapon classes, hull armour and stuff"

You can't with simple server-side mods. Each player will need the mod.
See the tutorials.
