Campin Carl

Thu Jun 05, 2003 1:05 am by Campin Carl

Fred,
The same thing happened to me but i fixed it. All you have to do iz make sure all the files you are editing and copying are NOT Read-Only. U can right click on a folder and then click properties, check read-only and apply it to all the subfolders and files, Then do the same thing except uncheck Read-Only this time. It should make all the files non-read-only and the mod should install fine. If the problem persists then try replacing the problem files manually (cut-and-paste them from your rebalance mod files). BE SURE TO MAKE BACKUPS OF THE ORIGINAL FILES IF YOU MANUALLY REPLACE FILES!!!!!!!!!!!!!

Hope this helps-
-Campin Carl

Fred,
Make sure when u cut and paste that u dont "copy" and paste but actually click and drag the file into the new directory. The files u manualy install need to be deleted from your mod file directory. In other words: keep the original game files u backed up and delete all the mod files u manually installed from the Rebalance mod folder. If the FLMM does not sense a file that needs to be installed it will skip over it and go to the next file. Look for the files that are causing the problem, manually install them from your mod folder, and delete the mod files u just installed from your mod folder.
