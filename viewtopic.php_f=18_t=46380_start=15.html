Bejaymac

Tue Sep 04, 2007 4:31 pm by Bejaymac

Cheese is probably correct in that FlModeltool is messing things up, it was designed for vanilla SURs and not custom ones, it also renames all of the component parts of the SUR as "ROOT".

CMPs, 3DBs, SURs & MATs can and should be done entirely in milkshape, the only other tools you need are UTF edit for minor alterations to the MAT (glow maps etc) and HardCMP for your hardpoints.

The only other thing I can think of is that I said to use shrink wrap, I've never tested it as I've only worked on ships and they use the bubble. I'll dig out an old SFC station tomorrow and see how it works, but iirc it's bloody huge so it will probably take me some time to do a SUR for any testing.
