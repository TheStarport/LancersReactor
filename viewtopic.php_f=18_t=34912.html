This is what you did i think... you just replaced an existing ship icon and replaced the material... also renamed the file.

This is a common mistake...

There are two solutions with this...
1. Use the commodity icon files that. The only downside is there is a chance that somebody uses your same icon file (the inside).

2. Use the CRC calculator. This will ensure that you use unique ids for your ship... check out the tut on this one on ship icons. This one will absolutely work.
