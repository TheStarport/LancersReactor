Thu Jul 03, 2003 10:45 pm by Browclops

Well, the simple way would be to reinstall all the Freelancer Server files. Start over, without any additional programs running that would affect the game state. Let it be for a while and see what your system can handle. Get a test machine set aside for testing anything that changes any aspect of the game. Install the add-on stuff on the test machine first, so it doesn't ruin your production environment. An ideal setup would be to have 4 machines; one permanent visible server, one permanent invisible (or LAN only) test server, and two client machines, one modded and one straight up. You may find in your testing that adding a particular change makes the test server more unstable if a modder comes on, in which case you may rethink the add-on, even though the add-on itself doesn't make the server any less stable. But I digress. The short answer is; Start over, move slow and test, test, and retest any change you make, and make them one at a time.

Browclops
