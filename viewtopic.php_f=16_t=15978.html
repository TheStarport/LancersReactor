WRONG, a fellow gave this tip to me in the multiplayer forum. I just need to know how to change my name, because i dont want to be 1 of 1000 trents.

Well its a bit´tricky :
1: First Go to C:\Documents and Settings\YOUR NAME HERE\My Documents\My Games\Freelancer\Accts
in that folder there a 2 folders Multiplayer and Singleplayer keep that in mind.

2:Start up the server create a new character,log off and shut down the server.
(hint* do not have any other characters on the server except the new one)

3:in the singleplayer folder find the save game you want to use "say if you want the ship you got right after finishing the story line then use the file named AUTOSAVE" copy that file an go to the Multiplayer folder.

4:in the multiplayer folder go to your characterholding folder(hint* it is much easier to find if you are the only ID that has been created) in the folder you should see a file with some numbers and such,if there is a file called "name" just leave it be it is not that one we need.Now copy the AUTOSAVE file into this directory now we have have the file with number & letters,the "Name" file and now the AUTOSAVE file.

5:Mark the file with number and letters (this is one of my files 0b-ae6d8b36.fl)
now press F2 you should now be able to rename it but don´t do that,instead copy the name from the file and then delete it,now highlight the AUTOSAVE file press F2 and copy the name from the other file into it.

6:Start the server and join it,now you should have a character named Trent select it and you are set to go.

WARNING: Do not use savegame files where you are in Omicron minor,Omicron Major,Alaska or any of the other systems that are only playable in SP or you will have to fly around in those systems till the end of time lol.
