I am not sure if this will help you wizekrax, but these conditions fixed my situation. Thanks for the suggestion cslim -- I actually did reinstall DX9

I was using the ZyXEL Prestige 600 series modem (with earthlink), and I swapped it with a Efficient Networks Speedstream 5260, and now everything is fine. I can only speculate that some internal settings on the modem itself may have been blocking the UDP ports (the modem was not shipped with the default password from earthlink, so I could not access it). Of course, it is odd that one computer would not have a problem but mine would -- but it worked.

I hope this can help you in some way.
