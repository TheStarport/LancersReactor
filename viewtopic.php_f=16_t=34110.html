The hole is gone. All jumpgates/holes work again.

It read
;After Jump
Activate Cordinate1 =1

Restore Power setting = 1

;Server only
Admin takeover = 0

JumpTunnel deactivation =
(it should read = 5.00s)

SO the hole is the Jumptunnel which has been left open!

I tried the JumDrive again after I had corrected the error (musta been caused during file compression) and no hole appeared.

Problem solved
