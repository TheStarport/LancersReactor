Did you check your Multiplayer ID? If you reinstall, you should write down the one you originally had (you can find it by running myflaccountid.hta in the exe folder) if you want to retain your identity on all servers. You can enter it when you install the game instead of accepting a random one.

But, if you can see the account and ID #s in the server menu, you COULD try entering each of those MP ids when you install the game, until you find the exact one you were using. The name of each account, the string of numbers and letters, is the MP ID of the person who logged in. If you played on your server one should be your old one, and if only a few people played on the server it shouldn't take too long to find.
