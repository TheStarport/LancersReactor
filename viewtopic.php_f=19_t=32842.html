Inability to Connect to the Global Server do to ZoneAlarm

We have all seen posts about not being able to connect to the Global Server. I have posted a thread about that same problem. For those people running ZoneAlarm, myself included, there is one simple way to fix this problem: Turn off ZoneAlarm. Simple, huh? For some reason ZoneAlarm does not like FL and, as I have recently come to beleve, other Microsoft games. The reason? I don't know. You can clear the ports for FL but ZoneAlarm will not let FL make a net connection. It's the same thing with Dungeon Siege, another game produced by Microsoft Game Studios. Where as non Microsoft games, like Diablo 2, FFXI, and WarCraft 3 can make a connection with no problems. Now i'm not saying that you should get rid of ZoneAlarm, personally I think that its a good fire wall with a few flaws, but if you do play, or are going to play, games like FL and DS online you may want to look into a new fire wall or find out how to make ZoneAlarm let these games through.

Am I understanding this correctly - you're behind a router and still running Zonealarm?
