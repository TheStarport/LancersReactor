You can do this yourself by adding

visit = 1

to all the systems in universe.ini


If you want a separate mod to do this, follow these steps:

0.conditions:
- a basic understanding of the English language
- necessary programs: FLModManager and BiniQDU.
- (obvious) make sure Freelancer is not runing
- deactivate any mod

1. Open FLModManeger's mod folder

2. Make sure FLModManager is not running.

3. Create the following directory tree structure in FLModManger's mod folder:

[ CompleteMap (or any name you like; this is the name of the mod)
---[ DATA
------[ UNIVERSE

4. Open ..\Freelancer\DATA\UNIVERSE and copy the file universe.ini to the folder CompleteMap\DATA\UNIVERSE you created in step 3

5. Open this copy of universe.ini in BiniQDU and add "visit = 1" to all the sytems; for example:

[system
nickname = Br01
file = systems\Br01\Br01.ini
pos = 2, 11
msg_id_prefix = gcs_refer_system_Br01
visit = 1 <----- add this
strid_name = 196614
ids_info = 66104

6. Save universe.ini; make sure you save it as a .ini file and not a .ini.txt file

7. BiniQDU might leave some flies in the CompleteMap\DATA\UNIVERSE folder; you don't need them, so you can delte them:
- bini.exe
- universe.ini.txt

8. start FLModManager, activate the CpmpleteMap mod

9. Start Freelancer and procees as usual
