MasCalaveras

Thu Mar 13, 2003 6:27 am by MasCalaveras

If you use the command line option, you should pick the "LAN" setting, which prevents the game from attempting to connect to the global server.

However, the server won't show up on the list until it responds to a ping (so you can see the name, options, etc.)

So if it isn't showing up, then you can't ping it for some reason.
