soundspawn

Fri Dec 05, 2003 6:45 pm by soundspawn

May not be the right thread for this request, but I can help you. Download the UTF editor (if you haven't already). Open up the mat file, go to the texture library, and open up the texture you wish to use. Click on the ".tga" node and click extract. Now in the UTF editor there's that spot above the hex value that usually says a bunch of garbage. If it says anything looking like DDS, then extract it as a .dds file. These are microsoft direct X textures, and you can find a plugin for photoshop to edit/save these. If you need the plugin, let me know. Once you have edited the texture, import it just like you exported it, but make sure it still shows ".tga" in the node name when you're done.

Cardamine, i've seen in other topics that you're not very good at INI manual editing. (ex : the hispania sleeper ship) If you are (even a litlle) good at FL editing, you can easily find the part of the file that contain the infos about it :

[Object
nickname = Hi01_hispania_sleeper_ship_1
ids_name = 261794
pos = -7511, 0, 26130
rotate = -12, 40, 50
archetype = hispania_sleeper_ship
ids_info = 66017
reputation = fc_ou_grp

So now if you want to add a sleeper ship in any of your system, you'll just have to put these lines anywhere into your system ini. Of course you can change the rotation, the position (can be done easily with FLExplorer), ids name, ids infos and faction. It's not so dificult ! (for someone that can make a working system without any other help than Bini, anyway !) If you have any sort of problem, just ask me and i'll surely be able to aswer.
