Holly god .. I FIXED IT... using deductive method. (eliminating all impossible whatever remains, however improbable, will be the cause :p )

It was a "corrupted" save game. I didn't remove save games during re-installation so that's why it didn't fix it. Apparently, the game writes autosave when you engage into fight or something and when you make changes (aka mod) this save causes trouble upon start up. So I deleted Restart.fl and AutoSave.fl and everything is fine again. phew.

+++edit
