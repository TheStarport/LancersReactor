The Monkeys 1.18

Hi, I'm Black Eagle!

I'm the creator of the Monkeys Mod and finished my new version. It should soon be available for download here on lancersreactor!

The aim of the Monkeys Mod is to add new things to FL (factions, systems, ships), which really fit into the existing FL world. I don't want to change FL, because I think it is a great game. However after playing it several times I could not encounter new things and also found some unfinished parts in the game. Therefore I created the mod, which adds four new factions to the game:

The Monkeys, the Robots, the Nomads, and the Order.

Of course all these factions were partly in the original game, but only during the story or on one planet (Planet Primus and Gammu). So I made them realy factions, with their own ships, bases, systems, history, etc.!

In this forum you can post your thoughts about version 1.18 and also any idea, you would like to see in the next version. I'm not very fast in modding, but I'm working on the mod as often as possible and will try to add all of your ideas, which in my opinion are good.

Some important things:

- Please read the readme, it is not so long and has some useful information in it.
- If you get problems with the mod (cannot activate, bugs, etc,) you can ask here, but there are some things, you have to care about:
- The mod is not compatible with any other mod, except of the "OpenSP for the Monkeys Mod" I released earlier!
- Please use the FL Modmanager 1.3 for activating the mod!
- Before you activate the mod, please deactivate all other mods and restore backups!
- If you still can't activate it, please do a fresh install of FL and the SDK and then activate the mod!

That's it! I hope you enjoy playing the mod and it would be nice to see you here in this thread! I'm not at home for three weeks now, but I think I will be able to visit the forums anyway. But it might take some days until I respond!

To all from the old thread: I hope to see you here again! Try the new version and say what you think about it. Please do also tell me if there is something you do not like, so that I can improve it. The commodity market is only a basic one (look in the old thread for details), but I will fix this with a patch as soon as possible!

See you!

Black Eagle

Hi everyone!

I'm very sorry that this bug is in the mod and that I could not sort it out earlier, but as you may know, I'm not at home until next week.

However I was able to download the mod and I found the problem. The patch is already sent to backedpotato and he will surely bring it on the site very soon!

The patch fixes the bug, that the Leeds system was not accessible and also some spelling mistakes in some infocards.

Please be sure to read the instructions on how to activate the patch. They will be on the download site. Again, I'm very sorry for this, but I really was in a hurry, when releasing the mod and therefore missed this bug. I hope you will now like the patched mod!!!

BIGMAG: The patrol files are all fully working. There is no mistake in them. I spent some days on getting familiar with patrols, but now I think it isn't that hard if you go through it several times to check for spelling errors. The bug was in the leeds system file and it was a simple spelling mistake!!!

About your starting reputation: You have to change this in the mpnewcharacter.fl (for Multiplayer) and in the newcharacter.fl (for SP. You need the SimpleSaveEditor for decoding of the last file). There you can change your starting reputation to any faction.
For other changes on reputation you have to modify initialworld.ini and empathy.ini! Please try to find some tutorials (there are many here on TLR), as I do not have so much time now!

For your problem with starting in Omicron Major: This should not appear, unless you have any other mod active (except of OpenSP for the Monkeys) or use old savegames!

AGAIN: The Mod is only compatible with "OpenSP for the Monkeys", which is downloadable from TLR. Do not activate any other mod, as I cannot guarantee it will work!!!
If you have installed the SDK and the Unofficial SP Patch 1.4 and activated only the Monkeys (and maybe OpenSP for the Monkeys) everything should now be fine with the patched version!

If you find any other mods, or have ideas or suggestions, just post them here! And of course many thanks for telling me about this bug, so that I could sort it out.

But now, just play the mod and have fun!!!

Edit: One other thing. I was offere a membership of Frostworks.com, but I'm not sure if I should aggree with this. Of course it would be nice to have my own Forum for the mod and I think I would not have a disadvantage.
