Ok, finally got this thing figured out. For those of you that are having problems try my steps.

1). Delete the FL folder completely or any saved game files.
2). Install A brand new spankin copy of FL and restart
3). Install the newest version of SDK available
4). Install FLMM
5). Go to FLMM's folder then in it go into the mods folder and create a Folder called FLRebalance 3.5
6). Sence this mod is deliverd in ZIP, Extract all of the mods contents into your FLMM'S/mods/FLRebalance 3.5 folder.
7). Edited: At your disgretion Move the .exe that came with the mod for FL into FL's exe and choose to replace the old exe with the new one. Or you can remove the old exe and save it somewhere and put the new one in there.
. Go to FLMM, Activate the mod (should show up under the FLMM'S mod list) FL Rebalace 3.5 Full, Now u should get an option to choose how u want to start your game off.

9). Enjoy
