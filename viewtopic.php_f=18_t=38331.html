Cold_Void,

I have noticed an odd thing about milkshape ... the .cmp importer seems to like 1.6.6 the best although there is a poly limit.. other later versions of milkshape seem to have aproblem with it.

Adding the mesa graphics .dlls seems to improve function with both versions of the .cmp exporter.. I use 1.70 mostly with the added .dlls and have few problems and very rare crashes.. one thing with the .cmp exporter tho that seems to be a factor is models that are not more or less centered on the x,y, and z axis.. off axis models often crash the exporter... now, I dont know if any of these factors really make a difference.. but this is simply what I have found.
