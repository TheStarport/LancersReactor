Greetings,

Some time ago I used to host a file page for Tribes and Tribes2, I recently wrote a portable system for allowing file uploads... Its pretty simple over all but I can set it up for any game (or anything else really) pretty quickly...
I have set one up for Freelancer to allow authors to post files for others to download, for free... This is good for people who dont have web space and need a free place to have hosting for their stuff...

Freelancer File Repository.

I have tested it with IE 5+ and IE 6 and things seem to be working correctly... If you want to use the site, great... I tried testing it out with Unreal 2k3 but the community never really started using it... Perhaps people here will find it useful.
