Sat Apr 03, 2004 12:32 pm by Chips

Earendill informed me of this some time ago.....so kudos to him over the info.

If you unlock the gates etc in the game, it will only work if you do a complete player wipe, due to the players save files telling the gates are really locked. Now, if you then import a save file or something else that has those gates locked.....it will then CHANGE all the save files and add that they are locked again (from what i understand) so it will then "Lock" all the gates once more - EVEN though the ini files says its unlocked. If you have made the server modded, but then used the old save games for characters, it has probabily done just this.
Alternatively, if you did it the other way round, this is probabily the problem as well
