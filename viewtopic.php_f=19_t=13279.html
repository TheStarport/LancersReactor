Yes, i've had the same problem.. Just add them in manually to the command line as mentioned above..

c:\games\freelancer.exe -s{IP Address}:{Port}

The easiest way to obtain the IP address is to use the ASE (All Seeing Eye) which will show you a list of servers for Freelancer. In the ASE you can easily get the IP address and Port number. Simply press Ctrl+C while the server in question is highlighted on the list. Ctrl+V will "paste" that information to any text editting window.

Beginning with the -s, add that IP address to the target area of your freelancer's shortcut properties.

Here's mine.. And this is the maximum number of IP's I could add in WinXP

D:\Games\Freelancer\EXE\Freelancer.exe -s128.186.141.13:2302 -s204.228.228.92:2302 -s131.107.83.10:992 -s216.231.38.24:2302 -s64.124.43.89:2302 -s66.93.130.60:2302 -s67.115.66.169:2302 -s64.139.36.11:2302 -s66.228.205.29:2302 -s66.74.224.223:2302
