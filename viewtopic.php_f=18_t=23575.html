Possible New Mod

A couple of weeks ago I posted a thread talking about somebody making a new mod of my design (I provide the storyline someone else make it) understandably it didnt recieve much response, although the idea was a good one. So now im embarking on a new idea this time with some good help. With the repositioning of the planets on the nav map it is possible to give Freelancer Hundreds of systems, Freelancer could be made on a modular design,every other week or so new systems ,new fleets, new races could be added. But the one thing that would run all the way through the game through every system is the need to survive. As this new Multiverse would be Total war nothing more nothing less.
The one thing that has been missing from all the mods thus far is a descent sized battle for users to enjoy and the majority of the users that play this game want to be in an Epic battle.
Hopefully as time goes on and we get more experienced that is exactly what we will give you.
Our idea is still in the early stages and we need a little help from experienced modders, to start with we need a tutorial on how to add new factions from scratch to a new system hopefully the easy way. Remember this would be our first mod but I dont think anybody has made one before on this particular scale. This could be permanantly growing forever changing. In time when we have the experience we could include trading in cirtain systems or short storylines into groups of particluar ststems, but for those ideas to work we would eventually need a lot of help from other modders and storyline writers.
Any and all feedabck on this idea would be appreciated unless of course your going to ridicule the idea in which keep it too yourself.

Well - for new systems, you can use FL explorer in the editing utilities forum. Make sure you DL both FL explorer and then the Vib 7+ files and put em in the FL explorer directory. I would offer to help - but am busy creating a large mod as well.....
