Ship library project

Hi

I'm thinking of the mod project containing DLL (names and infocards) and statistics for many ships published on LR. That way all ships can cooexists in one universe/mod.

You will have to download coresponding mods from LR, this project will only have statistics/description for ships and will allow them to cooexist. It should be compatible with most mods found here (not the big ones, for obvious reason), so you can have together your favorite mod and a bunch of ships of your choice to fly.

I'm not sure if you need this, so tell me so. It's a lot of work, but can be done.
The library will be rather small (no models inside), and can be constantly updated with new ships.
Also, if you don't like your ships in such a library (not models, statistics and infocards only!) just say it here.
If anyone want his ships to be in this library also drop a line here.

If there is enough interest in it the project will be created.



Tygrys

The idea in short is to combine all names dll into one big and put it on the second or third place. This way you can have any mod using its own DLL on #1 and the ship library on #2. The only things changed in ships description will be their internal numbers. You will have to download coresponding ship mods to get their models. The rest will be in the library.

You CAN have the Pinball with most other mods, it is designed to work that way. See the readme for details.
