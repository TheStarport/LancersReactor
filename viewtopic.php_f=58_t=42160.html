bizarre. that's some weird malarkey going on.

first, update your AV and run a full system scan. what AV is it btw?

secondly, run a registry cleaner like Norton Windoctor or Registry mechanic through your system.

if that doesn't sort it,

thirdly, uninstall Notepad from control panel>add/remove windows components, reboot, then re-install Notepad from control panel>add/remove windows components (you may need the Windows cd for that)

then go to folder options/file types and associate with Notepad .ini's and .txt's and anything else you want to view text-based

and while you're waiting for all that rubbish to finish, you can make us some more Lego!
