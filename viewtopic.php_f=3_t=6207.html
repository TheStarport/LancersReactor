That vid card might be your bottleneck if it's an MX version. The RAM would be the next consideration. If you're using WinXP the entire OS is probably clunky right now with 128 RAM. If you're on Win98 then you're probably fine.

You can tweak down the special effects within FL to run on some older cards and it will run just fine., But I'd consider the RAM upgrad at this point seriously. Even on Win98 that might help and be less expensive than a Vid card upgrade.

Processor wise I'd say it will be fine although you might have to tone down the graphics just a bit to get an increase on performance. You'll just need to test out some settings when you get it, but that system you have will run FL as is.

The mins are on the Home Page FAQ:

Freelancer requires DirectX® 8.2 or greater. (Setup will install DirectX 8.2, or download the most recent version of DirectX from http://www.microsoft.com/directx)

Minimum Configuration

Windows® XP, Windows Me®, Windows 2000®, or Windows 98 SE®
(Windows NT® and Windows 95® are not supported)
600 MHz processor
128 MB RAM
4x CD-ROM/DVD-ROM drive
1.3 GB free hard disk space plus space for saved games (additional space required for Windows swap-file and DirectX 8.2 installation)
16 MB video card with DirectX 8.2 compatible driver
DirectX 8.2 compatible sound card
Keyboard
Mouse
An Internet Connection for Multiplayer (56k v.90 or faster required)
Recommended Configuration
1 GHz or faster processor
256 MB RAM
8x or faster CD-ROM/DVD-ROM drive
32 MB video card with DirectX 8.2 compatible driver
A Broadband Type Internet Connection or LAN Network for Multiplayer
Recommended for Serving Multiplayer Games
Dedicated computer with a Broadband Type Internet Connection or LAN Network: DSL, Cable, T1, T3, etc.
