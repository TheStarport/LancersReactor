Hi all,

I have found a method to fix this problem of taking off crash when server and client is running on the same machine.

Procedure:

1. Use clean boot
http://support.microsoft.com/kb/331796/en-us#

2. Run FLServer.exe and Freelancer.exe

3. Press alt-tab to switch to desktop, press ctrl-alt-del to open task manager,
In Process Tab (Second Tab), right-click FLServer.exe -> Priority -> Immediate
right-click Freelancer.exe -> Priority -> Above normal

I have tried many times and work fine.
If anyone has questions about this problem, welcome to send email to me: tommy357@yahoo.com
