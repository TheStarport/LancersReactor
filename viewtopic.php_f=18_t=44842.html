Bejaymac

Wed Oct 18, 2006 3:14 am by Bejaymac

Sounds like a CRC conflict, when you launch FL loads all of the system & NPC models, textures etc. before it loads the players. If it finds that the players ship CRC's match ones already loaded for an NPC it gives the player the NPC's ship and settings. This normally happens because the CMP was copied rather than a clone being made, use the Model Cloner to change the internal mesh names (CRC) to stop this from happening.
