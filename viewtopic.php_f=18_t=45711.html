So you've used FLScan (not FLINIScan - never heard of that ) on the mod, when the mod is activated right?

However, it won't pick up everything... but it should mean your package is okay. My suggestion would be to remove everything from the package apart from engine, scanner and power unit.
Have only 1 ship set to sell (your ship) and now see if it works.
If not, change the ship it's using (in the package declaration it nominates it's ship hull to use) for an existing original Freelancer one... and try again. Does it crash?
Yes - then it's your engine/scanner/power unit.
No - then it's something to do with the ship entry in the shiparch.ini or indeed, the ships hull in the goods.ini

Double check all entries for everything at that point.
