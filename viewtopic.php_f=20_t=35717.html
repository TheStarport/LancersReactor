Earendil

ZoneAlarm has a nasty habit of still leaving it's firewall installed and running even though ZA was told to exit. Look for the ZA executable in the task manager (I misremember the name). Or disable the ZA start when starting the 'puter option and reboot.

The FL update/patch is ONLY for servers! Installing it on a client only install may produce unwanted issues. (there are dozens of posted issues mentioning the patch is for servers only and the issues that arise from client-side patchs) And yes, installing the patch does nothing for the displayed version numbers, both in the client and the server. For example nearly every server out there on the FL global server list has the patch yet they all show as version 1.0.
