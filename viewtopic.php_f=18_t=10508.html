Third

Mon Mar 17, 2003 9:25 pm by Third

Well, first off...other players only see your ship in space. I'm pretty sure that you don't see other characters at bases. Secondly, it's a mod running on your machine, so you'd be the only one to see it unless everyone on the server had the mod (in which case, they would look the same as you).

But if you want, there is a good example of one here.
Basically, it would require two changes and look like this:

1. Add this line into costumes.ini:
[Costume
nickname = DataDroid_trent
head = pi_pirate5_head
body = pi_orillion_body
righthand = benchmark_male_hand_right
lefthand = benchmark_male_hand_left

2. Make this change in mpnewcharacter.fl (in the exe folder)
;costume = %%BASE_COSTUME%%
costume = DataDroid_trent

Save the changes and start your server.
Open the game and create a new you...
(this is for MP, btw)
