Unless something goes wrong, all the files should be back to their original, encrypted state when the mod is deactivated. To make sure, you can do "Tools/Restore Backups", which searches for any original files that didn't get restored, and restores them. You shouldn't have to do that though; FLMM should properly restore all the files when a mod is deactivated. If it doesn't, please report the details to me.
