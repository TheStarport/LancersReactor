Corsair to the rescue! Okay, i know how to solve your problem for hardpoints, but the docking bay, somebody else should help you with that along with somebody helping you insert it into freelancer. Okay here we go.
step #1 Download and install HardCMP.
step #2 start HardCMP and go to file, open, and then go to your station.
step #3 On the right, there should be a list of things on the station. Go down a little, and click new. Click Rotary, and go to turret. Say okay and a red turret should come up. Use the X, Y, and Z cordnites to place it where you want.
step #4 well, this ones complicated. Go to a station's script file that you like, and then copy and paste. Type copy and paste as many turrets as you need and like, then exit and save.
step #5 go to your stations loadouts, and type in the name of the turret or turrets that you like in the correct amount. (if there are not enough, then create some more! )
step #6 (I know, this is getting long. Don't worry, #6 isn't very complicated.)
Ask someone else for help to install everything else.

The reason i know all of this is because i usually mod all of my regular ships to what i want.
Well, i really hope this helps.
Sorry if it doesn't, i don't mod stations.
