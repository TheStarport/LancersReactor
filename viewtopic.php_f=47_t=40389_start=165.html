The game is easy - at first read the rules in first post in this thread, then when you understand it, choose one faction in Sirius and start playing.

The game starts with all the bases, systems, etc. as they are in FL game, but many more can be built or explored later.

You do all your actions - battles, constructions, diplomacy etc. by simply posting them here. Then, a judge will resolve any actions based on random factor (research, battles...)
