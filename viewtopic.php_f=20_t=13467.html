Since I can no longer get on a computer that actually HAS this problem I do know of a way that you can fix it although it takes a little tweeking with the size values in "fonts.ini" and "rich_fonts.ini". These files are located in your "Freelancer\DATA\FONTS\" directory. Just use something such as Bini or BiniQDU to decompress them.

Here is a sample of what to edit in each file:

__Fonts.ini__

[TrueType
nickname = ButtonText <---- The ID for the font settings.
font = Agency FB <---- Edit this value for font.
fixed_height = 0.043500 <---- Edit this value for text size.

__Rich_fonts.ini__

[TrueType
font = 0, Arial Unicode MS, 22 font = ID#, Font, Size

[Style ONLY EDIT THESE IF YOU WANT TO
name = STYLE_CONSOLE
font = 0
bold = false
italic = false
underline = false
color = 255, 255, 255

I hope that this will help each of you fix your problems.
