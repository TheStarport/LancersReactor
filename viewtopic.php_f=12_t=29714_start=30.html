Esquilax

Wed Apr 07, 2004 3:10 pm by Esquilax

AVI files, etc, do not usually contain viruses, but because Windows does not provide an icon for PIF files as a default, the files can appear to be legitimate. Theoretically, anything that executes code on your machine can be a virus, but I have never encountered an infected AVI file. This is not to say that there aren't any however. Just remember Esky's First Rule of Avoiding Computer Infections; Don't open attachments!

On a related matter, I'm pretty sure that I received a worm two days ago; it was a "Please check in this ZIP file for your attachment" sort of message. Needless to say I took a laugh at its lack of subtlety, and deleted it.

I like "Esky" because that is an Australian drink cooling apparatus (see link), and thus it is amusing. "Esqy", "Eskie", etc, are fine. It's all good .
