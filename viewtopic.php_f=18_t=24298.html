The Mission MOD

Forgive me if this has been discussed before...doing a search for "MOD" and "mission" would be too tedious...you can imagine the number of hits...

I have seen a lot of ship mods, and several add-on mods and a few total conversion mods. What about a mission mod? Has anyone created a mod that simply adds new types of missions to the game? I'm talking about missions that are picked up at the bases from the NPCs.

So instead of just having "kill a bunch of ships" missions or "kill this guy" missions, there would be trade missions, escort missions, transport missions, rescue missions, etc (use your imagination). That is the one thing that would make the game much more like actual Freelancing. If I could walk into a bar and select from a multitude of different mission types, that would be awesome. Has anyone considered doing this, or has it already been done? How difficult would it be to do this?

I don't do any modding myself, so I can only guess at how easy/difficult it might be to do this. However, I do have some programming experience so my brain tells me it shouldn't be hard to simply add some new mission parameters like "when ship reaches X destination, condition=WIN" for transport missions or "when ship reaches X destination with Y cargo, condition=WIN" for trade missions. Could you add an object (escape pod or something) to a specific location in another system and then create a search and rescue mission where your job is to find the object, tractor it in and bring it back? Similar to the missions where you blow up the guys ship and then tractor in his escape pod, but in this case you would just be given general information on the possible location of the object and it would be up to you to actually find the thing and bring it back safely.

Anyway, like I said, maybe this has been discussed before, but I haven't seen anything in the download section that is just a "mission mod". If it's not too difficult to do, I would love to create one. I have a million ideas for different mission types. It would add so much flavor to the game, imho.

Wap
