Tue Nov 04, 2003 10:57 am by Chips

In system jumping is simple, very very simple:

Just make the goto line to another jumphole in your own system. The shortest_path.ini already has entries like:
Li01, Li01, Li01 That is all it needs, they all have it already!

For the jumphole example - check this out:
In your system file you do this, you can use whatever nicknames for the jumpholes so it doesn't matter, just make sure that it refers to the right part.

Example:

[Object
nickname = Iw03b_to_Iw03a
ids_name = 260610
pos = -83000, 0, 0
rotate = 0, -90, 0
Archetype = jumpgate
msg_id_prefix = gcs_refer_system_Iw03 ;says where jumping too -doesn't matter
ids_info = 66145
jump_effect = jump_effect_elite3 ;erm - my custom
reputation = li_p_grp
behavior = NOTHING
difficulty_level = 1
goto = Iw03, Iw03a_to_Iw03b, gate_tunnel_elite3
loadout = jumpgate_li_01
pilot = pilot_solar_easiest

Object
nickname = Iw03a_to_Iw03b
ids_name = 260610
pos = 0, 0, 10000
rotate = 0, -90, 0
Archetype = jumpgate
msg_id_prefix = gcs_refer_system_Iw03 ;says where jumping too -doesn't matter
ids_info = 66145
jump_effect = jump_effect_elite3 ;erm - my custom
reputation = li_p_grp
behavior = NOTHING
