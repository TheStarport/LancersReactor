Black 7 - you can download the mods i have made for them. Either Open SP where you can choose any Freelancer Ship to start with, Storyline - where you can start with any ship - or just the fly all ships mod, where you can fly any ship (check shipyards for sale points). Check the pathfinderstudios forum below these editing ones. You can find the correct script file and take it straight from there if you like. Can find the download in the stickied thread for mods made by me.

For the liberty Cruiser main gun:
[Munition
nickname = li_cruiser_forward_gun01_ammo
hp_type = hp_gun
requires_ammo = false
hit_pts = 2
hull_damage = 202
energy_damage = 0
one_shot_sound = fire_capship_forward_gun
munition_hit_effect = li_capgun_01_impact
const_effect = li_cruiser_maingun
lifetime = 2
force_gun_ori = true
mass = 1
volume = 0.000100

[Gun
nickname = li_cruiser_forward_gun01
ids_name = 263377
ids_info = 264377
DA_archetype = equipment\models\turret\li_cruiser_gun01.cmp
material_library = equipment\models\li_turret.mat
HP_child = HPConnect
hit_pts = 16860
explosion_resistance = 1.000000
debris_type = debris_turret_small
parent_impulse = 20
child_impulse = 80
volume = 50.000000
mass = 10
damage_per_fire = 0
power_usage = 0
refire_delay = 0.500000
muzzle_velocity = 500
toughness = 4.000000
flash_particle_name = li_capgun_01_flash
flash_radius = 15
light_anim = l_gun01_flash
projectile_archetype = li_cruiser_forward_gun01_ammo
separation_explosion = sever_debris
auto_turret = false
turn_rate = 90
lootable = false
LODranges = 0, 10
