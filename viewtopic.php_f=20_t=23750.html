2. i then Extracted the .cab files to the Freelancer directory with Winrar ( winrar extract all the FL folders insisde another folder called cab1 & cab2. so i cut and pasted the FL folders to the Freelancer folder then deleted the cab1 and 2 folders)

3. i copied the Freelancer excutable from the cd ( D:\game\exe\FREELA_1.EXE and pasted it to c:\freelancer\exe
when i ran the excutiable it worked !!!! yay..........

i have only tested single player i am not sure if multiplayer works or not.
