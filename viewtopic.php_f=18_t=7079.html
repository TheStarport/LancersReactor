INI compression

I was able to figure out the compression of INI files I built a small app to uncompress these files, so as a test i made the game load an uncompressed version of universe.ini and a modified weapon_good.ini, with the price of the Starbeam modified to 45000 credits. It worked I'm gonna add a small interface to my tiny console app, so you can start playing with these INI files, they're pure text files so all you really need is notepad after they're uncompressed. As soon as i hear from Gibby, i'll be uploading it into the Lancers Reactor.

Building the GUI version should be very quick work, as well as writing the document that describes the compression method. Unfortunately however, i haven't had the chance to even begin, since today was a very busy day Since i don't want to keep you waiting any longer either, i've uploaded the command line version as it is, so you can start playing with INI files imediately, with the promise of having a version with a Windows GUI by the weekend at most The link is for the command line app is:

download

Dragging a single INI file into the EXE should produce a txt file.. for example, file.ini -> file.ini.txt , if you rename file.ini.txt to file.ini the game loads the uncompressed version as well as your modifications

This allows you to edit a lot of the game, but not everything. Different formats are used in ships ("UTF" and "vers", iirc) and mission code (Lua scripts, as far as i've been able to determine). I've done some work on the UTF format, although not much... I'll let you know as things develop!
