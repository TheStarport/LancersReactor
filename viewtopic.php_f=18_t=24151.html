Go into FLMM mods folder, create a folder called 'My Systems mod' or whatever you wish for it to be!
Now, go inside your mod folder, create an EXE folder. Copy and paste the freelancer.ini from your freelancer EXE folder into this EXE folder.
Now open FLExplorer - click on work on existing mod - select your folder.
Once done, exit.
Get BiniQDU and open up Your mods Freelancer.ini and look below
Find this part
[Resources
DLL = InfoCards.dll ; Contains all infocard text resources
DLL = MiscText.dll ; Contains other bulk text resource: rumor, offer, and mission text
DLL = NameResources.dll ; Contains names of things, primarily.
DLL = EquipResources.dll ;Contains names of equipment (in string table) and infocards for equipment (in HTML resources)
DLL = OfferBribeResources.dll ; Contains random mission offers and bribes
DLL = MiscTextInfo2.dll ;Contains MP UI elements as well as other misc. text


Add this line to the bottom of this part (or whatever YOU called the mods dll file - by default it is my_mod.dll unless you renamed it!

DLL = my_mod.dll ; Custom systems dll
the ; just means the game won't read what is after it, so we put a note there to remind ourselves what this dll is.
it should look like this now

[Resources
DLL = InfoCards.dll ; Contains all infocard text resources
DLL = MiscText.dll ; Contains other bulk text resource: rumor, offer, and mission text
DLL = NameResources.dll ; Contains names of things, primarily.
DLL = EquipResources.dll ;Contains names of equipment (in string table) and infocards for equipment (in HTML resources)
DLL = OfferBribeResources.dll ; Contains random mission offers and bribes
DLL = MiscTextInfo2.dll ;Contains MP UI elements as well as other misc. text
DLL = my_mod.dll ; Custom systems dll

save your freelancer.ini using the save as command, and enter freelancer.ini (otherwise it will save a txt file instead - useless!)
Now, activate your mod, and get playing!
Hope that sorts you out a little

Usual probs - can't see a name for you stuff at all:
Cause - forgot to insert the line in the freelancer.ini
Still has no names - worked on the file more than once - it erases the names or something each time you work on the dll!
Try to jump to system - but it crashes ; You got an encounters problem - remove the encounters (zones, patrols) and try again - should work okay
Still crashes - look inside your systems ini file - you might have two [aseroids or two [nebulae parts, but NOT got two in the text, delete the one missing, or make the zone for it instead.
Still crashing? Check that yor patrols or ambient populations don't have stuff in them for factions - ie - who ever heard of zoners using transports etc. Limit it to cops and stuff. (ie remove traders, pirates etc)
Big cause of crashing is assigning things that factions don't have etc - ie types of transports etc... pirates won't do bh patrols formations or encoutners
