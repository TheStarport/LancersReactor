Best Firewall for server

Started this topic, because some admins run their server with their home pc.
I believe that only few have enough money to buy a full license for a firewall software just for home use, and they are using freeware versions (right?).

My case:
I have usually used Zonealarm, though it's pretty crappy because it's one of those wizard type programs (you cant configure it enough for you needs).
So after testing some firewall programs (freeware), I found that my choise was Sygate Personal Firewall (http://soho.sygate.com/products/comparison.htm).
Among those that I tested it seemed to be the best, it simply had more options to configure than the others. Though there might be better programs awailable?

So my question goes: what is your choise and why?
