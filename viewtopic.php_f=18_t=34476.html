Wed Sep 15, 2004 8:13 am by RimShot

rtg593,
maybe you don`t know this:

example of [SystemInfo name = St01: (data/universe/systems/st01/st01.ini)

pretty much on top of the file you find ..

[Asteroids
file = solar\asteroids\St01_nomad_asteroids.ini <--- (name of the file which contains all information about the general appearance)
zone = Zone_St01_edge_nebula <--- (nickname of the zone inside the system.ini; mainly just defines name, size, shape, etc)

same here for a nebula zone ..

[Nebula
file = solar\nebula\St01_edge_nebula.ini
zone = Zone_St01_edge_nebula

as you can see nebulas data is stored in: data\solar\nebula
and asteroid fields in: data\solar\asteroids


EDIT:
note: exclusion zones are linked to xxx_asteroids.ini in data\solar\asteroids folder as well ..

hope this helps
