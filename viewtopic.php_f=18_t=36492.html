Argh

Sat Dec 04, 2004 4:10 am by Argh

Actually... you could probably do custom BayDoors. I'm sure it can be done in the same way that we've learned how to do other things with moving parts, like custom Weapons.

That said... the problem isn't getting the doors to open... it's building a SUR file that would be "concave". There are such SUR files- the dockable objects all have areas that are concave. If you don't have a SUR file that has a "hollow area", then ships attempting to "enter" the transport will impact the object... and rebound.

But we do not (currently) have the tools to build new SURs. So... what you're talking about doing, while vaguely possible... would be a royal pain. It'd involve hand-hacking a SUR file (which involves a lot of prayer, sweat and hex-editing)... and you'd have to build an animated Bay Door as part of the ship (which, as I said above, is probably possible, but nobody's done one yet). So... if you want to try to achieve this... we're all interested in how it comes out, and we'll try to help with some advice. But you're exploring territory that nobody's ever entered... and returned alive
