Well first you need to install IFSO version 5 (I assume you already have that). NExt download and unzipp the IFSO data files for the rebalance 3.40 upgrade 2.1 (they're on Lancers somewhere). Make sure you unzip the files in the iron cross server operator directory, and not a subdirectory.

If you've done that then you have around 90% of the data. The next step is just a matter of trial and error ... when you edit or view a character you'll see a file load erros tab. Click this and it lists errors for theplayer. Some of these errors will cause IFSO so show "unkown" in fieds (like shipname) and sometimes it just causes a field to be blank. THe next step is to look at the file view and scroll down till you see the numbers list in file load erros. This will give you some idea if the number relates to a shield, cargo, weapon etc. It's then amatter of working out what the character has (I have some test characts I use).

To add an unkwon item (once you've worked out what it is) you edit the relevant .txt file in the Iron cross folder and add a line.

As I said I have files with a few additions and alos moved some stuff betwen files.
