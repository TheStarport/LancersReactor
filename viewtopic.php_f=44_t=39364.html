Spherical texture mapping

Im building a model of a person to put in a static object. What im trying to do is export the face textures and then map them to a model I built. The problem being, I dont think I can get them to wrap around the circular head. I know you can do this in other programs so I was wondering what the best solution would be. Do I need to texture it in another program and then import it into milshape? Will that even work? Or do I simply need to break the head into 4 or 5 groups and map each seperatley so simulate spherical mapping?

UVMapper Classic. It supports spherical mapping. Just export your model as obj, then open it in uvmapper classic, then go to the menu and set the mapping type for the entire face as spherical. You might need to use "relax uv's" setting to get a good wrap.
