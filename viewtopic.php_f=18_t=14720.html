Bata Testers Needed - War - Rheinland vs Liberty

I need a few "beta testers" to run trials of the Liberty vs Rheinland MOD before I release it to everyone. I've setup battleship encounters in specific locations and it appears to be running normal. What I need are a few people to fly around these battleship encounters and do the following;

1. Note the location of the battle, it should be the same as the location list I will give you.

2. Note the attacking ships. Turret fire should be comming from both fleets.

3. Check for any stability issues such as server lag, etc.

4. Watch for any glitches like ships vanishing in thin air, textures not appearing, etc.

You should be available for atleast an hour as you will actually have to travel to the locations where the battleships are. Good news is, the farthest distance a fleet position is placed is Hamburg. Also, there are only 5 locations and if possible I would like to have 5 testers.

It should be noted this is a server-side mod, as far as I know, so there should be no risk to your actual install of Freelancer.
