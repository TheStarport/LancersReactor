on another comp now so I can't help you out with the exact strings...
but basically I touched on this in another post.

first you need to go into keymap.ini and take user_z_rotate and user_z_unrotate or something like that to the bottom of the file under idr_cockpit heading. Then you need to go to your userkeymaps file and add/bind the user_z_rotate/unrotate commands to some key (eg I had used q and e). You can also add the user_z_rotate/unrotate along with its appropriate id to keylist.ini if you want to be able to rebind the roll function from the options > controls menu in game. If this is too vague or just not right, hunt back for my old post titled "I added rolling but..."
