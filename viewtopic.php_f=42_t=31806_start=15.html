PantherX

Mon Jun 14, 2004 2:44 pm by PantherX

Sorry to intrude but you may have problem if you name the material just 'glass'.

It would be better if you name it 'pr_glass" or whatever strikes your fancy. Other modders have used the name 'glass' and you can have conflicts in you have different ships with similar material names.

Also you can use 24 bit tga textures but you have better results and save space if you use .dds textures. If you are using photoshop compatible software you can get the .dds export plugin in the Nvidia texture tools pack at http://developer.nvidia.com/object/nv_t ... tools.html.

DDS files are half the size of tga and you don't have to flip them before you add them to the .mat.
