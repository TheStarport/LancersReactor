or this

I don't know any entry that is used for loot dropped by npc

But there is a solution !
This one:

open DATA\MISSIONS\LootProps.ini
[PhantomLoot
nickname = ge_s_repair_01
toughness_range = 0, 1
percent_chance = 16 <================= increase these values everywhere
num_to_drop = 1, 1
