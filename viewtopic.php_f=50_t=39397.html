Thu Oct 12, 2006 11:52 am by Bejaymac

Have you ever tried to add transparency and glowmap effects to a MAT file with UTF edit, if not then this might be a little to much for some people.

What you see on screen and this applies to everything you see ie HUD, Navmap, inventory etc. are comprised of several layers of models (3DB & CMP) stacked one on top of the other. Adding a transparency to the map 3db will make it see through, but only to show the solid layers that are behind it.

I've been trying to find the layers for the universe map for the past couple of hours, I've found the front 2 and made them invisable to let me see the ones behind them, but I've still got a couple left to find.

Once I've found them all I'll post a how to, but if you've never used UTF edit then get it and learn how to use it as I'm not posting an "idiots guide".
