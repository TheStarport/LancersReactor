to edit to opacity use UTF editor; find your material library and edit the Oc node in floating-point mode. 1.0/1.0 = 100%opaque 0.1/1.0 = 10%opaque

to add a general glow to the texture add an Ec node to the material entry - 1.0/1.0 = max brightness

also, you'll need to change the type from DcDtOcOt to DcDtEcEtOcOt to use both these properties together
