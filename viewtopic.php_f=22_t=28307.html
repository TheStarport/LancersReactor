Best Cheat Detection - Besides FLSO (highly recommended) there is naturally the v1.1 patch (for servers only!) and Lancerguard (mostly stops invulnerable shields, not allowed Nomad weapons, etc.). A server police or at least a cadre of trustworthy players to check for cheaters is extremely useful for those cheats that just can't be detected (like system wide verbal harassment, newbie killings, dock camping). I also use AkaTaka's tools FLAcctBrowser and FLAcctMon for ancillary work.

Best Remote Admin tool - I use M$' Remote Desktop Connection. I can do everything as if I am in front of the server except run task manager. It's a little bumpy in connection and holding a connection but it's more than adequate for remote admining.

Server-side Mods - Unless you have bandwidth to burn, consider these carefully. I have found only in bits and peices of info on server-side mods. I wish there was a definitive list and tutorial of what can and can't be done and how. It's there in the TLR editing forums but hard to find.

If you are thinking of uploading player files and such, it's a long, tedious renaming process for conversion to a new server. That is unless you actually take over a server's Multiplayer directory then you have the server ID and no two server's with the same ID can be on the global server at the same time (learned THAT lesson the hard way!).
