The idea behind FLSDCT:
¯¯¯¯¯¯¯¯¯¯¯¯¯¯¯¯¯¯¯¯¯¯¯¯
The idea behind FLSDCT is quite simple. As of 2008 the global server list of freelancer will be shut down, we decided to create an alternative solution. This solution should be more powerful than the old global server list. In the old global server list you would have to write down manually any URL leading to a server's homepage. And what you could put in for additional information about your server was limited.
FLSDCT gives direct links to the homepages of all servers taking part, and if you launch freelancer by FLSDCT, you can connect to all those servers directly, no matter if global serverlist is online or not. FLSDCT shows the minibanners of all those servers and also shows if those servers require a client-side mod or if there is an optional client-side mod by an additional icon. The icon showing this can be used as a direct link to the mod-download. Another icon shows if there is RPG at that server.
FLSDCT also has an admin-control-panel included, allowing admins of all servers taking part to edit all information about their server like IP, URL, mod-url and so on by themselves. Those changes take effect on FLSDCT for everyone using it as they are saved to a web-server, where FLSDCT gets it's information from.

How to use FLSDCT:
¨¨¨¨¨¨¨¨¨¨¨¨¨¨¨¨¨¨¨¨¨¨¨¨¨¨¨¨¨¨¨¨¨
Download and when necessary unpack it, you can put it wherever you want, it will work. Click the FLSDCT-Icon to start it.
You can click the minibanners shown in the FLSDCT-window to get to the homepages of the servers. Each minibanner has two additional icons, showing if the server is modded or if it is a rpg (role playing game) server. If the server is modded and it's administration decided to add that, you can click the mod/optional mod-icon to download the mod you need for that server.
At the bottom of the window there is a button "Freelancer - Server Direct Connection". If you click that, FLSDCT will launch freelancer and you will be able to connect to any server included, no matter if global server list is online or not. The servers will show up at multiplayer aswell as at LAN.
If you are an admin of a server taking part, you can enter the control-panel of FLSDCT by hitting "edit" at the bottom right, selecting your server and entering your password.

How does it work?
¨¨¨¨¨¨¨¨¨¨¨¨¨¨¨¨¨¨¨¨¨¨¨¨¨¨¨¨¨¨
FLSDCT.exe is a simple executable connecting to several webspaces (if one fails another is tried), to gather information and keep it's content up-to-date, opening the FLSDCT-window, and starting freelancer.exe adding the IPs of the server to it's start-command to make them accessable when globalserverlist is down.
