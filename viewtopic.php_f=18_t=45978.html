FriendlyFire

Sun Jun 17, 2007 8:55 am by FriendlyFire

I read THN in your sentence. Do you want to get it in a THN sequence on directly in the game? Because if so, it's possible. I've already done it. The effect loops fine and I bet that with the new EXE hacks Dev found, it'd be possible to overcome the "activation" issue (You'd have to get very close from the sun to start the effect, then you could go away and it'd still be there).

EDIT: Re-reading, I think it's in a system, eh? It's pretty easy to put it in. You just have to insert this into select_equip.ini

{AttachedFX} <--- Change these brackets
nickname = gf_nomadwormhole
particles = gf_nomadwormhole

And then add this new Equipment into a loadout, with a Hardpoint correctly set up (Make a new invisible model or whatever). From there, getting it inside a system should be easy!

Oh and, you'll need to make some hacks to get it to show up correctly. You should head onto Dev's Tutorial to see about that.
