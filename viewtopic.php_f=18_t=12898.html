thanx for the info - but i found out you don't even need to do that, actually, all you have to do is,
in equipment/goods.ini
find the package for the ship u want (for eagle it would be gf6_package)
then replace the line:

addon = ge_gf6_engine_01, internal, 1

with:

addon = ge_oe_engine_01, internal, 1

and it should work
