PhantoM,

I also had problems with beta 7 tho not the ones you describe.......

A couple of suggestions that may help....

1. disable all mods - the mod will not work with many other mods as it is unscripted
2. restore backups, and either move all saved games to another folder or use the option under options "automatically move saved games to different folders"
3. delete the mod from the mods folder using the "delete mod" (top right hand corner) available in the flmm main window.
4. install the mod again using the flmm folder (EpsilonV1O4.zip.flmm)
5. start a new game.

If the above fails then I can only suggest you download it again.... or use flmm beta 5

There may be another reason that I am unaware of ... maybe someone else can provide more information.....
