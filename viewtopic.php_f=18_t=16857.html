Wed May 14, 2003 10:03 am by Cpt.Kazo

sounds like you messed something up with the nicknames..all nicknames have to be absolutely unique! Otherwise you will encounter the real odd stuff.

The following object definition works for me

[Object
nickname = Li01_to_Li02 <- to edit!
ids_name = 260911
pos = 0, 0, 0 <- to edit!
rotate = 0, 0, 0 <- to edit!
Archetype = jumpgate
msg_id_prefix = gcs_refer_system_Li02
ids_info = 66145
jump_effect = jump_effect_bretonia
reputation = li_p_grp <- to edit!
behavior = NOTHING
difficulty_level = 1
goto = Li02, Li02_to_Li01, gate_tunnel_bretonia <- to edit!
loadout = jumpgate_li_01
pilot = pilot_solar_easiest

...ids_name, ids_info, msg_id_prefix can be used various times. I use this jumpgate object several times, only changing the marked lines and encounter no problems at all (except that all jumpgates have the same name etc)...but they do work!
