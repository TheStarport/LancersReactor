Tawakalna

Sat Oct 29, 2005 7:19 am by Tawakalna

in principle you shouldn't have a problem, provided you haven't done anything out of the ordinary in the bios and you're using an up-to-date driver suite. best thing to do first is set all the graphic options in the ATi suite to application controlled or performance, thats all the anti-aliasing and anisotropic filtering. You might well start off in a fairly low resolution for the game, 640x480 or 800x600 then work your way up from there. There are *some* games which this graphics chipset will only run stable in 640480 but so much depends on your own particular setup.
