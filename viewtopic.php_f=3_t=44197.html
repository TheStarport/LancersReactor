Ok, I am sure you already know what a mod does.
I'll give you a quick run through.

Here is the deal with mods. You turn on a mod and FLMM does it's magic and replaces those files, thus adding custom ships, weapons, system, etc.
When you turn off the mod, it will take out the files and the game will be as though the mod was never turned on.

BUT! FLMM is sometimes buggy and does one lazy of a job of replacing the normal files back, that's why you once in awhile do a clean install, which is pretty much going to your FREELANCER folder and just deleting FL and then reinstalling it.
There are also a set of "debugged/recoded" files(as I like to call them) named SDK files(they can be found on this site.)

All those files are is a set of decoded files that have fixed tons of bugs in them, it's alright if you don't have them installed, but it helps sometimes and you can just install the SDK to your FL folder besides doing a clean install.

So no, most likey when you turn off a mod, the game should work fine, you might have a chance of running into a bug becuase of a mistake FLMM made, so all you do is the clean install. Shouldn't be that hard.

EDIT : if what you are talking about is starting a game with the ship, taking off the mod and getting back in with the ship, then yes the game will crash becuase the mod isn't on and the save file is telling FL that you have a ship that isn't really there.
