FWIW Here's answers to a couple of basic MP questions I had:

1. Do I need the Freelancer CD in the CDROM drive of the server?

a. No.

2. I've loaded a FL server on another PC on the LAN but I can't see it.

a. Check that you're in the same IP address range and that you have disabled any firewall software which could interfere with directplay ports 2300-2400 (or set the firewall to allow the ports). For internet connection sharing, you need to be in the address range 192.168.0.x where x is between 1 and 255. The machine with the modem must be 192.168.0.1. The other address range you could use for a LAN is 10.0.0.x. Check that you can ping the other machine's IP address.

3. How do I use a mod in MP?

a. Use the mod manager FLMM. The files for each mod must be in a subdirectory off the mods directory where FLMM is installed. Use the interface to install each mod you want to use. Make sure you disable the feature to make files read only or many mods will fail to install (see the FAQ for FLMM).

4. Do I need the mod on the server and the client?

a. Yes. Load it on the server and all client machines. If it is not loaded on the client PC, the client will crash as soon as the player tries to leave Manhattan. If you use a combination of mods, you will need exactly the same combination on the server and all clients.

5. Can I run the server on the same PC I am playing the game on?

a. Yes.

6. Can I transfer characters from one server to another?

a. Yes, but only in a LAN. The "\My Documents\My Games\Freelancer\Accts\Multiplayer" folder contains a number of subfolders and a file called accounts.cfg. Copy these to the same directory on the other server. This doesn't change which machine "owns" the character. The "ownership" of the character is determined by the ID of the computer which created it.

7. Can I rename my character?

a. There doesn't appear to be a way to do it in the standard interface. It may be possible to do it by editing the file directly, but you could also do it with two computers. Run both in MP, create an intermediate character on the other machine. Get the two characters to meet in space outside Manhattan, and trade all the credits and gear across. Create a character with the desired name and trade the gear to them. The disadvantage is that you'll have to sell and buy ships, and you will lose a LOT of credits in the process. But it works! This method could also be used to "move" cash from a character on one PC to another - eg. you've just bought a new PC. You could also transfer cash and gear this way using a friend's help if it's an internet server. Just pray they don't decide to keep it!

8. Any other hints?

a. Check the readme.rtf that comes with FL and the FAQ that comes with FLMM.

Hopefully this may be some help to other MP newbies. Corrections welcome.

Rob
