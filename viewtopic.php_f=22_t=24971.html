FLSO users Major bug found

When using the Delete inactive players and a date format bug

It's the Date formats: here we use DD/MM/YY in US your using that odd format MM/DD/YY, FLSO just uses that format so can easily delete the wrong players
eg 5/10/03 is 5th Oct 03 but FLSO is reading the date as 10/5/03

So anyone using the delete inactive players, be SURE you have backed up before using this feature, hopefully this bug will be squashed quickly now its found.
