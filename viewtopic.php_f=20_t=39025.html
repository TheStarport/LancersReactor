This is a common problem, sounds strange I know but this is a "sound issue". I had the same problem. This issue may occur if Fraunhofer IIS MPEG Layer-3 codec is not installed. To fix this problem just install or reinstall the Fraunhofer IIS MPEG Layer-3 codec. Glad to be able to help!! = When this works for you please post that it worked THANK YOU
