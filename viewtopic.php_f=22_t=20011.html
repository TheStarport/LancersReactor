New type of server urgently needed

The FL community desperately needs a third type of server, perhaps with a server-only mod.

I propose calling it PvPwP ; Player Vs. Player with Permission.

The goal is to only allow PvP with consenting parties. No more PKing, Dock Camping, Back-Shooting.

Functionally, you open a Trade Request with the player you want to Duel.

Above the 'Credits' button is 'Duel'. If both players accept 'trade' they can kill each other.
