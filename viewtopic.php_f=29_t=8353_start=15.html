Thu Mar 06, 2003 11:55 pm by CptnEisen

Well, managed to get docking to a planet working, it was actually something Dawn missed out of that cool tutorial. Whenever u create a new base/system, u must also reference it in data\universe\universe.ini

eg:

[Base
nickname = Li01_01_Base
system = Li01
strid_name = 196766
file = Universe\Systems\Li01\Bases\Li01_01_Base.ini
BGCS_base_run_by = W02bF35

and:

[system
nickname = Li01
file = Systems\Li01\Li01.ini
pos = 7, 8
msg_id_prefix = gcs_refer_system_Li01
visit = 1
strid_name = 196609
ids_info = 66106


Oh - And no zones were used in getting this to work - Which also kinda means there's no atmosphere effect around the planet, only way u can kill yerself would be to fly right into it
