New patch prob?

I've been playing in Omi. Beta for some time now, so I'm quite familiar with how the missions tend to be offered. Yet suddenly this has changed. And not for the better...

Granted, this is only based on a few hours of play, but it seems that if ANYBODY is on an anti-GMG mission, nobody else can get an anti-GMG mission. ONLY anti-Bounty Hunter missions are offered. This continues until the GMG mission is completed (or failed), then suddenly the job board has GMG missions again. I've done the "new character" re-log near 100 times when I KNEW someone was on a GMG mission, and not a single GMG mission was offered in those 100 attempts. Only BH missions.

What this seems to be saying is, with the current patch, only ONE person/team can be on a mission against a particular faction at any time. Personally, I don't view this as a good thing.

I haven't experimented with what happens when both GMG and BH missions are "in progress" for some player, while another tries to get a job, but it doesn't look promising from here....
