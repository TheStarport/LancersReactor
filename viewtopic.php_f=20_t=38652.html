I'll go ahead and put up some of the more common troubleshooting tips:

Test after each step:
1. undo the 1.4 patch
2. make sure DirectX is at latest version
3. make sure you've got the latest NVidia drivers


If it's the 1.4 patch, it may have been corrupted in the download- I don't use it myself, so I can't say for sure. Try the FL SDK instead- search the downloads section for it.
