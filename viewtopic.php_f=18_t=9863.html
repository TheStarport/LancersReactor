You say the file already has .ini on it, but are you sure the .txt isn't hiding on it too? From any folder window, go to Tools->Folder Options->View and make sure the box "Hide extensions for known filetypes" isn't checked. If it is checked, you wouldn't see the .txt ending, only the .ini ending.
