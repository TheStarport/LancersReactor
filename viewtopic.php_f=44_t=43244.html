The nomad textures are double sided.. however it is very hard to do.. the .mat file has extra nodes and the type string has the following: DcDtBtOcOtTwo
:the (Two) being for double sided textures.. there are gradient panels built in to the texture node, plus there is a special reference to the nomad_fx.txm in the shiparch ini.. its possible to recreate the nomad look , but if you import the nomad fighter you will notice that it is a very odd mesh indeed .. but I think that unlike some other ships the .cmp importer doesnt mash it up. it is worth a look..

I have recently made some progress with this but the stumbling block appears to be the .txm files..

in short: you can make any texture almost completely transparent. you can also make it glow and control the level of illumination..make the whole ship glow.. but these are not new nomad style textures...

Would be nice if someone could crack this..tho I suspect its a tough nut..
