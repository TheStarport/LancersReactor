redeye

Mon Jun 30, 2003 7:30 pm by redeye

This is usually to do with LOD's (level of detail) settings on the ship as defined in shiparch .ini. The numbers define the various detail levels and how far away ships can be seen from. Normally changing LOD to 0, 1000 or 0, 9999 sort out invisible ship problems.

However as you are installing from the same mod it cant be as simple as that. The only thing i can think of is that your friends detail sliders are set up differently from yours. Therefore the game is looking for a different texture than on your comp and 'cause it cant find one the ship ends up invisible. Try changing the LOD to 0, 1000 in shiparch and seeing if that sorts it out.

No this is about texture sizes. Fell for this my self early on.

All textures must be square. EG 16x16 pixels, 32x32pixels all the way up to 512x512pixels. OTher wise they dont display on some graphics cards. Also Nvidia released a DDs file that creates the mip maps for you automatically which also helps slower systems and reduces the size of your mat files by a fair amount.

LR has the plugin online in its download area.

Giskard
