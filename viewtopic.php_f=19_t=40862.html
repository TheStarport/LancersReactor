If you have access to the FL server computer it should be quite simple.

- Start the freelancer server as normal
- Go to EDIT -> ACCOUNTS...

This should bring up all stored chars within the server, and their serial #'s. replace his chars serial # so that it matches the fresh install on his comp. with any luck he'd be able to log in on his computer now. see how that goes.

Your multiplayer identity on all servers is the account ID. Go to Start, Programs, Microsoft Games, Freelancer, Examine My Account ID, and copy down the code. When you first enter multiplayer on the new computer, enter the exact same account ID.
