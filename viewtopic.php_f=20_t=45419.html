I think this actually happened to me once before quite a long time ago.
But in my case, it was because I had used other mods before and what that did was corrupt my game files. (Even if I deactivated a previous mod and restored backups, some old data from other mods still remained)
What I had to do was completely uninstall Freelancer and all saved games, then make a brand new "clean" install.

The problem is if you ever play on a server running a custom mod, it makes changes in the primary "Vanilla" files.
So, even if you deactivate the mod and try to play Vanilla, you get odd things happening like you just encountered.

1. Uninstall Freelancer (delete saved games too)
2. (move any mod you want to keep out of the FLMM and store them elsewhere)
3. Uninstall FLMM.
4. Install Freelancer
5. Install FLMM
6. Transfer relocated mod files back into FLMM.
7. To play Vanilla Freelancer..DO NOT activate any mods.
8. Go on line and have fun.
