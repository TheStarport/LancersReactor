I have heard many server admins complain about a potential memory leak in the server program. These reports vary, however, in the length of time each server has been active. But, in general, it seems that around the 20-30 hour mark Freelancer becomes seriously lag ridden for all users logged in. At this point it becomes necessary for the server to restart. I know, this does not require a complete character wipe, but it seems quite a few servers cannot be active 24/7 (even though they have it in their name). And some ppl just feel that it is just a bit of a hassle to check up on their servers to ensure that everything is going smoothly and that it does not need a restart. Especially right now that spring break is going on (well, at least for my area and the various college campuses within 100 miles of me). So, this is just to let you fellas know that sometimes it's just not feasible to have a 24/7 server. However, if you can setup a daemon to restart a server in the event of a crash, then by all means, host your own server. Cheers.
