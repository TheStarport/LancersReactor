A somewhat rare occurance whenever I had corrupted files. (Possibly cross contamination from other mods using different SDK's and coding practices)
What you started in, is the Freeport 7 System that is used for the intro script to the story line. You'll notice a similarity to the Sigma-17 System where you'll find the debris of a station exactly in the spot where Freeport 7 was destroyed by the nomads. The only problem is, there is really nowhere to go in that system except to exit the game and try starting over.
As I said, it's a result of corrpted files. All I can suggest is a complete uninstall and fresh install of Freelancer. (However, before you uninstall Freelancer, make sure you make a copy of your MP account ID just in case you had one)


Sorry, but that seems to be a fact of life about using different mods when nobody is willing to cooperate in establishing a unified data base.
