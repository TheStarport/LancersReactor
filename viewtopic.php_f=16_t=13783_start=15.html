The only thing to consider is how much profit you make on each unit purchased and sold. The final price doesn't come into the reckoning, just the profit.

One decent (and fairly safe) run for beginners is to run boron from Pittsburgh to Stokes Mining Station, then beryillium from Stokes to Pueblo Station.

Using a rhino, with an initial outlay of 9,600, you'll make 120,000 profit.

80 units boron bought at 120 each = 9,600
Sold at Stokes for 900 per unit = 72,000 = 62,400 profit

80 units beryillium bought at Stokes at 300 each = 24,000
Sold at Pueblo Station at 1,020 per unit = 81,600 = 57,600 profit

It's certainly not the most profitable run in the universe, but it's a good way to set yourself up when you first start.
