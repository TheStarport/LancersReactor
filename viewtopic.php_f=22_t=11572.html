justinh

Tue Mar 25, 2003 4:01 pm by justinh

I've found that if your machine has multiple IP addresses assigned to its network devices (be it multiple IP's on one interface, multiple interfaces, etc.) the server browser will pretty much pick one of the server's IP addresses at random...

For example, my server has the IP addresses 66.238.194.140, 192.168.0.85, and 192.168.1.85. Any one of those IP addresses will be the one reported on the server list, it appears. For me locally, this isn't a problem, but for others it can be. If you get the wrong IP address, just try refreshing the server list; that seems to change the IP address my clients receive.
