What would be your "perfect" base in the Freelancer Universe ?
A short questionare for you to answer ........

Question 1: Station Type

a) A mined-out asteroid base
b) A Freeport-style base
c) A Kusari Research base (With all the solar panels outside them)
d) A normal-style base (Space station, medium-sized)
e) A Prision base (Borg-like-Cube type)
f) Other - Please State

Question 2 : Location of base

a) Hidden in an asteroid field, hard to find
b) Open space, trade-lane connected
c) Remote Open Space, not connected to trade-lane
d) In a Planet's Shadow
e) In front of a Major/Minor Planet
f) Other - Please state

Question 3 : Defence

a) High Security (8 - 10 Space turrets)
b) Medium Security (5 - 7 Space Turrets)
c) Low Security ( 2 - 4 Space Turrets)
d) No/minimal security (0 - 1 Space turrets)
e) Other - Please State

Question 4: Population

a) Highly Populated (Patrols go past it ALL the time)
b) Medium Populated (Patrols go past it occasionally)
c) Lowly Populated (Patrols rarely go past it)
d) No population (Abandoned)
e) Other - Please State

Question 5: Population Type

a) The Police/Law types
b) Space Pirate types
c) Bounty Hunter types
d) Open for all
e) Other - Please State
