Check out the list of compiled tutorials in the tutorials section for a way to become friendly with everybody. And if you want to see the games innermost secerets, you can check out the freelancer 2D editing utility in the utilitys section, which, when downloaded, alowes you to create new systems, jumpholes, bases, and almost anything else you can imangine. The only proble is that you have to install it right, and for some reason, some ppl (including me) have had trouble with it freezing alot. Oh, and if you want to take panoramic shots, I don't know if this will work, but I believe you can go into Mouse Flight Turret view (If you don't mind your ship being in the picture, so make sure you have a classy one, like the saber!) an take about 20 shots or so while at a full stop, each one moved either right or left from the last one a little bit (so you have a bunch of shots, all in sequence, of the thing you want to take a photo of) and import them into the editing utility called milkshape, usually used for creating new ships, where you can then put them together into a panoramic shot. (If you know how to use milkshape, which I deafinatly don't, so don't ask me for help. I just heard you could do this from someone else). Also, if you don't want your ship in the shots, then you can dig around a bit in the foroum, and I'm sure you'll eventually find a way to cloak (I found it once, but forgot). The only cach is, it involves the modification of an INI file (which you would do by extrating the file with bini and editing it, avalible in the freelancer editing utilities section of this site), but once it is modified, there is no way to fire your weapons in flight, and no way to deacloak, without quiting Freelance and remodifying the ini files. But it can be done. (Most of the best MODers are currently working on a way of hacking the EXE files so that you can decloak during flight.
