Pinger

Fri Dec 03, 2004 2:26 am by Pinger

There are only these known commands built in tio FLserver via the console.

/invite name <- case sensitive so you can remote invite someone in different system, note same issues for invite apply they cant be on a mission and the Origional Group inviter has to invite for them to see the group chat and map.

/leave quits from a group

/u message if you have a admin file in your players folder on a server then you can send a Universe message, handy when you in the server and want to let everyone know warning of a player for pvp restrictions / server lag info requests etc.
