If you have used Linux before you probably know about WINE. Basicly it allows you to run most windows .exe files on your Penguinated computer.
Head to http://winehq.com/ for more information. (FREE)
This specific versin of WINE http://www.transgaming.com/ (Payment required) is directed to play DirectX based games on Linux. Both of those tools run on Mandrake 8.2 - 9.1 I used WINE to run Counter-Strike (Half-Life) on Mandrake 9.1 This is the company that made The Sims for Linux...
If you are completely new to the Linux shell, I advise you to use KDE in Mandrake as your interface. Read the WINE FAQ and Documentation to fully understand how to use it.
It will be harder to run WINE on Winblows XP and NT and sometimes 2000 since it blocks the access to its regestry. Thus if you need a CD-key for a game, you will have to enter every time you run the game on Linux using WINE if you have Windows XP, NT or 2000.
The WINE FAQ is pretty good, so have fun...
