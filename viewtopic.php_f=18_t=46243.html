mh-dk

Wed Aug 08, 2007 3:06 pm by mh-dk

There's also more than one flam effect... What you're looking for is something like this to change:

[Effect
nickname = gf_br_smallengine01_fire
effect_type = EFT_ENGINE_FIRE
vis_effect = gf_br_smallengine01_fire
vis_generic = gf_min_smallengine02_fire

[Effect
nickname = gf_br_smallengine02_fire
effect_type = EFT_ENGINE_FIRE
vis_effect = gf_br_smallengine02_fire
vis_generic = gf_min_smallengine02_fire

[Effect
nickname = gf_br_smallengine03_fire
effect_type = EFT_ENGINE_FIRE
vis_effect = gf_br_smallengine03_fire
vis_generic = gf_min_smallengine02_fire

These are some Bretonia engine flame effect... The cruise flame is in the same as the normal flame you see... Which means multiple effects in one .ale!

You can always try to use windows search function to find the linking of files!

for example search for "gf_br_smallengine01_fire" and see which files are connected with that effect...

hope I helped a bit!
