You need to download:
Freelancer Rebalance Mod v3.40 file name Flrebalance340full.zip.flmod
After FlRebalance 340 Upgrade 2.1 file name Flrebalance340Upgrade2.1.zip.flmod
You can also download:
Flrebalance v3.40 Upgrade v2.1 patch file name Flrebalance340Upgrade2.1Bugfix.zip.flmod
and of course Freelancer Mod Manager v1.31

First you install FLMM 1.31 and auto-extract the above files in FLMM. Then from the main screen in FLMM you must activate Freelancer Rebalance Mod v3.40, FlRebalance 340 Upgrade 2.1 and Flrebalance v3.40 Upgrade v2.1 patch.
Each time you are going to activate FlRebalance 340 Upgrade 2.1 and then Flrebalance v3.40 Upgrade v2.1 patch you will get a message asking you if you want to overwrite the files. Press YES. At the end you must have all of them activate. Then you can load the program and enjoy the game...
