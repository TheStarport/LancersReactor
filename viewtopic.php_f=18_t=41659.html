Mace_166

Sun Nov 06, 2005 3:37 pm by Mace_166

Well, what can happen is that you have duplicate entries for the same base. In mbase it only reads the first base's mbase and uses that one.
Also, if you have problems with one base, all of the bases afterwards will not work, even if they have been added properly.
The easy fix is to copy the mbase entries that you want to the top of the mbases file, the better fix is to spawn at progressively higher bases in the mbase file and see if there are entries that no longer exist.
