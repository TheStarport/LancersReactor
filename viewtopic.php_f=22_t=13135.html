Alpha Centauri 66.150.31.84 is online

Hello All, New Dedicated Server is up for all to enjoy. Right now, it is open to all, set to 128 players while we (gaminhost.com) are testing how much cpu/memory/bandwidth it takes. Server is dual AMD MP 1900 with 2 gigs Memory, so the experience should be smooth.

If the server is down, it will be restarted as soon as we are notified of it's down status, please use serverdown@gaminghost.com to let us know that the server is down, with Alpha Centauri in the message header so we know what server to take action on.

We are looking for a server admin or 2 (possibly a clan, as we need someone(s) that can regulate the server.

Server is free to all to play on, we only ask a few things that will help us to keep this server up for a long time to come.

- PK'ing the new players - Please try to allow the newbies to get some experience up before you go on a rampage. Liberty is designated as a no-fire zone.

- Folks that want to cheat will be banned permanently, the server is here to enjoy the game, not abuse it

- Please give feedback, particularly if you are having any problems on the server. **NOTE** we have the server set to 128 players to see what the maximum limit is going to look like. When emailing an admin (matt@gaminghost.com), please include the amount of players logged in and the time in EST format

- While in-game politics are fine, please refrain from bringing "real world" politics into the game.

- Racial and other types of discrimination will not be tolerated and will result in a ban permanently, no questions asked

- Above all, have fun and enjoy. The server is being offered as a service to the gaming community with no strings attached.

We also might offer the ability to host other servers once we complete testing and see how she runs.

Thanks for your time, see you in the universe!
