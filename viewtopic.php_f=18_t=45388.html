Using solar models as asteroids.

I wanted to create a really large shipyard-like asteroid field. I wanted to use the models shipyard_a.3db in the Solar/dockable folder and space_tankl4.3db in the same folder. Currently I have replaced the models to the solar/asteroid/models folder and renamed with debris instead of space and shipyard. This is the asteroid_arch code.

[Asteroid
nickname = debris_tankl4
DA_archetype = solar\asteroids\models\debris_tankl4.3db
material_library = solar\Solar_mat_space_tank.mat
material_library = fx\envmapbasic.mat
envmap_material = envmapbasic


This is in the asteroidfield file at the solar/asteroids folder:

asteroid = debris_tankl4, 0.600000, 0.200000, -0.200000, 70, 20, 40
asteroid = debris_a, 0.200000, 0.800000, 0.300000, 90, 40, 0
