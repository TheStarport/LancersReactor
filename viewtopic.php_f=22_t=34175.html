Attention Server Admins wrt. FLstat

If you have a Freelancer server running FLstat and have submitted the URL to the Global Rankings , please could you double check the Server Information on the site. Look at the Server List and click on your server. If any of it is incorrect, then send the correct information to flstat <at> cryosphere.co.uk. Especially keen to know the current Mods installed (server or client), and any relevant homepages for those Mods.

Some other tips:
o If you have FLstat v1.2 or earlier, then you should upgrade to a newer version.
o It is best if the submitted stats include as many players as possible.
o It is preferred if the FLstat file is not re-processed by other programs, e.g. Excel, Frontpage or PHP.
o You can submit a different FLstat URL to the one your public web site. The Server Information will only show a link to the public page.

If your server is not yet included, then see here.

Cheers

Cryogenius - Author of FLstat v1.4 and the Global Player Rankings
