Fri May 09, 2003 8:44 pm by Braxton

It was done by editing a file called mpnewcharacter.fl (using notepad)

My advice is that you change it so you start with loads of credits. With all the money you need, you quickly get the best...everything. Once that's done, change the file back and enable the "avaliable on the internet" option on the FLserver program. (i.e. disable it at first so no-one can join)

If you want to make it extra easy, you could change all the factions to neutral/friendly with you. (0.91=fully friendly; 0 completely neutral) To change specific ones you'll have to check another thread that says which line represents which faction.

Hope the info helps.


Edit: Don't worry if this sounds jibberish, it will make sense when you open the file.

Tue May 20, 2003 11:30 am by bRydeR

Just unremark the money line in mpnewcharacter.fl and add a value to the end.

I created a couple of players @ opposite ends of the map with 99mil cash and friendly to all factions for admin purposes (tracking/scanning cheats etc). You can also change your "home" faction but I found it crashed my server when I joined. In fact just about any changes except reputation and money seems to cause problems.

You can do it "on the fly"; the server pulls the current settings whenever a new character is created, but it's probably best to password your server while you're doing it to stop anyone else creating a new account with those settings. It's a shame you can't set passwords, player limits etc without restarting the server.
