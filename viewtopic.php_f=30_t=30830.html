If I'm not mistaken, the scripter automatically assigns your ship to Manhattan. Read the "ReadMe" and you should find the time when ShipScripter assigns it to there. I can't remember if it assigns a base code (for example, for Manhattan, it is called "Li01_01", or what.
If it does assign a base code, you can control where your ship is sold by entering the base code of the base you want to sell your ship before you save the file.
If you don't have a list of base codes, do a search here on LR for it; I think there's one here.
Hope this helps; post another question in here if it doesn't.
