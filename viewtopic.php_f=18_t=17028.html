I'd say the most basic to edit are ships
uncompress data\ships\shiparch.ini
find the ship you want to edit: for the liberty light fighter look for
li_fighter_01
the most needed abrevations (sp?):
Factions:
li = Liberty
br = Bretonia
ku = Kusari
rh = Rheinland
bh = Bounty Hunters
pi = Pirates (Rogues, Junkers etc.)
co = Corsairs
ge = Civilian, Zoners
bw = Borderworld (Outcasts, Red Hessians)
or = Order
no = nomads
Ship Types (Some exceptions):
fighter = Light Fighter
elite = Heavy Fighter
elite2 = Very Heavy Fighter
freighter = Freighter

Basic stuff to edit:
mass = XX ; Mass of your craft, increase by large amounts to encounter a nice handling
hold_size = XXX ; The size of your cargo hold, there are some problems with tractoring if you increase it, but no crashes afaik
nanobot_limit = X ; How many Nanobots your ship can carry
shield_battery_limit = X ; How many Shieldbats your ship can carry
hit_pts = XXXX ; How many hitpoints your Hull has
hp_type = hp_gun_special_1, HpWeapon01, HpWeapon02, HpWeapon03, HpWeapon04 ; Which weaponclasses you can mount, you need one line for each class you can mount, hp_gun_special_1 = class 1, hp_gun_special_2 = class 2 ..., HpWeapon01 = 1st weapon, HpWeapon02 = 2nd weapon ...
hp_type = hp_fighter_shield_special_1, HpShield01 ; Same as above but for shields
hp_type = hp_turret_special_1, HpTurret01 ; Same but for turrets
hp_type = hp_thruster, HpThruster01 ; this line tells the game that this ship can mount a thruster, delete it if you don't want the ship to have a thruster
hp_type = hp_countermeasure_dropper, HpCM01 ; same as thrusters, but for countermeasures
hp_type = hp_torpedo_special_2, HpTorpedo01 ; same but for Cruise Disruptors
hp_type = hp_torpedo_special_1, HpTorpedo01 ; same but for Torpedos
hp_type = hp_mine_dropper, HpMine01 ; again same but for Mines

Just don't use your modified ships online, that would be cheating
