Ok to change the starting planet. Open your freelancer EXE folder which can be found at C:\Program Files\Microsoft Games\Freelancer\EXE.

When it is opened right click mpnewcharacter.fl and click 'EDIT'.

Once you have opened up mpnewcharacter.fl with Notepad it should have something like this (below)
([Player
name = %%NAME%%
;initial_rep = %%FACTION%%
rank = 0

money = %%MONEY%%
;voice = %%VOICE%%
;All must be Trent!
voice = trent_voice

costume = trent
com_costume = trent

system = %%HOME_SYSTEM%%
base = li01_01_base)

in the first part of Notepad. Scroll down until you find the words:
(system = %%HOME_SYSTEM%%) Delete %%HOME_SYSTEM%% and put in a System Nickname (e.g Li01 = New York, Hi02 = Omicron Gamma). Once done, you'll see a line beneath, system = %%HOME_SYSTEM%% that says, base = li01_01_base.
That is the base name. Li01_01_base = Planet manhatten. So change it to whatever you want.

And this is it. This is how to edit your MP Starting Location. Below are System and Base NickNames.
