Creating a modded and unmodded version of Freelancer

Probably not the right place, but I see some people are wondering how to create a ::"phantom":: copy of FL to run mods with. Simple folks, just copy your entire Freelancer directory to somewhere else. For example, I have 3 versions, one located in C://Freelancer, C://Games/Freelancer, and C://Program Files/Freelancer. Alls you have to do is create different shortcuts to the different EXE files, such as Freelancer: v.2.01, or Freelancer (normal). All versions work with the CD, but if you try to reinstall it, sometimes the CD won't recognize the freelancer directory. So, creating different versions is quite easy, but FLMM only mods one directory at a time, so you'll have to change the directory in there.
