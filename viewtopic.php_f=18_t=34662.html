On this note...

I wonder what it would take to have some game company make a basic space game like Freelancer with a full set of editors to go with it so you could make ships, guns, systems, rumors, missions, and all the rest. To whom do we appeal? Given the life of Freelancer and its continued poplularity world wide, and several precursor games I remember that were mainly text based about trading, it seems the need for such a thing is there, only the will to create an open game and risk selling it remains. Freelancer is popular because it can be modded. Only modable games have "legs"... that is, contiuned support. The fan following garnered by the Doom series by ID since the early days should have been an indicator of the "cash cow" such a game would be. The need is there, the fans are there, now where is the enteprenureal game maker who will fill it.

It should have a basic story that once played could be dispensed with or not. Then it should have the ability to write a new story using new or continued characters. It should have the ability to add new objects and fully integrate them into the game. Finally it should have a game engine large enough to do things not done in the original production. I would buy such a game. Hell, they could even offer upgrades for those who are not into modding, but might want to try new ships, a modified story, etc.

What do you think?
