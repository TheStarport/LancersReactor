1.55 Stable Release

WHAT TO DOWNLOAD

Version 1.55 FULL - download at our website

REQUIREMENTS TO RUN Freelancer: The Next Generation Expansion

Win 2k / Win XP (MS cut support for 98)
Latest 2k service pack
XP Service Pack 1
Direct X 9B
FLMM 1.2+
Winzip / Winrar installed in pc
Freelancer FULL game (English version) Not sure about international versions
Intermediate understanding of computer operations

Recommended Minimum Computer Specs:

750 Mhz
256 meg ram
Geoforce 2

---------------------------------------------------------------------------------

First time running Mods:

Download FLMM 1.2, http://www.usaezhost.com/site/igx89/flm ... 12BETA.exe

Install FLMM
Download the expansion vs 1.55
Double click on the file to install to FLMM (may take some time on a slow comp)
Activate the expansion in FLMM 1.2
Happy playing

----------------------------------------------------------------

1.55 UPGRADE INSTRUCTIONS (READ CAREFULLY)

Deactivate Freelancer: The Next Generation vs 1.4 in FLMM 1.2
Delete all versions of FL: TNG
Download the 1.55 FULL Install
Double click on the file to install to FLMM (may take some time on a slow comp)
Reactivate the expansion in FLMM
Happy playing

----------------------------------------------------------------

IF YOU ARE CRASHING AND SUCH WHEN PLAYING:

Make sure your computer meets the minimum requirements.
Make sure you are using the most recent version
Deactiviate and then reactivate the mod
Make sure your freelancer directory is not read only
Last Resort - Delete, NOT uninstall, your FL directory and then reinstall. Then reapply TNG
Last Resort - Delete all files in your mp account and sp save-game folder
(old mp accounts / savegames that were created with an earlier version WILL crash your computer)

SERVER ADMINS:

If people complain of crashes, ask what version they created their character in. If earlier have them create a new
character, as explained above, old chracters / save-games WILL crash your server.

For 1.55 I recommend a last p-wipe, future versions SHOULD be compatible with these new characters
Also make sure you if you are HOSTING, delete / move all your sp save-games.

REMEMBER: People equipped with a hull upgrade who use nanobots when they die WILL crash the server every time.
