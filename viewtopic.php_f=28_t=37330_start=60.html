Argh

Tue Mar 01, 2005 5:09 pm by Argh

It'd be a pretty arbitrary "drop", though- you'd just have to pull a 90 and then you'd "drop" forward.

It's not nearly so annoying a problem with bombs, which I've been working on making (yet another) version of for Beta 2... they really just shoot "down", but because they work with the ship's real forward momentum, they work fairly well and are almost impossible to use in a non-intuitive way.

I just don't think there's any good way to do a gravity simulation in FL's engine, period. Which is too bad, since from the way that early shots showed players interacting with planetary surfaces... and DA probably had it in the code at some point.

And if *anybody* knows how to make THN scripts interact with in-game objects... I'd love to hear about it. We could fix all sorts of problems with them, such as having objects with moving parts. So far as I have been able to determine, though... THNs are only able to be called by Scenes, which are part of Rooms... and you cannot see Rooms in Systems, let alone invoke a Scene inside of a Fuse or anything.
