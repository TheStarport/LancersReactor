Mission Mod idea

I'm not much of a coder (although I'd love to learn) so I don't know if this is possible, but I have this idea for a "Mission Mod"

As has already been said many times, the random missions in FL are in sore need of variety. Since pretty much all they have is seek-and-destroy (and possibly return to base) it gets a bit tedious after a while.

The idea is twofold:
1. Create new random mission templates; possibilities are escort, recon, investigate, patrol, infiltrate, etc (infiltrate involves a temporary rep change where you investigate an enemy's base and try not to get scanned). I'm pretty sure this wouldn't be too hard, since the demo story mission involves escort duty.
2. Create side quests; story-style missions with no direct influence on the story, but perhaps adding background. Incentives would be things like cheap, free, or exclusive info (location of wrecks, jumpholes, etc) or good items, and if all else fails, more cash.

Please feel free to slap me if this idea has been discussed already, but I can't find the search link, and if anyone cares to teach me how to edit ini's and code into mods, I would greatly appreciate it.
