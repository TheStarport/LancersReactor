you are indeed right, the 'navy alpha seven is in audio files, but not as you think it is.

Its like this:

West point base ini entry:
audio_set: lib_ba_1_fem (not what it is, i know, i just using an example)

and there would be a file somewhere that is called 'lib_ba_1_fem' that has various text strings:
'Liberty'
'Bretonia'
'Rheinland'
'Navy'
'police'
'alpha'
'beta'
'gamma'
'1'
'2'
'3'
'4'
'5'
'6'

The speech is simply generated by using each of the sound strings in sequence corresponding to the designation. Thats why the voice never sounds quite smooth.

As for modifying the code to change te player's designation, im not sure, but it may very well be possible, as it is done in multiplyer. Ive been called 'freelancer Alpha 16-1' before
