Well the "official" patch only makes changes to FLServer, so unless you use the Multi-Player server program, there's no point.

As for the "MOD patches". These patches are designed for games with a particular MOD installed. But don't install patches for MODs you don't have installed.

Though there's a patch called "unofficial 1.4 patch" or something like that. This fixes some minor errors in the game, like nav-map information, object names and other little things like that.

EDIT: And another thing, the "SDK" is not a patch. I'm not sure what it is myself, but I believe it has something to do with creating MODs. So unless you're planning on doing some programming/editing, there's no point.

the SDK is a set of debugged decrypted INIs, many bugs that were tracked down by modders have been fixed and so, yes it is a patch as well as a system developer pack - you need them to install many mods and they don't interfere with anything but some very large total conversions, and its a small download so ya i always install the latest version

the quickfix adresses video card issues and also bumps up eyecandy performance i think

don't install a mod patch unless you already have the mod installed in FLMM's mod folder - installing it overwrites some files in the mod to patch it, if its not there it won't do anything but install the patched files as if they were the entire mod
