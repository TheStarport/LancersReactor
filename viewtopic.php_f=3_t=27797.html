Earendil

Fri Jan 23, 2004 9:12 am by Earendil

If you have an editor and access to the FL saved game file, then this is what you need to do. Change the number which relates to how friendly or unfriendly a faction is, by changing it to 1.0. FLSO has this capability and I often change a character or two of mine to 1.0 to the, say, Corsairs just for fun. Then you see my character name in red brackets around my ship in space if you are unfriendly with the Corsairs along with the name Corsairs by my character name. This can happen in the real FL MP game but it is difficult. You would have to do so many friendly building things for the Corsairs that eventually they may be friendly enough to be above .9 which I believe is when the faction tag is added to the character name. Good luck!
