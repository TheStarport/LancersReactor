Not familiar with this mod but what you're describing seems an incompatability issue with the server version of the mod and Ioncross. Check what the version number is of the flserver.exe of the MOD and then open up the main IONCROSS folder and search for the FLServerOperator.ini

Look for this line,

FLServer - Version 1.0 Build 11 - FLServer

Change the 1.0 to the version of the mods exe file. Save and restart Ioncross. Hope you have the server files for the mod because Ioncross is not much use without them. Hope this helps.
