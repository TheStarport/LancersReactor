i was wondering how you can increase a scanners scan range so that it can detect objects from along way away, how would i undertake such a task?
  • Top

0blivion

Fri Aug 31, 2007 3:10 pm by 0blivion

Open up DATA\EQUIPMENT\misc_equip.ini either with BiniQDU or after installing Freelancer SDK and search for ge_s_scanner_01. There you will find two entries you might be interested in: range and cargo_scan_range. What these do is self evident; just set them to whatever you want them to be, save, and you're good to go.

One thing you might want to note, though: NPCs disappear after 2.5k unless it's a mission, so don't think it's not working if you don't see NPCs after 2.5k. If you want to change that, check out Dev's awesome limit breaking 101 thread.
