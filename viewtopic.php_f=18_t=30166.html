Freelancer Explorer update submitted by Louva-Deus

The Freelancer Explorer installer needs to be replaced with this new file.
It now supports Windows 95, 98, and ME installations and also removes the delete-all-from-install-folder so it is now safe to install it into Freelancer's EXE folder which many people need to do in order to get the program to work. The old installer will wipe out the EXE folder when you try to uninstall FLEXP.
Enjoy from Louva-Deus

For questions and answers see The Lancers Reactor > Forums > Freelancer Editing: Freelancer Editing Utilities Forum UTILITY: Freelancer Explorer

The Freelancer Explorer project was an endeavor to create a system editor for Freelancer. This dream started by Shsan has now been passed on to us and our goals for this project are not just to make a system editor but to make it easy to use and problem free.

Creates and modifies systems for Freelancer
Built-in support for Freelancer Mod Manager
Built-in partial support for infocard creation
