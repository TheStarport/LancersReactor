Starseeker is right Cain, you can change the audio files. I realized it after I posted my reply. But as he already said. It will take quite some time to change all audio files into new ones with a female voice on it.

Look in this directory for Trent's audio files:

Data\Audio\Dialoge\Trent
and
Data\Audio\Mixes

The Mixes directory is a pain. Not every file has Trent's voice on it, but many do.The files in the mixes directory usually contain more voices than just Trent's and some background noise as well. (Basically they are the audio files of several video sequences in the single player story mode)

It's a long shot, but if you have a voice morphing program like Starseeker said, it should be a bit easier, but it will take quite some time to alter everything.
