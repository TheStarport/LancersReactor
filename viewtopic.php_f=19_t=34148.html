New: Global Weekly Kills Table

New to the Global Player Rankings is a table of This Week's Top Players based on the number of kills made since last Sunday. Inspired by something similar developed by the Namyangju Server, this is designed to encourage healthy competition amongst all Freelancer players, regardless of flying hours, rank, etc. The previous week's table can be found by clicking on the link at the bottom of the page.

Edit: There is still the Top 1000 Characters, the Top Clans as well as the Server List .

Any comments or suggestions on how to make it better, let me know...

Have Fun!
