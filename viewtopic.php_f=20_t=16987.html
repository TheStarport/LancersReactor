Anything that effects your drive letters can cause this. (Adding or removing drives, using a Ramdrive, etc.)
I had the same problem after installing a new cd drive. I fixed it by running regedit (win98)
>start>run type in regedit
Click the little Plus signs (+)next to the following:
HKEY_LOCAL_MACHINE
then Software
then Microsoft
then Microsoft Games
then Freelancer
then 1.0 (this may vary if you've modded)

Look for the value "CDPath" and right-click it.
Choose "Modify" and change the LETTER to the drive letter of the CD drive you want to use.
Note: Change the Letter only! Ensure that a complete path statement is there before saving.
I.E. "D:\"

You must restart your machine for this change to take effect.

<<CAUTION - EMPHATIC STATEMENT!!!>>
REGEDIT is a powerful and wicked tool that can damage your operating system if used carelessly!! Backup your registry before using if you are new at it!! DO NOT fool around in there!! You have been warned!!
