I think the 2nd shield should work. I assume you saw the Hardpoint tutorial thread.

just edit the .cmp file and add HpShield02

I think cargo size = hold_size
I hope you're not changing the HpBayDoor01, HpBayDoor02 hardpoints.

in the shiparch.ini file you can change:

;Titan
[Ship
ids_name = 237021
ids_info = 66542
ids_info1 = 66543
ids_info2 = 66608
ids_info3 = 66544
ship_class = 3
nickname = co_elite2
hold_size = 140; change the size here
shield_link = co_elite2_shield01, HpMount, HpShield01, HpShield02
hp_type = hp_elite_shield_special_1, HpShield01, HpShield02
