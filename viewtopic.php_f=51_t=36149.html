When you start the server, do NOT let it take control of the flserver.exe file. Then in the configuration logs you will see FLServer Window Name in a field near the bottom.

FLServer - Version 1.00 Build 11 - FLServer


You will want to change that to say:
FLServer - Version 1.28 Build 11 - FLServer

(the above is what it should read!)
