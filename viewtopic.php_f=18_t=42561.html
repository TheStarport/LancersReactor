Well, the way that i do it is to add a lane acces zone overlapping the minefield using FLE and usually add both a circular access zone and a smaller ambient pop zone. All of which are again overlapping the minefield.
Then, i manually edit the minefields solar/asteroids file. There might be an easier way, i'm still using an older version of FLE, but if u look in the SDK for the mineield in New York u'll see..
[Exclusion Zones
exclusion = Zone_Li01_to_Li05
exclude_billboards = 1
exclusion = Zone_Li01_zone21_lane_access
....
etc
You, just add the names of the zones to be cut out of your minefield to that file. You will most likely need bini.exe or sumthing to decode the ini first.
