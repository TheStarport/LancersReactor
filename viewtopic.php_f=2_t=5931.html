A common fix for freezes:

It might be that you have some editing tools installed for starlancer like SLEdit, and that you have used it to switch ships, either ships of your wingmen or other ships that are important in the mission, if this is so unswitch all ships.

If you havnt installed any of these tools you could just try reinstalling SL, this fixes most of the crashes and bugs

Also try this: disable EAX in the sound options menu thingy, this causes about 60%+ of all the problems
