data folder -> characters folder -> Newcharacter.ini file - you cannot miss teh lines:
[Package
nickname = ge_fighter
strid_name = 11051
strid_desc = 11551
ship = ge_fighter
loadout = msn_playerloadout
money = 2000

[Faction
nickname = new_player
rep_group = co_ic_grp
base = Li01_01_Base
Package = ge_fighter
Pilot = trent

The other thing you can change is starting ship in here, as well as starting location. IGNORE everyone who mentions any fl files like newplayer.fl and stuff - cause all that file does is call for the values in this one.
