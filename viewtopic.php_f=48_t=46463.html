If you're running IONCROSS Server Operator and you get kicked for cheating, you will need: IONCROSS Freelancer Utilities
.

This is straight from the description, so you know what it does:
Used for creating the GAMEDATA files that allow the Character Editor and Server Operator software to work properly with modified versions of Freelancer, the IONCROSS GAMEDATA Creator enables you to quickly add new or edit existing entries to your GAMEDATA files.

If you have not applied that and have not updated IONCROSS Server Operator, then you will definetely get kicked for cheating. Also, if you're using any other software such as server operators and/or FLAC (FreeLancer Anti Cheat) Software then find a patch or anything that will update the file so it will not kick any cheaters. Also, Cheater's Death (CD), i am not sure you can update files or anything because i have only used it once.

Hope your server will now work successfully. Enjoy playing Freelancer!
