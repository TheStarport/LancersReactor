Here's how I found out what the patch did:

Create a folder in Microsoft Games called Freelancer (you will have to rename your freelancer install directory to something else)

In the new folder create a folder called EXE

Copy the flserver.exe from the install to the new EXE folder you created. You must do this, because it checks for this file when you double click on the patch.

Back in the Freelancer folder (the one you created) create another folder called DLLS then it it create a folder called BIN.

We're trying to match the Original Freelancer Directory structure here.

now, run the patch. it should say "Do you want to install the file?" and then click yes.

It will overwrite the copied flserver.exe and add the other files.
