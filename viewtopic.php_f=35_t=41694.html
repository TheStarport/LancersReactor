{Software} Freelancer Web Launcher v1.0

The final project is here! Freelancer Web Launcher is the ultimate vision of the Direct Connect project and is ready for public consumption.

Freelancer Web Launcher is an addon to Freelancer that provides permanent new functionality. This addon configures Windows to recognize "freelancer://ip:port/" links as legitament URLs that will launch the game with the specified "ip:port" and will automatically load that server into the Multiplayer Server List (LAN and Global). So now you can set up a webpage link for your server and visitors will be able to jump onto your server with a single mouse click. All you have to do is add code to your webpage something like this:

<a href="freelancer://213.202.247.73:2302/">Play on The Void!

This project was designed to act as a backup plan for the Global Server being shut down permanently but it also has other useful applications. Note to Mod Makers , one of these applications is that this works for Windows shortcuts as well as webpage links, so if you use a regular installer for your mod project that includes Start Menu entries, or if you provide a desktop shortcut for your server (Direct Connect style), then you can now add a direct link to your server to that group or as an even better desktop shortcut. Have one link to activate your mod and another to launch the game with your server preloaded.
