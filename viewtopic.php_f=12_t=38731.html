Freelancer-type web based game

I was hoping to get some feedback from some of you fine folks that appreciate the same genre as I do. I've been working on creating a shockwave game in some ways similar to FL. However, I'm still just starting, as I'm learning the skills as I go. However, everytime I play FL, I see features that I would like to incorporate into my own game and it's come down to this...what about a Freelancer "lite" game? It would be all web based and no local install necessary (maybe optional).

If one were to create all models and textures, referencing but not copying FL items, would this be legally and/or moralely accectable? If created, it would be a free game and the purpose is mostly for gaining programming experience for myself.

i envision a universe similar to FL graphically, with game play reminescent of Trade Wars (greatest old school game ever!)

Opinions? Here's a link to a design phase demo to show I'm not just blowing hot air. Understand that I'm still figuring out the basic movements ect. The only thing to interact with currently are the "hyperlane gates".

w+TAB = Fast forwad
