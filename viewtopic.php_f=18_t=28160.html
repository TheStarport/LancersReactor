Fri Feb 06, 2004 3:54 pm by ColinS

After loading model into Milkshape, open the Tools menu and select "Model Cleaner". If a dialog opens asking if you want to clone materials, click No (might do it anyway if no mat assigned to hardpoints). The final dialog is a list of items adjusted in the model. The flat triangle could be the problem .

Save model and try exporting again.
