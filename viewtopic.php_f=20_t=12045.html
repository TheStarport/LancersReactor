Maker

Sun Mar 30, 2003 5:40 am by Maker

You can run server/game on same machine and not be online.

1) Fire up the server
2) In the EXE subdirectory in the Freelancer directory is a file named
fldirectip.hta
Double click it. Enter 127.0.0.1 in the box provided
3) Click the "Add Server" button
4) Click the "Start Freelancer" button.
5) Choose Multiplayer
6) Choose Lan
7) Double Click on the name you gave your server when you fired it up
Have fun

Maker
