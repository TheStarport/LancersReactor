you find out when you look in nameresource.dll which number is "Battleship Missouri", it's in stringtable 11, first string convert this number to a ids_name:

string-1 : 1-1 =0 convert to Hex: 0=0
string table-1 : 11-1 =10 convert to Hex 10= 00A (allways 3 digits)
namesresource is 3rd resource file so hex number is 300A0
convert this to decimal and ids_name= 196768 look in li01.ini and you'll see that 196768 is Li01_03
