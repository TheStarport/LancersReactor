First time poster? Be courteous and READ THIS FIRST!.

Hello and welcome to the Freelancer forum! We are very pleased you stopped by to make a post here! But there are some rules you have to take in mind before doing so.

Let me put them in no particular order.


- First make sure you’re posting your question/comment in the right forum. Click here to see all the forums: Click here.
- Please read the Rules and Regs first. Some basic rules about the forums can be found there. This is the FAQ link for questions about the games Starlancer and Freelancer. We do frown on spoilers on the game itself in this forum. We have a place for those right here Fire away but 99.99999% of the spoilers have been identified by now.

Also, as a site we have decide there is a taboo of discussion on any forum. Read about them by clicking on the word about JOYSTICKS? and the dreaded cloaking mod!!
Oh yea, and if you want to chat it up about the potential of Freelancer 2, this click right here and read/reply to your hearts content.

Now if Fan fiction writing is in your blood and you want to submit your literary works, then just click right here and submit it directly to Eraser. If you want to see the published works of others, then simply click right here. There is the some good reading, worth the click. Of course there is a special forum area we set aside for the writers located right here.

If you'd rather get a good laugh to begin your Lancers Reactor experience, This link is guaranteed to get you in the mood. Feel free to add your own.

Are there still questions you want to ask? Please read this first:

- Maybe your question already has been answered! You can do a forum search on the subject you want information on. It’s simple, just type in your subject, select the Freelancer Discussion and search! This feature can be found here: Forum search. I'm sorry to say it but the search is removed due to the strain on the server. It's not sure if it's going to return.

You still have a question or comment? Here’s some posting help.

- Please don’t put CAPITAL letters in your topic name. This could mean you’re screaming (netiquette) and above all, it looks not good. We’ll read your question/comment and if we can, we’ll post a reply. Don’t worry.
- Is your question a technical one, please give us your system specs (CPU, RAM, mobo and videocard). If you get an error, please give us the full error (when possible).
- We don’t like it when you’re bashing someone, for instance things like: “I hate the game for the static trading prices!” Your voice can’t change anything anymore. Your only hope is to pray for add-ons/patches.
- If you want to give your opinion on the game, please look for another topic with the same subject. It is very annoying to see 5 topics with the same subject. If you give your opinion and it’s a bad one, please give arguments why you don’t like it.
- Topics with the subject “Joystick” support are not appreciated. All the things have been said about this subject. Please do a forum search on this issue.

Well, this is it. Now you’re off to post your topic here on TLR. The large amount of users are willing to help you! I hope you’ll like Freelancer and maybe you can drop by in our IRC chatroom: IRC chat

Have fun!

Vincent "Wizard" Hoogeboom

Posting images

Regarding Pictures on this, and other, forums

Please follow these guidelines:

1. once you have taken your screenshot - use PAINT (note - NOT photoshop) and resize it to around 800 x 600 and then SAVE it as a JPEG. This will make it around 50Kb per screenshot. USE paint, or if you HAVE to use photoshop - use a POOR quality setting, YOU CANNOT TELL THE DIFFERENCE FOR THIS IDEA, as this will keep them SMALL in size (around 50kb instead of 200kb).

2. Once you have saved it, now RESIZE the picture AGAIN to around 250x175 or so. Then SAVE THIS AS A JPEG FILE AND CALL IT THUMB!!!!!

So then you have two files:
1. Desktop.jpg
2. Thumb.jpg

Now - use some nice coding from the forums to create the IMAGE of the thumb on the forum which is an ACTIVE LINK to the picture size. This will ensure that loading this thread will take seconds, EVEN for a 56k modem user. So they see a small version (only 16kb or so) and click if they want to see more. Then they can see the full sized version (which can be any size you like) in a new window.

See my example below



The code for this is (*subsitute square brackets instead of the curly ones)
{url=http://www.pathfinderstudios.com/PICTURES/29.JPG}{img=http://www.pathfinderstudios.com/PICTURES/29thumb.jpg}{/url}

There we go, obviously change the links to YOUR host of pictures
The image above is only about 100x75 - so yours will be alot bigger of course - which is nice to . In effect you could do this in a thread containing one picture or around 300 x 225 or so - which is linked to the full size version
