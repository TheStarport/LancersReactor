Here are the differences for the 3 types of zone shapes.

sphere:

size = 2000

box:
rotate = 0, -30, 0
shape = BOX
size = 1200, 1200, 2400

ellipsoid:
rotate = 0, 50, 0
shape = ELLIPSOID
size = 6000, 6000, 10000
