Hi again unicornz!

CTD means that the game crashes, shuts down and you are back in windows.

Bejaymac is right. The only possible solution is that your Br01.ini is totally empty except of your new jumphole entry.

Try this: At first deactivate all mods. Then extract the FL SDK TWICE. First in your FL directory (this way it fixes some errors if you play vanilla FL). And then one more time to another directory. Then you can work on the files in the second directory, without working in your actual FL game directory (Working in the game directory is not a good ides, as you need to make backups manually and sos on).

After this, go to the Br01.ini and add the jumphole again (same code as in the thread) and afterwords to the Li01.ini and add the other jumphole. Then try if it works. In my opinion somehow your files got screwed up. Therefore you should start again and then it should work.

Using jumpgates will not work and you also do not have too many objects in NY (I think White was wrong with this). Just try to start again and if you do only these things (adding two jumpholes, without adding anything else or deleting anything) it should work!

Good luck!
