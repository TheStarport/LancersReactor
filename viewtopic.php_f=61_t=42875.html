First off...
Don't delete the LI01, FP7, or Into systems.
LI01 is part of Freelancer's setup, and is also part of the Intro system.
Fp7 is also part of the setup, and Freelancer gets touchy if you mess with it.
The Intro system folder also has three "rooms."
Those three rooms are actually the background for Freelancer's Main Menu.
So...
Deleting the Intro system is the main cause of your crash, as there's no background "room" for the main menu to display.
You can change the "rooms" and thereby change the menu's background.
Most "Cityscape" or "Planetscape" rooms will work, although, if they have flashing lights, they'll appear as flashing squares.
Bars, Decks, and other rooms can be used, but there won't be any people.

As I haven't tried it, I suspect that LI01 could be deleted if you replaced the LI01 info in the Intro.ini file with your starting system's info.
But that would also require changing the various .FL files that reference LI01, and if you're using the OpenSP mod, then the LI01 entries in M13.ini would need changed too.

Also, if you're using OpenSP, you'll need to edit the M13.ini file to tell Freelancer what system/base to start at.

Oh, and if you deleted all of the systems/bases, I presume, in Universe.ini, did you also delete them in the "Shortest_Path" ini's?
(There's three of those.)
As well as their entries in all of the "Market" ini's?
And in the mbases.ini?
I realize that you probably did most of that, but did you asked for ideas.

Watch your 6!
