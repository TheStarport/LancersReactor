For the level issue Go to C : Programs/MicrosoftGames/Freelancer/Data/Equipment/Market Ships and locate the ship in question. Example is -
marketgood = gf5_package, 24, -1, 0, 0, 1, 1, 1
Change the first number, 24 to 1 to get it at base level
The second issue go to Data/equipment/goods and find the package you seek example -
[Good
nickname = lamda_shuttle_civ_hull
category = shiphull
ship = lamda_shuttle_civ
price = 123576
ids_name = 239551
item_icon = Equipment\models\commodities\nn_icons\lamdaco.3db

Change the bold item to what ever price you want.

Hope this helps.
