Just a thought:

Open FLMM, go to the "options" menu item, and select "Choose a different freelancer directory" from the menu drop down.

Next, when the window opens up, make sure that it is "blue" on the Freelancer folder! Not the exe folder etc, just the Freelancer folder. Click okay, and see if it works then.

If not, then which mod is it you are trying out?
