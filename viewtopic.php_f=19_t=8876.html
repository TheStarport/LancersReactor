As of now there is no way to carry your account across servers. If you uninstall FL, get a new machine or otherwise remove FL from your comp you enter your old MP acount ID when you relaunch multiplayer from the new install (you just delete what it puts as your new ID and replace it with the old one). If Its a default installation you can get the ID code from the windows start menu. It should say "Examine My Account ID" in the FL folder".
