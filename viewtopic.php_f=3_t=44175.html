robocop

Tue Aug 01, 2006 2:33 am by robocop

A Vanilla server is one that just runs the basic install of Freelancer from off the CD. Sometimes a server that mods just server-side also calls itself a vanilla server but some consider that to be no longer vanilla.

In my opinion, a vanilla server is a server that does not require the client to install any mods in order to experience all that the server has to offer.

When you look at the global server list look for servers that are highlighted in blue. Under version they should say 1.0. When you select that server you will notice a box in the lower left corner of the global server list screen. Often there will be info in there that tells you whether or not mods are required and if a mod is required where you can get it.

Good luck.
