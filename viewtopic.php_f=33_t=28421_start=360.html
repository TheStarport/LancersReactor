archkaine

Mon Mar 28, 2005 1:44 pm by archkaine

If you're trying to use the OpenSP with the latest version of Rebalance (Rebalance 3.5.1 Delta 2). I'm sorry to say that I've made it impossible, due to some issues with Rebalance itself. I may, at a later date incorporate an OpenSP patch directly into Rebalance, but that depends on if I can get all of the other issues dealt with first. My apologies for this inconvenience. If you MUST have the latest Rebalance and OpenSP, then do the following:

1: Make sure that no mods are activated in FLMM. Open the sript.xml file found in the Rebalance mod folder in FLMM's folder (C:\Program Files\Freelancer Mod Manager\Mods\NameOfRebalanceModFolderHere). I forgot the folder name that I gave it, so whichever folder refers to Rebalance.

2: look for something that mentions exclusive, set that to no (exclusive="no", or just delete that part of the line (it defaults to no if there's no reference to it).

3: Save the file.

4: Activate Rebalance 3.5.1 Delta 2, then activate the OpenSP mod, say yes to any dialogs that popup... I haven't tried this, so I don't know if it will work.

5: If all that works, try running FL and starting a NEW GAME... (opensp won't work with pre-existing save files).

6: If you've made it this far, then you're ready to roll. Good luck.

I'm afraid that due to stability issues of the mod itself, I don't support the mod with other mods active, so use it with others at your own risk.

If you want to, you can look for the Rebalance 3.5 beta 3 and grab a copy of that til I can get the latest one ready to roll. I am not planning on quoting a release date due to the extensive alterations that need to be made.

Good luck on whatever you try Let me know how it goes for you.
