Hi there! I read elsewhere that you've just bought the game and want to start up a server...cool

However, a quick peek in the Freelancer Editing forum would have told you what you're after...

Anyway, to save you some time, here's the info you need anyway (from memory - apologies if anything here is inaccurate!)

1. You can't change the starting ship (easily). This info is held in an encrypted file called \DATA\CHARACTERS\NEWCHARACTER.INI. This file can be decrypted easily and modified...but you'd need to issue a copy of the file to every client that wants to connect to your server....which is more than a little bit sucky

2. You can change starting money, base and reputations though
These items are referenced in \EXE\MPNEWCHARACTER.FL which is (woo! yay!) not encrypted. You could, for example, get your players to start at a base that has a better ship that they can buy with their starting credits...

Change the %%MONEY%% to 20000 and your players will start with $20,000 instead. Which is nice.

Changing the system and base is a little tougher as you need to find the right codes. Look them up using this LR forum thread.

Finally, the reputations can be changed by altering the decimal values in the "house = <xyz>" section:
0.91 means a particular faction wants your babies
-0.91 means a particular faction wants to kill you and your babies

Again, faction names can be found by searching the editting forums...

However - can I advise that you don't go too loopy with your starting values, as joining a server with a phat load of cash and friendly to everyone makes for an extremely dull game....

HTH,
Dan
