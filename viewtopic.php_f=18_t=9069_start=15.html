don't forget to add the rep of the faction to itself...otherwise, bad things might happen

then add
rep = 0, fl_n_grp
to every faction already in...it then creates a new faction

Obviously, you can make the rep for the different factions whatever you want, as long as you reciprocate the rep to that faction.. i.e. if you put:

rep = .23, gd_bh_grp

then you would need to go to the gd_bh_grp faction list and put

rep = .23, fl_n_grp

The only thing I haven't figured out is creating a new ids info and name to add a name and description for the new faction(s)...but that involves .dll editing and I can't do that yet...
