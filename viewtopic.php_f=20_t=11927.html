Possible fix for ni servers listed in multiplayer

Ok after installing directx9.0a update to may already installed directx9 I found I had no servers listed in the multiplayer listings and 'connecting to global server' was showing all the time at the top.

Ok this is a 'possible' fix that worked for me --

When u update directx 8.0 to 9.0 or 9 to 9.0a a small file should run once to register all the directx dll files to finish the upgrade, this file being called -- DXDLLREG.EXE - this is found in Win XP in (drive):/windows/system32.

When I first rebooted my pc this file was mistakenly opened by Quick View Plus and as far as I can figure out it crashed at starting and stopped the dxdllreg.exe program from completeing its registering of the files. I know this to be the case as when I ran dxdiag I had errors saying missing registry entries under the network tab for directx8 entries.
After I rebooted again and ran the dxdllreg.exe file I again ran dxdiag and found that all the entries under the network tab saying that entries were missing was now gona and all was saying ok.
As I say a possible fix for some of u may be to manually run this file found in - windows/system32 dir - just double click it and then look in dxdiag to see if the errors are corrected and even if not maybe try the game to see if servers are now showing in the listings.

After I did this and noting else low and behold I ran Freelancer and servers were again appearing in the multiplayer listings

Hope this helps some of u out there
