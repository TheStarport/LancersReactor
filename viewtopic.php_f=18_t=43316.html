Accushot

Sat Apr 22, 2006 3:11 pm by Accushot

You can't do it. The ships you see in cutscenes are just empty shells. If, for example, you want to make ships move you have to create a path and drag them along it yourself. Pretty much everything has to be done manually - even getting an engine flame requires you to create a separate effect and attach it to the back of the ship.

You'll have to do as Manhattan_Guy says above; create a bunch of explosions in amongst the ships (or attach them to areas of each ship model) and set them off in a way that looks like an explosion. Perhaps even spawn some debris models and move them in a trajectory away from the explosion if you're feeling adventurous.
