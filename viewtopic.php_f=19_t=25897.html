Create a NEW shortcut for Freelancer. It should be a copy of the FL shortcut that is under start, under Programs, etc. Open the shortcut properties. I have found you can do this through the file manager/explorer only. Not through a right-click of the NEW shortcut in Start menu itself. Add the information moby posted above in the box next to Target. If there are quotes in the box, stay WITHIN them. Good luck...

By the way if it is my server (or any server whose IP address is NOT static) realize I make my IP address dynamic so players need to connect to the global server to log onto Boston.
