1. I reccomend NOT renting a server. It will be harder to manage, you will have to setup VNC on your PC and the server to control it remotely and manage the server. just get a cheap PC ($400) and host. Get 1GB or 2GB of RAM, that will speed up things considerably, and around a 3.2GHz or equivalent proc. Once I can be bothered, I'll get you a link to a rig I think will fit the bill. GPU and hard drive don't really matter, unless you want to run the game while hosting it on the same rig, which I don't reccomend.

2a. My connection is a Road Runner Cable one, 1Mbp/s download / [some unkown tiny number upload, and it works great. Whenever you have a chance though, go for more upload vs more download. Uploading is the big bottleneck on servers.

2b. Freelancer will only work on Windows PCs. Too bad. You can get an OEM copy of Windows XP at www.newegg.com for cheap, though. I reccomend Windows 2000 because of it's stability. XP should work fine too, but don't use Win95/98/ME. they have serious stability issues.
