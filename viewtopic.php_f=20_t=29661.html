Welcome to TLR! The first thing is to make sure that you have the latest drivers for your graphics card and sound card. You should also turn 3D Sound off in FL (if you have it selected) because it can cause problems. Finally, consult the stickied threads at the top of this forum if you haven't already done so, because a lot of fixes are listed there.

As for the FPS, I cannot recall a command off-hand, but a third-party program like FRAPS can show you your FPS. You can find it on Google.
