Yes, this has already been thought of, to some extent. I was planning to do an experiment with this at some point, but I haven't gotten to it yet. In theory, you should be able to do this pretty easily... in theory... by naming a file in a MAT that contains animation instructions. This is already done in a variety of places in the stock game, so I would be surprised if we couldn't make this work.

Whether this will actually work... I dunno. If you're wanting to try this out, I'd suggest building a cloned version of standardeffects.txm, or just using that Material as a named DcDtOcOtTwo texture on a "ship" consisting of a flat plane. DcDtOcOtTwo Materials render two-sided and will use opacity channels. For more information about Material types, please see my Tutorial, "How to Skin", in the 3D Modeling Forum.

As you can see, I had the experiment set up (in my head) but actually executing it will take at least an hour or two, and I've had other things going on
