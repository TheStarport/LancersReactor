Bejaymac

Thu Feb 08, 2007 8:34 am by Bejaymac

All newer version of Milkshape (1.7.7 onwards) will crash on start up if you don't add "CMPImporter.dll" to the buggyplugin.txt file, all older versions will run without a problem, but it still won't import properly for most people. This is down to a compatability problem the plugin has with graphic drivers, directx & your OpenGL version (a bit like VISTA atm).
