ATI Radeon 9200SE - some issues

Specs:

Pentium D 805
Asus P5V800-MX
Sapphire Radeon 9200SE 128 meg AGP8X graphics card
512 megs DDR RAM

So.

I installed Windows XP on it + SP2 and the VIA Hyperion drivers to make sure the AGP Miniport worked OK and everything. I installed a special version of the Catalyst 6.11s provided on ATI's website which are designed to work with the 9200 series of cards.

My problem is, if I change 3d graphics settings, and use anything besides "Application controlled" for Antialiasing or Anisotropic filtering the game becomes embarrassingly choppy. Considering that Freelancer was designed to work properly with 16 meg video cards on Pentium III 700 processors this is actually kind of annoying.

I'm using the 1024x768 resolution for the game.

Any suggestions? I haven't really monkeyed with the AGP aperture or anything like that.
