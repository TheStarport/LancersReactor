Are you talking about just playing FL MP or running a server? If you are just playing (in other words you are a FL client) then you need no INCOMING ports thus you should need to set up NOTHING in your firewall. Most firewalls (except for ZoneAlarm) are INCOMING port auto-blockers only. So if you can get to the Internet then you can play FL MP.

Running a FL server and you need only one INCOMING port and thus need to set up a firewall tunnel (sometimes called a virtual server) in the gateway (I call routers gateways because they do so much more that JUST route!).

You say you can't play? What are you seeing when you try? Are you accessing the FL global list server in FL?
