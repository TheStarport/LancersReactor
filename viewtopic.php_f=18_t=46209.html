unfortunately there is a limit to the length of the texture you could use for a billboard. i know DDS cubemaps at least will save in a resolution of 1:6 height/width but whether the game will allow an odd size like that for a normal texture is questionable

the good news is that if you're willing to fiddle with it you can move the texture right to left, then jump down to the next line, scan again, jump etc until you reach the end of the texture, at which point your animation would return to the beginning of the loop. i will be posting a materials semi-tute soon, which will explain how UV animation works, if not how to use it (its a little tricky)
