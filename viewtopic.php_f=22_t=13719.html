Chatting from the Console can be done in one of three ways (or all at the same time if you're so inclined).

CHAT w/universe: Sends server wide messages. Console can only see replies if they are done by players hitting enter and arrowing up until you see Console and typing in their message. Players see WHITE text from the Console ingame.
CHAT w/system: You can chat with everyone in a particular system. Everyone in that system can chat with everyone else including the Console if they make it a system wide chat. This is indicative of the BLUE color of text ingame.
CHAT w/player: You can have a one on one conversation with one player but they have to hit enter and arrow up until they see Console to reply. This is indicative of the GREEN text ingame.
