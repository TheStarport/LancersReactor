Sun Jun 25, 2006 12:39 pm by H.Q. Victor

Which ever ship you want to try, rank too high, mod the level lower yourself. Open MS Games/freelancer/data/equipment/marketships

marketgood = Scorpian_package, 30, -1, 1, 1, 0, 1, 1 (Original)

marketgood = Scorpian_package, 1, -1, 1, 1, 0, 1, 1 (New)

The first number is the rank required level. Simply find the ship you want and change the rank level, simple. Download search for Misc Data base file and it gives the code/listing for each ship in the Vannilla game.
