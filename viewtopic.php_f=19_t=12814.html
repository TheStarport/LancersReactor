Yeah, if you lose that CD key after reinstalling, you will lose your online chars. It isn't printed anywhere on the case or disc either. I'm not sure how it's made, probably some crazy random generator based on your system ident, or something (otherwise it would be the same after reinstalling). It is explained perfectly in the server readme on the CD.
- User has a unique ID. During first launch of the game, the ID can be entered, it is PROFILE dependant. How important this ID is is perfectly explained in the text coming with it !
- Server has a unique ID. A mix of both IDs are made to obtain the final server-player's ID. This prevents "bad" server operators to retrieve player IDs, from their public server and "use" them.
