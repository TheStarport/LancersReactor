Creating links is fairly simple.
When you're making your reply or posting a new subject you'll see Forum Codes in blue, click on that and it will give you examples of how to...
You can also click on edit in someones post that has what you're looking for and that will show you how it's done. Note: You can only edit your own post unless you're a moderator, moderators can edit or delete posts.

Don't forget you have to add brackets, I wont show the brackets in the examples because then they wont show up, the forums will think it's a command, so I'll replace where the brackets would go with the greater and less than sign < >.

Bold: enclose your text with and
Italic: enclose your text with and
Image: insert an image with <img=link_to_my_image> where <link_to_my_image> is a link to an image.
example of an image link..
<img=http://www.lancersreactor.com/images/mypicture.jpg>
Link: enclose your linked text with <url=http://www.mysite.com/mydir>text goes here can be the actual link itself</url>
Code: enclose your text with <code> and </code>
Quote: enclose your text with <quote> and </quote>

All the smilie codes are there in the Forum Codes too.

I've already posted this in the Technical Issues thread about the Global Server, but I'll repeat it here...

If you know the IP of the server you want to play on, it's very simple to connect. Right click on a shortcut to Freelancer, click Properties. See the "Target" field? It should have someting like:

"C:\Program Files\Microsoft Games\Freelancer\exe\Freelancer.exe"

Alright, now just add -s, followed by your server's IP and port. For example, -s123.456.78.90:2302. It should look like this when you're done:

"C:\Program Files\Microsoft Games\Freelancer\exe\Freelancer.exe" -s123.456.78.90:2302

When you start the game, select Multiplayer, then LAN. The Server will show up regardless of Global Server status.

There are also other Direct Connection programs out there that let you connect when the global server is down. Sometimes they have a bunch of servers listed. You'll have to search for them, because I haven't been keeping up with the most recent ones.
