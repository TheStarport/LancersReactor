ElectricBrain

There is no zoom feature.

The draw distance varies depending on the object you are talking about. For planets/space stations, the distance at which you first see them is faaaaaaaaar, like around 100K which takes you maybe...15 min to travel in cruise.

Other objects like in asteroid fields are drawn in at a much closer distance, say around...3K, but FL compensates for this by having asteroid sprites in the background. This boosts performance cuz you have less things to render and the effect is well done.

Things like other ships appear at about 3K away as well.

Tue Feb 18, 2003 12:40 am by milod

The correct (although less commonly known) term is "3D billboards." A sprite is technically a transparent bitmap used to render a character or object in a 2D graphic system, such as Mario the plumber in Donkey Kong. A billboard is a single flat four-sided polygon that is always oriented so that the plane normal faces the camera, however the camera moves through 3D space. The ships in the early Wing Commander games were rendered using this technique. Some flight simulations use this technique to render large numbers of trees on the terrain model.

The advantage to using billboards for far away objects such as asteroids is that there is only a single polygon (actually, as many as four triangles after clipping) and a single texture to manage. This greatly reduces the amount of data that needs to be sent to the graphics card and increases rendering performance. If the video card supports fat point clouds, even more bandwidth can be saved.
