Mon Apr 14, 2003 8:06 pm by Ut

The ini files come compressed, so you need to decompress them. Go here:
http://www.lancersreactor.com/t/editing/download/
and download BiniQDU. It decompresses the ini's into .ini.txt files. You can edit the .ini.txt files, and rename them back to .ini files. Make sure you've turned off hidden file extentions first, though.
