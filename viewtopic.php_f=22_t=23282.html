But the server won't be here for all of time. There will come a day when it will suffer the same fate as browclops MkI. We can only prey that day is far away. I can ensure you right now though that it is no where in site at the moment.

Modders won't live long once found. I knew we should have avoided posting on lancers, but whats done is done. Short of passwording the server, there is little I can do about it. We might be closing new account creation soon though. Till we weed out some of the bad apples.

Too all of those waiting for account transfers, it might be a little while more. For me to upload the new characters is shall we say, much more complex than it is for other servers. Mainly because I have to do it all remotely. When I browse through the accounts, or transfer characters, I have to download the whole every single folder character folder via FTP, browse them, find yours, find yours, browse the files from Browclops, find the matching ones, move the characters from that folder to the new one, then surf the FTP once more for your files and upload them to the correct folder by a manual search. Its very time comsuming. And for safety, I have to take the server done while doing this, to make sure nothing gets corrupted during any of the downloads or uploads by the server writing to these files. And downtime for the server is bad.
