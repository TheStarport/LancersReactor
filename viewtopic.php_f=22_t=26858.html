Unfortunately Cardamine, this is a LOT of work for a sysadmin. For myself there is moderating the server's boards, advertising the server by posting on other forums (like here), checking the server for cheaters, talking to the players, setting up events, troubleshooting server hardware and software issues, etc. To randomly setup players starting locations I believe would take a third party program unless you are dedicated like Captain Tylor and can do that every night. Also unless it's done automatically (like with JoeBoomz' excellent FLSO) it would get old fast moving, editing, monitoring and reinstating cheaters that in my mind aren't worth a microsecond of effort in the first place.

Instead these are excellent suggestions for events which happen occassionally on many servers toward an effect at gathering together those that are interested. My server runs these usually once every weekend thanks to a player called WAN_SOLO who thinks up and referees many of these events. As interest mounts and more players wish to help, more events can be added.
