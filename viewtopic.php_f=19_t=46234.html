druid

Fri Aug 10, 2007 2:13 am by druid

Okay, so your mod version (none) is right and you are at least connecting. You are undocking and last a few secs so it does not look like a cheat/error detection. It really looks as though you just have a poor connection so like the others said, you need to check pc settings not freelancer.

Anit-virus set to allow FL can still have background checks running, zonealram can have 2 sytem checks at about 30,000kbs using up CPU power. Switching it off altogether can work.

Vista as said has problems running FL if any other anti virus is present. I have had reports from at least 3 players in a week who have had to completely remove other AV from the pc to run with Vista.

Check (ctrl alt del) task manager to see if you have other background processes or auto updates running too. iTunes, Apple, HP, Quicktime and loads of other crap will set to run on start up when you install it and after you put enough of that into your machine you start to creak.

Oh yeh...if you are using wireless try running a cable to the dsl and playing via LAN. Also download a direct connect tool so you can play without going through the global server. I don't have a link for one now but a search on the TLR homepage should get you one.
