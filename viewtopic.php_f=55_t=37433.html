By network card I mean a desktop card with a RJ-45 connector...
Just like a sound card or video card but a network card.
Some computers have one integrated into the motherboard.
The cards plug into PCI bus master slots on the motherboard of your computer.
So you may have network capabilities and not even know it.
They cost around $15.00 at your local electronics store.

If you are playing around with MOD's
1. Make sure you move or delete any saved games you have.
Previous saved games cause a conflict with your shinny new mod so they have to go...
Saved games are located at:
My Documents\My Games\Freelancer\Accts
There are two folders
One for Singleplayer
And one for Multiplayer
Move or delete the files in both of those folders...Singleplayer and Multiplayer.

2. Deactivate any MOD's that are active...Open the Freelancer Mod Manager and make
sure that no MOD's are active, if there are deactivate them...

3. Delete the Freelancer folder.
By deleting the Freelancer folder you make sure that you will get a nice new clean
install of Freelancer...that's what you want is a new clean set of files...
Freelancer is located at:
C:\Program Files\Microsoft Games\Freelancer

4. Reinstall Freelancer
Pop in the Freelancer disk and do a complete install...
Server files included...

5. Now you can run the Freelancer SDK v1.3
Browse to and install to:
C:\Program Files\Microsoft Games\Freelancer

6. Now you can activate the MOD you want to play with the Freelancer Mod Manager.

7. Start up flserver.exe
C:\Program Files\Microsoft Games\Freelancer\EXE\flserver.exe
NOTE: The Mod Manager has an option built into in to start flserver for you...
Wait until the FL Server finishes .....loading......
Start Freelancer.
Click on Multiplayer.
Click on LAN.
Start a new Character.
Have a blast!!!
