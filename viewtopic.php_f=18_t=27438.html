FreeLancer INI uncompressor (command line, v1.1)
File size : 19,99 Kb
Category: Utility
Date: 3/11/2003 6:51:29 PM
Description: This small command line program uncompresses the INI files used by FreeLancer, allowing it's modifications (FreeLancer can use uncompressed versions of these files)

Version 1.1 has a small fix, allowing the uncompressor to handle correctly INI entries with 0 values (source code on the INI document has been updated as well)

Created by Mario "HCl" Brito

You can also use Freelancer SDK v1.3 made by Louva-Deus
Freelancer SDK (software developement kit) is set of decompressed (text) ini files (over 1200) from the original game of Freelancer, you can use this package to make your mod making life easier since it contains fixes for all the major errors in the original source. It also fixes the errors caused from decompression using bini.
