I guess I don't believe there is filtering. I see servers from all over the world. Is there an instance where one player can see a server and another cannot at the exact same time?

I could believe this if different global servers are enabled throughout the world, which is what I think you are trying to find out. From what Chips posted under [Listserver, anyone see anything [udifferent[/u than what it shows? The semicolons are like comment commands so the only FL global server enabled is the MSN one.

As for listing on the FL global server I have seen some odd things. Such as copying over the Multiplayer directory to another server to test it for an upgrade. Now two DIFFERENT 'puters on the Internet with the SAME FL MP server ID are petitioning to the global server to be listed. If they regularly update at timed intervals, which I think they do, then the last update from a server will display it's IP address. I had this problem when players were trying to log into one server but often got the other.

Other global server problems I have seen are ports being selectively closed and I am not talking about just the INCOMING client port. Also lockups by flserver will maintain a listing on the global server but allow no logging on to the game server because flserver is locked.
