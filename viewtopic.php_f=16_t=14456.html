The only things visible in Alaska are:
1) A LOT of mines
2) A 'prison' which serves as the setting for a long cinematic sequence in SP. You never actually get to use it as a base.
3) A base which is neutral to you, but you cannot dock with. The only times during SP you have access to it, you're supposed to be going somewhere else, so you'll get the dock denied message.
4) Some weapons platforms near the Gate to York.
