Gyroscope

Sat Mar 04, 2006 2:18 pm by Gyroscope

[Player
name = %%NAME%% ((Name of your character. Default is Trent.))
;initial_rep = %%FACTION%%
rank = 0 ((Starting rank, default is 0.))

money = %%MONEY%% ((Starting credits, default is 2000))
;voice = %%VOICE%% ((Trent's voice - do not change this.))
;All must be Trent!
voice = trent_voice

costume = %%BASE_COSTUME%% ((Trent's costume))
com_costume = %%COMM_COSTUME%% ((Not sure.))

system = li01 ((Starting system))
base = li01_01_base ((Starting base))

Change %%MONEY%% to the amount of your choice.
Change the system to the system nickname of your choice (not in-game name)
Change the base to the base nickname of your choice (not in-game base name)

So, say you want to start at Omicron Alpha with 1 mil, you'd make it:

money = 1000000
system = Hi01
base = Hi01_01_base

Not sure about ships, however. What you could do is make the start location where you buy the ship and supply enough credits to do so.
