Reynen Client / Server Mod 2.0 BETA 5b RELEASE

Download FLMM Version 9.0 MB ----- Readme.txt

Download RAR Version 6.2 MB

The website is also now up and completed.

Reynen Client / Server Mod Website

Unzipped The mod is about 28 MB and I HIGHLY recommend NOT running any other mods with it as it changes over 150 files.

Please see the readme for complete details, this mod is HUGE and completely revampes the Freelancer universe!

Enjoy! This is over a month and a half months worth of work so I hope you all really like it. Please post comments and feedback!

I will be hosting a server with this installed. Look for Reynen's Mod Server. (12.252.29.222) Other servers can be found in the readme file.

There have been some major changes in version 2.0 BETA 5

- Added cargo train
- Corrected TIE infocards
- Changed the starting screens
- Added Edge Stations docking rings
- Added Outer stations docking rings
- Added Kusari stations docking rings
- Added Border stations docking rings
- Added Rheinland stations docking rings
- Kusari gunboat main mount now swivels
- Stealthblades for theTIE's are now buyable
- Fixed map bug that displayed huge dock rings
- Fixed a problem with one of the Rheinland bases
- Changed and updated freighter ships and info cards
- Fixed problem with Defend willard station mission 3
- Fixed problem with Defend benford station mission 4
- Captain walker's ship will not be destroyed so easily
- TIE stealthblades now have starwars sound & graphic FX
- Increased timer for mission 8 from 2 minutes to 20 minutes
- Trent now has new clothes (need to start new game to see them)
- Fixed "Bouncing" with Battleships when cruising toward a target
- GB, CR, & BS thrusters are have now been "fixed" to prevent abuse
- Due to complaints about prices being to cheap...I've raised ships to match the readme.
- Created a temporary fix for GB, CR, & BS cargo and tractor problems (need to find the root of the problem) ( something is taking up 400 cargo slots, so I added 400 to each to compensate )

Update May 11th:

- Due to complains about the race, cruise has been set to 400.
- Added buyable cargo train to planet Houston.
- Collision has been raised from 5 to 10
- Max ammo has been increased to 100

For installation help and trouble shooting please see the readme file

KNOWN BUGS / ISSUES

- If a station is destroyed with a player in it, the server will crash.
- Battleships have real problems in asteroid fields
- If you try and buy gunboats consecutively, game will crash
- Nomad gunboat will not use a thruster
- Stations that sell ships, but do not have ships for sale, the vendor will pop up then close.
- Freelancer will crash if you try and run without the mod and have moded save-games
- When you first buy a gunboat, cruiser, or battleship your entire cargo is available. Once you save and reload your game / or log out and back in for MP, you will lose 400 cargo slots. I have created a temporary fix to this.
