@thing, Welcome to TLR.

In your My Documents folder, is a folder called My Games. That is where the saved games are kept. Look in the single player one and see if they are there. Question, have you installed any mods? There should be a Autosave and any regulare saves there.

My Documents/My Games/Freelancer/Accts/Singleplayer/Autosave.fl
If you see an additional folder, the game moved the saves to this one. You will need to copy and paste the saves back into Singleplayer folder for the game to see them.
If you have opened a mod, you will need to close it before you use the saves as it can cause the game to crash. Post back on your situatioon if you need any other help and some one can help you further if you need it.

Edit- As far as mods, there are many. Look over the thread of names, and discriptions. Evolutions, Hostile Universe, rebalance, pryvateer, ect. try different ones. FLMM has some built in one too.
