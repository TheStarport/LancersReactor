empathy.ini shows how much each action the player performs affects the reputations with the factions. Using this file you can see that if you kill bounty hunters, for example, it upsets the zoners (positive empathy) and makes most criminals happy (negative empathy).

To set up which factions hate each other you need to go into initialworld.ini, these are the actual reputation between the factions and are not effected by the players in any way.

To control how the factions feel towards the players you can edit mpnewcharacter.ini (for MP games) and I can't remember how to do it for SP games other than editing mi13.ini when using the OpenSP mod. There are some tutorials about this in the tutorial forum.
