captain howdy....3 things every ship needs

.cmp - the model file...exportable through milkshape

.mat - the texture (material) file - created using UTF

.sur - the surface file...use one from a ship that is about the size as yours

what chips is saying is export your cmp as the same name as the cmp of a standard freelancer model...and do the same with the .mat...if you do that you wont need to code it...to test it anyway
