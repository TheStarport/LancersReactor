Cheers Bejaymac, i didn't know that one. So, djkaos, if you just deactivate all fs mods and just delete the freelancer.exe and flserver from the mod folder and activate. It will work ok, just remember , you can't join any FS 174 servers as your version number will be wrong.

@Sarges Slaughter, hiya, you could try deleting any old server details that may be in:
..My Documents\My Games\Freelancer\Accts\MultiPlayer

The client version is for playing on multiplayer servers, or running a basic server
The OpenSp is for single player
The server op is for changing the servers start up details and loadout.

You need to be a bit carefull with using the server op with FLShell as you can easily start at a level higher than 10 and the restart options will not work.

So, for my money, delete any previous multiplayer files from other servers.
You can host Frontierspace without permission, i'm happy with people playing freely as much as they like.
