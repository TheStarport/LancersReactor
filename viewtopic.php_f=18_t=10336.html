death match?

Ok, anyone else noticed that dog fighting in MP sucks? It takes a long arse time to kill anyone. I'm going to play around with a few things but here is basically I'm going to try and create a death match mod. New play balance would go something like this :

1) Takes place in one system - this concentrates everything around a planet or if there were TDM, two planets as bases
2) No nanos or shield bats - yeah, those have to go... if one person has tons of shield batteries and the other just wasted them all in a fight, who is going to win?
3) shields charge quickly - this way you can go from one fight to another. Your shields will be back and your enemy won't be any better off.
4) no random encounters - its DM afterall
5) all components are free - you customize your ship
6) ship classes - ok, this is kind of in the works and I'll have to test balance. It would go something like this

light fighters - thin fast charging shields, more manueverable/smaller profile = lower level stuff, smaller reactors, some will be able to mount class 10 guns

heavy fighters - thicker and very slightly slower charging shields, sort of a middle of the road choice, better reactor to support more high level weapons

very heavy fighters - these are going to be slow, great reactors, thickest shields, great guns, slowest turning.

freightors - these are going to become more like gun boats, shields are not going to be as strong as vhfs, best reactors will go on these to support massive fire power. You'll have to have good aim for these because you won't be dodging much.

7) Class 10 weapons will have massive power drain - you aren't going to be want to mount 7 class 10 guns, you need to try and balance your fire power a bit or you'll never have enough power to fire those guns.

Longer range weapons - tachyon weapons have 700m range, but I think there should be weapons with ranges up to 1000m. Combat should be all close quarters

Ok, I'm sure I left some stuff out, so I would like some input on death matching ideas before I start changing stuff. I think most of these changes would add a lot to the PKing in FL because as of right now there seems to be very little actual one on one combat. Anyone even interested?
