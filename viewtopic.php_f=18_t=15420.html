sorry dude dont use messengers,

ok first things first,

your script puts double entries into the ini for the 3 ships your trying to make buyable -the fact it did anything apart from make your game fall over is more by luck than judgement.
you do NOT just paste in lines of code you'd like to see.
to make this work you need to firstly set all the ships currently buyable on manhatten to not buyable, ie set gf2_package, lf_package and lfr_package to
-1, 0, 0, 1, 1, 1
once youve done that you can make the ships you want buyable DO NOT replace gf2 with bwf or anything like that just change the bwf line to read -1, 1, 1, 0, 1, 1 at the end.
Look at the code i put in the thread about your script problems and look for the REASON the code is written the way it is.it may help to have the original ini file open in wordpad (use bini from the downloads section of this site first drag the file on top of bini and release and it makes a text copy you can view in wordpad)
the reason the code only does one line at a time is because there are lines in between the ones your changing you dont need to change.

THE MOST IMPORTANT THING IS TO MAKE BACK UP COPIES OF YOUR INI FILES BEFORE YOU START DOING THIS - JUST IN CASE!!

before you try this i think you need to go read all the tutorials again or look at the code other people have used in their mods and use it as a template

with a little dedication this stuff aint too hard to get a grip of - i had never replaced a ship before i tried to fix your code in the other thread (my mod just adds one ) but i looked at rapids code from the tie mod and worked out what he was doing then did the same

stick in there and you'll get a grip of it in no time
