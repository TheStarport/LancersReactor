For opening *.txm, you should take HLC's UTFEditor and go to the node called Texture Library, then choose the map you want to extract, enter in and select the MIP0(zero) , MIP1, MIP2, ...(there could be 1 but sometimes there are about then click the export button. Save it as *.tga and then you should be able to open it with photoshop for example. But if you find MIPS, you should save it as *.dds. I Never tried to because you must download plugins for photoshop and it didn't work for me. There are some post whitch talk of this. Just try to search in the forums "DDS" and may be you'll find it.

For importing, just make your picture and save it as *.tga. Then open UTFEditor and create a new node in the Texture Library and an other in the node you've just created. Call the first as you want and the second MIP0. If you want to put others, you can put them in the first node you've created new node called MIP1, MIP2,...

That's all.
