kingrat

Wed Mar 03, 2004 8:06 pm by kingrat

Sometimes with the ATI based cards, even with an updated driver the program will not recognize it. Check your card settings in the display properties section. I know with NVIDIA cards there is an option to have the program set the settings. If that is not checked, then the program is locked into using the settings that are set in that dialogue box. The other option is to check your version of directX. You need to be using ver. 9.0b. I had to upgrade to this in order tto solve some graphics problems. I hope this helps. Good Luck
