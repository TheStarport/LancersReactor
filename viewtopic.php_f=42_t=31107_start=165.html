warzog

Wed Nov 17, 2004 4:57 am by warzog

jenxter:
Question 1:
NO!
This release was a complete rewrite because in the last version, there were to many bugs induced by trying to adapt the Sirius Sector of Freelancer to the Gemini Sector of Privateer.
I discovered, during the disasterous last beta, that the FL engine doesn't like having the Sirius sector messed with.
So...
With this beta, I left Sirius as is, I just moved it out of the way, and made it invisible/inaccessible.
The next beta release will have the same Gemini Systems, Bases, etc.
But...
It will attempt to repair some bugs, and to improve general gameplay.

Observation 1:
As my computer is XP, and I know of no one who can alpha test using Win98/ME, I can't make claims as to compatipility.

Perry is system #48.
If you open the GS48.ini file, you'll find:

[Background
basic_stars = solar\starsphere\starsphere_stars_basic.cmp
complex_stars = solar\starsphere\starsphere_rh02_stars.cmp
nebulae = solar\starsphere\starsphere_bw06_fp7.cmp

Each of the 70 GS##.ini's have:

[Background
basic_stars =
complex_stars =
nebulae =

If you replace the data with the info from GS48.ini, all systems will basically appear as Perry.
(They'll still have their unique planets, bases, holes, and fields.

Observation 2:
If you can produce better Privateer-like music, that would be great!
Several things you need to take into account:
1) Freelancer's engine ONLY plays WAV files, not MP3's.
2) Music files need to be as small as possible.
--In Freelancer, tunes that run in a continuous loop are 200kb-500kb
--In Freelancer, tunes that run only once are 600kb-1.3mb
3) We would prefer tunes that can be looped continuously.

warzog

Thu Nov 25, 2004 10:22 pm by warzog

"OLD" save games save your location.
This version of the mod doesn't use the same system designations as Freelancer, as older versions did.
The Freelancer systems are there, but all of the market ini's have been stripped.
(To keep the mod size down.)
Sorry 'bout that.

To use an old save game, edit it, and change:
one of the "sys_visited" entries to: 2334889548
one of the "base_visited" entries to: 3096232199

Also change you're current location to those numbers.
Which are System GS52, and Base GS52_01.
Then restart the mod, and Freelancer, and load-up the edited savegame.
(It'll place you on the New Detroit base.)

If your save game is from the current version (v4.0) of the mod...
Then something's become corrupt...
And that means that you need to uninstall, and then reinstall everything!
(Freelancer, FLMM, and the mod.)
