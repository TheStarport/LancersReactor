How do you find the node or ISP range?


node is a section of IP's, usually the last section of an IP address so you ban last 0-255

e.g. 100.1.1.1.200 so the ban would be 100.1.1.*

for * you would ban the range 0-255

The ISP depends on the IP, you would have to resolve the IP and find out where it starts and find the Exact range the ISP owns, as some ISP's have simmilar IP ranges so you could ban/block someone innocent. So won't go in to detail for that one. But http://www.dnsstuff.com/ should help with that, or any ping trace utiltity usually has the ability to find the IP range owned by the ISP
