Sun May 09, 2004 6:32 pm by calshire

Well, what kind of chat room do you want to put in there? Sort of like an IRC style one, or like a continuous refreshing web-board style?

The IRC style, iirc, needs a server to hold the chat room first of all. The web page can then load a client that connects to the room. The bots shouldn't be too hard, since there's lots of tutorials on the internet.

The continuous refreshing web-board style will need some sort of cgi capability, which would probably be way beyond what you're doing (I'm assuming basic high school stuff).

If you want some good references, check out www.w3.org. It has a lot of documents on what each command does, etc.

Hope that helps.
