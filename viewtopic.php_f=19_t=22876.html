Players will do whatever is fun. If it isn't fun for them, then they will quit doing it. If a victim isn't having fun with what is being done to them, then they will find a way to make them stop. I certainly don't think all servers should be like this. Just one server.

Noob killing is in there because it makes it the most challenging to advance. It is already too easy to get ahead. A noob can get away from a noob killer by asking for help. Dock camping makes it more challenging and you can normally get away from a dock camper too. The fun is that you can turn the tables and do the same to the victimizer, but it is challenging.

I stress that these types of server rules should only be on one good server to see how it works out. It could work well if you get a group of good players. The lack of community rules would have to be very clearly stated in the server news and description, so that people would not complain about stuff like that.
