Frist off, you need to have Fl-ID's, This will enable you to view the names from the dll. files, sort of takes time to get use too, but it works.

Second, if you don't even know what your looking at, I would say download this. That has every items,ships,weapons,etc nicknames from the ini. files. Now if you don't got Microsoft Excel you can just down it from here.
Now remember that is only the viewer, so you'll only be able to read those files for microsoft excel and not edit them.

After that you should be all set to start to edit the ID's. Just becareful not to over write something.

Also might want to get Resource Tuner Link below

After you find the name, you can see what dll it is in and open that dll with the Resource Tuner then find the string by looking at FLIDS in will tell you exactly what the string number is, and it is easier to change or just browse the dll for all the names after you open it. It is called Table####

To help you out if you open the nameresources.dll then string folder scroll down to 2525 and that is where all the ship names begin.
