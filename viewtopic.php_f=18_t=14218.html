jo hi@all

Sorry ,
but the mod I made is a client-only mod, no server mod. I don't have neither time nor inclination to give support for a server you decide to set up with my mod. Most people complain it wouldn't work correctly, and what would I know what kind of bugs could be surfacing for others?
I only can say that for me, it works like a charm. Okay, given, some bugs I know of and work hard to solve them. But I just cannot talk for hours every evening just discussing bugs (or thought-to-be bugs) or other problems via ICQ for people who think they have to set up a server with my mod.

I'll only write this once. If somebody thinks he absolutely has to set up a server with it, then he should de-install Freelancer and manually delete the Freelancer account data in his profile folder under My Games. Experience showed me that this is the most common cause for problems with my mod.
Then, you can re-install Freelancer and the mod. It should work properly now.

Okay, once again: I don't give server support, because this is a client-only mod.
Alright, check it and judge for yourself.

cu then
