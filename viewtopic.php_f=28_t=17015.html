...this is not the right forum for such questions.
But anyway:
In principle you can add new factions without messing around with dll's or so.
Each faction is defined only in the ini's nowhere else.
So to add a new faction write the stuff into all ini's dealing with factions...ad some names and infocards and you have created a completely new faction.
To see which ini's you have to edit do a search on the complete /DATA directory...you should find 13 files
