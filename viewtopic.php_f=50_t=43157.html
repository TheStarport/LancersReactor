You said you can't open a saved game with the Ioncross Server operator? How about after you download it from the server to a folder on your local machine and then point Ioncross Server Operator at it? (Set the folder path name to FLServer.cfg in FLServerOperator.ini, this will be the same as the saved games for MP folder)

I haven't done this and wouldn't want to, but anyway I'll try to help give you a pointer or two...

I wasn't able to open any MP saved games in Notepad or Editpad, they're encrypted. But I could open any of them in Simple Save Editor by TheJKWhoSaysNi.

It's available here:- http://www.lancersreactor.com/t/downloa ... asp?id=771

HOWEVER: the name of the savegame is a hex number, like for example 0b-89bd486f.fl, and I think you will not be able to easily discover a player's file related to a specific character name. The lookup file is normally in the Ioncross folder, it's called FLServerOperatorPlayerList.ini and is the decrypted FLServer.cfg but if Ioncross can't get to the FLServer.cfg file or accounts.cfg file then it won't be available to you, the files themselves will still be encrypted.

But - OK, if you can manage all that, then you just need to open the file in the Simple Save Editor.

After this it's fairly straightforward but still needs great care, all items including the ship and the equipped items are in hashcodes and these are pretty standard knowledge now, they are listed in the Ioncross GAMEDATA files so that's a good place to start.

Or you may need additional tools such as CRC Tool and the Ioncross Gamedata Generator to help you out - sorry, I know you'll hate me for this but I don't have time to trace the links for those for you, just do a search on the names in the General Editing / Editing Utilities forums, and on the www.lancersreactor.com/t/downloads webpages. They are there, I got my copies from thence I promise!

But overall... I would try hard to get another server that you can have remote admin access to, for safety.
