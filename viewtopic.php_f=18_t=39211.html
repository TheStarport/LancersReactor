Bejaymac

Tue May 31, 2005 6:43 am by Bejaymac

To get the base to work you need to change the entries-

from -> file = Universe\Systems\Li01\Bases\Rooms\Li01_01_cityscape.ini
to -> file = Universe\Systems\TM25\Bases\Rooms\TM25_01_cityscape.ini

Do this to the rest of the file & then copy all Li01's room ini's to TM25's room folder and rename them to match the base.You have to give every base an individual set of room ini's or you'll warp back to the origional base.


*edit* Just noticed,add "_Base" to the nickname in universe.ini,you might want to change the voice as well,hearing a woman when you can see a man is a little strange.
