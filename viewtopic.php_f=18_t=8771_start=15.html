Fri Mar 14, 2003 12:22 am by HCl

Indeed, the conversion process itself will most likely discard animation data and such, so it'll probably be the work of an UTF editor to add those animation chunks in. I'll be able to give more details when i reach that part, my priorities regarding ships are: mesh format -> hardpoints / mount points -> animations.

I'll be working on a Milkshape plugin because it's the only 3d software i have installed here. After i finish documenting the format and release the source code of the Milkshape exporter, somebody else will be able to work out on plugins for other 3d software.. i hope my humble contributions for the FL community will turn out to be useful regardless of the 3d software you use

Also, yes, the UTF editor should allow you to add / remove .wav files from the .utf files
