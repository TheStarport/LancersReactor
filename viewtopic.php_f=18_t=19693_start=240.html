Chips

Sat Aug 30, 2003 9:23 am by Chips

FLMM v1.2 moves saved games in and out as too which of the mods you load... so you in effect 'lose' your saved positions when loading a different mod! You should be able to get round it by going to your save games folder and moving files from one folder to another - but you will crash if there is a difference betweent them (ie - one has a ship that the other doesn't, you are starting in a system that doesn't exist) i suggest that you use something like the remove storyline mod to sort out getting rid of the storyline for every mod you load.

Okay - this is too start with alot of cash instead.
If you are in the starflier (gf1) then open up the goods.ini and scroll down till you find the following entry:
[Good
nickname = gf1_hull_m01a
category = shiphull
ship = msn_playership
price = 6000
ids_name = 12013

[Good
nickname = gf1_hull
category = shiphull
ship = ge_fighter
price = 6000
ids_name = 12013
item_icon = Equipment\models\commodities\nn_icons\cv_starflier.3db
ONe is for the ship with mission 1a (juni gives it too you). Now see the price - change this to 9000000 (9mil) and you should be able to sell your ship for about 7mil - easy money! Save and enjoy the cash (change both to be sure you get lots of dosh) You can change other money by doing the same for other ships hulls that you are flying, but if you ever try to buy that ship again - it will cost 9mil!

If you're using BINI then, in Windows Explorer, just drag the .ini file onto the BINI icon. A new file will be created called <FILE>.ini.txt. Delete the original file and remove the .txt from the end of the new file, which is now unencrypted.
