[Good

addon = co_elite_power01, internal, 1
addon = ge_s_scanner_01

addon = contrail01, HpContrail02, 1
</source>

First off, those are cavalier engines (not that it really matters), and second, there are two of them. Which is very odd, but that's how it is. Since both engines provide 48000 thrust and 599 linear drag, you'd think that the ship would still only go 80, but since it goes 160 I suppose the linear drag from one of the engines is ignored. Which is also very odd, but that's the only explanation I can come up with. Just remove one of the indicated lines and it should fly normally.
