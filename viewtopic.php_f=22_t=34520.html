As I mentioned to you here, you cannot do that IP address to FL character name matchup through FLConnMon any more because AkaTaka no longer provides the server for FLAcctBrowser for it's decrypting work. AkaTaka is the author of both of these programs.

To match FL saved game file names to character names, that's easy! Use IFSO, select the character in the Players tab view and hit the "View Player (non-edit)" button. Under the File View tab you'll see the entire path to the file name! It's helpful because you can also see the character's account directory name! Good luck!
