Victor

Tue Jun 24, 2003 6:10 am by Victor

Make sure you deleted the entire line, something like: addon = SlowSmallPurple, HpRunningLight01, 1

The alternative is to edit lights_equip.ini yourself go to the entry for each light you don't like and change it. You could set it's color to 0, 0, 0 or bulb and flare sizes to 0.000001 to remove it or you could simply change the color or blink rate to something more pleasant. Changing this file will effect all ships in the game as soon as you save the file and fire up freelancer, and it looks as though you can run it edited client-side only without any problems. For me I think the game looks a lot better when most of the lights don't blink at all and the ones that do blink have longer delays and shorter durations.
