Mod to make the race v. Hovis EASY

I bought Freelancer the day it came out. I had a great time playing it, but I have had an extreme amount of difficulty beating Hovis in the race. However, I have developed a simple mod to fix this.

1. Down bini (in the utilities list)

2. Copy bini to this directory
c:\program files\microsoft games\freelancer\data\missions\m06

3. Run bini.exe on the npcships.ini files in each directory to uncompress it into npcships.ini.txt

4. Open npcships.ini.txt in notepad.exe. The first ship entry is for Dexter Hovis, and is listed as follows:

[NPCShipArch
nickname = MSN06_Dexter
loadout = MSN06_Dexter
level = d6
ship_archetype = bw_fighter
pilot = MSN06_Dexter
state_graph = FIGHTER
npc_class = lawful, FIGHTER

5. In Notepad, edit the fifth line to read:

ship_archetype = rh_cruiser

6. Save the file and exit notepad.

7. Rename the npcships.ini file to npcships.ini.bak (to save it in case you made a mistake).

8. Rename npcships.ini.txt to npcships.ini.

9. Play.

The result is effective, and funny. Dexter Hovis, the racer, shows up for the race in a cruiser too big to fit into the gate. I flew the race in my Clysdale, at normal speed (79) and handily won. This mod did effect the graphics a bit, before the race, but this didn't effect the outcome, and should not have any effect elsewhere in the game. By editing the npcships.ini file in each mission directory, it's possible to change the type of ship npcs fly.
