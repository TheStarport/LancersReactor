MIPS are for dds files
MIP0 are for tga files

so if you are extracting textures form MIPS, make sure the extension is a .dds not .tga.... Now for viewing this you can only use photoshop with a dds plugin from nvidia.

so, if you want a tga texture.... use milkshape with cmpimporter and import the ship with the corresponding mat file and check the check box of export tga files to a directory or something....
