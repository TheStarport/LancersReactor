Sherlog

Sat Jun 21, 2003 12:02 pm by Sherlog

I have thought about the police/militia angle as well but I found that there are two basic problems: it is difficult to verify claims of wrong-doing against somebody unless you happen to be on the scene yourself so that you can see what is happening, and the regulators need to be able to deploy fast . I think the only exception are equipment cheaters that edited their ship/weapon stats in the files; but you cannot catch cheats that modify the game in memory because they can revert to normal when the regulators show up.

As I said elsewhere, the game could do it easily and therefore it should. I.e. if there is a case of non-consensual PKing then the game should deploy a wing of souped-up Patriots/Drakes/Banshees (equipped according to the toughness of the offender), lawful bases should turn hostile and all exits out of the system should be blocked with gunboats/cruisers + fighter wings. The police doesn't have to be super-tough - I think it is okay if a skilled player can beat them, and the APB should wear off after a while (doesn't apply the cheaters/modders who the game should simply boot immediately). The basic idea is simply to make tollers/pirates/etc. face higher odds; so if there are couple of kids in Titans camping a dock then they would eventually deal with about a dozen Banshees or Eagles and that should give the lamers something to think about.

This solves both the verification and rapid deployment problems because the game 'knows' about all that is going on and it can spawn fighters right at the scene of the crime.

Of course, the game currently doesn't do this.

Players doing duty in a permanent police/milita could get special equipment like long-range scanners and cloak, and they could use a special-made array (or cube/sphere) of jumpholes for rapid deployment. Players willing/able to handle a call would simply load their police character which would stationed in a base adjacent to the jumphole array or already floating in space in front of the array.

Of course, somebody has to vet the people applying to enter the militia and create their police characters (savegames) for them ... And the force could be spread rather thin if a bunch of rogues do coordinated strikes in several places at the same time (just like in RL). But I think the FL server capacity is several orders of magnitude to low for such a 'community' solution to be feasible currently.
