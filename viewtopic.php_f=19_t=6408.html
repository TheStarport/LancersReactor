It is possible to connect to another server, even when the list servers are down.

To do so, do the following:

Execute the freelancer.exe application with the -s<IPORT> option.

For example, to connect to the Elite server:
freelancer.exe -s131.107.83.10:2302

(which is the static IP,and current port the server is running on)

Other server operators may want to post a message as well with the current IP of their server.

Stinger

Tue Feb 18, 2003 12:09 am by Stinger

To connect to the TLR server:

freelancer.exe -S67.39.67.197:2304 (updated)

To start a server it's very similar:

flserver.exe /P2302 (that last number is important to post since it's the DirectPlay port)

All this is done two ways. One is to go directly to the command line of your computer (START/RUN/CMD) and get to the directory where the files flserver.exe and freelancer.exe are located. Then simply follow below:

TO CONNECT freelancer.exe -s67.39.67.197:2302
TO START SERVER flserver.exe /P2302

The other way is to add this on the end of your desktop shortcut, but being a DOS man, I encourage the former technique.
