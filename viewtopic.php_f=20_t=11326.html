If you go to the Linksys website (assuming you have a Linksys router), the download page for the firmware doesn't have the *Z* on it, however if you look at the FTP link below which is from Linksys, you can clearly see the *Z* after the version number.

****** PLEASE NOTE - THIS FTP LINK IS FOR THE BEFW11S4 v1.0 WIRELESS ROUTER ONLY ******

ftp://ftp.linksys.com/pub/befsr41/befw1 ... -1442z.zip

Go to your manufacturer's site and download the appropriate firmware upgrade. I have a wireless router (and 4-port switch), which is why I'm using this particular firmware. I looked through some of the other Linksys router products, and most have a similar version number, some with a *Z* attached. As far as I can tell, the Z stands for ZoneAlarm support, which shouldn't affect us any. But like I said, I updated the firmware, and now I'm playing online without any errors at all and no special configurations.

I sincerely hope this helps some of the others out here having the MP problems.

I'm sorry I can't vouch for any other router manufacturers except Linksys, as all I own is a Linksys. But if anyone else if having these issues and manages to cure them with a firmware update, whether it's a Linksys or not, please post the results here.
