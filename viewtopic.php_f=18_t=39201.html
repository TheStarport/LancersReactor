FriendlyFire

Mon May 30, 2005 8:21 am by FriendlyFire

1.- Do anyone Know if it is possible to create a Joystick Mode???
---- A Joystick Mod is not possible.
2.- If it is Yes or No, please, explain Why?
---- Because Freelancer has no Jystick Support.
3.- Did anyone try to make the Joystick work?
---- Yes, using some programs that simulate key pushing with Joystick Button pushing. Button One would, for example, do as if you pushed "A".
4.- if the answer is yes, please, explain what happen...
---- No one really uses it because it is impossible to aim correctly with this, because the Joystick moves the cursor to move the ship, BUT the cursor is the aim. Also, precision is no way with this. NPCs would kill you easily.
5.- How many of you would like to get the joystick working on freelancer?
---- I guess not many. before more wanted, but whenever you really get the trick with mouse and keyboard, you love it.

Hope this helps,
