Restoring MP Account ID

I've been playing Freelancer with a couple of friends while we were housesitting. I hosted the server. I did that too with the server. We also all did the account lookup thing and got our account ids.

We copied all those files back to the new computers, BUT we can't find any way to restore the account ids we used while housesitting. I heard before that the account id is made up based on the computer you're on but that can't mean that everytime you switch computers you have to start all over, right? There has to be a way you can switch computers and still play? Help!

you missed the vital backup of myflaccountid.hta located in HD:\Freelancer\EXE, that holds you Online ID number.

So unless you still have access to the pc's that had freelancer on them and its still installed you would have to find that and double click that file to retrieve the number, if lucky then you would have to reinstall freelancer and at the point of going online freelancer prompts you to verify the ID number and or edit it, thats when you change it and thats the only time that popup appears.

Otherwise what you have to do is:

using IFSO get a list of the characters, noting 1. The folder names for each of you characters 2. the Exact spelling and case of each character.

Then with the FLserver closed move all the player folders out of the multiplayer folder to a safe place dont delete them, then restart the server and then each of you creates new characters with the EXACT same spelling and case as before. Once you have done that again note the folders for each of you.

Then with the server closed move only the .FL files from the backup folders into the corresponding folders for each of you, NB 1. Don't move the NAME file 2. if windows doesn't complain of overwritting any of them then it wasn't named properly in the first place and have to start over for that one.

Restart the server and all your charactes should be back.
