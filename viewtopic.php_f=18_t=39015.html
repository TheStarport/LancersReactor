No, the error message clearly refers to the Alchemy effect called GravityField which is contained in an ALE file. Alchemy effects are simply eye candy used in the engine and this error simply means for some reason the effect couldn't be written to memory properly.

Alas, to create a "gravity" effect on objects the only way would probably be to use some kind of Fuse which has been discussed before but they only fire once per session so it would only last for a moment.
