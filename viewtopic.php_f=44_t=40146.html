Got to admit that i'm not a real expert on this, but if you are using textures you need to make sure they are:
64x64, 128x128, 256x256, 512x512, 1024x1024, 2048x2048, 4096x4096

I really wouldn't go that high though. If you are usind DDS, you shoud be able to get away with a texture map of 1024x1024.
I think there was some rule about the TGA being uncompressed too. Might be worth taking a look at?
