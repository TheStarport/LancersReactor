For everyone that hasn´t got the TBP server on the global list:
You don´t have to change the shortcut to direct connect,just go into the EXE folder located in the games folder,look for a file called "fldirectip.hta" and doubleclick it,a menu pops up and you can write an IP adress in it.
TBPs IP adress is 128.186.141.13 type that in and hit the "Add server" button after that hit the "Launch Freelancer" button,now when the game starts up just go to Multiplayer and the first server to show up should be the TBP.
