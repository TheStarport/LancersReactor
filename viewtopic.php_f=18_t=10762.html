Archanon

Wed Mar 19, 2003 8:40 pm by Archanon

well gunboats are easy... have to use rheinland military or nomad faction though... They are the only one with gunboats...

edit area_gunboats.ini:
[EncounterFormation
ship_by_class = 1, 1, sc_gunboats
pilot_job = assault_leader_job
make_class = wanderer
formation_by_class = gunboats
behavior = wander
arrival = all, -object_all
allow_simultaneous_creation = yes
zone_creation_distance = 0
times_to_create = infinite

[Creation
permutation = 0, 3

add this line to your "system_name".ini:

[EncounterParameters
nickname = area_gunboats
filename = missions\encounters\area_gunboats.ini

and add this info to one of your zones:
encounter = area_gunboats, 2, 1.000000
faction = rh_n_grp, 0.600000
