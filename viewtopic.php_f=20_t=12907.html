Nog

Sat Apr 05, 2003 1:20 am by Nog

There is a ini file that probably needs to be deleted. (Freelancer is trying to initilize the new card as if it were the old) Record your multiplayer number and uninstall/reeinstall the game. Ensure the entire game directory has been removed.

Please note that I am only giving you a canned solution for a problem that happens with many games when video cards are upgraded. Basically, the card is no longer the one that the games ini files says it is, but the game doesn't think to test the card again. It just tries to boot as if it were the same card, hense the crash. Usually deleting the file that has the card specs on it is a fix, but in this case that file contains so much other info that just removing it may not work.
It may not even be the right file. But if you wish, try removing the file flconfigdatabase.txt and then restarting the game. you can also try the same with Freelancer.ini. They both contain video data but one looks like just a user info file (the TXT) and the other seems just to load up default data and contains the master server list. (the ini).
