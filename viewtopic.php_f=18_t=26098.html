the_seeker

Tue Nov 18, 2003 3:25 am by the_seeker

1. Model Ship
2. Texture ship using a TGA with material name the same as the file name
3. Add triangles to represent hardpoints for engines (hp/fixed/hp_engine01), thrusters (hp/fixed/hp_thruster01), mount (hp/fixed/hp_mount), tractor source point (hp/fixed/hp_tractor_source) and weapons (hp/fixed/hp_weapon01 or hp/revolute/hp_turret01).
4. Press the export button
