Sat Jul 28, 2007 4:27 pm by Ogu

There's no need for that. In my experience, you can have a mod stay at the same version forever - you just change the other numbers with Reshacker (such as the build, etc).

Mod 1:
FILEVERSION 4,0,1254,6
PRODUCTVERSION 4,0,1254,6

Mod 1, but with a patch:
FILEVERSION 4,0,1254,8
PRODUCTVERSION 4,0,1254,8

When Asgard did this, no one with build 6 could connect to the server if we were running build 8, even though at the lower right of the main menu and on the Select Server screen, they each showed 4.0. So I don't think we'll ever "run out of version numbers" (not to mention some mods that had version numbers don't even exist anymore)...And please don't tell me this doesn't work unless you've done it yourself.
