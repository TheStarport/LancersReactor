Can''t play the demo-native resolution problem

This really is more of a gripe, so I'll apologize in advance, but I was unable to play the demo for what I consider a pretty flaky design decision.
You don't need to know my system specs, sufficed to say my pc is hooked to an old 36" gateway Destination CRT monitor, the max resolution of which is 800x600. Fine so far right? I run my desktop in windows at 800x600, and I have NEVER installed a game which defaulted to a higher resolution than my desktop, this one did. Ever tried to change display resolutions blindly while clenching your teeth and staring at 4" tall letters that scream "OUT OF RANGE" at you?
The readme helpfully suggests that performance issues may be addressed by turning the game down to 800x600, but as far as I can tell this can only be accomplished within the game.
Thank god Microsoft doesn't believe in saving all your config files in proprietary file formats or anything...oh crap.
I can't find a cfg file to make the game work, maybe somone knows where it is.
I can't see what excuse Microsoft has for installing the game at a higher native resolution than the operating system THAT THEY MADE is set to. I realize that this will be very isolated problem, but it seems to be one that never needed to be. The game only runs in 4x3 aspect ratio, I understand, but why on earth does it not then default to lowest possible 4x3 resolution, especially considering other peoples troubles with stuttering frame rates, isnt it ALWAYS better to ship something that will work and let people turn options UP?
