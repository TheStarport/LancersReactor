hey bud,

teh best way to do so is this...

copy the entry that you want...for instance...the starflier

[Power
nickname = ge_fighter_power01
ids_name = 263705
ids_info = 264705
volume = 0.000000
mass = 10
DA_archetype = equipment\models\hardware\li_fusion_reactor.3db
material_library = equipment\models\hardware.mat
capacity = 1000
charge_rate = 95
thrust_capacity = 1000
thrust_charge_rate = 100
lootable = false


then just paste the copy at the bottom of the .ini file like so...

[Power
nickname = ge_fighter_power02 <------change the 01 to 02 (just an example)
ids_name = 263705
ids_info = 264705
volume = 0.000000
mass = 10
DA_archetype = equipment\models\hardware\li_fusion_reactor.3db
material_library = equipment\models\hardware.mat
capacity = 3000 <------ change Capasity
charge_rate = 190 <------- change Charge
thrust_capacity = 4000 <----- Change Thrust Capasity
thrust_charge_rate = 320 <------ Change Charge Rate
lootable = true <----make this available so it can be looted and sold


the go to the "Misc_Good.ini" and make this power plant "sellable"
and add it to a base...buy the sucker and fly away happy

there is a LONGER way to do this....make ALL power plants sellable so you can sell the generator you currently have and buy a new replacement one
