Black Eagle

Sat Apr 01, 2006 12:45 am by Black Eagle

Hi!

Be sure that you have SDK 1.3 installed. If you have the newest version 1.5, the mod will not properly activate, as the scripts are made fro the SDK version 1.3.

So, download the SDK 1.3, reinstall FL and then install SDK 1.3 and activate the mod. This should work then.
If however, you want to use SDK 1.5 by all means you have to correct several things in the scripts to fit it to this version of SDK. That's what I made and it works also, although it is some work.

Hope this helps!

Black Eagle
