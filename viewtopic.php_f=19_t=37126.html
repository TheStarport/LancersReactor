If you have UK ADSL then your upload will be 256k or if using Cable then could be 400k, so at best you can host 20 to 24 players.

As for using the pc and running a server, it won't happen as FLserver and any admin utilities need all the ram and cpu power you have and that is for a standard server not modded. Mods add extra Server load and so the player limit has to drop.

So unless you have another pc to host the server I wouldn't recommend you USE the pc that is the FL server.
