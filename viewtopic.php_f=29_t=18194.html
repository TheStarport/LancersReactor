1 - See if you are using the FLMM. In that case, all you have to do is turn off all the mods. Note: FLMM has an option in it to protect files from editing or deleting. Once more, deactivate it.
2 - The attributes of some .ini files was changed to 'read only'. To check it, open properties window (right click with your mouse over the file that you want)uncheck the 'read only' option.
3 - You have the same file in use with other application. Just 'close' that .ini file opened by that program.
