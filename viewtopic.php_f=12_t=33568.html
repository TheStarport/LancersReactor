Teamspeak vs Ventrilo

2 major chat proggys, both very popular, but which one is actually better?

I did a little comparison, and here are the results:

- Codec (Teamspeak):
Teamspeak has more codec options, which enables the admin (and user) to get a better channel codec. Users can chance the codec on their own created channels.

- Codec (Ventrilo):
Ventrilo has also several Codec options, but users (the client) can't change the codec if needed.

----------------------------------

- User Friendliness (Teamspeak)
Teamspeak is quite user friendy, it is easily accessable (gah, sp?) and little changes can easily be made. A few simple clicks and you can create your own channel, give it a topic, name, password, codec, etc. Advanced user settings can easily get confusing. Connecting to a server is simple.

- User Friendliness (Ventrilo)
Ventrilo is not as user friendly for the beginner, but advanced users can easily use it. Connecting to a server can be made easy, but if you want to connect to a new server, you have to go through multiple tasks, which can be very confusing. Clients cannot create their own channels.

----------------------------------

- Compatibilaty (Teamspeak)
Older versions of Teamspeak can connect to newer servers, making it very easy.

- Compatibilaty (Ventrilo):
Older versions cannot connect to newer versions of Ventrilo. This can lead to you having several different versions of Ventrilo on yuor PC at one time to connect to your favourite servers.

----------------------------------
{Note: This section will not be counted in the final overview}
- Misc (Teamspeak):
Funny voice messages when connecting, changing, disconnecting to/from Teamspeak. Funny extra if you have SA rights: If you want to ban yourself, funny messages like "It's not smart to ban yourself" come up.

- Misc (Ventrilo):
None known.

----------------------------------

From this overview, I can say that Teamspeak is easier to use and it keeps it simple as Server Admin. Users can get their own privacy if needed.
*Good for those who like it simple and just want to talk*

Ventrilo is a bit more complex, and is not quite as good. It is not quite as user friendly and can be very confusing at times. Advanced options are easier to control though.
*Good for those who like to chat a bit more advanced*

----------------------------------

BP: Teamspeak has more options
