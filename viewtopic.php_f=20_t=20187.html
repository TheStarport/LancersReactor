Mon Jun 30, 2003 7:31 pm by logic

Those of you that are freezing up, make sure your systems are clocked down, make sure your registry files are optimized and clean. Plus make sure all of your background programs are not running. I am running an Asus 865, 2.4 gig drive, hyperthreading off, Geforce4TI 4200 running open GL only D3D is off in the DXDIAG. Mine hasnt crashed yet. Still going on. Now if it works and your systems dont crash that will save the fixers alot of time. Remember that applications that are running in the background also use memory addresses, and when one application hits the same memorly address another application is using, it will put your system into a hardlock. So all in all, try turning off overclocking and make sure your system has a clean and optimized registry. Also make sure the drive is properly defragged. Again I have had no lockups. Hopefully that will fix the freezing and lockup problem so atleast one of the issues are resolved. If it doesnt work, Well I tried..
