Pinger

Wed May 28, 2003 3:55 pm by Pinger

If you cant get there ping, then probablly there not active/crashed but still in the master server cache.

If you want to add them to LAN then just make a copy of your desktop icon and add the server ip and port number in the target line.

eg Elite would be

HDD:\GAMES\Freelancer\EXE\Freelancer.exe -s131.107.83.10:992

NB you have to have its correct Query port or it wont work. The default is 2302 but some use custom query ports.
