HOW FAR CAN MODDING TAKE US?

Before I start, I know I am going to sound "way out there" to most of you.
Likely that's because I don't have a clue how far "modding" can take us.
Here's the situation;

Can modding with FreeLancer go so far as to allow us to create a player-player interactive station or world? Will the tools allow this, or can we get tools that will?

I know that Star Wars Galaxy will be bringing this to us soon on a pay server.
I appreciate the value of a player run free server.
I have several good programming freinds who might be interested if its possible to create a player world within Freelancer.

So the question remains, can we only make minor changes, or can we modd as deeply as we want? Where are the limits?
