Hi!

I guess you mean the effect, displayed during you jump to another system. It is specified in two files:

Data/FX/gate_tunnel.ini and Data/FX/jumpeffect.ini

You can change the color of this effect and also the time, how long this tunnel is displayed. However, you cannot (except you are able to edit ale files, which noone was able to edit until now) change the entire effect and create a new one. Besides you cannot simply add this effect to any other object, as it will only work on a jumphole or on a jumpgate. This effect is called by some triggers, which are only working with jumpholes/gates and therefore it wouldn't be possible to add them to a weapon to creata a jumpweapon or so. I t would simply not work.

However, changing colors and so is possible and looks quite nice.

Hope this helps!

Black Eagle
