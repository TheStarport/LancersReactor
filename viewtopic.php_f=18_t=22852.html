Project Privateer

This message is to test the waters for one of two new mods for Freelancer. Project Privateer is intended to create a remake of Orgin's Wing Commander: Privateer using the freelancer engine. This project is being directed under the auspices of Darkside Industries. Darkside Industries is a new company formed by various software techs from a variety of companies. In an effort to gain both fundiing ans industry recognition DSI has secured permission to create a number of OFFICIAL expansions and MODs for freelancer. What this means to you editors out there is that if there is enough interest and involvement the Microsoft will PURCHASE the MOD to use as an official release with all of your names attached. Please post with ideas for the project or your willingness to help

DSI Industries

What's New Pussycat:
* More than 20 new ships: Broadsword, Caernaven, Centurion, Concordia, Demon, Dralthi, Exeter, Ferret, Free Trader, Gladius, Gothri, Hellcat, Hornet, Jalthi, Paradigm, Rapier, Raptor, Sabre, Scimitar, Stilleto, Talon, Thunderbolt, and Vindicator.
* Faction and system names changed to reflect Wing Commander.
* All weapons from Privateer, and a few other favorites.
* Variable power and armor levels.
* Guns, afterburners and shields now draw power.
* Missiles now take up space in the hull. Tradeoffs will have to be made bwtween ammo and cargo.
* Difficulty level retooled, for what should be a smoother curve.
* Mission payment redone - climbs steeper, caps at 44,000.
* Weapon stats - all guns are faster and have shorter ranges.
* Speed for all fighters and cargo ships multipled by 3.75.
* Missile ammo now takes up space in the cargo hold.
* Certain ships are capable of carrying only certain weapons, shields and power - see the readordie file for more info.
* Cruise speed doubled to 600.

Installation:
Installation should be fairly straightforward - you can extract it to your root directory, then move the Freelancer folder into C:\Program Files\Microsoft Games (or whatever the folder above your FL directory is). The mod is designed to work with FL Mod Manager, which, if you don't have it, is available at http://www.lancersreactor.com/t/downloa ... asp?id=143 (download OpenSP as well, if it doesn't already come with it.. Once you've installed it, drag the Freelancer Mod Manager folder to C:\Program Files.
