Daekar

Sun Apr 06, 2003 7:14 pm by Daekar

If you look in DATA\COCKPITS\ you'll see folders for the different types of cockpits. If you go further in, you'll see some *.ini files that define the *.cmp files for the game as well as set some camera data. In the "models" folder there should be the *.cmp file for the cockpit. This is the same format as the ship hull files and has a reference(s) to a *.3db file in it, and so theoretically a new cockpit could be made in the same fashion as a ship hull.
