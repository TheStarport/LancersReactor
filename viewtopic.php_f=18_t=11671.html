Sat Sep 06, 2003 2:55 pm by Ijuin

The autopilot seems to prefer to use jump gates IF there exists both a route to your destination that uses gates and one that uses holes. In other words:

Holes only route = Use holes
Gates only route = Use gates
Gates and Holes routes = Use gates

I suspect that this was done to reduce the processing load that would have been needed if the program had needed to compare the multiple routes in order to determine which one was the shortest.
