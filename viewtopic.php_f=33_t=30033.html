Well, I'm not absolutely cetain that this will work, but you might give it a try if no one else responds with a better idea.

But first, let me ask you this: you said you have the Evolutions Mod? If you do, you cannot have both Evolutions AND Rebalance Mods activated at the same time. I think this will cause problems, so make sure you DON'T have both activated at the same time.
If you don't have both mods activated at the same time, then try this:
Deactivate all of your mods in the Mod Manager. Uninstall FLMManager. Uninstall thr Freelancer game itself. Now do a fresh install of Freelancer, then install the Mod Manager, and finally install your mods again. Sometimes the game gets a little wacky when you download new mods, so a fresh install of the game sometimes works.
I know it stinks to have to start over (especially from level 40 ), but it might fix the problem.

Ro9ue

Mon Apr 05, 2004 4:47 pm by Ro9ue

Any mod in FLMManager that is colored red means that it conflicts with your activated mod(s). Most of the time, this would mean that files might overlap and there WOULD be problems in the game. For example, if you had the Rebalance v3.4 activated, the Evolutions mod would be red. You shouldn't activate both at the same time because the game would crash and lock up.

~*~ However, in this case, the Upgrade 1 is like a patch for Rebablance v3.4; it overwrites a few files in the Mod. It is this that makes the new bases possible, so you WANT it to be red. Even though it will give you a warning about conflict when activating both Rebalance 3.4 and the Upgrade, press "OK" or Yes whenever it warns you.
Like I said, in patches, there will be a conflict because that is what the patch/Upgrade is MEANT to do. Don't worry about the Upgrade being red in this case - it's want you want.
