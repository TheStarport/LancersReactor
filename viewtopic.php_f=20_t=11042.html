YuriEd

Fri Mar 21, 2003 11:47 am by YuriEd

I had the same problem too, although I never had problems with the trial version. After installing the game Norton Disk Doctor showed lost sectors and invalid entries in the FL installation dir. Fixing them didn't help, afterwards I could get past the intro, but the game still crashed nearly every time I talked to NPCs in a bar. After uninstalling the game I defragmented my harddisk and let Disk Doctor check for errors and fix them, then I installed the game again and now it runs fine for about 25 hrs game time.

Though, there are 2 things I noticed: First, on my computer NDD finds invalid entries in the Freelancer dir every time I install the game (either trial or full version), but fixim them is easy and permanent (while the lost sectors reappeared in the crashing FL-installation after restarting the game). Second, in the crashing installation, the .ini where FL stores user options (somewhere in My Documents/My Games/Freelancer, don't remember the name right now) showed me a screen resolution of 1024x768 at 24bpp, while the trial version as well as the working full version run at 32bpp.
