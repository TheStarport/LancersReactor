Buyable bases?

I heard something about this ages ago, is it practical? Does it require either scripting or server admin chnging the files?

Clanwise, all players from the same clan would be allowed access to the base and the defending local forces would attack anybody not friendly with them.

Argh

Sun Sep 11, 2005 11:35 pm by Argh

Um... you can alter which Faction you're aligned with, in MP, when your character is created. I'll bet you could make it so that you could use an XML script to just alter this, and voila... any new character you created was aligned with whatever faction.

Then bases that were "owned" by your chosen Faction would be friendly to you by default, and bases you "bought" wouldn't necessarily be friendly to anybody who hadn't also "bought" that base. And, instead of using Bribes... make players take Missions to improve their relationship. That way, players who'd become friendly with the Faction owning the "private base" wouldn't be friendly with other groups' Bases... because the Base Factions could be set up to mutually hate one another, making being buddies with multiples of them, or Neutral to All very difficult, if not quite impossible... add that to making certain things only able to be bought if Friendliness was above some fairly high level, and you'd virtually ensure that players would have to make meaningful choices at some point. I did something like this with Warriors of the Sky, and it actually worked pretty well- people worked really hard to become friends with, say, the Proletariat of the Air... because they had special aircraft that nobody else got. Etc., etc., etc.
