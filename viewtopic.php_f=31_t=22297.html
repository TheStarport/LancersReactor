1.54C Patching instructions (READ CAREFULLY)

http://ebiia.swishmail.com/fldownload/b ... patchc.zip

Download the patch
Unzip the patch
Select all the files in the patch directory
Press ctrl-c to copy all the files
Goto where FLMM is installed usually C:\program files\FLMM MOD MANAGER
Goto the mods DIR, \FLMM MOD MANAGER\Mods
Goto TNG directoey \FLMM MOD MANAGER\Mods\Tng154full
Mkae sure you files are NOT read only
Press ctrl-v to paste the copied files into the directory
Select yes when it prompts to overwrite files
Run FLMM you should now see, Freelancer: The Next Generation 1.54 Patch C
If you do not you did not install the patch correctly, redo it.
