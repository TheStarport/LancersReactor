Well, there's good news on this front, but in order for this to be effective, people will need to be active.

We have already set up two avenues for communications between FL server admins. One is a mailing list tied directly to this web site. It allows for email communications that are directed only to members. The other is a website set up by one of our long standing FL enthusiasts and it's located here -> Lancer Admins dot Com . If you check that site, you'll see it's not been used in a long time for whatever the reasons (probably because it take some effort on the individual to click on the site to check for messages and if nothing new is posted for day, chaces are you'll forget it exists and not visit anymore.)

Taken both alternative into account, it might be better for realtime communications if the former mailing list is utilized for this sharing of ideas with each other. To get on this mailing list, simply send me an email directly requesting your inclusion to this MP server mailing list, I'll start bugging the poop out of Gibby to add you and then all you'll need to do is send emails to an address and everyone will get the message in their inbox. When you sent the info, be sure to specify what email address you want to have registered if you have more than one to play with. My email addy is freelancer@ameritech.net or pierre@beol.net or p1erre@ameritech.net. This is a great idea, participation is the key.
