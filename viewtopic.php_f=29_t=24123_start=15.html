Captain Tylor

Fri Dec 17, 2004 6:21 pm by Captain Tylor

You seem to have a good understanding of it all. Check with shiparch.ini to make sure the class restrictions are correct. It's possible to go over the class restriction on character creation, but once you try to undock from a base or planet, the 1.1 server patch will kick the player for cheating. The best way is to test it out yourself if you're ever in doubt, before commiting it to your server.

Sounds like you wish to add weapons that go beyond class restrictions of the ship you're using. Instead of using:

equip = li_gun01_mark03, HpWeapon02

try using:

cargo = li_gun01_mark03, HpWeapon02

and you should be fine.
