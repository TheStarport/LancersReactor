wrecked ship - the aggressor

i just stumbled across this in the li01.ini file
its the wrecked ship AGGRESSOR, found searching with "suprise"

[Object
nickname = Li01_suprise_bh_fighter_badlands ;objects name
ids_name = 261709 ;wrecks name, in this case AGGRESSOR
pos = -4870, -1000, 39426 ;where it is in the system
rotate = 6, 38, -61 ;rotated model
Archetype = suprise_bh_fighter ;within the solararch.ini
visit = 16
ids_info = 66458
loadout = SECRET_bh_bh_fighter_li01 ;what they carry, its within the loadouts.ini

from the solararch.ini, found searching with "suprise_bh_fighter "
[Solar
type = MISSION_SATELLITE
nickname = suprise_bh_fighter
LODranges = 0, 80, 150, 1000
DA_archetype = ships\bounty_hunter\bh_fighter\bh_fighter.cmp
material_library = ships\liberty\li_playerships.mat
surface_hit_effects = 0, small_hull_hit
mass = 100000.000000
solar_radius = 800
shape_name = NAV_surpriseX
explosion_arch = explosion_li_elite
hit_pts = 3600
destructible = true
fuse = fuse_suprise_bh_fighter, 0.000000, 3601
fuse = fuse_suprise_drop_loot, 0.000000, 3590

[CollisionGroup
obj = bh_topfin_lod1
dmg_hp = Dptopfin
dmg_obj = bh_fighter_dmg_topfin_cap
separable = parent_impulse = 0
child_impulse = 0
debris_type = debris_small_ship
hit_pts = 1800

[CollisionGroup
obj = bh_btmfin_lod1
dmg_hp = DpBtmfin
dmg_obj = bh_fighter_dmg_btmfin_cap
separable = parent_impulse = 0
child_impulse = 0
debris_type = debris_small_ship
hit_pts = 1800

from the loadouts.ini ,found searching with "SECRET_bh_bh_fighter_li01"

[Loadout
nickname = SECRET_bh_bh_fighter_li01
archetype = bh_fighter
equip = gd_bh_gun01_mark01, HpWeapon01 ;list of equipment easy to change
equip = gd_bh_gun01_mark01, HpWeapon02
equip = gd_bh_gun01_mark01, HpWeapon03
equip = gd_bh_gun01_mark01, HpWeapon04
equip = gd_bh_turret01_mark01, HpTurret01
equip = ge_s_cm_01, HpCM01
cargo = ge_s_cm_01_ammo, 20

if i had time i'd search out the rest, but looking at what i've just found it seems quite easy to make your own, or at least change an existing one to suit your needs. from the few sytems i looked at most seem to have 3 wrecks within them.but, i havent time to look at them all, i'm to busy figuring out zones atm, i'll write a basic tutorial on them if anyone is interested???
