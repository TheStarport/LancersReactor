First guess would be you're behind more firewalls than you can configure (aka, yer service provider (college?) is blocking the needed ports).

2nd guess is you don't know how to correctly configure your router (assuming it contains a firewall). Don't mean this as an insult, just that I've seen too many people who said they were "doing it right" when in fact they weren't. I'm just stating it as a possible cause. Personally, I freed up ports 1 to 9999 when I started, and am narrowing the list as I find which programs need what ports : )

3rd: My only problem with blocked ports showed itself when I actually tried to log onto the game server (connection timed out), I've never had a problem reaching the global server unless it was down .

Those are the only 3 real issues I can think of, honestly. Hope someone else has some more ideas...
