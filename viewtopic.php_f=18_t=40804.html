New Player lootable thread..

Okay, I edited TATE's excellent compilation that made use of Wasabe's instruction in an earlier thread and came up with a slightly modified version of this mod.

Right now I have only freighters as being lootable. If a freighter dies the death message is not displayed but if a fighter dies the message displays fine. So, it appears that just adding the fuse line to the ship disables the death message. Someone described it to me as having one of two choices. A ship can be either looted or killed, but not both. Therefore, if a ship is looted it did not die...

Does that help any?

Now, here's another quirk. If you start out with a freighter your ship is eternally lootable even if you buy a fighter. If you start out with a fighter though, you are not lootable until you switch to a freighter. I have not yet tested to see what would happen if you started out with a fighter, switched to a freighter, and then went back to a fighter.

Does anyone have any ideas to solve these two problems?

1). Death message not displayed on lootable players

2). Players that switch to a fighter from a freighter are still lootable

Any input greatly appreciated.
