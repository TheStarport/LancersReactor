New GUI system editor

Hello all from Wildman Productions

Ever since I started the team Wildman Productions we've only been focusing on support for 1 game, Populous: The Beginning. But now thats starting to change, I've been playing Freelancer for quite a while and now I have proposed that we start giving support for it.

So, as a first project torwards Freelancer we will be working on a new, user friendly, fully-functional system editor for the game. If time and programming leeway permits this may turn into a 3d system editor that shows the system as it would actually appear in the game - while you are editing it.
