Keep an eye out for this virus...

Usually I wouldn't post a topic about a new virus because a new one is detected every few minutes, but this one is a bit tricky. Essentially, the "Beagle" virus's 10th variant is actually an encrypted ZIP file, with the main body of the email containing a password to the archive. Because the virus is encrypted within the ZIP file, it will not be detected by some virus scanners. The virus (technically it is a worm), also uses addresses similar to "support@lancersreactor.com", because it copies the domain name (in this case "lancersreactor.com" ) of the infected computer, and uses names such as "administrator", "staff", and "support". So if you receive emails like this, watch out. As I always say; DON'T download attachments!
