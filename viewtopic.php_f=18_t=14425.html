Open SP v1.1 Mod Tweaking Tricks

You can pilot all the Freelancer fighters
including Juni's Defender without spoiler but with Top Big FIn
Ozu's Dragon Heavy Fighter with Red Color
All of the SP Character's Fighters incl The Junker CSV Ship

For example:

1. Open the M13.ini
2. Find this line:

Act_SetShipAndLoadout = ge_fighter, msn_playerloadout

3. Replace the ge_fighter, msn_playerloadout with these:


Act_SetShipAndLoadout = ku_dragon, MSN09_Ozu

"Ozu's Dragon Heavy Fighter but Red in Color"


Act_SetShipAndLoadout = ge_fighter4, MSN07_Tobias

"Tobias' fighter"

Act_SetShipAndLoadout = li_elite2, MSN11_Juni

"Juni's Defender Heavy Fighter with Top Fin instead of a Spoiler"


YOU CAN DO YOUR OWN LOADOUT WITH THE SAME FIGHTER. For Example:

1. Open loadout.ini
Look at this:

[Loadout
nickname = msn_playerloadout_def2
archetype = li_elite2
equip = ge_re_engine_01
equip = shield03_mark10_hf, HpShield01
equip = infinite_power
equip = ge_s_scanner_02
equip = ge_s_tractor_01
equip = ge_s_thruster_04, HpThruster01
equip = armor_scale_12
equip = special_nomad_gun02, HpWeapon01
equip = special_nomad_gun02, HpWeapon02
equip = special_nomad_gun02, HpWeapon03
equip = special_nomad_gun02, HpWeapon04
equip = special_nomad_gun02, HpWeapon05
equip = special_nomad_gun02, HpTurret01
equip = mine01_mark05, HpMine01
cargo = mine01_mark05_ammo, 50
equip = torpedo01_mark02, HpTorpedo01
cargo = torpedo01_mark02_ammo, 50
equip = ge_s_cm_03, HpCM01
cargo = ge_s_cm_03_ammo, 50
equip = LargeWhiteSpecial, HpHeadlight
equip = SlowSmallBlue, HpRunningLight01
equip = SlowSmallBlue, HpRunningLight02
equip = SlowSmallBlue, HpRunningLight03
equip = SlowSmallBlue, HpRunningLight04
equip = contrail01, HpContrail01
equip = contrail01, HpContrail02
equip = DockingLightRedSmall, HpDockLight01
equip = DockingLightRedSmall, HpDockLight02
cargo = ge_s_battery_01, 30
cargo = ge_s_repair_01, 30

and then
2. Apply the archetype and the nickname in the m13.ini

Act_SetShipAndLoadout = li_elite2, msn_playerloadout_def2

"Juni's Fighter with my own loadout configuration"


THAT's ALL, HAVE FUN TWEAKING OpenSP v1.1 Mod
