Yes, NPC's can use torpedos, but you have to tell them to. Go into pilotspopulation.ini file, and for all the npc pilots's that you want to use torpedos, change the lines like this:

attack_preference = fighter, 5000, guns / guided / unguided

To this:

attack_preference = fighter, 5000, guns / guided / unguided / torpedo

Basically add the "/ torpedo" to the end of the line. Nice and simple eh?


EDIT: After I posted, I realized that Lancer's Reactor doesn't handle the straight up and down line, so all the lines are changed to forward slashes. Don't use forward slashes, but instead use the line (shift \ on your keyboard).
