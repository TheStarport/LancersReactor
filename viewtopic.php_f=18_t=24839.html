Model File Formats - Basic Information Request

This is a call out to those who have technical knowledge of the various file formats used with models in Freelancer ie .cmp, .sur, .3db etc. A friend of mine is interested in developing a plugin for 3dsmax that will automatically generate all the output necessary to have a working model for Freelancer(minus ini/infocards). In order for this to happen I need all the available information necessary on the file formats used.

Now we don't want a book since much of this information is available in other threads but I think it would not only be useful for my crew but for other people as well to have one thread with all the basic information in it. So basically what I am looking for is a list first of all the file formats used for models in the game, not just ship models but all models including this curious missile model I have heard about. Next I want a basic explanation what information is stored in each particular format and then if possible some links to any threads/sites with technical information regarding that format.

This is a lot to ask and I know the typical response is to search for it on the forums but I ask that you would humor me in this request since I am sure this particular project would be very useful for the entire community when completed. We look forward to providing the Freelancer mod community with an easy to use high quality plugin for 3dsmax.

***********

.cmp - utf format, contains vmesh/lod/hardpoint data - cmp exporter source - cmp/3db importer for ms3d
.3db - utf format, simplified .cmp file which can contain textures, also references .cmp vmesh data, also used for ship/item icons
.sur - bounding box/surface detection for shields and models
.mat - utf format, contains tga textures in DDS format
