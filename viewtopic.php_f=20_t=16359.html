I'm not sure if 2k is the same but in XP there is a "feature" called Prefetch which is supposed to make regularly used programs load faster.

See if you have a folder C:\Windows\Prefetch (Or whatever your windows folder is) and if you have, Delete everything in it.

Chris Carter

Sat May 17, 2003 3:20 pm by Chris Carter

Have a look in the system temp folder (default will be c:\windows\temp\) for a file named something like '~f1d055.tmp', there may also be a folder with a similar name - delete/move these files and folders.

You'll find you can now restart FL - well, it worked for me anyway.
