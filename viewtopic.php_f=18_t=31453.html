The cruise desruper is about as close as you could come. It is possible to add the effect to a mine and add a tracking effect, as I think this has been done.
add this to a mine.
HP_trail_parent = HPExhaust
seeker = LOCK
time_to_lock = 0
seeker_range = 2500
seeker_fov_deg = 90
max_angular_velocity = 8.710000
cruise_disruptor = true

Chips

Mon May 17, 2004 12:52 am by Chips

Think he is referring to the energy_damage line in files - which will erase the "shield" of ships, but if large enough, also takes down energy.

Example: - still on the above cruise disruptor, but can do it for any explosion weapon:


[Explosion
nickname = cruise_disruptor01_mark01_explosion
effect = li_cruisedis01_impact
lifetime = 0.000000, 0.000000
process = disappear
strength = 100
radius = 50
hull_damage = 37.400002
energy_damage = 0
impulse = 0

Those lines there - edit:
radius = 1000
hull_damage = 0
energy_damage = 1000000

Those stats will literally blow out all shields, and possibly power plants within a 1km radius (radius in meters). lifetime can be fiddled with in the lifetime line above this, but i don't see the point in making explosions of weapons last for ever, only ships
