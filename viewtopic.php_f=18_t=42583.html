Hi!

First, for having the cruisespeed showing right, you need to edit the exe. However, as the normal exe is not editable, you need to get the cracked exe and edit this one.

So, if you have the cracked exe, you can open it up with an Hex Editor and do the following:

- Select "GoTo" and go to the following offset: D5936
- At this location, you will find two bytes, 0F and 85. Change them to 90 and E9.
- Safe the edited exe

This should make your cruisespeed show up right.

And if you want to have the tradelane speed show up right as well, do this:

- goto offset: D5984
- there you will find the byte 7E. Change this to EB.
- safe the file

Hope this helps!

Black Eagle
