1) The ids_names on the map are:
46001 LIBERTY
46002 BRETONIA
46003 KUSARI
46004 RHEINLAND
46037 SIRIUS SECTOR
All in the Resources.DLL

2) The Nebulae seen in the bars are in the bar's CMP file.
Unless you can add it in, you're outa luck.

3)There is a program called "Hash-Generator-FE" that requires some doing to set up, but will give you the codes.
And then...
Edit "InitialWorld.ini" in the Data folder.
