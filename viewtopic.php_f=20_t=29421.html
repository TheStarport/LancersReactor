Ok, thanks for all your help...

I found a way to work around this issue as stated in a Microsoft help thing...

To work around this problem, disable the movies in the Freelancer.ini file. To disable the movies, follow these steps:
Locate the Freelancer.ini file in the following folder:drive:\Program Files\Microsoft Games\Freelancer\EXE
Open the Freelancer.ini file.
Edit the [Startup section. Type a semicolon ( in front of the three lines that start with movie_file = --for example:
[Startup
;movie_file = movies\MGS_Logo_Final.wmv
;movie_file = movies\DA_Logo_Final.wmv
;movie_file = movies\FL_Intro.wmv
