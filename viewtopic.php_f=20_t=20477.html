Just Some Guy

Sat Jul 12, 2003 10:03 am by Just Some Guy

In the developers discussion they note how they made the game with the particular desire NOT to have it played with a joystick. They thought that would somehow limit the players who would use it if they included that option, because it would inhibit the way or proscribe the code used to interface with the game as it is now. So... they made it this way to avoid a joystick.
I like joysticks for flying things, and yet haven't really missed it that much in this game. Joystick/throttle combo's like the Saitek have a lot of buttons and can be programmed and would have been neat to use, but I have to attach them to the arms of my chair, and then they get in the way of normal activities on the computer, so this is OK too.
