** Modding For Newbies **

Installing mods, this is how to install mods

If the mod is from a zip file (uses winzip, not flmm(freelancer Mod Manager)) all you do is fallow the instructions in the readme. If the instructions are not good and there are .ini files left behind and you don't know were to put them, and they are not in the ReadMe then don't worry.

If the mod is a FLMM mod (uses Freelancer Mod Manager) then click on the file you downloaded, but first make sure you get it at http://www.lancersreactor.com/t/downloa ... asp?id=143

After you are done Downloading it then install it. Then you can click on the .FLMM files and they will automatically be installed into Freelancer Mod Manager.

Freelancer Mod Manager is good but still has it&#8217;s flaws the delete autosave options does not work on my computer may on others, but if it does not fallow the Fixings down below. And that should stop most the crashing&#8217;s if not all.



Fixings


When you install a new mod, referring to newbie&#8217;s, and the game crashes after the end of the three intro movies (going from game the desktop). Then do not reinstall the game this will continue to happen if you do that. You need to manual fix it on your computer desktop go into:
My Documents/My Games/Freelancer/Acct/MultiPlayer/(the only folder there, it varies from computer to computer)

When you are there should be a few files a name.file and a few (Numbered).Fl files the FL files all you do is delete them and this should work, if you want to keep them just make a new folder and put them in that folder.

Now for the single player, all you do is go back to Acct/Singleplayer

That is all you put them in a new folder or delete them then this should stop the crashing.

You NEED TO DO THIS FOR EVERY NEW MOD INSTALLED!!!!!

If you got any questions then post it here and I'll try to get back as soon as I can or some one else will help you.
