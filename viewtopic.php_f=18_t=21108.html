I reckon finding decent pictures of it will be the hardest part It is surprisingly difficult to get pictures of ships, even video game ones, that allow you to construct an accurate model.

Anyway once you have the pcitures you will need a bare minimum kit of :

Milkshape3D - Even if you construct the model with something else you need to have Milshape to export it to FL's .cmp format
Milkshape -> .cmp Exporter - Obvious
Paint Shop Pro - For making textures
FLUTF editor - For adding your textures to the game. Also to manually tweak gun/turret Harpoint stats (rotation limits etc.)
BiniQDU - For decompressing/editing .ini files to add your ship
FLIdReferer - For Working out infocard reference numbers
Reshacker - For adding your ships infocards (name, gun/turret mounts etc.) to a .dll for use by the game

There are other things that will make you life alot easier. FLEdIds' latest incarnation will allow you to type an infocard and will add it and work out the IDS_ numbers for you. Any other 3D editing program will be better than Milkshape Trouble is you then have the hassle to correctly importing it into Milkshape.

There are a few great totorials kicking round. Zens ship adding tutorial is great and has good screenshots and code examples - although Zens website doesnt seem to work anymore someone has posted a link to their own mirror of it. **Tutorial** Lightwave 3d -> Milkshape also has alot of useful information. HCls Milshape .cmp exporter has a good example ship with it and a readme. For basic construction tips the milkshape help files are surprisingly good. Root around in the tutorials forum and see what you find.

Finally what about altering the name a wee bit Fluttering Petal sounds a bit like the villan in a 'blue' movie to me...
