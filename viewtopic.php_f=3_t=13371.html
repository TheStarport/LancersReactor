That's right! You've discovered mining! An excellent part of the game. Some asteroid and debris fields are "mineable". If you click on the mining icon in your nav map, it will highlight those areas that are mineable. Some fields produce better commodities than others, and some produce more commodities per asteroid/debris than others. Takes some exploring.

But basically, yes, this is a fully intended feature of the game and provides a whole new financial path to explore. Have fun!
