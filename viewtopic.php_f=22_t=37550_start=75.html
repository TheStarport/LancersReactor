Prolifik,

My server is set up to automatically start the FLserve after a crash or reboot. Here is how I did it:

1st I use IonCross Server operator, so I set the PC up to run only one account and auto log in on boot up. Then set IFSO, My remote Desktop server, and Tiny upload (My FLstat uploader program) to start up with the puter (Put it in the Start>All programs>Start Up folder).

Reboot and viola!

The only hang I have is the password for the uploader to access my website keeps dropping all but the first three characters, but that is ok as I check the thing regularly.

Anything you want to automatically start up is easily done by dropping it in that folder from your start menu.

If there is a better way to do this, someone feel free to chime in. Im no expert I am just relating what I did.
