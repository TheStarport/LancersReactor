If you want to use Gmax to design things for freelancer you have to go to their home page (www.discreet.com) and download the tool called tempest, it was used for Quake 3 but it can export to .MAP files which milkshape can then import but I don't like the interface of tempest so I do this:

Open original Gmax and make your model then save your model
Next open tempest and open your file
Then export it as a .MAP file
When you have done this open milkshape and import your file

Hope this helped =-)
