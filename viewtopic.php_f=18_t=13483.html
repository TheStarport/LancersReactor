This is an interesting idea, but the sheer amount of work involved makes this unlikely. You're not talking about a mod, this would be an expansion. The code would have to be rewritten to allow player interaction, the server would have to also store an entirely new sort of information.

In my opinion, it;s just not going to happen. The actual designers might do this, it would make a good selling point for an expansion, but the mod community just can't handle this kind of project, not without source code and design tools used by the developers.
