There's an limit server option on near the bottom of the screen when you're at the server logon screen that you can click and there should be a checkbox for "limit to the servers I've visited" or something like that. Secondly you could click the VISITED heading on the top of the server listing and that should be able to list the servers that you've visited, which would help cut down on the amount of time you spend looking for a particular server if you've already been to it.
