Captain Flint

Mon May 12, 2003 5:13 pm by Captain Flint

The answer can be found in solararch.ini. There you see the example of the original Li01_03 (l_dreadnought) as well as the original solar for the r_battleship. You will need to add a new solar for the prision ship before it can be used as a station...

[Solar
nickname = g_prison
type = STATION
DA_archetype = SHIPS\UTILITY\PRISON_LINER\prison_liner.cmp
material_library = ships\utility\utility_liner.mat
material_library = fx\envmapbasic.mat
envmap_material = envmapbasic
LODranges = 0, 600, 800, 1200, 2000, 3000, 6000, 20000
surface_hit_effects = 0, small_hull_hit
mass = 100000.000000
loadout = MSN01b_Prison_Ship
docking_sphere = berth, HpDockMountA, 10, Sc_open baydoor
docking_sphere = moor_medium, HpDockMountB, 5.000000
docking_sphere = moor_large, HpDockMountC, 5.000000
docking_camera = 0
solar_radius = 400
shape_name = NAV_smallstation
hit_pts = 999999961690316250000000000000000000.000000

Note that the loadout is located in ships\loadout_utility.ini. You can then use g_prison as a solar object in your system ini.
