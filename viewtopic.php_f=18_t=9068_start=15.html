About coordinates : it uses a x,y,z layout. 0,0 is the center of the map, positive x values will be eastward, positive y values will be southward.
The best way to place an object is to calculate relative coordinates. I did that for a jump gate. Aligned with Newark Station on the x axis and with Trenton Outpost on the y axis.
I'm not perfectly sure of this but it seems one unit is about 80 meters in-game. 50 units would make something like 4kms.
I also think I'll make a little program which calculates trade lanes coordinates and angles but I have to see how they work first.
