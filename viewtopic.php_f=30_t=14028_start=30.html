Ok to uncompress the INI files.

1. move bini to your desktop
2. open exlporer and go to the *.ini file you want to decompress
3. make sure you can see the explorer window and bini icon at the same time
4. drag the *.ini file (not the folder) over the bini icon and drop
5. shortly you will see the file *.ini.txt appear in the directory listing
6. rename *.ini to *.ini.bak (back up the orginial compressed file)
7. rename *.ini.txt to *.ini

Ok again one important advice for everyone.
You hafe to look that the lines of the systemcode are clean, I mean if the error for example "Li01_12 " is not a valid name or so appears, you should take care that no space or comment is behind the codelines. If you get that error for ex. "Sun_Li01 ",then go to the line with the nickname of that object. If it is "nickname = Sun_Li01 " then change it into "nickname = Sun_Li01". Do the same if you get an invalid INTEGER (=non float number on pascal) just look that the lines are clean.
