So, for Aravis, to change the color of the menus, just open a file of the above list of files with UTFEditor. You'll see in the most part of it nodes called :

edgecolor.tgaagain4.tga
edgecolor.tgaglowedge5.tga
edgecolor.tgaglowedge8.tga
edgecolor.tgaglowedge10.tga
gradientblue_glowedge10.tga
gradientbluefinal.tga
30.tgaframetexture.tga
backgroundpattern.tga
darkblue.tga

To change the color, juste extract them and open it with Photoshop.
Then, change the Hue to whatever ou want.
In the end, import it in ALL the files listed above.
This is the most hard (and boring part of this tut).
