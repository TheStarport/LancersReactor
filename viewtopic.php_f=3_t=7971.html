The ship that she flies, with a little bit of modding, can be piloted... but after digging through the source, you'd have to create a loadout for the vehicle, as well as package type for it. In the source files, it's referred to as the elite 2 (you can see the model files in the ships/liberty/li_elite2 folder)

Tis a pretty cool ship, but as formentioned, can carry heavier weapons than the elite class ship, but also weighs more and is not as maneuverable as the defender.

But since it wasn't intended to be a piloted ship, there are a lot of visual effects that were removed from the model (ie damage effects to the engines/wings). I'm looking into creating a better mod that would allow that ship to be purchased, at a premium price, since it is a heavier version of the defender. In my opinion though, the anubis is still a more effective heavy fighter as far as the npc ships go. :/
