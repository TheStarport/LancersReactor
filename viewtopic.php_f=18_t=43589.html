Louva-Deus

Fri May 19, 2006 6:18 pm by Louva-Deus

Future versions of QuickFix will include a better solution for server stats and administration than is currently available via FLHook, FLStat, FLShell, FLAC, FLCD, etc.

For players the latest patch/fix/SDK is QuickFix 1.0.

For server ops the latest patch/fix/SDK is QuickFix 1.0 Plus.

For mod makers the latest patch/fix/SDK is Freelancer S.D.K. 1.5 Beta (rel: 03/14/2005).

The unofficial patches by Buck Danny are classified as modifications and are not true patches according to the =EOA= (whether or not you agree with us is your choice). The 1.4 release by Buck Danny is unique because it was released based on a modified version of the Freelancer S.D.K. (version 1.3), but it is still considered to be a mod and not a patch.

Check out the latest beta's of the Freelancer S.D.K. 1.5 and Freelancer Explorer v2.x
