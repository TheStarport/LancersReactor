FIX Connecting to global server forever message

Quick
Re-install DirectX to latest version. Make sure DX is really updating the system! Also Freelancer session have to be allowed with firewall.


Advenced
In most cases your DirectXirectPlay drivers goes dead. Possible after installing other games or software. To check if its really the case do as follow:
1. open Start -> Run
2. type: "dxdiag.exe"
*DirectX diagnostic dialog should open*
dxdiag will ask you to check the drivers with WHQL, choose "yes" (you have to be connected) and it will update your digital signatures if they arent updated YET.
3. choose network tab
4. check to see the OKs under registry and that all files have the same version and not beta.
5.(optional) If you have friend who can assist you. Ask him Press "Test DirectPlay" and so do you.
a. make sure TCP/IP selected
b. ask him "create session" and number of his IP
c. press join session and enter his IP

Finaly 1: If you cannot see the OKs (4) or you failed to connect (5) Your DirectX is corrupted.
Solution:
You have to re-install your DirectX. DX have some annoying feature, if its already see the same version or one above it quits and says DX is already installed or "done". You have to upgrade Directx file in this case. If you already have the most recent version try first remove DirectX and then uses this uninstaller: Click here and install your last version again.
Finaly 2: You have all green but you fail to connect.
Solution:
You probably have firewall that prevents you from using particular port. Please shut it down and try again. If you were successful make sure you have rules in your Firewall to allow Freelancer.exe to communicate and allowed ports (4662, refer to disk manual for more info)
