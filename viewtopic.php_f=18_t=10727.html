Well, I know this place is rather cluttered, but this is answered around here several times.

You have to edit the market_ships.ini... this page should help you.
http://zeus.duwx.com/flsdk/tutorials/shipmarketref.php

Note that a base can sell only THREE shipsat a time. If you add one, you have to remove another. You will need a tool to uncompress the INI files, to be found here:

http://zeus.duwx.com/flsdk/downloads.php

Sorry @ lancersreactor for linking to a foreign site, but I myself had a dead hard time finding the BINI tool around here because it's so well "hidden". =/

Digital_Ronin

Wed Mar 19, 2003 12:07 pm by Digital_Ronin

Added:
each section stands for a base, each line for a ship. gf6_package is the "Eagle" for example. Looking up which ship is which package is rather confusing, unfortunately. Read a bit around here to find out.

marketgood = gf6_package, 30, -1, 0, 0, 1, 1, 1

means a ship is not available. to make it available, change it to

marketgood = gf6_package, 30, -1, 1, 1, 0, 1, 1
