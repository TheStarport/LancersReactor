Hahukum Konn

Sat Apr 07, 2007 9:47 pm by Hahukum Konn

Had FL on a Celeron 2.8 (LGA775) for a while with 512 megs of DDR RAM - with a 24 user limit it wasn't too bad, but a Pentium dual core really hits the spot when it comes to being able to handle 32-48 users and do Ioncross maintenance stuff in the background.

With the Celeron, if I used the "Update Player Files" or "Re-Generate Player List" function in Ioncross I could count on lagging my own server quite handily, but with the Pentium D 945 it barely notices.
