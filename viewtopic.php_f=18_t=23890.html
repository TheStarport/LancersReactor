That should do ya. Just paste this into your Shiparch.ini (which you open with notepad like any other text document). If all you see when you open it is gobbledegook, symbols and such that you don't recognize, then just get BINI. It's a program that you can find on this website, and it decrypts the .ini file so you can read it. Basically, you just drag the .ini file you want to decrypt to the icon for BINI and let it go over it. The rest is automatic, as it creates a new file called thefile.ini.txt where 'thefile' is whatever the name of it was before. Just change the end of it from .ini.txt to just .ini and you're in business. It's really not all as complicated as I just made it sound, and if you need more help, just email me and I'll send you the file.
