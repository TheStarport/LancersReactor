Add the data from the mat file to the cmp file using the utf editor, or exporting
the materials from Milkshape using the mat exporter in update mode.

If the crosshairs are not displayed after adding the materials, try re-exporting
with the mesh a bit further away from the origin (0,0,0) along the z axis.
