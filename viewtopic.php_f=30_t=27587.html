ColinS

Fri Jan 23, 2004 5:09 pm by ColinS

firebase

A ship with multiple components will need the Fix data from an existing ship
importing into the Cmpnd->Cons->Fix node with the Origin x, y & z values changed to 0.00000 or 0.000001. The tutorial i refered to previously is called
"Ships/Weapons w/multiple components" by redeye. See step 2 for details on how to change the origin values using a hex editor. The tutorial was last edited in July last year so you may have to do a search.

The text in each Cmpnd->Part_xxx_lod1->Object name node will need changing to match a name in the fix data. The matching .sur file will also need copying and renaming as described in the tutorial.

That should make the whole ship appear in the game.



ColinS
