Administrative commands are executed from the Flserver.exe program, kick, ban, etc. are all located there.

If you want full server control i would recommend getting IFSLO Mk5. it has a very large array of server commands, most of which allow you do execute in one step, rather than the manual step-by-step proceedure you normally have to go though. the Jolt.co.uk server runs this.

All of the standard commands are in the flserver toolbar, and it has a 'console' mode if you want that, but it dosent have any entereable commands, its more like a status monitor.

Hope that helps
