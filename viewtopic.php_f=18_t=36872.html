I haven't been able to get music to play in the main menu, so I can't say if it actually can be done.

As to the rest...
Have you edited the "Music.ini" file?
That is where music files are given tags, and identified.

Have you added the music anywhere?
If you simply replace an FL music file listed in "Music.ini" it'll work whenever the old music would've worked.
If you are trying to change the background music of a story related cinematic, most won't work because the music is often embedded in the cinematic's audio file.
If you created a new music entry in "Music.ini" you must add:
[Music
space = New_Music
danger = New_Music
battle = New_Music
To the system ini files where you want it to play.
OR...
[Room_Sound
music = New_Music
To the bar, planetscape, deck, trader, shipdealer, or equipment room(s) where you want it to play.

If you've done all of the above, and it still doesn't work:
1) Is your "attenuation =" value set to low in "Music.ini"? (higher minus numbers lower the volume)
2) Check for typos---very common cause.
3) Check that the path to the file is correct.
4) Is the music in WAV format?
5) Play the file to make sure it actually is a good music/audio file.



Watch your 6!
