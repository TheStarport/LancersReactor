Thu Sep 16, 2004 3:51 pm by Pinger

Sounds like a corrupted character or characters. So with FLserver closed backup your player folder to a safe place and then delete each of your .FL files in your player folder from My Documents\My Games\Freelancer\Accts\MultiPlayer.

Next start the server and make a new character if all goes well you will join your server again. If so the character you have been trying to join from the backup is corrupted so you can't use it. So put back the backup folder making sure you delete the new character first and then try each in turn and delete each that causes a crash.
