Fri Apr 04, 2003 9:27 am by Thundercleese

Here's how to make the Battleship Missouri (and consequently all l_dreadnought's) destructable in Single Player. Note that I haven't tried this with Multiplayer.

1. Open UNIVERSE/SYSTEMS/LI01/li01.ini
2. Find the Object with a nickname = Li01_03 (that's the Missouri)
3. Find the Archetype of the Object (Battleship Missouri's is "l_dreadnought"
4. Open SOLAR/solararch.ini
5. Find your Solar archetype (a [Solar with a nickname = l_dreadnought)
6. Set destructible = true
7. Give it a reasonable number of hit_pts (or just a measly 5000 if you want to make sure it worked)
8. Go blow it up

Note that the explosion will suck. The battleship will simply disappear and its turrets will give off nice big fireballs. Consider looking at MSN_d_l_dreadnought for a reasonable explosion_arch, CollisionGroups, and fuses.

If you just wanted to make a single object destructable, you'd need to create a new Solar Archetype for it. Just copy the existing one and nickname it slightly differently.
