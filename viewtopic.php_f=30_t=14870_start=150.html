Sun May 11, 2003 10:13 pm by shsan

sealnova, i have just posted a new version. can you try it?
Since i have no idea what could be wrong in your case, i have fixed other stuff that could cause trouble in some weird cases.

MLEdele, you can add nebula (see field, it's now part of the list of templates : field, nebula and clouds ).
The exclusion zone is buggy tough, i don't add it in the field/nebula file yet some it's not working (you can do it by hand). I'll fix it later.

beowolf, I am preparing a manual for the tool. I hope to have it ready soon.
But for now, here is my advice.
Install the game in default path a <letter>:\Program Files\Microsoft Games\Freelancer ...
install Freelancer Mod Manager in a default path too <letter>:\Program files\Freelancer Mod Manager.
Make sure no mod is activated before launching the explorer.
Launch the explorer and add what you want.
When finished select save a new mod in the menu
It should display the correct path for FLMM.
Click on new folder and type the name of the mod you want to create.
Validate, click ok.
This should produce a mod, with a dll inside and a DATA directory containing all the files you changed/created.
Quit the explorer.
Launch FLMM and activate the mod.
Test it.
Deactivate the mod.
Launch the explorer and select NO at the first question, and Yes at the second
choose your mod and continue the edition....and so on

Hope this help to clarify the procedure
shsan
