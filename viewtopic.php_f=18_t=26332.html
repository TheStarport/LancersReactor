bakedpotato

Sat Nov 22, 2003 11:51 am by bakedpotato

If your getting a runtime error you need the vbrun file.
You can get the file you need at http://www.ioncross.com/

If you cant find the visual basic runtime files on the IONCROSS site you can get them at Microsofts downloads page. Just search for Visual Basic...

After you download the vbrun file put it in your Windows\System folder.

This program the Server Operator isn't for making mods it is for modifing multiplayer saved games and and for running a Freelancer server. The Character editor is for modifing singleplayer saved games. My fault I got confused...Again.

The closest thing I can suggest for an editor is Tinyfish Freelancer Ini Editor v1.4.4 beta
This is an editor for Freelancer ini files. It provides a graphical user interface with a grid view with organized data so browsing and editing become easy. I always use it to find items and fix bugs in downloaded mods, re-balance some parameters. It should be useful for both players and mod makers...
