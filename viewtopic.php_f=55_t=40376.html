>If there is a server dedicated to traders.
>If there are any mods for traders.

The Hamburg City Server is not dedicated to Traders only, but has certain features that make gameplay very interesting for traders (in fact, there are a lot of traders there) while the server still is sort of a vanilla server with optional mod adjustments.
The most interesting part for traders would be the escort feature.
You can group up with some fighters and ask them to escort you.
You need to type in "/convoy" and you are ready to go (escorts must type /escort playername). Once you reach your target base, escorts will receive a server-generated bonus based on the profit you made. This way it actually makes sense to be an escort -> more RP -> better gameplay
