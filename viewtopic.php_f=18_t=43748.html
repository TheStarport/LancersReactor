Sat Jun 03, 2006 6:07 pm by Mace_166

I don't think its possible to change it that drastically. In Diff2Money.ini you can set the amount of money for each difficulty level. Not much help there..
In mbase.ini you can set the difficulty of missions at set bases.
This is from SDK:
nickname = Li01_01_Base
local_faction = li_p_grp
diff = 1<--increase to 19
msg_id_prefix = gcs_refer_base_Li01_01_Base

[MVendor
num_offers = 8, 12

[BaseFaction
faction = li_n_grp
weight = 13
offers_missions = true
mission_type = DestroyMission, 0.000000, 0.112387, 30<--increase to 32, 99, 30
npc = li0101_lnavy_001_f
npc = li0101_lnavy_002_m
npc = li0101_lnavy_003_f

The numbers just mean:
the minimum level, maximum level, percentage of this faction at this base
the highest in normal FL is 28.5, 53 at Crete/Malta so the above should give you whaat you want. But, the problem is that the factions in Liberty do not have very high loadouts and ships so only the Outcasts could give the very high level missions unless you added to their faction_prop entries.
Also, I believe that the encounter in the system is also a factor. If you have rogues at D1 then you can't expect to get 250k missions.

mknote

Mon Mar 19, 2007 2:21 pm by mknote

Between this:

faction = fc_c_grp
weight = 30

And this:

npc = ew0401_corsairs_001_f

You need something like this:

offers_missions = true
mission_type = DestroyMission, 0, 0.112387, 30

You can fudge around with the numbers a bit in order to get the correct difficulty levels.

MK
