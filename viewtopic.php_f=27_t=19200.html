Sun Jun 15, 2003 6:05 pm by barn.door

Guys, try this- Simple Tutorial for installing 1.6b, this worked OK for Me,
1. defrag your HDD
2. run scandisk
3. disable all screensavers and auto-dim
4. stop as much running in the background as you can.
5. running FLMM 1.1 with, Options, Make modified files read only etc..... unchecked.
6. back up all current mods, then deactivate and delete all current mods, i had rebal 1.4b & ghost docking ring running
7. install & activate rebal 1.6b, it takes some time, even on my P4 2.4Ghz with 512mb ram.
seems to work OK, not really gone far yet, and, if you can multi-boot and image your drive, that helps alot to.
