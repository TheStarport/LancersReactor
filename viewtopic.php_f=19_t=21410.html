Joining clans, as or be asked in a server. Best to get some experience first online.

As for servers only pick servers that:
1. actively admin them selves
2. have a contact email/website
3. have a No Pking/PvP[Player killing/player v Player rule for New York or its a night mare with idiots dock camping Manhattan
4. have active players, ask about in the server if its laggy, used a lot etc. Your making it a online home so ask questions. No point in investing 30+ hours on a character to loose it, after the server dies if it was just a home based server.
