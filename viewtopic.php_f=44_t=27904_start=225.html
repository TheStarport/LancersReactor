X-man and Thazell, I know i'm not the one to be giving much advice an modeling but I believe you should be able to do it easily. Import your model into milkshape. Then select the face you want to add your hardpoint to. Then use the face tool to devide that face by 3 or 4 (make sure you rotate the faces so they all are facing ooutward like they are supposed to). Repeat til you have as small a face as you need. Duplicate the face and rename and texure it as you want. Repeat the proccess til you're done. Then export the model or save the new model, update the script and you're on your way. The details on how to do each step is covered in both this thread as well as in a couple of other forums in TLR. I believe one is called Noob questions done by Chips, also a thread for modifying existing FL ships already in game. Just check the main forum list or do a search of the various forums for adding hardpoints. Hope that helps.

hardpoints eh,

There is avery simple method of making hardpoints.

Editing/adding existing hardpoints in a freelancer model is another ball of wax entirely and as mentioned there are several threads that contain info on doing this. Both the methods mention above work but I may have an easier method for imported or new models.

Go to the model window in MilkShape and select "vertex" then the top view - normally bottom left window. Make 3 vertex's in approximately an equilateral triangle, Then go to "face" and connect the faces.. one triangle - one hardpoint made. Scale to suit, rename, move into position, copy and either mirror or move to suit. Hardpoints MUST be listed after the ship groups.. failure to do this will result in a .cmp that does not work

The centre of the triangle is the actual hardpoint and it doesnt really matter what plane it is in but I like to start in the "y" plane.

Since so many people have trouble with these I will upload a complete or nearly so ms3d file to TLR.. Simply use this file, merge with your model file save as the model file and move the hardpoints into position..

once I have done this I will be back to provide a link.

junior1210,

Is there anyway to tell scale accurately in fl? Yes...

I keep on about this but will say it again. Import the pilot figure from fl..DO NOT rescale it... then scale your SHIP irrespective of SIZE to the pilot figure. A freighter, fighter or battleship.. it makes no difference, build the ship around the pilot figure.. once happy scale in x,y,z by a factor of 100 ... it should be spot on for Freelancer.

If the weapons dont fire at all, then it may be hardpoint orientataion. do the weapons fire at all? in any direction? try turret view and see if they will fire.. if they do but wont fire forward.. it is more than likely the hardpoint orientation... if not.. are your hardpoints listed last in the ms3d file?
