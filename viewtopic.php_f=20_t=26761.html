I assume you want to re use your old MP ID by what you are saying , there is a very simple way to do this

print out your CURRENT account id

Open Regedit

Search for your account ID (just type it in without any spaces and or '-')

replace 2 characters in the string you found (just random the only function it serves is to generate an invalid ID)

Start freelancer

It will pop up a box telling you that your ID is invalid and give you the opportunity to fill in a new one

Type in your old ID
