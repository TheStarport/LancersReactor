The simplest solution would be adding a hardpoint in hardcmp.
http://lancersreactor.com/t/Download/Do ... sp?ID=1257

Open up the ships cmp
1. Look at the bottom right and click the new button.
2. In name scroll down and select HpShield
3. Set type to fixed
4. Cheak the box that says Use numbering scheme.
