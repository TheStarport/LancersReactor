Yes it can. The Data/audio/music folder. There is a mod I love, the SWMM that adds the music from Star Wars to the game. Look at it as to what and where music is added. I say that, as posting all the information is long, though I will show piece -

[Sound
nickname = music_danger
file = audio\music\sw_danger.wav
type = music
attenuation = -6
streamer = true

[Sound
nickname = music_dangerous_chase
file = audio\music\sw_chase.wav
type = music
attenuation = -6
streamer = true

*note* The music is in Wave format.
