Gung_Ho

Fri Aug 08, 2003 6:41 am by Gung_Ho

A common fix for v2.2 is to delete the databases created by the program and run it from scratch so it recreates them. If the program was interupted while initially creating the databases it will throw you all sorts of errors until you delete them:

FLDS-Language.mdb
FLDS-Settings.mdb
FL Datastorm.mdb

Try deleting all three of those, then fire the program up and give it time to rebuild them (it takes quite awhile). That might fix some errors.
