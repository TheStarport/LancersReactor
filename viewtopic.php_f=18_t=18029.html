Well, if you're going to host your own server for yourself to mess around with, go to your Freelancer directory and go to your EXE folder (example: c:\program files\microsoft games\freelancer\exe) and find the "mpnewcharacter.fl" file. Open it up with notepad and change

money = %%MONEY%% (or whatever it says)

and change it to

money = 999999999 or whatever you want.

It won't work on a public server, that's another file that I don't even know about so don't ask.

You could use a trick... If you edit the cargo in the file located at Freelancer\Data\Ships\loadouts.ini. Use Bini to decompress the file. Note, the new filename that you need to edit is: loadouts.ini.txt and after you have made your changes, you must do 'save as' and 'cut' the '.txt' extention from his name.

Ex:
----

[Loadout
nickname = msn_playerloadout
archetype = ge_fighter
equip = ge_gf1_engine_01
equip = shield01_mark01_lf, HpShield01
equip = ge_fighter_power01
equip = ge_s_scanner_01
equip = ge_s_tractor_01
equip = ge_s_thruster_01, HpThruster01
equip = li_gun01_mark01, HpWeapon01
equip = li_gun01_mark01, HpWeapon02
equip = LargeWhiteSpecial, HpHeadlight
equip = SlowSmallOrange, HpRunningLight01
equip = SlowSmallOrange, HpRunningLight02
equip = SlowSmallOrange, HpRunningLight05
equip = contrail01, HpContrail01
equip = contrail01, HpContrail02
equip = DockingLightRedSmall, HpDockLight01
equip = DockingLightRedSmall, HpDockLight02
cargo = ge_s_battery_01, 3
cargo = ge_s_repair_01, 3
cargo = commodity_cardamine, 20 ; <-- ADD THIS LINE HERE

-------------------------------------------------------------------------------------------
NOTE: Don't forget to hit the key 'SPACE' after the last edited line or if you just use 'copy'/'paste' you must include the space, too. 'Cut' this text: ' ; <-- ADD THIS LINE HERE'.

The number that follow 'commodity_cardamine, ##' is the max. cargo of your ship.
Next, all that you have to do is sell all the cargo at the local market.
If you start in Manhattan, cardamine = $1500 x 20 units = $30000 bucks!!!!!!!!
- 'Not bad for a start, rookie!'
