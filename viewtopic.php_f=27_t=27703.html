Your backups haven't been done correctly. easy way to do this is to download and install the SDK from the EOA players consortium. It is all the games ini files decompressed for you. Once installed - you will need to manually copy and paste the whole contents of the file into the data directory. It will say its overwriting files - so this should sort it. Otherwise, find its constants.ini, engine equip.ini and the st_equip.ini files - and paste them over the Freelancer edition of these files istead.

Constants.ini is in the data folder
engine_equip.ini is in the data\equipment folder
st_equip.ini is in the data\equipment folder to
