NightHound

Thu Nov 25, 2004 2:34 am by NightHound

faction_prop.ini is in data\missions\faction_prop.ini

add this to data\Fx\fuses.ini

[fuse
name = death_player
lifetime = 0.1
death_fuse = false

[dump_cargo
at_t = 0.000000
origin_hardpoint = HpMount

then in each ship put this in the shiparch.ini located in data\ships\shiparch.ini:
fuse = death_player, 0.000000, 1
