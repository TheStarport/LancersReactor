Erm - the Open SP mod needs to be activated AFTER the main mod is activated. Otherwise you just get.....well - open SP without evo. Its basically Open SP with options to start in systems - including the new ones. Thats it - so it will need evolutions activated first to make sure that there is a mod there for you to play!

@Outcast - thats fine - if FLMM didn't move your save games, and it never crossed your mind to make a zipped backup of your save games folder (takes about 10 seconds) to keep save games - then I guess never trying a mod is your best course of action. As for your install problem - i am exceptionally sceptical you actually read the troubleshoot.txt

Why do I say that?


· When I try to open FLMM, it gives an error, and i cannot see Evolutions!

The mod has been checked, and so shold unzip perfectly. If it doesn't,
then the chances are that its corrupt. Try downloading once more, and try another location too.

--------------------------------------------------------------------------------

· During the activation of the mod, i get an error and it fails.

In this case, its usually because a bit of scripting cannot find a location in the files.
This is more likely to happen in the server version than the client, as it does contain scripting.

If it does, then follow the install steps again by deleting the Freelancer Folder,
installing the sdk and then trying the mod once more. After this, it may be a corrupt file again - try
downloading the mod once more.

--------------------------------------------------------------------------------

· When I try to deactivate the mod it says its had an error, and asks if i want to restore backups.
What does this mean?

It means that it cannot find a file that it wants to remove. In this case it will need you to restore
backups in order to deactivate the mod. However, after anything like this you should reinstall the SDK
at least!


now the top of the file contains the following:
###############################
Section 1 - Installation issues
###############################

· How do I install Evolutions 1.28?

This assumes you already know what FLMM (Freelancer Mod Manager) is, and how to use it. If you do not, the FLMM is available in the downloads section.

The Steps:

1) De-activate any mods.
2) Delete your Freelancer Folder
3) Reinstall Freelancer
4) Use FLMM - go to tools tab, select "go to save game folder" and delete everything
5) Download and install the SDK Ver 1.3
6) Download and install the CLIENT VERSION of the mod
7) Activate the mod, and have some fun

If your running a server, download and install the microsoft patch after the SDK!!!


For other mods though - seriously - just ZIP up your save games folder before installing ANY mod - and then you have those save games saved away for yourself.
