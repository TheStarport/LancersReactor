Yes, I run my server off of a 'puter that has only the server files installed. On the FL disk it gives you this option. If you select the other option and install the game itself, you get the whole enchilada - the game and the server files.

I play the game (client side) on my main 'puter that I use for everything else including games. Since I am on a LAN I just ensure I log onto my server through the LAN (it lists in the global server list how to connect to all the FL servers, in my global server window I see one LAN server and the rest Internet). Logging in the server through LAN prevents you from using some of the precious bandwidth through your Internet pipe which can thereby reduce the amount of Internet players that can play. Good luck!
