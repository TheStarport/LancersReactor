adding music

put your new music file in the data/audio/music folder

then in the music.ini file

[Sound<<<<put closing bracket
nickname = music_your_tune <<<<<< nickname of your music file
file = audio\music\music_your_tune.wav <<<<<< path to your music file
type = music
attenuation = -6
streamer = true

in the system ini you want your music to play you will find this section usually near the top of the file

[Music<<<<put closing bracket
space = music_your_tune <<<<<<<<nickname of music file
danger = music_br_danger
battle = music_br_battle
