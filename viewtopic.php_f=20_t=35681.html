Uhm...Freelancer doesn't have a "product/serial key" only a key generator for MP.

The only way to find out what that MP ID code is to run the myflaccountid.hta and then writing it down (or printing it) then re-install the game. Then when FL generates a new ID, simply delete the key and type in the MP ID you have on a piece of paper. If done correctly, you'll be able to access your characters again. If the key isn't accepted by FL then you should double check if you typed it right.

Edit;
I just realised that he wrote that he already reinstalled the game. The old ID is gone now. The only thing you could do is what Pinger said.
