bakedpotato

Thu Sep 11, 2003 2:37 am by bakedpotato

Hello all
As with all the mods I post I tested the FightSpace mod and it runs fine on my test system. I am using Windows XP professional but I also have computers hooked up to my home local area network that run on Windows XP home edition and the FightSpace mod runs fine when we play multiplayer on the lan.

If you have been using mods especially mods that add dll files to the Freelancer exe folder read the following.

If you have had problems with mods crashing try the Tools\Restore Backups in the Freelancer Mod Manager. If it is still crashing you need to uninstall Freelancer then reinstall. Deactivate any mods that are active in the mod manager first. Check to make sure that the Freelancer Folder was deleated by the uninstall routine. The default location for Freelancer is:
My Computer \ C: \ Program Files \ Microsoft Games \ Freelancer.
