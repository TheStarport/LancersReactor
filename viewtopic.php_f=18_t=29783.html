Changing the costumes of players in the game is actually very simple. All you need to do is edit the costumes.ini file in the Characters folder so that the inted character has the desired clothing.

There's also a small exe hosted here called FLCedit or FreeLancer Character Editor. This program allows you to change the heads and costumes of Trent, Juni, King, Tobias, Orillian, Quintaine, and Sinclair & then save them to either your own save game file or the costumes.ini file. Please note that while all male bodies are compatible with all male characters and all female bodies are compatible with all female characters, not all heads are compatible with all characters, but you're just going to have to try them out and see for yourself.

If you have any further questions fell free to e-mail me at barraganap1804@msn.com
