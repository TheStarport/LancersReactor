Chips

Tue Mar 23, 2004 9:05 am by Chips

Also - i made a thread about problems with mods inside this forum. Check there for problems - it won't solve some of yours (like where is save games) but it will sort others out perhaps.

Deactivation problems can also occur with read only files and stuff. Use options tab option in FLMM, and then mae sure that the "make modified files read only to prevent tampering" is DESELECTED. This will make them all available to be edited/deleted. Otherwise, go into mods folder, and manually select the mods folder name, and right click -> select properties. Now make sure the read only/compression boxes are NOT selected.
