use a bini decompressor on the weapon_equip.ini in the Freelancer\DATA\EQUIPMENT directory - you will have to run the bini decompressor on the file if you have not already installed a mod that changes this file, consequently decompressing it -
search the text in the .ini file for the weapon numbers: (all three are for the nomad laser in it's various iterations - best to alter all of them so your change will be consistant)
263523
263524
263525
for refire rate, the setting that you should alter for each of these entries is "refire delay". this setting is in the form of a decimal - how long in seconds before another shot will be fired. I.E. a refire delay of 0.5 will be 2 shots / sec. 0.25 is 4 shots / sec. 0.10 is 10 times / sec. you get the idea.
questions, gimme a holler - hoghunterzero@hotmail.com
