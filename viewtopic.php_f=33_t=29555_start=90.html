Ro9ue

Tue Mar 30, 2004 4:35 pm by Ro9ue

If you have FLMM properly installed and the other stuff (like SKD 1.3), download the Rebalance v3.4 MOD to ... i guess where ever you want to, so long as you remember WHERE that is. I personally put all downloads into a folder "Web Downloads" on my desktop, and I put all ones reguarding FL into their own folder "FL" inside Web Downloads. This makes it easy to find downloads.
After you've done that, go to that folder or whatever place you saved it to. Double-click the mod (should be called " Flrebalance340full.zip " ), and it should unzip to FLMM. You'll get a message like "Rebablance v3.4 has been successfully unzipped, and is ready to be activated" press OK, and FLMM should pop up with FLBebalance v3.4 there. then just click the mod, press Activate, and you should be good to go.
