Argh

Mon Feb 07, 2005 4:32 am by Argh

One of the things I'd really like to see is a lot more usage of custom Bases. Building a Base with operating doors, etc., isn't too bad if most've the Base is a custom CMP put in a position to connect with a standard Base object that supported animated doors- for example, space_police01.cmp

Liberty would be much more interesting if there seemed to be neat things to visit just for their architectural value, not just another pretty background. And it'd be nice if things had some character most of FL's objects are terribly generic and plain looking, and make use of really boring tiled texturemaps.

I had an idea for a base that I looked at executing at one point, for example, which would be a Space Restaurant. I was planning on putting a Zone around it which played some sort've appropriate music, so that players would hear music as they got closer, and then use a combination of techniques to make the interior Rooms really interesting.

I guess my point is that what I'd like to see in a mod of this kind is NOT some lame attempt at a story, and not some uber-ship that's tucked away in the new systems as bait for players. Instead... if you're just going to do a very small number of Systems... take a lot've care, and build them into something truly interesting and unique, with a lot've interesting feeling to every object, instead of just another mod that pretends that assigning things name strings and descriptions in a DLL and doing a few colorful lights is enough to create an interesting world.
