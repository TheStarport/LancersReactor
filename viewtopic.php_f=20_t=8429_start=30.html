Tue Mar 11, 2003 8:09 pm by Newt

This is just speculation, if you're having problems generating an MP unique ID under 98/ME, please try this out and let us know.

More info on 98/ME profiles.
For '98 AND ME users, here's how you enable user profiles. This would then force you to login as a specific user and would create a unique profile for the user:

Step Mouse actions
1 On the Start Menu:
Point to Settings.
Select Control Panel.
2 In Control Panel:
Double-click the Passwords icon.
3 In the Passwords dialog box, select the User Profiles tab.
4 Select "Users can customize their preferences and desktop settings. Windows switches to your personal settings when you log on."
5 Select OK. (it may ask if you wish to reboot at this point, go ahead and reboot)

Then reboot, when it comes up with the login box, DO NOT CANCEL IT, put in a username, you can leave the password blank. Don't forget your usename, Freelancer and other apps will save settings (and save games) under your profile, so you wouldn't want to login and not be able to find your savegames or bookmarks.

Reinstall (if necessary) and try to generate the unique ID for MP.
