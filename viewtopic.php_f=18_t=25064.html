Hey wingc3,

That's a great idea, I've experimented a little with doing this and it works great. Only thing is I need it in wav format, cause that's what the game uses, and it's larger than mp3. Vad's Wing commander music site has a lot of midi's for wing commander. The process is pretty simple. You simply rename the file you want to use to the name of the file your replacing. It can be done with flmm with no problem whatsoever. If everyones interested in doing this, I'll make a mod out of it. It would be great with the wing commander fighters I'm creating.
