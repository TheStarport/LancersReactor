(or if someone else knows please let me know)
I'll put in new options to disable those features that you tell me can't be on your server. I will then send you a new script that you can copy into your Black Market folder and replace the old script.
This carries two conditions however.
1.) That the mod remains a Black Market mod. ie the bases and commodities remain. Without those this is no longer my mod as I designed it and I'm not going to hack out all my work and still have it called Black Market.
2.) That I don't have to put in an option for too many things in the mod. Plain and simple if it looks like it's going to take me forever to do it I'm not going to.

Here's what my mod changes:
1.) Adds 23 bases*
2.) Adds 17 new commodities
3.) Changes all vignettes in 20 systems to unlawful,lawful
4.) Triples the number of vignettes in same systems
5.) Changes scan rate and scan announce for navy and police factions
6.) Changes all player pilotable ships banking angle to 360**
7.) Changes all ship chase camera turn up angles to 0 on player pilotable ships**
8.) Changes all ship chase cams horizontal turn angles to approx 75% of former value on all player pilotable ships.**
9.) Changes multiple stats on the 6 player pilotable freighters
10.) Changes Zoner empathy and initial rep values
11.) Adds interference zones to 20 systems
12.) Changes PTough graph points.***
13.) Adds six "new" ships.****

* = Obviously the bases include changes to univers.ini/a new system folder to keep all my base folders and rooms organized/changes to system inis/ changes to market inis/appending of mbase.ini etc.
**=These are already optional
***=Optional
****=No new cmps or mats. Just changes to shiparch. New engines/powerplants. Changes to good/equip inis etc.

Like I said earlier #1 and #2 are not negotiable. Without them this is not Black Market. Anything else just tell me what can't go on a server to make this a "server side" mod and I'll try and include the options for you.
Be warned, however, that changes to things like the vignettes, mbase.ini, and zoners can seriously degrade the effectiveness of Black Market.
For instance the bases that I made have elevated num_offers for the mission vendors because of all the vignettes that I made available. If you remove the vignette changes and additions you're going to get most of the NPC's in my bases handing out nothing but empty talk boxes. This is because their first priority is to fill the mission quota. Once FL assigns them that task they fulfill it. If all the vignettes eligible are already accounted for then the NPC has nothing to hand out.
