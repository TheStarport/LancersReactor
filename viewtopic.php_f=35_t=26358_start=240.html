Fri Jul 08, 2005 2:32 pm by ColinS

networkcity

If a non FLE mod is active when loading your mod, the names will be taken from
the dll of the active mod. Deactivating the mod in FLMM will make FLE extract
the names from the dll in your mod's folder. If this is not the case, check the
original Freelancer.INI file was completely restored when the last mod was
deactivated. If the modded Freelancer.INI file was not restored, there will be
7 dll entries in the (Resources) section, instead of 6, causing your mod dll to be number 8. The ids_name/info numbers in the mod ini files will be between 458753 and 524287 for dll number 7.
