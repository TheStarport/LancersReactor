It's a bit boring. But I have made it successfully.
there are 3 files u must edit:
1. the .ini file you define the system, u must add some zone with "mission_eligible = true" property. (maybe need some with "mission_type = lawful, unlawful" also)
2. "mbase.ini" in the MISSIONS directory, add your base and npcs.
3. "vignetteparams.ini" in the RANDOMMISSIONS directory, add your the base's faction(if it is a new faction) to some kinds of mission u like.
