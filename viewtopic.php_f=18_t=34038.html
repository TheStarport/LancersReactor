clan "controled" bases (might be possible)

I just thought of this....
Well technically clans could control bases, but it would require MASSIVE work from side of admin (so not sure who would actually do it). But here is what I think anyway...

A clan controls an system, and would like a static BS placed near a jumphole, to defend the system. How this could be possible is this way, admin of the server sets up the price of the base and weapons it would carry. Lets say base costs 900mill with best weapons and all. The clan pays this amount to admin, later this money can be used for different kind of competitions..etc. Anyway, the base can be put up there, by an server update which could be released for example every first week in a month. So once the update is out, the clan would have an base. Now for rest of the work, admin would have to make a new faction (for that clan) and make it hostile towards the players that clan is hostile towards, and allied towards.

Now I know above IS possible, since there were so many mods that added new bases and faction. But the main possible problem could be player wipe. I know that for some changes there is no need for player wipe, for some its needed, and since I am not sure if adding a new base and faction would require a player wipe, I am not sure that this idea could work either...

Therefore, would changes mentioned above require a player wipe?? And what do ya think about the idea??

another alternate is this:

by default, if u make a new faction, all existing chars will be neutral to it.
now make lets say 20 (or how many u want) factions that are all-hostile to players when they are created.
later in the game when you need a "hostile" faction just use one of these.
initially, all players would be hostile to this, but its not much work to update the reps of a clan towards this. of course all new chars would be hostile as well.

a bit of maintainance work, but it works.
and you only need to wipe once in the beginning.
the amount of factions you create can be seen as a buffer of how many clans "can" have their own factions and bases.
