Victor''s MP Enhancement Mod 1.3

Lancers Reactor is now hosting the mod, so here's a thread everyone can use to post bugs, comments, complaints and suggestions.

Version 1.3 can be found here thanks to the Lancers Reactor guys.

I'll try to keep this original post updated with any known bugs or issues as they are posted and if they can be reproduced by myself or my beta testers. If you'd like to take a more active role in testing the mod and contributing work or ideas please visit our MSN group here . We have a closed server online and the server password is available after you have been approved to join the MSN group.

Known bugs/issues:

- Infocards are incomplete for a few of the capital ships

- Infocards for bases/planets are untouched, they may be inaccurate

- Docking with large ships is imperfect, heavy traffic and odd angles may cause the player to get stuck in the docking cinematic

- Occasional crashes will occur in large battles regardless of the ships involved or the location, they are normally few and far between

- Occasional crashes in Omega-7, see below for a quick fix

- Crashes when Rheinland Battleship explodes, see below for a quick fix

OMEGA-7 bug temporary fix

If you run a server or play in SP mode you will need to remove the modified file or it will crash when you enter omega-7.

To remove the file from a FLMM installation:

1) Deactivate the mod

2) Browse to drive:\Program Files\Freelancer Mod Manager\Mods\VictorMod1_3\Data\Universe\Systems\Bw03 and delete Bw03.ini

3) Reactivate the mod and you're all set

To remove the file from a non-FLMM installation:

1) Find your original Bw03.ini file if you made a backup, otherwise browse the Freelancer CD and look for it in one of the .cab files.

2) Copy the original over drive:\Program Files\Microsoft Games\Freelancer\DATA\UNIVERSE\SYSTEMS\BW03\BW03.ini

Replace "drive:" with your appropriate hard drive letter, for me this is F: for most people it will be C:

A patch will be released soon

Another bug in 1.3, this time with the Rheinland Battleship.

If you run a server or play in SP mode you will need to remove the modified file or it might crash when a Rheinland Battleship (and sometimes other capital ships) explodes. This one was a pain to track down

To remove the file from a FLMM installation:

1) Deactivate the mod

2) Browse to drive:\Program Files\Freelancer Mod Manager\Mods\VictorMod1_3\Data\Fx and delete explosions.ini

3) Reactivate the mod and you're all set

To remove the file from a non-FLMM installation:

1) Find your original explosions.ini file if you made a backup, otherwise browse the Freelancer CD and look for it in one of the .cab files.

2) Copy the original over drive:\Program Files\Microsoft Games\Freelancer\DATA\FX\explosions.ini

A patch will be released soon, probably when it fixes at least three bugs.
