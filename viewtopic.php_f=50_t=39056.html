First, most of these questions are more binded to Freelancer Server Forum.

Look at this for a detailed walkthrough on how to set up a server :
Click here

To put a mod on a server, simply download Freelancer Mod Manager (Click here), and download from TLR the mod(s) you would like to run.

When you activate a mod in FLMM, the name of other mods will change color. Red means this mod is not compatible with the one you activated. Orange means it could work, but it could also bug.
Normally, the biggest mods like Evolutions or The Next Generation are not compatible between eachother.

I also warn you that most of these mods only offer a client side version to download freely. To get the server side one, you would need to e-mail them a request.

Oh, and yes you can play on your own server.
