Ah - did you try to use the SDK instead - as the mod relies very heavily on scripting. At which point, the SDK is the one that will garuntee you have the same files as the mods been tested upon (see EOA Players consortium for SDK, or downloads of TLR).

For number 2 - do a FULL server wipe - erase ALL characters (none to be left at all!!) and that should (note the should ) fix the issue. Corrupt files, or running it without the mod at any point could have caused this - as save games with locked gates will corrupt ALL the saves with unlocked gates, and then more trouble occurs. Also - you might have to watch out for options with IFSO clashing with the mod. I never tested it with the Operator, mainly cause i didn't know anything about it at the time

Hope that fixes you up, as i am an ini junkie, and unfort not a server whizz!!
