Fri Jul 28, 2006 2:57 pm by bakedpotato

You didn't give much information, such as were you using any mods
but here's a suggestion...

Delete the Freelancer folder, not uninstall but delete.
C:\Program Files\Microsoft Games\Freelancer
By deleting the folder you make sure there will be no files left in the Freelancer folder which happens sometimes (especially if you have used mods) after just uninstalling the game.

Open up your saved games folder...
Documents and Settings\your name (or the name you gave your document folder)\My Document\My Games\Freelancer
In the Singleplayer folder if there is a file in there named Restart.fl - delete that file.
Then pop in the Freelancer CD and the install screen will come up.
Now install Freelancer.
Happy gaming!
