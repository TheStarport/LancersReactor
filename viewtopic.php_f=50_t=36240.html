Tue Nov 30, 2004 5:42 am by warzog

This is a bit Complicated...
Info is stored in "voices_space_female.ini," and "voices_space_male.ini"
Each of those list which voice, "atc_leg_f01," "atc_leg_f01a," etcetera, have which messages.
"atc_leg_f01," "atc_leg_f01a," etcetera, are all UTF files.
UTF files, in Freelancer, are a sequential set of audio messages, all stored in a single file.
The "voices_space_female.ini," and "voices_space_male.ini's" allow one to set up how the messages are stored, how/when they are to be called up, and when they will be played during the game.
Each UTF file has several hundred messages stored, and they're a bit daunting, to me.
There is a UTF editor program, here on LR, that will allow you to rip the UTF files, and hear each message individually.
I haven't messed with it much, so I'm unclear as to how one goes about making a new UTF, with new messages.

Watch your 6!
