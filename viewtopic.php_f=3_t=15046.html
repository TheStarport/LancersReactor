Sapient6

Thu Apr 24, 2003 11:21 pm by Sapient6

- There is no cheat-protection in Freelancer beyond having the character files stored on the server
- Cheats are rampant
- Cheats have been known to crash servers badly
- Cheats have been known to corrupt non-cheaters' character files
- Cheating griefers galor!
- FL lacks useful server tools, so there's little server admins can do to detect a cheater
- Some server admins are in witch-hunt mode and are banning players for "cheating" with little cause at all

You may want to steer clear of FL online at the moment if you can't stomach sharing the internet with a lot of cheaters. DA is currently working on a patch to clean the cheating problem up a bit, but:

- unless a game is entirely server-side, which leads to extremely poor game performance, it can never be entirely cheat-proof.
- the only hope with any game is that the cheating be infrequent enough as to only be an occasional and brief nuisance

So if your goal is a cheat-free game you may not want to play anything online. However if your goal is a game that just isn't rampant with cheaters then once the patch comes out you may want to give the situation another appraisal and see if the problem hasn't gotten better (that's my plan, anyhow).
