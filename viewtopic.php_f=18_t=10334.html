*Tutorial* Names for a new systems and the bases

If anyone wants to know how to name there new selfcreated system and their bases here is the answer.

[First you need the Tool Reshack (for the dlls)
You open with it the namenameresources.dll, then you put in it the names of all the systems, stations, bases etc. you want to name (I have chossen the string table 3501 because there is enough free space for all the names). Aftre this you save the dll.

[Then you edit the name of your system
For that you must edit the universe.ini. In it you go the system entry for the system you want to edit. There is a value strid_name there you must put the value of yout string (look in reshack) + 196608.

[The stations
You edit the stations precticaly the same way. You edit their strid_name value in the universe.ini and the ini-file of your created system.

[The names are for the navigation map and the radar
[You can edit this way the name of your jumpgates too.

you could also just copy nameresource.dll, and rename the copy (NOT the original!!!) to nameresource2.dll and add this name to your freelancer.ini...
Now you've got an entire dll file that's free... just start at Stringtable 1... also this is easier when you want to distribute your mod...People don't have to add all the names manually to their nameresource.dll (if they have mods of their own for instance)

Only thing you have to do is toadd your dll to the freelancer.ini file...
since it's the seventh resource file the ids name for number 1 (New York) would be 0x70001 = 458753 In this file you can change all data safely..
