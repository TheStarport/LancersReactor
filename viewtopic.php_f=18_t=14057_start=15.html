C06alt

Wed Apr 23, 2003 4:02 am by C06alt

problem 1
edit gun arcs with utf edit

problem 2

copy a .sur file from another ship and rename it the same as your .cmp file
and place it in the same directory. i suggest using a player transport .sur
to match the size of the Lambda

problem 3

loadouts are saved in save games so if at some stage you saved the game in your Lambada and you are restarting from that save game,
new equipment will not be there. you either need to buy the ship from a shop
or start a game with it.

problem 4

caargo bays have animations on the player transport ships, but someones going to have to crack the format for that.

problem 5

some times you need to invert a texture so it shows up right in fl
try fliping it in a paint program and put it back in your mat file.
