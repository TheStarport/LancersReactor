Annilyna you can find the correct ports that need to be opened by looking in the readme file in the Freelancer folder (2300 - 2400 should work, thats what mine uses). You might not have the ports open to the right IP address. Run IPCONFIG by going through your Start menu then selecting RUN. This should tell you the IP Address of your computer that your routers port need opened to.
