Hi!

At first a question?

Do you know how to make patrols and if you think so, did you do some patrols with original factions for testing and did they work? If not, learn at first how to do patrols (there are many tutorials for this here on TLR) and mod some new ones, so that you are sure that you can create working patrols. It's not that hard, but you should do this first.
Why? Because setting up patrols for new factions is working the same way as for original factions. I got working patrols for several of my own custom factions and I did them the same way as my patrols for original factions. Just be sure that you finished your custom factions completly. For this you will find all the necessary stuff in the different faction tutorials here on TLR (like the one Manhattan_Guy posted the link for).
For example, your new faction should already have NPC encounters, some bases and so on. If they do not have this add it, before you go on to patrols.

Hope this helps! Otherwise ask again.

Black Eagle
