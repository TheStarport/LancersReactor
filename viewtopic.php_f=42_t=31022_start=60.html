warzog

Wed May 12, 2004 12:23 am by warzog

The MAT file is the texture file used by FL for everything.
Unlike MAX, cmp files do not include the texture in them.
It's a seperate file.
The MAT file.
If you've downloaded the mod...
Look in the Data\Ships\Geminiwc folder.
Every ship has these 3 files:
1) CMP-the model
2) MAT-the texture
3) SUR-the "surface" file that actually takes hits from weapons, or collisions.
