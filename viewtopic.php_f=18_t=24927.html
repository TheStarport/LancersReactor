Smiley, go into the EXE directory in your freelancer game dir, then open a file called
'mpnewcharacter.fl' in notepad (You shouldnt have to unencrypt it). the 6th line down should look like this:
money = %%MONEY%%
if you want to start with 25000, replace it with this:
money = 25000
There ya go, any number up to 999,999,999 should work. and its a server side only mod, client doesnt need it, but it wont hurt if they do.

Sun Oct 05, 2003 6:27 pm by deathwalker10982

subxero, for SP, go into data/missions/m01a and open m01a.ini (you have to
unencrypt it using bini) find the line that reads :
Act_SetShipAndLoadout = ge_fighter, msn_playerloadout
this tells you your starting ship that juni gives you for your first mission. ge_fighter is the cheesy starflier package you start with. there is a list of all the ship packages somewhere... just replace it with the package you want
as for money, in that same file, find this line:
reward = 2000
change reward to whatever and thats how you get your starting money, but you cant have an openSP mod activated. once you finish the first mission, you will have enough money to last a long time.
