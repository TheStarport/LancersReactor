Freelancer runs minimised and attempts to maximise

Hi

I am attempting to run a clean installation of Freelancer, however am unable to as it starts minimised. The QuickFix patch bypasses the into, and the Freelancer splash shows, but then the game minimises to the desktop,and it seems as if it is constantly attempting to maximise but fails. The problem was the same before I installed the QuickFix patch.

System:

XP Home SP2
AMD Athlon 64 X2 Dual Core Processor 4200+
2048 MB RAM
MSI nVidia GeForce 7900 GT/GTO
DirectX 9.0c

I am not attempting to use any mods. I have installed AMD's dual core optimiser, the latest video card drivers, reinstalled directx and installed the QuickFix patch, all of which has achieved nothing. On the two rare occasions that the game has run properly, I have got to the menu and tried to increase the resolution higher than 640x480, however this then caused the problem as above.

I have searched the Official Freelancer FAQ Database and these forums but nothing has worked.

I tried changing the size setting in PerfOptions.ini to 1280,1024 but this only made the problem more obvious. I haven't managed to 'fluke' my way into the game since then.

The problem does not occur in any other games. AMD's dual core optimiser fixed a recent problem with animations in other games, but that seems irrelevant. I tried setting the affinity of the process to a single core (which fixed issues in another game that were due to the dual core), but this had no effect.

I can hear the sound perfectly well - it's the only indication that the game is running. On the two occasions that I did get to the menu, there were no other apparent graphical problems.

Any help would be greatly appreciated.
