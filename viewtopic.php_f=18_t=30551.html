Tate The Trader

Sat Apr 17, 2004 4:42 pm by Tate The Trader

Droid X2 I know the to all neutral server "problem", it is in the mpnewcharacter file. In your file there is a line called:

;initial_rep = %%FACTION%%

You must type your starting faction here and remove the ; in the beginning of the line. Like this example to start with liberty navy rep:

initial_rep = li_n_grp

This way it solved the rep problem if I run a server.
