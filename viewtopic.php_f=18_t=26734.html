Freealancer takes the first mat file that is designated for that ship class and uses that.. so if the existing mat file for say liberty ships still exists then it will use that mat file..... and ignore any others. and all the liberty ships use the same mat file..... All freelancer textures even though they are id'ed as .tga are in fact .dds why they did it this way I dunno.. You will have to create a new file for say the patriot.. rename it (patriot_ mk1) and then refer in the appropriate files to that texture.. otherwise the patriot MK1 will still look like the original. I have had this problem with creating new weapons models and it took some head scatching before I was able to work it out.. i'm sorry if this is not a good explanation but it is a very difficult question to answer...
