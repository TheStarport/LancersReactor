Piercing an Ubershield

Ok for one I do not know if i should place this here or in the Modding forums , i decided on placing it here since it is an issue that concerns all MP players and is getting out of hand real fast , and nobody seems to be willing or able to supply the solution for it

ubershields are a nuisance and can be dealt with in two ways , either have the serveradmin install lancerguard on the the server or make one small modification to one file rendering the ubershield useless

personally i prefer that a server admin installs lancerguard since it will make things a lot easier on all players , however since this is not the case , below you will find a way to quickly render an ubershield useless


locate the file ST_equip.ini
unpack it with BINI (to be had here on this board) if it is not already unpacked
open the file with notepad or a similar editor
search for heavy_fighter_invulnerable_shield
replace the following lines

regeneration_rate =100000
max_capacity =1000000
toughness =10000

with the following

regeneration_rate = 1
max_capacity = 0
toughness = 0

and the result will be that everybody toting an ubershield as produced by FLMM or manual modding will have NO SHIELD making them easy pickings even if you are only equiped with standard guns.

The main advantage of this is that you do not need in any way or form increase the fire rate of your guns (which causes tremendous lag) and that is it hardly a mod any admin can object to since it only makes their servers less prone to crashing and abuse

Feel free to transfer this message to other places, and to head off any requests NO i am not going to turn this into a flmm mod , it is just not worth the trouble to do so.
