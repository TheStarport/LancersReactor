@OMEGA (et. all)

One thing that always seems to be an issue with computers and modems and routers. They work better when you shut them down and restart them. I do that myself on occassion to "clean" up the line. Broadband users tend to not reboot very often either (I'm living proof of this, been connected for 4 days plus now). My suggestion (albeit it a generic one) is to start with a fresh boot. Be sure you unplug any power and phone lines from a DSL modem. Same with Cable, turn the router off for a few seconds. Then plug evrything back in and see if it makes a difference.

It could just be heavy internet traffic as well.

Stinger

Sun Feb 16, 2003 4:38 pm by Stinger

Freelancer uses the standard DirectPlay8 ports 2300–2400. Ports are allocated starting at 2300, working toward 2400.

This port range is used by both the game client and game server. The game server uses it to connect to the list server and to accept incoming connections from game clients. The game client uses it to connect to both the list server and the game servers.
