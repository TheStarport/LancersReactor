While Theoretically possible... the PITA factor on that project'd be about 11. You would essentially need to add several new systems... to each system already in the game. you would also need to modify the starspheres in these new systems to make it look like you were flying through an atmosphere, as opposed to space. you'd then need to change the object entry for each and every dockring you wished to go to a flyable atmosphere instead of straight to base.

BUT: this would have some interesting side-benfits: multiple (almost unlimited!) bases on each planet without multiple dockrings.

The look and feel of flying in an atmosphere, though the physics would still be Space physics, no gravity on planets.

But, if anyone feels up to the challenge of modelling new starpheres to make planets, modelling new objects to make cities, and doing all that lovely coding, it could be done. I just doubt it would be worth the effort.
