newbie modding questios

Okay, I've been reading the tutorials and have downloaded SDK and UTC. I haven't seen the question and answer I'm looking for. Actually, I have a two part question;

1) How do I edit a saved game file to increase the level; Say, from level 5 to 32?
2) If I can't do that, how do I edit a ship's level requirement in the shiparch.ini file? for instance, if I wanted to plant a Titan at Detroit Munitions base that I could purchase for 50,000 credits as a level 5 player, how would I do that?

Ok here goes...

1) To edit saved games you have to download ether the Ioncross character editor or the simple save editor.

2) To edit the level requirement of a ship open shiprach.ini, find the ship you want, (i think the titan is co_elite2) then find the line saying 'ship_class = 10' and replace the ten with the number you want.

And Thats it.

Sat Dec 02, 2006 2:22 am by ugur

You have to change in market_sihps.ini the level requirement for a ship:
marketgood = coe2_package, 32, -1, 0, 0, 1, 1, 1; the first number after coe2_package is the parameter for the level requirement.

You also have to enable that particular ship to be bought on that station in market_ships.ini
But that's not all if you want to place it on Detroit Munitions. You'll need at least a ship dealer there. The FL Explorer will allow you to change the characteristics of that base so you can sell a ship on it.
I think you can keep the basic base template, but you'll have to change the rooms in it to perhaps another template that has a ship dealer.
