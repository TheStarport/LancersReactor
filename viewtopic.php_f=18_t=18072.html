As you can see the tradelane has the name: 260920. But if you look that up, it says nothing, so you can just use 260920 for every new tradelane you add. The tradelane_space_name part is the important part that you need to mention right, so in this case 196776 in number 16 is Fort Bush and 261658 in lane 20 is Colorado Jump Gate.

So if you go fom Fort Bush to the gate it will say: Fort Bush -> Colorado Jump Gate and if you go the other way around it will say: Colorado Jump Gate -> Fort Bush.

That's all about naming tradelanes.
