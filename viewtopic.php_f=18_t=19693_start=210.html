@ evolution.
OKay - to make a server where poeple can mod? Just don't have the patch or anything else! Keep the barebones of the server running for everyone to be able to mod like crazy. However - beware. Some people take the piss on those severs and so it will crash one heck of alot as they overdo it (ping will be skyhigh).
Otherwise - install your own versions of mods on the server, and leave something in the downloads section for everyone else to download. Only problem is that starting a new server is hard - just by the fact that its hard to get noticed that is! Good luck with it.
For the battleship at Ithaca - you mean you want a station there for you to use? which then sells the hammerhead?
Will give it a go. Basically you just need to add a station (osiris or something similar) to the li01.ini at the bottom. Copy and paste rooms from another battleship, then edit the rooms names to have the name of your station (as in li01_13_bar.ini etc. Now change your marketships.ini to include this station and to be able to sell the hammerhead. Then edit what stations selll what were (weapons etc) and finally add it to the universe.ini as well. It is a bit of work - but if i get much spare time - i will have a crack for you.
