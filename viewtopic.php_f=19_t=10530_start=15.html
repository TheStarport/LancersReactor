StarNova

Mon Mar 24, 2003 1:07 pm by StarNova

Go look in your C:\Microsoft Games\Freelancer\EXE folder and find a file called mpnewcharacter

Open the mpnewcharacter with notepad and look for something like this:

[Player
name = %%NAME%%
;initial_rep = %%FACTION%%
rank = 0

money = %%MONEY%%
;voice = %%VOICE%%
;All must be Trent!
voice = trent_voice

costume = %%BASE_COSTUME%%
com_costume = %%COMM_COSTUME%%

system = %%HOME_SYSTEM%%
base = %%HOME_BASE%%

And change it to this

[Player
name = %%NAME%%
;initial_rep = %%FACTION%%
rank = 0

money = 15000
;money = %%MONEY%%
;voice = %%VOICE%%
;All must be Trent!
voice = trent_voice

costume = %%BASE_COSTUME%%
com_costume = %%COMM_COSTUME%%

system = %%HOME_SYSTEM%%
base = %%HOME_BASE%%

That should do it. Good Luck and be sure to backup any files before making any changes to them.
