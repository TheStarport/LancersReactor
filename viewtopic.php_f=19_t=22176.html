I have posted the solution to this in several threads here and on Lancers HQ.

It's not your system, the Global List Server is down. It happens occasionally.

the file FLIPDIRECT.HTA in your freelancer program directory will allow you to use server IP address to connect directly... simply double-click it. When inputting the port be sure to use ??? instead of a number if you don't know the port. Some servers don't use the standard ports. From what you said about your firewall, this should not be a problem for you.

Some server IP's are posted here in the Server forum. Otherwise I suggest you write down such info somewhere so you have it handy.
