1. You need to use FLED-IDS. You can find a link on making "Infocards" in the Tutorial forum.

2. Again, check out the tutorial forum for editing and creating new engines.

3. Just set the Thrust really high and have a high drain, that will let you go fast, but not for too long. Though you can always drift at high speed using engine kill.

4. Combinable means that you can add things on top of each other. I think you can do this with lots of things, but I don't know any right off.

5. I don't think you can make an engine run out of fuel, but you can make the thruster run out of energy. Just have 0 regen and a low drain. It will recharge when you dock.

6. Get a free email address that has a large file attachment size and email the mod to Lancers. You could also look for a free online hosting site (there are planty) and send the link to whoever needs it. Depends on how big your mod is I guess.

7. It's simple. Zip it using winzip or something, then rename it and add .flmod to the end as the file extension. It will look like "Mymod.zip.flmod" - make sure your Folder options are set to show extensions.
