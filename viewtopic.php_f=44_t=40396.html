Argh

Sun Aug 14, 2005 12:10 am by Argh

Any changes made to the HUD were done using primarily UTF Edit, which allows us to alter the contents of the various 3db files used for things like the HUD. That's all that's going on here.

As for Cinema4D... erm... well, it's kind've crappy for low-poly stuff, imo... it's really not designed for game work. Other than that basic problem, it's a pretty decent modeler. Unfortunately, that's a pretty major problem... and if you aren't super-committed to C4D, I'd strongly suggest using GMAX instead, if you want to make models... or pick up a student copy of Rhino3D (which is what I use), because both of them are better modeling environments for game modeling.
