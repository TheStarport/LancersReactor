Simply find the "EXE" located in
HD Letter here>>X:/Programs/Micrsoft games/Freelancer/EXE

then scroll down til you see a file called

mpnewcharacter.fl

you can sort by file type look for "FL file" (should only be 2 and one of them is

*mpnewcharacter.fl

(right click) the *file, select (open) or (open with), next a window will (pop up) saying windows cannot open the file and abunch of blah blah MS crap..

at the bottom of this window are two options to choose from..
(choose) the bottom one "select a program from a list"

**Next a window pops up**
in this window is a list, and one selectable option**
select notepad

**Option(all read)"Always use this program to open or use this type of selected file"
For god sake..read it again!
(being on the safe side)

"Always use this program to open or use this type of selected file"
SELECT (NO) *UNCHECK IT*

if you dont you just, screwed yourself royal!

But we'll assume u unchecked it..and have select Notepad as the program to open the file..

once its open look for a line with this in it..

money=%%MONEY%% (delete ->>%%MONEY%%)
and add the amount of money you want NEW players on your Multiplayer server to start with..ex) 25000

Table
----------
25000=25,000
40000=40,000
1000000000=1,000,000,000

that simple..no programs needed..just Notepad!

Steps again, but short explaining..
1.look for file mpnewcharacter.fl, in your EXE folder

2.right click the FL file, open with notepad
(make sure that the "default options" is not checked)

3.Look for the line "MONEY=%%MONEY%%"

4.change "%%MONEY%%" to the amount you want.

save file and your done..

Ps. dont change the name of the file, or it wont work..
