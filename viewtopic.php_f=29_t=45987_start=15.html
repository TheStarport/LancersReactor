Rankor

Tue Jul 24, 2007 11:52 am by Rankor

In the particular universe/system folder where you'll find your planet listed.
If for instance your planet size is 4000, make sure the atmosphere is larger and also "planet_death" about the same size as the atmosphere.
example:

[Object
nickname = St04_01_Base
ids_name = 196842
pos = -589, 0, -2712
<span style="text-decoration: underline">Archetype = planet_icemntcld_4000</span>
ids_info = 65809
spin = 0, 0, 0
<span style="text-decoration: underline">atmosphere_range = 5000</span>
burn_color = 255, 222, 160
Base = St04_01_Base
reputation = fc_or_grp


[zone
nickname = Zone_St04_order_planet_death
pos = -589, 0, -2712
shape = SPHERE
<span style="text-decoration: underline">size = 5000</span>
<span style="text-decoration: underline">damage = 120000</span>


================================

Also, make sure the "damage" is set high enough to hurt a ship.
The default damage is something like only 2,000 which is not enough.
So, you simply manually edit the damage portion to a larger number.
