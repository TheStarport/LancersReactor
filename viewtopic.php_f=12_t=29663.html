OK im back, this is your most likely option at this stage, however registry keys can be hidden in a number of places so nothing is guarenteed. I'm also assuming your using Internet Explorer.


First and foremost only follow the steps Im giving you here cause you can really do some damage to your system if you screw your registry up.

1. Click start and select RUN
2. Type "regedit" into the field and click ok.
3. Navigate to the following directory: HKEY_CURRENT_USER\Software\Microsoft\Internet Explorer
4. In this folder you hopefully have a "MenuExt" folder, if not the key is hidden somewhere else and this isn't the right fix.
5. If you do have a MenuExt folder, go into it and you should find your offending registry key. If not exit regedit and once again this aint your fix.
6. If you do see the offending key (and make sure it's the right one), delete it, close regedit and restart your 'puter.
7. Once you've restarted, the option, with any luck, should have dissapeared.
