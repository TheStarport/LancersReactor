Tue Aug 05, 2003 4:54 am by T-hawk

Doh! I got the name wrong, hehe

It's called RivaTuner, and I got it from here:

http://guru3d.com/rivatuner/

The option you want is here: in the Driver Settings box on the first tab, click on the little Customize triangle on the right, and click on the second icon for Direct3D settings. Go to the VSync tab in this menu, and turn the Prerender Limit down to 0. (Also turn synchronization with retrace off or you'll get a terrible performance drop. You'll have to live with "screen tearing" but this is not badly annoying in an open-space game like FL, since most of the background is usually stable from frame to frame.) The utility's documentation warns that Prerender Limit = 0 can be unstable with some games and suggests using 1, but 0 works fine with Freelancer.

I'll wager a Sunslayer against a Javelin that this is indeed your problem. You've got a GeForce 2 MX, same as my system.

One other tweak that you can do: in the Freelancer\EXE directory, edit the file FLConfigDatabase.txt . Search for GeForce in the file, and add the line NoTripleBuffer in the appropriate section (you should be able to figure out how.) This reduced my mouse lag problem somewhat before I found the RivaTuner setting, and it feels like it still makes a slight difference even with the prerendering turned off.
