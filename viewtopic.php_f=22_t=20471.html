FLPING (available from http://www.flserver.com/), as well as being able to output to the console, can generate a HTML table with server stats which can be merged or scripted for publication to a webpage.

The source code is also available so if you're a programmer you could probably adapt it to your own use.

Alternatively you could re-direct it's output to a text file and use a script to pick out the information you need. I use this technique to populate the server status on http://www.cyberphobia.net/freelancer/.

I think Axdrenalin was actually asking for a program to show all the players on the server. I have written such a program (called FLstat), and I know that Axdrenalin is now using it very successfully on his server.

FLstat loads all the character files (big thanks to Jor and Sherlock for decode algorithm), and sorts them by Rank, Money and Kills. The resulting output (see example) shows most of the info that you get on the player stats screen in the game.

Update: You can download the latest version (v1.3) from TLR, or you can read this page for more details. This newest version includes significant bug fixes and speed ups.
