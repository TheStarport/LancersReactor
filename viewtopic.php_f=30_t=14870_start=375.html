shsan

Fri Jun 13, 2003 9:00 pm by shsan

The rotation number describes a rotation in space.
the X axis is the one across the screen horizontally (left right)
the Y axis is the one into the screen
the Z axis is the one vertical on the screen (top or bottom of the screen)

So if you rotate only in Y, it's rotating the object in the plane of the system.
a rotation of 180 degrees in X, will put the object upside down.
combining the angles will give you any possible orientation.

I am afraid that in 2D it's not easy to represent that. If you want to do one only in Y, then you can use the jumphole rotation to have the number. first change the rotation with the dedicated interface(right click), then ask to change the rotation with the generic menu (use shift-right click).
Hope this help.
