Ideas for modders...

Hi all.

I have to admit, I don't know anything about modding this game, but I came up with an idea that some capable modder could do...

Could it be possible to add some kind of material usage to ships? Like H-fuel, oxygen, water and so on? This would add some depth for example to trading in long routes since you should make stops at stations for refueling, taking more oxygen etc. Or if there isn't any bases or planets on route that sell them you'd have to take supplies with you.
As I see that many people are very interested of flying big capital ships, this could add some challenge to that too as the huge ship would consume lots of fuel and other supplies.

Ofcourse there is still big holes in this idea like for a starting players the h-fuel is far too expensive etc. but i think that if we put our minds to it, these problems can be solved.

What do you think about this idea?
Is it even possible to do?

Regards:
Noir
