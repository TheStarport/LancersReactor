Different Starting ship MP..Definitive help wanted

Hi all,

I have searched these forums for days trying to find a definitive answer but all the postings lead nowhere and end up with just lots of different techniques.

I just got off the irc chat with the The TNG mod team where I got some brief help with changing the starting ship using a server side only mod for a new player in a MP game.
I don't think it would make any difference but I am running the latest Rebalance mod as well.

What I was told was that all server side changes to get a new ship were made in the \DATA\CHARACTERS\newcharacter.ini which I did and here's how it looks:

[Pilot
nickname = trent
body = trent
comm = trent
voice = trent_voice
body.anim = Sc_MLBODY_STND_000LV_A_31
thumb = interface\intro\objects\MULTI_blackdragonmale.3db
comm.anim = Sc_comm_body_unfold_female, Sc_headCOM

[Package
nickname = bh_elite .....Baracuda
strid_name = 11051
strid_desc = 11551
ship = bh_elite
loadout = gd_bh_bh_elite_loadout01
money = 80000

[Faction
nickname = new_player
rep_group = co_ic_grp
base = Li01_01_Base
Package = bh_elite
Pilot = trent

The only other changes are to the MPNewcharacter.fl which look like this:

[Player
name = %%NAME%%
;initial_rep = %%FACTION%%
rank = 0

money = 80000
;voice = %%VOICE%%
;All must be Trent!
voice = trent_voice

costume = %%BASE_COSTUME%%
com_costume = %%COMM_COSTUME%%

system = Li01
base = Li01_01_Base

I didn't change the %%PACKAGE%% at the end of this file because I was told I didn't need to, although I have read on other threads that it had to be changed to the package that was used like bhe_package.

Now after making the changes I try to log onto the server and it says something like character id and password are taken no matter what I try its always the same.

Thanks again for the help on irc and for any replies to this thread.
