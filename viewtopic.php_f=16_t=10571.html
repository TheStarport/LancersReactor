pmak0

Tue Mar 18, 2003 5:26 pm by pmak0

That must be the hardest mission in the game so far; I just spent like 45 minutes retrying it over and over until I got it right.

It doesn't help that there seems to be no way to skip the animation sequences, so I have to listen to Hogis repeat the rules each time I fail twice.

Things I noticed that might be useful to anyone else attempting this mission:

- If you bump Hogis near the beginning, he'll say "Watch it!" but he will fall behind.
- As soon as the race begins, hit Shift+W to engage cruise engines.
- Make sure to steer properly into every gate; if you miss even one gate, you probably won't be able to recover. Some gates require tricky turning.
- Load up on a lot of Countermeasure Flares before the race, and use them freely whenever you hear "Incoming Missile", which happens in the latter half of the race. I think those are Cruise Disruptor missiles (not sure, never actually got hit by it).
