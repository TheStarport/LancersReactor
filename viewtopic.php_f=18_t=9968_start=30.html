Sat Mar 22, 2003 8:13 pm by X-Lancer

any ideas...?


You are probably not saving them in Unicode. If you have WinXP and using Notepad make sure you select "unicode" and not ansi in encoding field. Above that I have found that those XML files (if you extract them using ResHack) have some kind of 2 byte header with values (255, 254). I dunno if that's a part of unicode or a part that xml parser uses or some kind of identifier for the game engine but I am keeping it intact and all seems to work fine.
