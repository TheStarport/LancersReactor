For some reason something in your system is changing the mod extensions.
Nothing has a .zip.zip file extension.
So something is changing the file name to .zip.zip from .zip.flmod

If you cant find out what is causing this such as your firewall to fix the problem
just rename your files.

Example if it's a mod right click on the file name and choose rename
then rename the mod to whatever the mod file name is
lets say the mod file name is close_encounters_in_sirus.zip.flmod
and after you download the file it comes out as close_encounters_in_sirus.zip.zip
rename the file to close_encounters_in_sirus.zip.flmod

If you are just trying to extract (unzip) the files rename the file by deleting the .flmod extension so you are left with a zip file.
example close_encounters_in_sirus.zip.flmod
rename the file to close_encounters_in_sirus.zip
so now you just have a zip file and not a flmod file.

the .flmod extension tells the Freelancer Mod Manager to unzip the mod file to the Freelancer Mod Manager\mods folder...

If you're still confused post here and some smart character will help you out.
