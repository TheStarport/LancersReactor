Messed Up Screen On Exit - SOLUTION!!

I am fed up of constantly seeing new threads about the EXACT same problem. I would appreciate it if an Admin 'pinned' this thread. So if someone has this problem, instead of all of them starting their own thread, they can simply refer to this one.

Now onto the matter at hand. Based on observations I've made, I have 2 different solutions for the problem.

Solution 1: When I alt-tab, it does the same thing as when it exits (messed up screen)...but when I alt-tab out again or exit, it doesn't happen. So my first solution is that you alt-tab out and back in before you exit. I don't know how it'll work on anyone elses computer, but it usually works for me.

Solution 2: Another observation I've made is that it NEVER happens when the resolution or the color depth is different from the desktop settings. Therefore my second solution is that you simply make the in-game settings different from the desktop settings. e.g. most of you will be using 32bit color for the game, so just change the desktop color depth to 16bit. Afterall, I don't think anything on the desktop actually needs 32bit color. Well, nothing that I know of.

That's it. It might not sort the problem for everyone, but I hope it helps a few people.
