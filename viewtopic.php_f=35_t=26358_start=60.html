@Greame: First off, the ATL71.dll file must be installed in the System32 folder. There are two files, one for Win 2000/XP and another for Win 98/ME. Make sure you have the correct version installed. Also, make certain that the mod is deactivated in FLMM before attempting to edit anything. I can tell you right now that if you click "YES" to the warning message about your custom dll file having the Windows signature that this is 100% of your problem. I have seen this problem happen when attempting to add new rumors in the bars, I haven't seen it with infocards, but there's always a first time I guess.
