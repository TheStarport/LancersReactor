Fri Jul 15, 2005 10:51 pm by Argh

To fix the problems with System/Planet/Solar names... go get Warriors of the Sky, Beta 1, where I did so with XML...

WOS Beta 2 has a more elegant implementation, but I'm not releasing it at this time, as the codebase is still very messy and several technology steps aren't finished yet, but basically, aside from new Factions (which must be named in a DLL according to very strict numbering conventions- there's a thread about this in the Tutorials, I believe), you can name / describe just about anything with XML. In WOS, I basically process the entirety of Universe and a lot of Mbases.ini...

Oh yeah... last note: if you do this correctly - see the notes that come with the XML Toolkit mod for information about how to implement parsed XML/FLMM scripts correctly (basically it has to do with code execution order issues) then the DLL entries created by FLMM will be the same for every client and the server- names will be right. This has been tested and re-tested in MP, so don't believe the naysayers who say that XML mods screw up names in MP... they just didn't code things correctly
