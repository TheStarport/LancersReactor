Hi there...

Norfolk shipyard is Li01_07_base. Go into your marketships.ini file and look for that (hopefully it is already decrypted, otherwise use bini or bini lite to decrypt it, which will make a text file which you will have to rename to the ini). Any ship that is actually for sale will have these numbers after it....

0, -1, 1, 1, 0, 1, 1

If there are more than three ships like that, only the last three in the list will be for sale (only three ships can be sold per base) and you will have to change the numbers on some ship you don't want or just "comment out" the line with the unwanted ship with a semicolon ( which will make it non-readable to the game.

But you probably already knew that.
