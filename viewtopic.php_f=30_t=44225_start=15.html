@ Wizou
Thank you for putting in the manual selection of the FL directory.

To see if a hole or gate is locked, take a look into the \DATA\initialworld.ini file.

Explaination:
"npc_locked_gate" are those gates and holes that are locked from NPC use, but can be used by players.

"locked_gate" are those gates and holes that cannot be used by either human players nor NPCs. That's what needs to be parsed.

Welcome to the DLL hell. Various Programs requiring various versions of a DLL file. If a program tries to access something inside a DLL that's not there (because the version of the DLL on your system is too old), it spits out error messages like yours.

Provided by the download link below are two versions of this DLL file you mentioned from my system. One of them should work, however I'd be surprised to see if there wasn't missing at least one more such DLL file. Only thing you could do is to feed the missing DLLs to the program until it's satisfied.

Normally those DLL files are found inside your Windows system directory, but putting them into the same directory the program is in should work, too.

Here you go:
http://rapidshare.de/files/28363032/mfc42u.rar.html
