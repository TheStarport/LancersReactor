Get the ini decrompressor (BINQU or something, in the downloads section)

In the market_ships.ini look for the line

marketgood = coe2_package, 32, -1, 1, 1, 0, 1, 1

... and change the 32 to whatever level you want. There will be several entries for Leon, Crete etc. Can't remember the base name .. EW03_01 perhaps or Hi02_01, not sure. Just change them all.

*edit* PS remember to back up your ini's first ...

Look around for links to how to decompress and save the file, there are lots of posts about it. Half the fun is finding out stuff yourself but this is a start.

GS Destroy

Tue May 27, 2003 5:36 am by GS Destroy

First, download BINI. Unzip it to a directory. Now, find the ini file you want, and right click on it. Click open with (or go to open and then open with. something like that.). Now instead of selecting a program from the list, click browse. Go to the directory where you unzipped bini. double click the exe labled bini in that directory. this should take you back to the list, and bini will be selected. go to the bottom and make sure that the box that says "always open this type of file with this program" or whatever, is UNCHEKED. now, click open. It should create a copy of the file in the same directory as the original ini file, only this one will be followed by .ini.txt. You can edit this one, then rename the old one, and delete the .txt off of the new edited one so it has the EXACT same name as the original, including the following .ini. It cannot have .ini.txt, or the program will not run. Good luck.
