Hmmm,

My understanding of how the MP game works is that the character is stored on the server. The only way then to transfer them effectively would be to share the MP server files. This can be done by copying the file, or if you know how to figure out who's is who's IN the file folder, then just the files within that you want transferred. I am not sure if this would work but its worth a go.

The files are small so they should easily be transferrable over any filesharing IM program (Such as Yahoo or MSN) and you can always just e-mail it.

Just if you do this (assuming it works) remember to transfer the updated file BACK to your local server so that you have the updates from playing together.

One downside, I do not think you can use a SP game character in MP. I could be wrong there as Ive never tried it.

I did not think the IONCROSS Server Operator would do this but I could be wrong there. I know the SP IONCROSS Character editor wont work at all on MP files.

If there is another way Ive no idea, but thats how i would do it were I of mind to..
