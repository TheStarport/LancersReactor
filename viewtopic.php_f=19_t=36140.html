Earendil

Sun Jan 09, 2005 5:16 pm by Earendil

I agree with Pinger what the average FL server needs except for the Internet connection. These average 'puters coupled with a good ADSL line (like 1.5M/384k downlink/uplink) can get you nearly 15 players with a little lag on an original FL server install. I run an Athlon XP 3200, two ADSL lines and an NPC-reduced server-side mod so I can handle over 30 players.

I have heard of FL server's running on 600 MHz 'puters but player amounts are real low, maybe 2-4 players.

Any higher spec'd 'puter must have accompanying Internet bandwidth like T1, T3 lines and higher or else the server power is a waste. It should be obvious determining what a server needs to run must take the Internet connection's bandwidth into effect.
