I had deleted any other version of reynan's nods, and have no other mods active (essentially from a clean install of FL & FLMM). I've played a bit of SP with no trouble, and even for a few minutes on B[tch's server, before it crashed, presumably because someone tried using the old mod. So, to sum up:
1. Delete, using FLMM any old version of Reynan's mod.
2. Deactivate, using FLMM all other mods.
3. Extract all files from Reynenmod206.rar (6.4 M into a single folder.
4. Place that folder in ...Freelancer Mod Manager/mods/
5. Activate, using FLMM, Reynan's Mod 2.0 BETA 6
6. Enjoy

Hope this works for y'all too.
