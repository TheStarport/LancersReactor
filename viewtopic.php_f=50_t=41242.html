Black Eagle

Sun Oct 09, 2005 3:15 pm by Black Eagle

Hi!

One question: Did you start a totally new game, or did you use an old savegame?

Make sure that after activating your mod the folder My Games/Freelancer/Singleplayer is totally empty before you start the game. Then start a new game and go to Omicron Alpha.

This is, because the problem you're describing is known and appears, if you add new factions, but use old savegames. Then the game only looks into the savegames for faction relationships and cannot find your new faction, so that it is neutral to everyone.

So, just start a totally new game (no old savegames in the savegame folder) and the problem should be solved.

I hope this helps!

Black Eagle
