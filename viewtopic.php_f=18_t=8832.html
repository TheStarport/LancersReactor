how to get into locked systems in MP (spoiler)

sorry if this has been posted already but i haven't seen it.

here's what i did (i admit to being a total editing newbie). decompressed initialworld.ini in the data directory and opened it with wordpad. i'd read that it only had to do with sp. i thought it was a lost cause until i heard a freighter say "hauling sidearms to the alaska system." that sparked my interest, and sure enough the jumpgate worked on my own mp server.

well, what i did was remove all of the locked gates lines from the initialworld so it ends up looking like this

<pre><font size=1 face=Courier><font size=1 face=Courier>
[locked_gates
; Mission 4 ;
; Mission 5 ;
; Mission 6 ;
; Mission 9 ;
;
; Mission 10 ;
; Mission 11 ;
; Easter Egg Systems - Player is allowed to go to these systems, but not NPCs </font> </font></pre>

since i ran the server off the same files i ran the client off of i'm not sure if it's server or client side. it should work in other systems but i have yet to try it. hope this helps someone. (flame the newbie)
