example only see the bold area. If you changed it in the freelancer/data/ship.ini, then it should be that way in the game. Move the mouse curser over the Hold bar and it will show you what it actually is, not what the info card says.

type=FIGHTER
mass=400
hold_size=999
linear_drag=1
fuse=li_fighter_gas01, 0, -1
fuse=li_fighter_smoke01, 0, -1
fuse=intermed_damage_smallship02, 0, 400
fuse=intermed_damage_smallship03, 0, 200
max_bank_angle=35
camera_offset=9, 63
camera_angular_acceleration=0.05
camera_horizontal_turn_angle=23
camera_vertical_turn_up_angle=5
camera_vertical_turn_down_angle=30
camera_turn_look_ahead_slerp_amount=1
nanobot_limit=220
shield_battery_limit=220

Use Subxeros misc database to give you the nicknames that the game uses to find ships in the shiparch.ini file, and once you change values, RELOAD the game and then try it.
