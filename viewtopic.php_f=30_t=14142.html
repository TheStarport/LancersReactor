two ways
1) in ms-dos, find bini and run it like so:
bini.exe <path to compressed ini file>

2) drag the compressed ini file over bini and let go.

in both cases it should create a file that is named the same but with .txt at the end. then rename the .txt one to .ini just remember to backup whenever you change something in case you do something wrong
