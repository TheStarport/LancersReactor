errm.. Maverick.. i think you mean the MAT file. That stores the materials (textures) used on that model. The CMP file contains the actual ship's model as well as information on the position of hardpoints. Anyway, you can open up the MAT file in a UTF editor (one is available to download on the main Lancers Reactor site) and export the data of the leaf nodes (likely called MIP0, or MIP5) within the "Texture Library" branch. I've never done it before, but if the node is called MIP0, export, and save it as a .TGA file.. if it's called MIP5, export and save it as a .PDD (i THINK, although not too sure about that). Anyway, good luck making it work, and good luck to Maverick fixing the mod.
