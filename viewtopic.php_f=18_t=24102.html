Projectile Color found

All the weapon project effects/color can be found in /Data/FX

Most of them in the beam_effects.ini
However the projectile colors can be easily modified, but it won't work on Multiplayer unless the server and client have the same color code.

[BeamSpear
nickname = Li_laser_03_beam
tip_length = 10
tail_length = 100
head_width = 1.500000
core_width = 1
tip_color = 0, 0, 0
core_color = 60, 60, 215
outter_color = 40, 40, 210
tail_color = 60, 60, 215
head_brightness = 1
trail_brightness = 1
head_texture = star
trail_texture = wide
flash_size = 3
