(slight spoiler) If you're still in the story, you're not going to be able to get a banshee.

If you're in the "free-form" game (post-story SP or MP), then either take a bunch of hard missions and blow things up (you'll have to travel to hard bases to get hard missions), or buy a freighter and make high-value trade runs.

A really, really, cheesy way to make money when you're broke is to blow up supply depots near bases, steal the goods that fall out, then run away and sell them somewhere.
