parabolix

Tue Feb 28, 2006 12:45 am by parabolix

Just to start off...After years of modding Freelancer I can offer some quick insight...

-Creating a new class of ships, the assualt fighter, a suped up heavy fighter


This isn't possible. Freelancer supports just 4 classes of ships, 0, 1, 2, 3, or, Light Fighter, Heavy Fighter, Freighter, Very Heavy Fighter - respectively.

-Replacing all the original systems, and adding some for a total of 66 systems.


That's quite a number of systems you're aiming for. Be sure you're ready to take on the challenge. Also, it's a bad idea to have many people working on systems because when you make a system you not only edit that system but any system that has connections to it (E.G. New York -> California).
