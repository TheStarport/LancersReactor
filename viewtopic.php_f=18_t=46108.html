Ok, since you obviously can't read any tutorials on how to add new ships, I'll at least tell you this; It doesn't matter where you "drop" a ship or weapon. It all matters on how you've coded the .ini files (or how you script them in a .script file if your going that path), and if you have their file path correct. But in order for FLE to read it, you have to have the shiparch filled out correctly and the goods file filled out correctly for ships. For weapons, it's the weapon_equip file and the weapon_goods files. Most importantly it the good files as that's what FLE reads, but you need both for it to properly work.

On a second note, in order to put anything that's not yours in a mod, you need permission from the creator. Just letting you know before you have problems.
