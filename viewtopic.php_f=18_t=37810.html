More information about what you did would help...
But most likely if you have been editing your files you made
a mistake...one that the game engine cannot compute...
So it chokes and crashes...
But you said that Freelancer wont start, so by telling what you did, like which files did you edit and so on might help.

That's why it is suggested to back up your files before you start changing things.
You can loose months worth of work if you don't have backup files...

If you cannot find the error you will have to do a new clean install of Freelancer
and then install the SDK or the Unofficial patch...
