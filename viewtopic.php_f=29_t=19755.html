In the .zip file are the files in folders like DATA and EXE? If they are all you have to do is change the end of the zip file to .flmod. (i.e. name.zip ---> name.zip.flmod).

if not.... well you will have to find which folders the files go in (i.e. shiparch.ini goes to \data\ships\ and market_good.ini goes in \data\equipment\) and create those folders. put those folders with files into a folder named after the mod in the ...\freelancer mod manager\mods\ directory. BUT, you most likely will not have to do that.
