Simplest wat to fix things. Delete and reinstall freelancer. 2nd, delete and reinstall FLMM, 3rd, download and install SDK1.3, 4th install your mods in FLMM. Do not install updates, or any other hash file. Open your saved games folder My Documents/My Games/Freelancer/accounts/Singleplayer and delete any saved games. Open FLMM and activate the Mods you want, noteing any in orange may conflict with others and those that show red will conflict with others and list those that it does conflict with.

The
said this "there was a problem while inspecting one or more of the mods in the \mods folder.
deal is, a mod may have been corrupted or missing files. The SDK fixes a lot of this. Give this a try and let us know how it goes. I have done this my self and it fixed a lot of problems. *Note, if a mod is crrupted, redownload that mod to fix it, getting rid of the old one first.
