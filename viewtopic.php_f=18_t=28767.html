@illero: does ur view move with teh ship or is it fixed? if its fixed its most likely an ini error (typo).

creating custom cockpits:

i will only handle the ini-part of it (not hte cmp - i just use existing ones, but those can also be custom made)


somewhere in shiparch.ini (in the section of ur ship):
...
cockpit = cockpits\liberty\l_gunship.ini
...

in the data\cockpits\liberty directory u create a file named l_gunship.ini
this will contain the cockpit and also the turret view info

something like this:
[Cockpit
mesh = cockpits\liberty\models\li_freighter_cockpit.cmp ;<---- the "look" of the cockpit interior
int_brightness = 0.500000
head_turn = 45, 10

[CockpitCamera

[TurretCamera
tether = 0.000, 20, 105 ; <-- offset of the turret view, in this case 20 up and 105 back
yaw_rotate_speed = 2.000000
pitch_rotate_speed = 1.500000
accel_speed = 5

thats it, there u go with a selfmade cockpit. I suggest to make a custom cockpit entry for each custom ship, this way u can set the turret settings for each one individually.
