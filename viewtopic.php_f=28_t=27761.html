Titan Mod feedback

Titan mod v2.2.5 made by Eugene



The Titan Mod gives you your choice of a Titan, Sabre, Eagle, Hammerhead or an Anubis at the beginning of the storyline and your choice of What Weapon, Torpedo/Cruise Disruptor, Shield, Mine, Counter Measure, Engine Speed, Thruster Speed, Invincibility, Infinite Power and Money.



Important NOTE:
Download and than extract titanmodv225.zip
By default it will extract into the folder you downloaded titanmodv225.zip into and make a folder called titanmodv225
next install Titan mod v2-2-5 a.zip.flmod
then install Titan mod v2-2-5 b.zip.flmod
then install Titan mod v2-2-5 c.zip.flmod
into the Freelancer Mod Manager by double left clicking the mouse button on the .zip.flmod file name.

Then you can
Activate titan Mod v2.2.5 a first
Activate titan Mod v2.2.5 b second
Activate titan Mod v2.2.5 c third



Even if Titan mod b and c shows possible incompatibility, still activate it. you will see that it truly is compatible. if you do not activate a b and c and combine all three sections of Titan mod, the mod would not work.



Then you can start Freelancer from either the Freelancer Mod Manager under File or you can start Freelancer the way you normally do.

Changes from the last version:
V2.2.5
A bug removed
Instead of all of the same guns, there are options for each gun.
This is a pre version of the mod, so there are three sections.
This may work for people that had problems with the last version.

Many bugs are removed, namely from some weapons.
More options, like Ships other than Titan, starting cash, and infinite power.
The ship selected shows up in the cinematics.
Changed reward at end of first mission to start of mission.
Rearranged modding format.
V2.1
More options , invincibility, countermeasure droppers.
Mines, torp/disrupter option and thrusters, engine speed option, both are split up into two options each to minimize confusion.
Bugs removed.
V2.0
Added five options to maximize freedom.
Removed weapons in cargo hold to make the game more fun, and less exaggerating.
V1.2
Added many more cargo, all the most powerful weapons in the Game.
Added huge amounts of cardamine.
removed bugs
V1.1
Added more cargo and combinable weapons.

V1.0

Included Nomad weapons
Placed missile turret

Put 71 batts and bots
An Advanced Thruster instead of a normal one.

Have a great time!
