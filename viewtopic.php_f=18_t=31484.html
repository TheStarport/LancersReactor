Anton

Tue May 18, 2004 3:05 am by Anton

In initialworld.ini you will need to comment out or delete these lines :

locked_gate = 2284213505         ;Li05_to_Li01 ;Li05_to_Li01
locked_gate = 2293678337         ;Li01_to_Li05 ;Li01_to_Li05

Then (and this is important) start a new character! Your existing MP chars and SP saves will store the locked/unlocked gate information...

Tue May 18, 2004 8:39 am by Chips

If its in single player - its not possible.

If its in multiplayer - then the server needs to be modded to open it up, as your save files on the server (your character) will have it set to be locked by default.

The server will require a total player wipe to make changes like this, because if it doesn't, then player file corruption occurs, and the gate is still "locked", even though the ini file says open!!!

it is not client side
