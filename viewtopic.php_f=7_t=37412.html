There are programs available here to extract the individual HOG files that the movies are packed inside of it.

Using those you can identify the .BIK files, that's the format used to store Starlancer Movies. To view the, you need a program called Rad and Bink, I think. Or something similar to that. You can download it online for free, just do a search for .BIK file movie viewers or something along those lines.

End result will allow you to view all movies in the game for all possible outcomes of every mission. This program also allows you to rip the movies into AVI Format, but alot of them have sound stored in another file so that isn't always a good idea.
