I was about to say there's a Star Wars mod called the Free Worlds and they changed a few costumes of the NPC,
Then I see that you're from the Republic Mod. Oh well !
----------------------------------------------------
in the file \data\missions\mbases.ini
find a person, and you'll see the line:
voice = rvp126
then go to \data\audio
you'll see that voice file. Use the utility UTF to open files, and import in new audio .wav files.
-----------------------------------------------------
in the file \data\missions\faction_prop.ini
you'll see the space costume such as helmets
- download the Freelancer Head Accessory Collection to see pictures of helmets, sunglasses, hats, etc...
you also see the pilot voice files in that faction_prop.ini.
------------------------------------------------------
to make new helmets, read
- Kasdia's guide to make new dds texture

http://www.lancersreactor.com/t/editing ... asp?id=206

and the guide on how to make new icons
http://www.lancersreactor.com/t/forum/topic.asp?topic_id=28277&forum_id=29

Hope these will help
