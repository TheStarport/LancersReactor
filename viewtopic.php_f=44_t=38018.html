Are you using 24- or 32-bit TGA's?

I have had similar problems using 32-bit TGA's whereby the areas of the model using them would be invisible.. Resave your textures as 24-bit and create a new .mat file.

Or you could just use .dds files and not have to deal with these shenanigans .

Ok I've had that problem also and I've never been able to figure out why but it always happened when I the first group in the list is the cockpit glass. Anyway this is how you fix it.

In milkshape highlight the 'glass' area of the mesh and duplicate it then delete the original. Make sure you assign the glass texture to the new group and then export.

That always did the trick.

There is probably a better way and if anyone knows one please share it.
