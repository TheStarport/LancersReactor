Actually, your problem there was that you installed an Open SP mod with the Freeworlds 1.66. Freeworlds 1.66 and later already has an Open SP built into it (though albiet, Freeworlds was designed with MP in mind rather than SP, so some things don't quite work out right in SP mode), so you don't need to install an Open SP at all. To run Open SP, just activate Freeworlds, load Freelancer and click new game.
