Aldebaran28

Fri Apr 01, 2005 12:48 pm by Aldebaran28

Check your version numbers. The current version of Freeworlds which is running on most Servers is Freeworlds 1.65 (1.66 is still under development). It can be downloaded from Lancers Reactor here or from the Freeworlds home site at www.freeworldsmod.com (which is where all the development news is also located). Make sure you have a fresh install of Freelancer and that you have also downloaded the Mod Manager (the program which enables you to active mods). Once you have installed mod manager, double click the Freeworlds file which you downloaded and that will automatically install itself into the mod manager. Then open up the mod manager, select Freeworlds 1.65 and then click activate. You will need wait a short while as the Mod Manager does it magic. Once that is done, close mod manager and open up Freelancer again. You should find the splash screen has changed and your version number on the title page be changed to version 1.65 (the version number is printed in the lower right hand corner of the title screen).

If you are still stuck, there is a section on the Freeworlds main forum at the address given above which deals directly with technical issues with the Freeworld mod.
