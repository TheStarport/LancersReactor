Earendil

Wed Oct 29, 2003 6:42 pm by Earendil

You can get all system chats (the blue colored chats) at the server! I regularly do this for NY system but not for any other systems as I would have dozens of chat boxes up all at once. The catch is someone must be in that system before you bring up the system chat box or it doesn't list it as a system chat box to be able to be brought up. While the system chat box is up all chats in the system from anyone in that system sent out as a system message will show up in that system chat box! I'm catching quite a few PvPers in Liberty (specifically NY system) because someone responds on system chat with something like "Hey N00bship, stop hitting me! Now I have to spend money on nanos!" or something like that.

To change someone in the game from one place to another is more than just changing the current location. Since the player is seeing and hearing things from that location, it is stored in memory and being accessed at that time. You would have to initiate an equivilant jump gate sequence from anywhere to anywhere OUTSIDE the game and I don't know of anything doing that. This is easy when you logged off because there is only one thing linking you to some place, the location as saved in the FL MP saved game file. The audio and visual clues are part of FL data files and are not be involved in a saved game except when starting up.

Earendil

Thu Nov 06, 2003 7:51 pm by Earendil

A player should see the chat message in green if CONSOLE is directly chatting to that player. Players will see the chat message in white if the CONSOLE is chatting to the star system (I think) or the whole universe (I know).

If CONSOLE is directly chatting to a player and the player wants to directly chat back to CONSOLE, hit enter and open chat window, hit up or down arrow to direct the message to CONSOLE and type away.

If CONSOLE is sending a message to the star system, he will see any system message any player in the star system sends. But if CONSOLE is sending a message to the universe then there is no way a player's message can be seen by CONSOLE even though he direct a message to console. It has something to do with how flserver's chat windows are setup.
