Team Speak - Free Voice Chat

The PortaLive.com Team Speak (Free voice chat) server is now up and operating so you can play multiplayer games like Freelancer, (my personal favorite) Planet Side, Battlefield 1942, Black Hawk Down, Metal of Honor and a ton of other games and talk to other players at the same time. Nice for talking to your wingman or fellow sniper.

Team speak also makes a good way to communicate with friends and relatives, since the sound quality is excellent and theres no long distant charges and no best of all no typing.

Of coarse you will need a mic.
Or better yet a Mic Headset combo.

You will need to download the TS2 client software which is FREE
at ----> http://www.goteamspeak.com/news.php click here
After you download and install the FREE software (client side version) from Teamspeak start up the software and this is what you will need to fill in.
Click on Connection, then click on Connect.
After: *You can just copy and paste*
Label, put this in: PortaLive.com Network Server
Server address, put this in: 69.93.236.42
Server Password, put this in: portalive
Nickname: choose your own nickname. Example: URA#1
Log on as Anonymous with a user name the first time and then you
can register with the server if you want to. Registering with the server allows you to create you own private channel.

A special thanks to: AlbinoEatPod and Devon from Portalive for providing this service to the gaming community.
