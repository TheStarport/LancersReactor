slug609

Fri Apr 04, 2003 2:29 am by slug609

Well, I hope I don't make you mad... but I don't think using a trainer in MP is a good idea. I understand that the servers aren't working out for you, but I honestly don't think it will be fair to everyone else... and besides, if you can only play for a few moments at a time, maybe you should consider running your own server instead...

Just find the program called FLserver.exe, located on your hardrive (typically c:\program files\microsoft games\freelancer\exe\flserver.exe), and run it. Pick a name for it and make sure the option is not checked to list your server on the internet. Keep that program running and run the regular freelancer program. Then select your server on the "LAN" option and play in your own MP universe, albeit with no other players... but hey, it will be stable! Just a thought.
