Maverickdan wrote "Hi everyone, now ive had a search around most of the forums i can find but i cant find anything on adding a new faction to the game "

It can be done. I am building a mod that incorporates es OpenSp by Xerx (because I don't like fixes missions etc.) and I modified the following files to add new factions:

DATA\initialworld.ini

DATA\MISSIONS\Empathy.ini
DATA\MISSIONS\faction_prop.ini.txt

DATA\MISSIONS\M13\M13.ini <----- this is specific to OpenSP; no idea if an other file should be used when not using OpenSP as a foundation.


The follwowing files also relate to the factions, but modifying them is not absolutely necessary:

DATA\MISSIONS\npcships.ini.txt <----- if you want your faction to fly in new npc ships (the section in faction_prop.ini.txt points tio npc ships)

\DATA\EQUIPMENT\commodities_per_faction.ini.txt

DATA\RANDOMMISSIONS\killablesolars.ini.txt
DATA\RANDOMMISSIONS\rmlootinfo.ini.txt
DATA\RANDOMMISSIONS\vignetteparams.ini.txt
