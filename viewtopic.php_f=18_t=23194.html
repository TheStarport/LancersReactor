Wed Aug 20, 2003 9:24 pm by Imagine

Sorry about that. Milkshape is at
http://www.swissquake.ch/chumbalum-soft/ms3d/index.html
Will put that in the tutorial too.

I didn't go over dll editing at all, most of the reason is that I don't know how to do it . I think the only difference is that you can put in a new ship without replacing an old one if you edit the dlls. HFH_Mack has a good tutorial on dll editing here.

As for hardpoints, you can put them on a ship in milkshape really easily: no math involved, just put a triangular marker where you want it. CMP export handles the basic orientation data.
Hardpoint editing only gets hairy when you try to do it outside of milkshape, ie, editing the .cmp file with utf editor. It used to be that you have to do it that way if you wanted to add hardpoints to a ship already in FL, but now there's a CMP import utility that will put the ship back into milkshape, where you can re-hardpoint it.
