MP trade routes profit analysis

Did some testing tonight, these are the results for the more famous routes. Route times include time spent planet-side (ie buying & selling stuff, repair, etc), trade lane interruption & pirate evasion (I always run in a freighter), and gate queues. All times rounded up to the nearest minute.

-----------------------------

1) Cadiz Base (buy Alien Artifacts) -> Planet New London via Cambridge jumphole (buy small arms) -> Cadiz Base

Trip time: 15 minutes
Profit: 6008 credits/unit-cargo/hour
-----------------------------

2) Planet Manhatten (buy Luxury Consumer Goods) -> Ainu Depot (buy Cardamine) -> Planet Manhatten

Trip time: 32 minutes
Profit: 4277 credits/unit-cargo/hour
-----------------------------

3) Planet New Tokyo (buy Luxury Food) -> Planet Honshu (buy Optronics) -> Planet New Berlin (buy Diamonds) -> Planet New Tokyo

Trip time: 26 minutes
Profit: 4230 credits/unit-cargo/hour (perishable involved, assuming using a Dromedary)
------------------------------

4) Cardiff Mining Facility (buy Beryllium) -> The Ring (buy Engine Components) -> Planet Cambridge (buy Water) -> Cardiff Mining Facility

Trip time: 30 minutes
Profit: 4222 credits/unit-cargo/hour
-------------------------------

5) Ruiz Base (buy Alien Organism) -> Cambridge Research Centre {via Tau-37} (buy pharmaceutical) -> Planet Leeds (buy Small Arms) -> Planet Malta (buy Oxygen) -> Ruiz Base

Trip time: 48 minutes
Profit: 3690 credits/unit-cargo/hour (perishable cargo and radiation involved, assuming using a fully fitted out Dromedary)
------------------------------

6) Planet Manhatten (buy H-fuel) -> Detroit Monitions (buy Small Arms) -> Montezuma Base (buy Cardamine) -> Planet Manhatten

Trip time: 21 minutes
Profit: 3569 credits/unit-cargo/hour
------------------------------

7) Roppongi Station (buy Luxury Consumer Goods) -> Ainu Depot (buy Cardamine) -> Roppongi Station

Trip time: 11 minutes
Profit: 2766 credits/unit-cargo/hour
------------------------------

Planet Manhatten (buy H-fuel) -> Buffalo Base (buy Cardamine) -> Planet Manhatten

Trip time: 9 minutes
Profit: 2367 credits/unit-cargo/hour
------------------------------
