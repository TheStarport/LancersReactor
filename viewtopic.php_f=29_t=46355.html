**Tutorial** - Applying Factions for SP!

Has your game ever crashed when you have created a new faction? Or is the faction just not showing up properly? Then this is the correct place for the Whys?, Whats? and Wheres?

This thread, and below, will explain to you how to solve this mystery. Some people may have heard that it is impossible for factions in Single Player. But those people are wrong!

The Cause: You have not defined the factions correctly for Single Player. It will show up in Multi Player because you have defined them correctly using 'mpnewcharacter.fl'.

Modding a '.fl' file is not that hard once you have the correct software and a knowledge of editing 'mpnewcharacter.fl' and other '.ini' files. All you need to begin with is a Freelancer Saved Game File Decoder. I could suggest one but it cannot convert back to Encrypted message once it is decrypted. So it will stay readable forever.

If you want to securely hide the contents of your '.fl' file then you should not use the one i am about to give you. Use another one created by somebody else that can be found on the forums. But, like me, i like to keep things readable to all factions (Humans).

The program is created by the author of the famous mod activating and deactivating utility of all, Freelancer Mod Manager: Matthew 'IGx89' Lieder. The file is called FLS1 and can be downloaded from this site. It does not show up on any of the search databases for some reason but i will give you the link:
FLS1 <--- Link.

These are the tools you'll be needing in order to proceed with this tutorial:
Any INI editor. QuickFix is good.
Any Freelancer Saved Game Editor. FLS1 is good.

These are the following files you'll be editing:
DATA\MISSIONS\M13\M13.ini
EXE\newplayer.fl

Starting the actual tutorial:
Open up Newplayer.fl with any Freelancer save game decrypter (Reader). You'll notice that it is similar to 'mpnewcharacter.fl'. Add the new factions to the list like you did in 'mpnewcharacter.fl'. Add the voices of the NPC's in the vnpc section of the Newplayer.fl file. You can change any other thing in here if you want.

Save it and close it. Now open up the M13.ini file inside the M13 folder which is located in the MISSIONS folder. Scroll down to the bottom of the page and find SetRep. Add the exact value you had in the Newplayer.fl file. Save it and it should be finished. Now that you have defined the faction for SinglePlayer.

Read below for FAQ's & Troubleshooting if your game is still crashing when one spawns.

-----F.A.Q's & Troubleshooting Assistance-----

<span style="text-decoration: underline">My game is still crashing when one spawns. What do i do?</span>
Are you sure you have defined the correct factions in initialworld.ini and empathy.ini file.

<span style="text-decoration: underline">My new faction is not spawning or patrolling. What do i do?</span>
You probably have not added a patrol path or have not defined a ship for the NPC to fly in.

<span style="text-decoration: underline">My new faction is not shooting anyone when there are enemies and the enemies are shooting my new faction. What do i do?</span>
You have probably not set the correct entires in either Initialworld.ini or Empathy.ini.

<span style="text-decoration: underline">What does IDS_SHORT_NAME mean?</span>
It is the name shown in the HUD. Use FLED-IDs to create a new IDS string.

Any more questions will be added shortly. If need furtherassistance, please make a reply topic.
