Tue Mar 25, 2003 2:33 am by troop5

Well i doubt it can be done at all without some serious modding of the hardcoded stuff.

You can make the AI do pretty much anything you want in SP using triggers and stuff, but you cant.

If you open up the universe.ini you can add a base, either create or use reference to another existing base, that part can be done.

Open up the actual system ini like li01.ini, everything you can dock with like planets and dockable stations and battleships are defined as spawned [Object
and they have specific position coordinates and then references to dock with a base. If you create an [Object and put dock references, you are just basically creating something. I dont think there is a way to define yourself as an object, because its not meant to be used that way.

You can try if you want, i can give you my rh_battleship, and the ini files for it. If you have a LAN you can try and figure out a solution.
