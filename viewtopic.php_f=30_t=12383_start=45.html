TheCoredump

Wed Apr 09, 2003 8:19 pm by TheCoredump

Here is how I prepare my mesh for MilkShape under GMax (free) :
If you have sub/multi materials you should know that each sub mat. correspond to a face id, the first sub mat. has id 1, the second has id 2, etc
Select all id 1 faces and detach them in a new object and them apply your first sub mat. to this new object. The new object has now a simple material.
Do this for all object and all multi mat.
At the end, each object should has its own simple material.
Now you can export
