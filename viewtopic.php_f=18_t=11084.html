Orion System

There really is no install for it. It's just a sfx archive, thats why its in .exe format... Just extract it wherever... C:\Temp or whatever.

When you've done that you can see that the archive keeps the original freelancer directory structure.. So if you look in each of the directories where I modified files, they should match up to what you have allready in C:\Program Files\Freelancer\DATA ..\..\EXE, ect...

So in order to install it.. Make a back up of the files being changed in DATA, EXE, read the readme for a quick reference of which files I modified. Then take the folders I included in the archive "DATA and EXE" and copy those to C:\Program Files\Freelancer and "overwrite" as wanted...

There are some other changes to this universe as you will notice if you read the read me, some cash changes ect. This is why I didn't make this an installable mod so to speak, other then keep the directory structure for easy reference.. So you can weed out some of the files and options you don't want.

This mod is for 1.0 retail...

Read the "Read Me.txt" For more information on what was changed, and where it was changed inside the ini's A side from that, there's no extra surprises.

If you only want the system installed, without the ship changes, engines, ect.

You need only:
DATA\SOLAR\ASTEROIDS\Orion*.* - The test mineable asteroid fields.
DATA\UNIVERSE\Universe.ini - The listing of the Orion system.
DATA\UNIVERSE\LI01\Li01.ini - For the jumpgate to Orion
DATA\UNIVERSE\Orion\*.* - The system itself.

You don't have to populate either base with weapons, or commodity by adding "DATA\EQUIPMENT\market_*.* they just wont sell anything if you don't have those entries there.

Also I missplaced a file, when creating the original archive, i'll have to either send that to EB, or make a new archive with that included... Its not a terribly important file, it just for base population.. So people are in the bar, etc.

There's no secret to getting it to work, if you followed the instructions.. You can use mod manager also to install it or remove it. I'm not sure what problems your having it with it.

Here are the notes from the latest archive.

Extract this to wherever, and then manually copy the files over, I do not recommend installing this directly, unless you've read the read me.txt and understand what i'm changing prior to extraction.

A simple installation would consist of making a backup of the DATA directory... Edit your freelancer.ini to add in the new DLL file, drop the extra.dll in EXE directory, copy all the files and directories included in this archive, to the DATA directory in C:\Program Files\Freelancer\DATA

You can also install this mod with mod manager created by IGx89. To do this type of installation.
Extract all files to: C:\Program Files\Freelancer Mod Manager\mods\Orion

Under this mods\Orion directory should contain all of the subdirectories for the mod. So you will see Orion\Data\* ect. Then run mod manager, and click activate mod.

Note: this is not supposed to be a tutorial, or have an install I guess I would consider this reference material of how to do a system, populate a base, add entries to a dll, and so on and so forth. The mod will work, i've tested this on many different machines here. Also IGx89 has gotten it to work as well, in testing it with his mod manager.

If you need more information. Check out the forums at: http://www.lancersreactor.com/ I've created a topic you can post your questions about this mod.
