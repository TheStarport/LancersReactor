Mon May 19, 2003 11:21 am by Chris Carter

Next time you start up FL make a note of the time, once you've finished your FL session do a search for any files that where created/modified during your session. Look for any files that have a .tmp file extension or any folders that start with a tilde (~), move/delete these files and try to start up FL.

If this doesn't work then I'm afraid I can't help any further - I run Win 2000 Pro and I've had the same kind of problems as you are experiencing, not only with FL but with other games, removing these temp files/folders has always fixed the problem.

Hope this helps.
