Actually, quick fix isn't nessisary. I edit my holds all the time with out it. Takes just 1 minuet to do. Download the SDK file. It is a decoded data file for freelancer, click on it and it will extract and install itself in the freelancer folder. Second, open My computer/c: drive/programs/microsoft games/freelancer/a nd then right click on the data folder. Click properties. Uncheck the "Read only" box then close it. Click the data file to open it, click ships, then Shaip arch file, find the ship you want and then edit it. close and save and your ready to go. Do a search for the Misc Data base, its a file that contains all vanilla ships/weps/and soforth in the game and thier data designations. It helps when hunting for things.

Its really not all that difficult to edit files like this.

Edit - For setting the ships for sell, data/equipment/marketships.ini click to open, scroll down to {[BaseGood
base = Li01_01_base}
marketgood = cof_package, 8, -1, 0, 0, 1, 1, 1 ;Ship not forsale
marketgood = li_freighter, 0, -1, 1, 1, 0, 1, 1 ;Ships forsale

It must show this lay out for the ship to be available. Its heasyer to cut and paste the name "li_freighter" over an existing ship.
