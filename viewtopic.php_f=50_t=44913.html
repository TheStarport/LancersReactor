Wed Nov 08, 2006 1:24 pm by _xXx_

Ok, ship needs 3 files to make it game:

.CMP: The Model
.MAT: The Textures
.SUR: The Hitbox/Shield Bubble so to say

Once you have these 3 files for your ship, you must add an entry to the following:

shiparch.ini: Defines what the ship can carry, how it flies, etc.
loadouts.ini: Defines what comes on the ship stock when you buy it.
market_ships.ini: Tells the game where the ship can be bought.

Lastly, you need to have added Ship Name and Info Card to the DLL files.

Best advice, pick a vanilla ship copy all of its info from the files, change the info to match your ship, place the new info into the correct files. There are countless tutorials for all of this. This is just a simple run down of how to do it. I would suggest reading some of the tutorials before attempting this in full.
