Reynen Client / Server Mod 2.0 BETA 6b RELEASE

Download FLMM Version 9.1 MB ----- Readme.txt --- Timestamp: : 12:25 PM 5/22/2003

Download RAR Version 6.4 MB ----- Readme.txt --- Timestamp: : 12:50 PM 5/22/2003

Look for Reynen's Mod Server 206b (12.252.29.222)

If the above .zip.flmod files do not install then you will have to download the rar and install the file manually.

Simply create a folder named Reynenmod206b in your mods folder in your FLMM DIR.
Ex. C:\program files\freelancer mod manager\mods\Reynenmod206b
Put all the files from the rar in that directory. Run FLMM activate and your good to go.

The website is also now up and updated.

Reynen Client / Server Mod Website

Unzipped The mod is about 30 MB and I HIGHLY recommend NOT running any other mods with it

This mod changes about 350 files.

Please see the readme for complete details, this mod is HUGE and completely revampes the Freelancer universe!

Enjoy! This is over a month and a half months worth of work so I hope you all really like it. Please post comments and feedback!

Other servers can be found in the readme file.
