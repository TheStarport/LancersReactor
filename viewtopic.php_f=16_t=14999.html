What exactly do you want to know about the Freelancer Locator?

Either click on the map to get the internal coordinates for a location, or enter coordinates you found in the INI files into the text fields and press the Show Button.

The Clear Button will remove any marked location.

To adjust the NavMap Scale, either select a reference system from the combobox, or set a custom scale value (enter value, then press the Set button, or spin through the values).
