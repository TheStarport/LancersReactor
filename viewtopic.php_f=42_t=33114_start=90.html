warzog

Tue Nov 02, 2004 7:41 am by warzog

Update:
The missing robots wasn't due to a corrupt base file.
It was due to an improperly set up set of entries in the mBases.ini file.
In order to save time while populating the 99 bases with the 540 people, and the 396 robots, I created several templates.
Some bases, planets, have "Planetscapes," others have "Cityscapes."
Some of those have individual rooms for each activity, some use the landing pads.
For some unknown reason, Oxford was set up as a station.
(Meaning, it had decks.)
Since it has no decks, the robots didn't show up.
I went through all 99 bases, and found about 4 that had this error, and it's been corrected.
Thanx Quixotal!
