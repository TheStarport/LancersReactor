Warzog - Chemus is right, "flspex.txt" is located in;

C:\Documents and Settings\%USERNAME%\Local Settings\Application Data\Freelancer

The only reason that I can think of (off-hand) that would cause the file location to vary, would probably be because XP is saving the file under the same directory as the username of the person who is running the FL process. That is to say that it may not be "Owner" (which is the username that I assume you use because it is the default), but "Local Service", etc. Therefore, flspew.txt will most likely be located in the above directory, but under the username of whoever XP thinks is running "Freelancer.exe". Run FL, go back to the desktop, and use CTRL-ALT-DELETE. This will show you who owns the process (I assume that you know this, however).
