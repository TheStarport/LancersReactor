Did you back up your saved games first? If so, moving them back to the root sp directory should solve it. If this does not work, back up your saved games to a temp directory and uninstall FL then reinstall it. Also ensure that you are running Direct X 9b. If not, download it from Microsuck and install it BEFORE you start playing again. Also, believe it or not, your video card could also be causing this problem. I run a Ti-4600 nVidia card, and I was having simular issues. Updated the drivers, and things are peachy now. Good luck and I hope thios helps.
