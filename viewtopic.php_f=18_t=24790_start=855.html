Vortimer, the answer to your question could probably have been found with a little bit of searching on the forums. But because I'm feeling nice, here you go...

Steps to putting the Sabre on Battleship Hood:
1. Install Freelancer SDK if you haven't already. Clicking the name brings you to the download link.
2. Find the name that Freelancer uses to recognize the base you want to edit. In your case, since you are editing the Battleship Hood, the code you need is Br05_01. Br meaning Bretonia, 05 meaning the 5th system in Bretonia, 01 meaning the first base in that system.
3. Find the name that Freelancer uses to recognize the ship you want to add to the base. You want to add the Sabre, so youd use bwe2_package.
4. Open market_ships.ini (this can be found in your ##/Microsoft Games/Freelancer/DATA/EQUIPMENT folder, note ## refers to the root you have installed the game to, it should be Program Files on your main (C) drive) with wordpad or notepad.
5. Press Ctrl+F to activate the search function of the program you are in, it will have one. Search for Br05_01_base.
6. You will see a LOT of text under it. But all you need to worry about is the bwe2_package line. It should look something like marketgood = bwe2_package, ##, #, #, #, #, #, #. Change the numbers (RESPECTIVELY!!) to the following:

marketgood = bwe2_package, 10, -1, 1, 1, 0, 1, 1

The first number (10) defines what level you need to be to purchase it, the next level (-1) defines how your reputation needs to be to purchase it (-1 means you can have any kind of reputation with them) and the next 5 numbers are needed in that form (1, 1, 0, 1, 1) to make the ship for sale. Important to remember however, every base can only sell THREE ships. You will not have to worry about the Battleship Hood since it only sells the Cavalier, however, if you were editing a base like Planet Manhattan, you'd have to make one of the ships there (Patriot, Startracker, Rhino) not for sale.

To edit the price of the Sabre:
1. Navigate to goods.ini (its in the same folder as market_ships.ini)
2. Open goods.ini, use the search function (Ctrl+F) to search for bwe2_hull.
3. You should find an entry that looks like this:

[Good
nickname = bwe2_hull
category = shiphull
ship = bw_elite2
price = 505000
ids_name = 12017
item_icon = Equipment\models\commodities\nn_icons\bw_heavy.3db

4. The only line you need to edit is price = 505000, change that line to price = #####
5. The choice of the price is yours, however, the ship, by default, comes with a level 7 HF shield which will cost more than $50,000 with the ship. You can keep the shield on the ship which will cost that much more money when you buy it, or you can remove the shield from the ship by doing the following (however, you'll need to buy your own shield for the ship.)
6. To remove the shield from the ship, keep goods.ini open and search for bwe2_package.
7. You should see an entry like this:

[Good
nickname = bwe2_package
category = ship
hull = bwe2_hull
addon = ge_bwe2_engine_01, internal, 1
addon = bw_elite2_power01, internal, 1
addon = ge_s_scanner_01, internal, 1
addon = ge_s_tractor_01, internal, 1
addon = shield01_mark07_hf, HpShield01, 1
addon = LargeWhiteSpecial, HpHeadlight, 1
addon = SlowSmallWhite, HpRunningLight01, 1
addon = SlowSmallWhite, HpRunningLight02, 1
addon = SlowSmallWhite, HpRunningLight03, 1
addon = SlowSmallWhite, HpRunningLight04, 1
addon = SlowSmallWhite, HpRunningLight05, 1
addon = SlowSmallWhite, HpRunningLight06, 1
addon = SlowSmallWhite, HpRunningLight07, 1
addon = SlowSmallWhite, HpRunningLight08, 1
addon = contrail01, HpContrail01, 1
addon = contrail01, HpContrail02, 1
addon = contrail01, HpContrail03, 1
addon = contrail01, HpContrail04, 1
addon = DockingLightRedSmall, HpDockLight01, 1
addon = DockingLightRedSmall, HpDockLight02, 1

8. Do NOT change anything there except for the line:

addon = shield01_mark07_hf, HpShield01, 1

9. We want to DELETE that entire line (and remove the line space it leaves after you delete it) but NOT touch anything else.
10. Make sure to save all your files after editing them. If they do not let you save, go into the folder (KEEP THE FILE OPEN!) and find the file. Right click the file you are trying to save and go to Properties. If the Read-Only box is checked, uncheck it. You will be able to save your file now.

byelogurov:

I would try lowering the Impulse (standard is 0) and Strength (standard is 100), setting them closer to vanilla values. Also you should try raising the detonation distance, the missile will not explode unless it comes within 2m of a target according to your code. With a missile as fast as yours (there is no point, actually, to set it that fast, as without a speed hack, the fastest anything can go is 990) accuracy would be very hard to achieve, so youd have to raise the detonation distance as it would rarely come within that proximity of a target.

Also, the turn rate under the gun heading, should stay at the vanilla setting as it effects the launcher and not the missile. You should also lower the refire rate as any missile with an explosion that big, fired at a rate that fast, would cause your CPU to turn on you. The maximum refire rate for a Freelancer weapon is around 40 I believe anyway, the engine cannot process anything faster.

I assume you did the model/effects correctly, yes?

dragon,

Ahhh the "joys" of modelling ...Those "black" faces would be reversed faces. That's what happens when faces are created backwards (order of vertices). To fix this, simply select the faces and press Ctrl+Shift+F. It is a bit tricky to get all the faces selected, but you can turn on Wireframe Overlay in the 3d window to get a wireframe mesh to appear over the model to make it easier to see what you have selected.
