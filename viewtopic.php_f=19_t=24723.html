Earendil

Thu Oct 16, 2003 7:53 pm by Earendil

No, no Gamespy. When you connect as Jat71 said and the list of servers "magically" appear, it is because FL is logging you into a global server that lists all the flserver programs connected to the internet. When those flserver programs start they look for the global server to list themselves. This is so those wishing to play MP FL can connect to the global servers and see what flservers are listed. It is possible to directly connect to a flserver though a LAN or the internet but you must know the IP address and the port to acccess. Some servers (like mine, I change it usually weekly) have IP addresses dynamically allocated which means they aren't static (aren't permenant).

So in short, viewing the global server is like using Gamespy except the viewing is built into FL.
