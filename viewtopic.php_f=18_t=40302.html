Manhattan_Guy

Sat Aug 06, 2005 1:19 am by Manhattan_Guy

Of course your missing something. The game needs to know where the new file is. This is defined in DATA/AUDIO/music.ini. Just copy one entry and replace the name and the file locations with the values you need.

The game does not automatically detect all files in AUDIO/MUSIC. You could put the music file in the ships folder as well, as long as you correctly define it in music.ini.
