Tue May 13, 2003 1:07 pm by Silver71880

We had this question a few weeks back as well. The game gives you the impression that Freeport 7 was in Sigma 17. There can be found debris in that system. The truth is that the system with Freeport 7 is a separate system that you won't be able to visit.

universe.ini:

Sigma 17:
[system
nickname = Bw06
file = systems\Bw06\Bw06.ini
pos = 12, 6
msg_id_prefix = gcs_refer_system_Bw06
visit = 0
strid_name = 196638
ids_info = 66119
NavMapScale = 2.500000

Freeport 7 system:
[system
nickname = FP7_system
file = systems\FP7\FP7_system.ini
pos = 15, 6
visit = 128
strid_name = 196657
ids_info = 66115
NavMapScale = 3

As you can see the positions within the Sirius sector are different.
The strid_name stands for the name of the system and if you look that up it says: Omicron Major.

So Freeport 7 was in Omicron Major then? Nope
First of all, a system can not be in another system and secondly:

Omicron Major:
[system
nickname = St03
file = systems\St03\St03.ini
pos = 14, 3
visit = 128
strid_name = 196657
ids_info = 66115
NavMapScale = 2

as you can see, the position of Omicron Major is not the same as the Freeport 7 system
