Fri Aug 27, 2004 11:43 pm by Pinger

The character you attempted to join in is corrupted so crashing the server, you will have to delete it.

This happens when either:

1. lagging badly when F1'ing or disconnecting {unlikely as its your own server}

2.if you are F1 switching a lot between 2 characters, as eventually the server will write to one character with mixed information and so corrupt it. That is because the server holds the character in memory for 30 seconds after it quits before saving it to the file, so eventually you F1 switch just at the Wrong time.
