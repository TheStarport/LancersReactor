Joe! You CAN transfer as long as you start from scratch one of the two servers! This would be your new server. If you get a new account.cfg file (I believe) then you give the entire old Multiplayer directory to Bobway and he takes your account.cfg. Then you create a new one. Either a reinstall on the old server or an install on some other 'puter and take that account.cfg. It is the files under the Multiplayer directory that count to decrypt the player saved game files and their directories.

If you intend on using any old saved game files on the new server (with a different account.cfg or whatever flserver uses as a FL MP ID for the server) then you have to rename the file to a file name created on the new server.
