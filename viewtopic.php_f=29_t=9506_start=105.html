Cpt.Kazo

Sat Mar 22, 2003 4:29 pm by Cpt.Kazo

How to make CapShips dockable to the mooring fixtures on planets:

Open 'solararch.ini' in /DATA/SOLAR and search for 'docking_fixture'.

Change:
docking_sphere = moor_medium, HpDockMountA, 5.000000
To:
docking_sphere = berth, HpDockMountA, 150.000000

...now the top moor of each fixture is dockable

This is even better than with the shipyards, because you can select such a fixture and sy 'dock' and it will dock there.
BUT!...when launching from the planet, the game will still use the docking ring...so no chance to dock with the Rheinland BS because it will not be able to launch again!
