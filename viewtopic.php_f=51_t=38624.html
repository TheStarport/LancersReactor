Instructions for a trouble free gaming experience.

Move or delete your old saved games.
This is important because your old saved games will cause Freelancer to crash.
Location for your saved games folders is at:
My Documents\My Games\Freelancer\Accts
Multiplayer and Singleplayer
Move or delete any files in those two folders.

Do a fresh install of Freelancer.
If you do a fresh new install of Freelancer you will be sure to get rid of any files that may conflict with the new mod you're about to install.
Delete your Freelancer folder.
Location for Freelancer is at:
C:\Program Files\Microsoft Games\Freelancer

Pop in the Freelancer CD and reinstall Freelancer.

Download and install the Freelancer SDK v1.3
The SDK in a self installing exe file.
Browse to C:\Program Files\Microsoft Games\Freelancer and install the SDK in that folder.

Install the Mod.
In Windows Explorer just drag the file to the Freelancer Mod Manager v1.31 and it will install the mod into the mod manager then you can activate the mod. You can also double left click on a mod file (flmod} and the mod manager will install the mod into the mod manager then you can activate the mod.

Now you're ready to start Freelancer and have a blast.
You can start Freelancer from the Mod Manager.
