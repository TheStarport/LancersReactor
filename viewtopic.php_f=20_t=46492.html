Removing Vista if you want

XP does not support Dual Core processors?

XP2 (XP with Service Pack 2) does.

There is a Hotfix available that will allow Windows XP2 to run dual-core processors properly in the latest PCs.

Here is the link:-
WindowsXP-KB896256-v4-x86-ENU.exe

This article describes:-
How to install the Hotfix

There is also an update for the hotfix for Genuine Windows users:-
Hotfix v4

So for those of you who wish to return to XP, you can get the Hotfix and apply it after installing XP. Make sure you only use XP with Service Pack2 though! And you will need to have your drivers CD for your PC handy too, or download them from the internet and burn them to CD or save them on your USB Memory stick first.
