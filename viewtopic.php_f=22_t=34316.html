EG MAIN HQ server: description and FAQ

New life of EG MAIN HQ 24/7 server

IP: 81.176.64.168
Port: 2302
Name in global server list: EG MAIN HQ 24/7
Web-support: http://www.elite-games.ru/egmainhq.en

All pilots and clans are welcomed to try their skills on our server. The server has its own developing storyline, in which all standard factions are involved. Thanks to unique server-side mod, player can start the game for almost every standard faction and after that he will have to play the role according to the Rules of Conduct. One character can have only one role that can not be changed. All interfaction relations have been considerably rebalanced, and sometimes they are changed according developing storyline of roleplay, which is periodically updated on web-server.

The client-side mod on our server is optional, what means, that it is not necessary to download it if you want to have a look at the server. However, using client-mod will give you more interesting features.

What you get without client-side mod?
- possibility to play for almost every faction;
- extended features via console commands;
- loot-mod for all ships, except those used by NPCs only;
- NPC-ships loot nanobots and bataries and actively use them; the time intervals between reloads are similar to those used by real player;
- NPC-ships actively attack players;
- reinforced NPC-patrols in House systems;
- assigned missions' range (in money and difficulty) is considerably extended.

What you get with client mod?
In addtion to above:
- visiting home clan and other additional systems;
- possibility to use two heavy transport ships: Armored transport and CSV transport (full description is here)
- officially registered clans receive their own tag with possibility to adjust reputation to other registered clans.

Every registered clan receives credit of 10 mln. credits for the period of 3 weeks. The information about registration procedure can be found in section 3 of the Rules.

Have a nice playing!

The initial reputation to all faction is negative (red). You need to restart the game for one on the faction. It is simple - just send the following command to console:
/restart<space><faction>
<space> - space button, a lot of pilots forget to push it;
<faction> - the name of faction, for which you are going to play.
e.g.: /restart Corsair - restarts for Corsiars; /restart BAF - restarts for Britonian Armed Forces.
The table of all factions can be found here.

If you have any questions and/or problems - don't hesitate to post here.
