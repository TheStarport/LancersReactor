It's really quite simple. Mods have their own servers. I you wish to play TNG 1.7, for example, find a server that supports TNG 1.7. This is true with any mod out there. If you have a mod installed on your system, and go to a server which isn't supporting that mod, you'll crash it, and get many people unhappy. Find the mod you like, then find the server which supports it. It's not that tough to figure out.
