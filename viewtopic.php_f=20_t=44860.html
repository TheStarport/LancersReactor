not sure about the console error - i see it all the time myself... but the server does list. is it accessable from the net?

i have heard that FL does not like XP x64...

the main CTD error in freelancer sounds like an audio codec problem- if you look on the disc you will find under the redist folder a couple packages... wmfdist is windows media 7. i think it includes an mp3 decoder that FL needs - some codec packs overwrite it. This fix has worked for me before, but i'm not going to promise it will definitely work - i'm not sure what effect it might have on windows media player 9 or 10 though (or whatever version they're on, i stopped using it a long time ago when media player classic came out with all the same features and none of the crap XP styling)

the Quickfix pack is a more important bugfix than the 1.1 update, which is really just a few cheat patches that serverop software can detect now... besides correcting 1000's of bugs in INIs, Quickfix adds support for more cards and disables 3d sound (which has been known to cause errors)
