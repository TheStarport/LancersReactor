F!R

Fri Jun 08, 2007 7:06 am by F!R

.... as stated above: there are only some basic engine fx.

But you can (mis)use weapon effects to create your own effects:


Done like that:
=====================================00
in weapons_ale.ini (weapons_ale.ini is correct)
------------------------
- VisEffect -
nickname = wts_engine21_proj
alchemy = fx\weapons\rh_plasma_04.ale
effect_crc = -73976642
textures = fx\planetflare.txm
textures = fx\sarma.txm
textures = fx\standardeffects.txm
textures = fx\miningship.txm
textures = fx\hexagon.txm
==========================
and in effects.ini
------------------------
[Effect
nickname = wts_engine21_proj
effect_type = EFT_WEAPON_PROJ
vis_effect = wts_engine21_proj
