Crash-Drive

Wed Apr 09, 2003 7:31 am by Crash-Drive

Sure, just change the list server hostname to "localhost" (or your local IP on the LAN) in the freelancer.ini file. By default, the FL server tries to connect to fllistserver.zone.msn.com, which is kinda dumb for a LAN-only game, but they didn't provide separate list server settings for the two types of games.

Crash-Drive

Thu Apr 10, 2003 5:16 pm by Crash-Drive

I use the localhost setting when running both server and client on one machine with Win2k, and the client finds the server immediately. If you're using just one machine, maybe your system isn't correctly resolving "localhost" into the local IP on the LAN, so try putting the local IP explicitly. If you're running the server on a different machine than the client, you have to tell the client the server's IP with a command line parameter. The README explains how to use the -s parameter.
