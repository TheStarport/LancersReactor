Ok, some tips:

Push 'F6'-key for NavMap, F3 for dock, F2 for 'Go To' (after you've created a waypoint in your NavMap). If you want to use your mouse effectively, push your space-bar to enable/disable mouse-flying. Use your 'Tab'-key to afterburn (a little more speed) if you're under attack and wants to flee or make combat. IF you've killed an enemy, use the 'b'-key to pick up the loot. I downloaded the trialversion first,was curious about the game, then In I tried it a few times. It was like "Wow !! What a cool game!!", so I decided to buy the game, which I did. This was in early December, almost a half year (or more) after it was released.

If you have LAN or network card, start a server on your own computer.

1. Start FLServer, name your server, wait till its finished loading.
2. Start Freelancer.
3. Select Multiplayer.
4. Select LAN.
5. Select New Character.
6. and you are free to do whatever you want without any storyline to follow.

PS. You don't need other people on the LAN or Server to play.

On help with the server, see the server forum.
