Just to clarify: anyone can use it; I believe bakedpotato was making a distinction between "real" serverops and people running their own local server.

The records of a player's status is stored on the server, not on the player's own machine. Unless you have access to the server (either it is a local (physically accessible) machine or you have write permissions via some form of remote connection), you can't upload a modified file.

I just tried this to make sure:
* Clear two machines of their %userdocs%\My Games\Freelancer\Accts\Multiplayer\ folders (just renaming them will do fine)
* Run a server on Machine A
* Connect to it from Machine B
* Create a new charachter and quit
* Observe the newly created Accts\Multiplayer\12-a3b4de56\ folder and accounts.cfg on machine A
* Observe the complete lack of any such files on machine B

You can modify the player just fine by running IFSO on Machine A, but running it on Machine B gives a "path not found" error (because you have just moved the multiplayer dir if there was one).
