Wed May 05, 2004 8:12 am by Chips

Use the newcharacter.ini file in the characters folder. THAT is where it takes the values for the mpnewplayer.fl file from. Hence why it says %MONEY% - its actually calling for the line dictating the starting money in the newcharacter.ini file instead. You can also change the starting location, ship, loadout (make server custom if you like - but be careful if you don't know what your doing, as they may get booted/cause server to crash!).
