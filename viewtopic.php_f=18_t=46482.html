Those changes have to be made in the Alpha Channel of the .tga image. They need to be in light colours or preferably white, because the Alpha channel sets the opacity, so dark colours fade out of view.

Most common image editors don't show the Alpha channel, so you need to find one that does. I use DXTBmp, it's very simple but also very easy to use, it sends the Alpha Channel to Windows Paint for editing.

And it also saves DDS DXT images with MipMaps for other work that I need sometimes.

It's available from here: - http://www.mnwright.btinternet.co.uk/download/dxtbmpx.EXE

If you want to add your own startup screens too then it's these files that you need to replace or edit:-

DATA\INTERFACE\INTRO\IMAGES\startupscreen.tga (800 x 600)

DATA\INTERFACE\INTRO\IMAGES\startupscreen_1280.tga (1280 x 960)

Hope this helps.
