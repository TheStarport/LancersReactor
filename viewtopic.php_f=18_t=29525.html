OK, first it looks like your file is still encrypted...look here for how to decrypt it.

Once you've done this it's really pretty simple. Search the file for "nickname = or_elite" (The anubis' nickname) and then look for a line under this that says "hold_size = ". I think you can take it from here...

[Ship
ids_name = 237041
ids_info = 66581
ids_info1 = 66582
ids_info2 = 66608
ids_info3 = 66583
ship_class = 1
nickname = or_elite
LODranges = 0, 75, 150, 1000
msg_id_prefix = gcs_refer_shiparch_Ordhf
mission_property = can_use_berths
type = FIGHTER
DA_archetype = ships\order\or_elite\or_elite.cmp
material_library = Ships\order\or_ships.mat
material_library = fx\envmapbasic.mat
envmap_material = envmapbasic
cockpit = cockpits\liberty\or_elite.ini
pilot_mesh = generic_pilot
nanobot_limit = 29
shield_battery_limit = 29
mass = 150.000000
hold_size = 70 ;<--- EDIT THIS
linear_drag = 1.000000
fuse = intermed_damage_smallship01, 0.000000, 1050
...
(continues)
