@Sp/der -
Due to an oversight on my part I failed to notice that there were multiple versions of the atl71.dll file included with this program. In the folder where you installed the program you will find a folder named ANSI_FOR_WIN98 which contains the correct version of the dll for users running windows 9x, just copy that file over the one in the program folder.


@Darksiege - FLEXP supports basic ids creation/editing, you should read up on the ids and dll editing tutorials so you know what you are doing.

As always you should not only consider using this program to create your systems, it is a good starting point but you should learn to manually go in and work on system files. FLEXP can do a lot of things and makes many things much simpler but you should double-check its work manually.
