Sat Aug 14, 2004 12:36 am by Chips

You have to start from scratch on a server, but once you have, the character is saved by the server - so you can rejoin and continue at any time (as long as server is 24/7). You can have up to 5 characters on servers, so you can start a variety of characters, and its easy to swap from one to another. However, each new char or new server you start on will require you to start in a starflier once more.
Good servers? Find one wtih a low ping, but reasonably active - so that you can interact with other players.

Sat Aug 14, 2004 11:37 pm by Chips

That sounds VERY much like the firewall. Find your firewall settings. If you use Macafee - then bottom left you will see a logo - hover the cursor over it, and it will say the name of the firewall. Right click, and an options list will come up - probabily with level of security and stuff. Click on the options tab if it has one, or just double click on the icon once more to start your firewalls actual application.

There maybe a large tab looking like a computer saying "Internet applicaions" - there may not, but you want to find out internet permissions - so have a look around on top nav bar or icons on the page. When you find it, click and you should see a list of things with permissions for the internet. You should be able to find something that says "allow new application" or "add new permission" or something like that - click it. Now it should open a familiar browswer window - just like when you save a document. Navigate your way to your freelancer folder:
C:/program files/microsoft games/freelancer/exe/
Inside the exe file find the freelancer.exe file - and double click to allow it access. Once done, it should add it to the list. Now you can exit your firewall, and try once more.

If you ever play a modthat changes the version number of the Freelancer.exe - a la TNG, Evolutions etc etc - you will have to REDO this proceedure to allow the new version through!!

Hope that sorts you out.
