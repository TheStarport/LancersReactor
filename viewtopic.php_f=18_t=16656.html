Question-Anyone know the settings that control what ships are displayed first...
I have not tested this yet because, well to be honest I never thought of it but I'll let you try it and if it works great, let us know. Try this. Here are three ship packages the titan, dromedary, and the dragon, put them in one of the [BaseGood entries, you dont even need the rest of the packages they wont be for sale anaway so we dont need them, just these 3. Start the game and see where their displayed then switch them. Move the second line up so that it becomes the first entry then start up again and see if it changed the display showcase.

marketgood = coe2_package, 0, -1, 1, 1, 0, 1, 1
marketgood = bwfr_package, 0, -1, 1, 1, 0, 1, 1
marketgood = ke_package, 0, -1, 1, 1, 0, 1, 1
