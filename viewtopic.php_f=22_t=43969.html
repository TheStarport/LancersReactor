How well will this PC run FL Server?

Hello all! I'm not sure if this is the right forum to ask this or not, so if it isn't please excuse me.

I'm currently running a FL server that runs the new Frontierspace mod. It's running great and I've had virtually no issues at all with it. I have a couple of regular players that are on it every day. However, I've never had more that 3 people logged in to it at once!

It is a P4 1.6GHz with 512MB of RAMBUS memory and Windows XP. It's kind of old, but it is the only PC I have available to host the game. My internet connection is just a cable modem. My question is, how many players do you guys think a configuration like this could support? Like I said, I've never had more than 3 people on at once and it performed just fine. I just don't have any idea how it'll handled the load if I get a lot more people on it!

What do you guys think? Thanks in advance!

Tue Jun 27, 2006 11:02 am by Ogu

If the cable modem is decent, then you probably have to worry about RAM before CPU. If you're able, just watch the usage of the RAM by FLServer.exe and also the CPU usage (the task manager in Windows XP does this). When you start seeing one of them being completely used, then you know you can't have any more players than that, and it would be best to keep the limit a few players below. However, this does require that you catch a bunch of people online at once.

I'd guess that you could have 10-15 or so on a good day. Make sure you have no spyware, and that FLserver.exe is the only program running (aside form IC, or something like that).

If you constantly have 100% CPU being used, the server is likely to crash if everyone gets in one area and start fighting.
