It shouldn't take any tinkering to get the mod into the Freelancer Mod Manager.
Mods install by double left clicking on the Mod file name...
Example MyMod.zip.flmod
double left clicking should install the Mod into the Mod Manager no tinkering required. Then after the Mod Manager tells you that the Mod has been sucessfully installed you can activate the Mod in the Mod Manager.

You may have to get rid of all your saved game files...
Go to
My Documents\My Games\Freelancer and delete all the files in the singleplayer and multiplayer folders.

If it still don't work deactivate the Mod in the Mod Manager and then delete the Freelancer folder
C:\Program Files\Microsoft Games\Freelancer
and pop in the Freelancer disk and reinstall Freelancer...

When I do this (which is almost daily) I use to install the Freelancer SDK but now that the Unofficial Patch is out I have been using that instead because it has the SDK files in the patch along with a ton of fixes...
