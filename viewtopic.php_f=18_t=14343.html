Do you want an osiris flying around or a base?
If you look for the base this should help, it's the dockable osiris from the SP.

File: \SOLAR\solararch.ini
Archetype: o_osiris

Example:
[Object
nickname = St03_osiris
ids_name = 458804
pos = 18412, 0, -18955
Archetype = o_osiris
ids_info = 65747
dock_with = st03b_02_base
faction = fc_or_grp
archetype = o_osiris
label = osiris
base = st03b_02_base
difficulty_level = 3
loadout = o_osiris
