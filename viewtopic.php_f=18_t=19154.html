Zytec

Thu Jun 12, 2003 12:34 pm by Zytec

Are you using the old or the updated version of the voyager mod??? I have the update, and I noticed the same problem. The answer is simple. In the folder where you installed FLMM to, there is a folder called mods. Go in here and there should be one called voyager. The problem lies with what is held in the data part. Inside the data folder there is a ships folder. This directs where the next folder will be installed. The folder inside it is called voyager. By looking at the script we can see the problem - the program is looking for a folder called startrek, not voyager. To solve this, simply re-name the folder. Done. Hope that helps.

Seems that the cockpit is messed. When you try to view from your cockpit you'll see all blank from the middle to the top of the screen. The cockpit section still missing in this mod. The author must include that files with the mod or this never works properly.
What I've Done - How To
I've replaced the script file itself from all the entries that have 'startrek' path included to 'voyager'. Ex: \data\ships\STARTREK\voyager.xxx to
\data\ships\VOYAGER\voyager.xxx
After i've saved the script file, i rebuilded the flmod mod by selecting the files that the mentioned mod uses: script.xml, readme.txt, swshipinfo.dll and the DATA directory. With that files selected click on right mouse button and choose 'add to archive' (I use the Winrar to compress and decompress files), choose zip format, then add the following to the end of the file name: '.flmod'. Shows like this: voyager#########.zip.flmod
Then OK to compress the new archive and all that i need after, was activating it by clicking in his icon (i used the windows Explorer to do that. That's all!
