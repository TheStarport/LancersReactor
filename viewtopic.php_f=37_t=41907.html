5i1,

Pity about your problem...Let's try to help you through this...

First off, go into FLMM, make sure all mods are turned off. Once you know that all the mods are turned off. Uninstall Freelancer. Do not reinstall. Uninstall completely, removing all data/saved games/screenshots (if you want, you can back these up to another folder). Once you have uninstalled. Go into your program files folder on your C drive (unless you use an alternate drive such as D for your main installations), and look for a folder "Microsoft Games." If you find one, check whats in it, and unless you have any other MSGS games installed (i.e. Halo, Dungeon Siege) delete the entire Microsoft Games folder. Also, check your saved games directory (default C:\Documents and Settings\USERNAME \My Documents\My Games\Freelancer\Accts) and delete any .fl files (these are saves) if there are any in there (you could infact, delete the entire Freelancer folder from My Games safely as it will be reinstated after a fresh install).

Once all the traces of Freelancer are gone, do a new install, you could even install to a new directory (E.G. "MS Games" but I don't recommend that unless installing to the default directory doesn't work. Then extract the mod to FLMM. Make sure you are using this version of FLMM. With no other mods activated, activate Epsilon. You do not need, and should not install Freelancer SDK or any other utilities before or after installing Epsilon. After activation in FLMM, Epsilon should run smoothly. Let me know if this works, if not, we can try to help you more.
