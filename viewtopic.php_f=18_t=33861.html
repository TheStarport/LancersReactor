ok u need to download the freelancer simple save game editor by TheJkWhoSaysNi found here http://www.lancersreactor.com/t/downloa ... asp?id=771

open your save game file and find the line that says

rumour = (your idsinfo no), 1 or 2 depending on what u set up in mbases,ini
e.g. the line could look like
rumor = 393353, 1

delete it and save the file then next time u meet the npc the will offer up the rumour again
