Some more details for a few points:
1) Where can I obtain the "Adv Champion Heavy Fighter" level 10 graviton shield? (I heard it can be dropped by ships in the Omicron Alpha system but I tried to no avail, not sure if u have to be at a certain level, i was at 34)


In some mods these shields do indeed 'drop' but in the original game they do not drop and you would not want them anyway because NPC shields have virtually no regeneration and there are no HF/Fr variants (all shields have the light fighter type capacity). What the mods do is 'drop' real shields as phantom/bonus loot and/or put real shields on NPC ships, instead of NPC shields.

The line of available graviton shields ends at class 7, except for the HF (heavy fighter) variant which you can get as class 8 inside the reinforced shopping cart known as Titan. IOW, there are no class 8 LF/Fr gravitons, and no class 9/10 gravitons of any type. The line of available moleculars ends at class 9, and only positrons go all the way to class 10 (the much-favoured Adv. Brigandine).

2) How many rowlett's revenge turrets are dropped at the start of mission 3?


As the others have noted, it is only one (alas). It is dropped as phantom loot if you kill the leader of the pack. Contrary to what others have said, it is a very fine turret and easily the best class 3 one that can be gotten. Perfect for outfitting a Rhino, if only there were more than one ...

4) How do you obtain battleship kills?


By killing cruisers and destroyers - or battleships that are actually ships and not solars . See the thread Battleship kills that do get counted for a discussion of this problem. My personal record is 17 registered BS kills during one single-player campaign run, but it should be possible to get at least 20. A couple of relevant threads are linked from my overview/linky topic Life after level 38 - campaign fun overview.
Look for FLDQ-related threads; the relevant discipline is 'Tmax runs' .

7) ... is there a list of equipment (guns, etc.) that cannot be bought, only dropped from ships?


Rowlett's Revenge and class 10 kit (CODENAMEs, Nomads) have been mentioned already. But there is also the mighty Ripper Mine Launcher, which can only be looted from a ship graveyard but not bought (except in MP, from other players).

8) any other information that you feel that would help me... like things that cant be gotten if ur not in the story, things that you can't get unless you do something... any easter eggs, etc.


The only bona fide easter eggs in the game are Rowlett's Revenge and the robot/monkey planets. However, if you know where the wrecks with lootable high-class equipment are then you can make your ship much stronger than the game designers intended.

For example, you can loot a Ripper mine launcher and 20 Ripper mines has soon as you have a boat that can stand the heat up there in Omicron Alpha (Drake or Barracuda with class 5 shield or better recommended). This is some seriously heavy ordnance which allows some serious fun to be had during the campaign.

Another example, the Balboa wreck in California yields a pair of class 5 particle guns and a wreck in Texas yields a pair of class 5 Catapult missile launchers and 20 missiles (you can get more missiles if you kill Unioner Falcons, which is easy if you have a ship like the Dagger); Outcast Daggers will drop class 4 particle guns if you ask nicely and are not too poor. This means you can max out a Cavalier or Crusader with the finest weapons that it can carry, long before you are even allowed to buy class 5 stuff. See the thread Plan for a Maximum-Fun Campaign for some ideas how to utilize the excess firepower.

Also, during campaign missions you can loot hostile depots with impunity - sometimes even twice during one and the same mission - because during missions your reputation sheet is essentially 'frozen'. The FLDQ threads contain a discussion of this effect, which was discovered by T-Hawk.
