I fixed the same problem that Jabberwacky and a coupleof others were having by using regedit to force the machine to use the advanced instead of the nemo codec. I changed:
[HKEY_LOCAL_MACHINE\SOFTWARE\Microsoft\Windows NT\CurrentVersion\Drivers32 "msacm.l3acm"="l3codecp.acm"
to
[HKEY_LOCAL_MACHINE\SOFTWARE\Microsoft\Windows NT\CurrentVersion\Drivers32 "msacm.l3acm"="l3codeca.acm"

If you are using windows 9x, this change needs to be made in system.ini.
change msacm.l3acm=l3codecp.acm
to
msacm.l3acm=l3codeca.acm

This was after I installed the wma8-9redist so that the correct codec was in place. This told the machine to use it. Fixed the "Codec shows up but all the menu's don't do anything and it disappears right after viewing problem."

Good luck.

Tue Mar 11, 2003 12:43 am by pci

Windows Media player uses a different driver than the game.
The game is use ACM (audio codec manager) which is an older standard.

You have to run WMFDist.exe from the Freelancer CD to manually install the correct audio driver.

pci

Wed Mar 12, 2003 2:03 am by pci

if you're using Win2k or WinXP the simplest thing to do is...

Run RegEdit
Open HKEY_LOCAL_MACHINE\SOFTWARE\Microsoft\Windows NT\CurrentVersion\Drivers32
Look for msacm.l3acm or msacm.l3codec... delete them.
run WMFDist.exe from the Freelancer CD

the right codec will be installed and Freelancer willl play sound.
