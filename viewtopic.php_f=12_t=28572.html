I''m Running Privateer in XP!

Just thought that I'd share with anyone interested

Actually, I'm using a DOS emulator called DOSBox: http://dosbox.sourceforge.net

Reading the readme file is necessary, but I have some pointers. First, the directory selected as the 'root' should be one level below your privateer directory. The dosbox.conf file needs to be modified to get Privateer to work right. Set the line 'ems=true' to 'ems=false'. With the EMS disabled, Privateer runs fine on my PC. I get sound, and can use the joystick ok (it seems to have a really big deadzone to me). I recommend turning off as many other programs as you can manage; it runs more smoothly. With a 2.06 GHZ Athlon, I have it running at 7000 cycles with nary a hiccup. WARNING occasionally I have a crash back to windows, apparently during animations/cinematics. I have saved and loaded to a fare thee well, and it works fine. Save often.

Happy Flying -Chemus
