From my experience, if you add a ship and make it purchasable in a specific location then it will be available there in both multi and single player games. I've made 2 ships myself and simply added them to Manhatten. I started my testing of those ships using multiplayer (hosting my own server) and then, by a fluke, I ended up in single player and the ships were available there for purchase too. As long as it isn't a Server-Only ship mod you should be able to use it, although I'm not sure how you would add a ship to your Freelancer that is only available on a specific server. Not to say it can't be done.

I don't think anything about Freelancer is server-only. The servers run off the same ini files as the single player game, it just runs off those files on the server's HDD.

The thing with adding ships is, if you connect to a server that doesn't have it, you get booted, and the server may crash. If you connect to a server that has a ship that you don't have, and you see that ship, the same probably happens. New ships have to be both server and client side. Otherwise, one program doesn't know what to do when it encounters ship references it doesn't have.
