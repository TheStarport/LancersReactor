n4tune8

Thu Apr 24, 2003 2:18 am by n4tune8

To actually make yourself neutral to all sides but zoners:
In newcharacter.ini:

[Faction
nickname = new_player
rep_group = gd_z_grp <== that's the important part
base = Li01_01_Base
Package = ge_fighter
Pilot = trent

In mpnewcharacter.fl:

[Player
name = %%NAME%%
initial_rep = %%FACTION%% <== remove the ; in front of this line
rank = 0

and then comment out (by putting a ; in front) all the lines that start with house =

The zoners will be your friends, and everybody else will be neutral or very close to neutral. Good luck!

fangmans

Fri Apr 25, 2003 5:40 am by fangmans

Use the command line Bini.

Put it in a directory by itself.

Copy the .ini file you want to hack from the FreeLancer subdirectory and paste it into the Bini directory.

Drag the .ini file icon onto the Bini.exe icon.

In a few seconds a new file will appear in the directory, .ini.txt.

Make a sub directory of your Bini directory and move the original .ini file into it.

E dit the .ini.txt file with wordpad. DO NOT use note pad.

When finished rename the .ini.txt file to .ini.

Move the new .ini back to the FreeLancer directory where it came from and overwrite the original.

You're good to go!
