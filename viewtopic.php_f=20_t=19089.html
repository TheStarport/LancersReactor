Seanus

Fri Jun 13, 2003 1:05 pm by Seanus

Don't change the mod extension.

1.Create a new directory in the flmm\mods directory called something inteligent like "rebalance 1.4",
2. Extract all the files to this directory.
REPEAT

3. Try and activate the mod.
4. When falls over, uncheck the RO attribute on that file (in the mod directory, not the game (destination) directory).

UNTIL mod=activated.

It will fall over several times, but on a different file.
