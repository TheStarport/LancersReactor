Please follow these steps if some sounds and speech are still missing after installation.

1. Uninstall any 3rd party codec pack or the Nimo Codec pack from your system.
2. Uninstall Freelancer.
3. Go to the following directory, C:\WINDOWS\system32 and delete the file "l3codeca.acm". If there happens to be another file named "l3codecp.acm", delete this too.
4. If your system is Windows 98 or ME, open the system.ini file located in your system folder; C:\WINDOWS\SYSTEM. Delete ONLY the line that starts with "msacm.l3acm=". In the case of the Nimo codec pack, this will read "msacm.l3acm=l3codecp.acm"
5. If your system is Windows 2000 or XP, open the system registry editor (regedit) from the RUN command window.
6. Locate and delete the following key: \HKLM\SOFTWARE\MicrosoftWindowsNT\CurrentVersionDrivers32\msacm.l3acm
7. Load the Freelancer installation CD and locate the REDIST folder. Within the REDIST folder, double click on the file named “WMFADist.exe” and follow on-screen instructions.
8. Install Freelancer.

Remember: If you are not familiar or comfortable with modifying the contents of the system registry, then please ask a qualified person for assistance.

I had this exact problem. It's caused by a sound event not completing (i.e - the code doesn't register that the cut scene has finished because no sound completion event is generated).

This is DEFINITELY caused because an incorrect codec is being used to play the audio.

Uninstall the Nimo codec if you have it installed,

Follow all the tips for audio trouble in the readme file. Re-install.
Remove the codec references in the registy (V IMPORTANT)
Run the codec re-install in the redist folder on the cdrom.
FOLLOW ALL INSTRUCTIONS IN THE README THAT APPLY TO FRAGMENTED AUDIO

This SHOULD work. Remember that this IS a sound issue - reinstall if neccessary - remove other audio codecs and their reg entries if necessary! You CAN fix this.

The game cannot be continued if the audio event is not completed!

Sorry to have to specify this in black and white - if it still doesn't work.. ...you need to keep trying! This CAN be fixed on any system with a sound card - its just an audio issue. Any problems feel free to reply - i'll try and help u out some more - maybe arrange to send me a savegame and i'll progress u to just after - but it's likely this could happen again.

Hope this was some help.
