Gnavpot

Mon Apr 11, 2005 8:15 am by Gnavpot

I can't help with your problem, but perhaps I can help you getting a more meaningful error message.

Start Windows Explorer and open you CD. You will find a large file with the name cab2.cab. This is a compressed file containing the game files.

Doubleclick on cab2.cab and you can see the contents. Find the file which is causing the error (br_02_cambridge_cityscape.cmp) and drag it to a folder on your hard drive. If there is a problem reading the CD, I assume you will be told so.
