Accushot

Mon Aug 15, 2005 10:05 am by Accushot

Create a new file in the same location as universe.ini called something else, like "universe2.ini". Then, in freelancer.ini add the line "universe = Data\Universe\universe2.ini" after the other line that says "universe = ...". This works because Datastorm only reads the first universe entry, so anything in the second file will be hidden and not shown in Datastorm.
