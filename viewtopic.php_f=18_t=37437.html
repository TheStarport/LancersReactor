Mon Jan 31, 2005 9:44 am by Lethal Dosage

When using FLE, to add jump holes and gates:

:: Double click on the system you want to make the gate from (Not right click and edit!).
::Right click on the system screen, create jumphole/gate.
::Go back to main map by double clicking system background.
::Choose the system you want the gate to take you to.
::Right click on system background, create jumphole/gate
::Enter the needed info

Think thats basicly it - however I would only advise using FLE for two reasons:
1) You are VERY new to modding Freelancer and just want to get to grips with the files and how things work.
2) (like me) You can never get the co-ordinates right when making systems in the .ini files, so you add whatever in FLE to get the co-ords, then make the real system in the .ini's using the co-ords given.
