Good for you! A little extra safeguard, so this kind of thing doesn't happen again. Go to Control Panel -> System -> System Restore (It's a tab at the top) and make sure it's turned "ON" and make a system restore point. You can access System Resotre and make a System Resotre point through Start Menu -> All Programs -> Accessories -> System Tools -> System Restore. Through this you can simply restore your computer to what it was when it just started, or at any other System Restore point, which is very convienant, and better than reformatting all the time.

Special Notes: Make sure you virus scan before you create a System Restore point. Creating a virus backup will cause large amounts of problems because System Restore has to be turned OFF for a virus scan to scan the back ups, because Windows puts those files under protection. It's also suggested you set up and configure all utilitzes suchs as virus scanners, firewall, netwrok settings etc. (basically the tedious stuff that you don't want to be repeating) and then make the restore point so you don't have to do that stuff again. If you want more System Rstore points (Windows will trash the older ones every now and then to save space), allocate more hard drive space to System Restore by using the slider in the System Restore tab.
