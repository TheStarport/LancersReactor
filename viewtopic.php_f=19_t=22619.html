Global Players Stats (Repost)

(Original thread here in the Server Forum)

Inspired by the thread "Best of the Best?", I've been working on merging the data from all the FLstat servers I can find, into a single list.

So, here is my Freelancer Global Rankings . You'll notice that I'm sorting on number of kills rather than rank, because a) it makes a nice change, b) some servers start you with more money and c) it's too easy to cheat to get more money/rank. Note, these only show NPC kills, not player-vs-player kills.

If you have any constructive comments, or if you have FLstat running on your server and want it adding to the Global Stats, then please post a reply or drop me a line.

Cryo.

Cryogenius

Thu Aug 07, 2003 9:00 am by Cryogenius

I wanted to create a list based on number of kills, in order to create a new dimension to the game, and to give players something to aim for other than just making more money.

I considered sorting by Rank, but I really can't see how the rank of a character on one server can be compared against the rank on another. This is because of a number of reasons, namely: starting money, cheats, and mods that change either the prices of things, or the ranking system (if there is one). Admittedly, it is possible to cheat using the cloaking device, but hopefully admins will prevent this on their own servers.

However, I shall take on board all of your comments, and I shall work on providing two tables: one sorted by rank, other sorted by kills. (I did also want to sort by systems/bases/holes visited, but again this is distorted by mods).

I see no need to sort by name, as the browser's "Find" command should let you find your own name (or server) easily enough.

Anyway, keep the comments coming - it's all good!
