The Ioncross program will let you unlock the gates, even when useing a mod.

IONCROSS Freelancer Character Editor mk.V

Just start a game, or have one going, create a save, not an auto save, exit the game open Ioncross, click on the saved game file and it opens. Bottom of front page contains 2 boxes, check on for all gates to open, the other to reviel all map. Save and close, and restart saved game, gates and hole open.
