Buck

Tue Oct 11, 2005 10:03 am by Buck

I have actually created an exe version of the Colours Elite mod that literally copied the entire contents of the Freelancer folder into a new folder called ColoursElite. It then installed the mod into this new directory and placed a shortcut to it on the desktop. The beauty of this was that players need never change mods cos they use the Colours version on my my server and the original Freelancer with FLMM on other servers.

It worked perfectly for 80% of my community but for the others it was problematic due to installing in different locations, using Win98 etc. etc. I scrapped it and resorted back to FLMM but with a little more work, like scanning the host machine and finding things for itself, or asking players to point the installer to the correct location, it would work.

I never got round to continuing with it but I may do one day. I used InstallShield to do it and wrote a script for copying the freelancer folder.
