Have no fear, this guy is here!!!!! OK, first of all you didnt burn the files to another Computer did you??? If you didn't and they don't work, heres what to do. This one is easy, all you have to do is

go to the folder ...microsoft games\freelancer\data\ships
left click on shiparch.ini, select properties
uncheck the box beside Read-Only

Ive had this problem before, some mods and editing programs write protect the file automatically for some strange reason. oh well, I hope I helped

Here is another tip. I was getting some error messages with FLMM 1.1 with the Starfighter also. In another post the author of the program gave two suggestions to someone else having a problem. Only the second solution worked for me and here it is. If the mod is set up for FLMM 1.1 (which the ship or ship pack is):
Download the mod using the OPEN command instead of save. I tried it and it not only loaded everything successfully it worked successfully! (I got the former message without the thing working once before I tried this.)
