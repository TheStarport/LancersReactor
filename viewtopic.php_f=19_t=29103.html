Pinger

Fri Mar 05, 2004 8:00 am by Pinger

On any single cpu server running anything in the background other than FLadmin and or IFSO will steal cpu cycles so adds to server load and eventually lag.

Best thing for a FL server is:

1. a clean install of windows
2. FL
3. IFSO
4. FLadmin
5. FLstats
6. Nothing else running except a firewall especially if your doing Player IP blocks/bans. So no virus scanner etc, those you scan the server pc remotely via lan. But being a server you shouldn't be putting files on it that often so a virus shouldn't be an issue.
