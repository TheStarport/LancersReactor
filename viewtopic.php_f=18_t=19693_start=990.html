Mon Aug 14, 2006 11:56 am by parabolix

Jayn3, you can install Freelancer SDK (found in the downloads section of this website) which will convert the .ini files to readable text. Then go into your Freelancer folder (Program Files/Microsoft Games/Freelancer by default) then Data, then Equipment. You'll see a file called weapon_equip.ini. You can find all the weapons in there.

My tutorial here will walk you through creating new weapons, you can also use the info there to help you edit weapons.

Ogu

Tue Sep 05, 2006 6:10 am by Ogu

1. All you do is use winzip to zip up your mod, then add .flmod as the file extension. Your mod will be called "mymodname.zip.flmod" - make sure you have your folder options set to show extensions and you won't have any problem.

2. I think you'd need to change the effect on the Cavalier engine to that of the Rheinland engine (or change the engine altogether). I don't think there's a color value anywhere that you can just change from blue to green for the Cavalier engine.
