The Ioncross datafiles for Frontierspace (or any other mods) should be copied into the:
Program Files\IONCROSS Freelancer Server Operator mk.V.1folder.
You will need to delete any other multiplayer server details and start flserver before running Ioncross. Then it should work fine.
(edit)
When you get the server up and running, feel free to post on the Frontierspace developers thread. All of the servers running the mod are listed in the thread topic.

Chips

Mon Mar 05, 2007 1:59 pm by Chips

Have you got two firewalls running? If so, get rid of the windows one (turn it off) and trust your main one.

If it kind of "freezes" at the server listing, saying they're available but not listing any - then it's your firewall, as FL can't connect to the net.

Try re-permissioning your Freelancer.exe file once more, or alt-tabbing out an see if a warning about internet access is up.
