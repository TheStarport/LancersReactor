I have run FL server on a 2K3 box and to tell the truth it has a ton more overhead than it should. I have since switched to XP pro and simply reboot the box at least every week.

On to your question. It seems that I had to enable directX to get it to function correctly. No I do not mean install but after you install direct X9c then run dxdiag and it seems like there were a couple of check boxes hidden deep in there that will allow the box to use directX. It is turned off by default for security reasons.

Try that.
