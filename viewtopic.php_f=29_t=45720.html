Now to start the actual tutorial.

Firstly, download the FLS1 program by IGx89 or any other Freelancer Save Editors. Then open up the file you downloaded FLS1 from and right click and select Extract All. Open up the extracted file and find a reg key file, double click on that reg file. Then open up the EXE folder. Find the newplayer.fl file, right click and select "Decrypt to new File". Now delete the Newplayer.fl file. At the bottom of the exe folder, you will find a NEWPLA~1.FL.txt sitting there. Rename that file to "newplayer.fl" (Without the quotes). Now open up that file and edit.

Inside you will find:
