Fri Dec 30, 2005 2:44 pm by H.Q. Victor

There are a variety of mods avaiable, but, if you want to change some ship level requirements, that can be done very easy.

base = Br01_01_base
marketgood = bf_package, 10, -1, 1, 1, 0, 1, 1
marketgood = bfr_package, 20, -1, 1, 1, 0, 1, 1
marketgood = bhf_package, 30, -1, 1, 1, 0, 1, 1 ;Change the 30 to 5 if you want.

base = Br01_01_base
marketgood = bf_package, 5, -1, 1, 1, 0, 1, 1
marketgood = bfr_package, 5, -1, 1, 1, 0, 1, 1
marketgood = bhf_package, 5, -1, 1, 1, 0, 1, 1 ;Making it level 5 lets you buy it sooner.

Its easyer this way than using a mod, just choose the base/ship and modify it. Remember though, it is for SP game, not MP as using it there will be considered a cheat.
