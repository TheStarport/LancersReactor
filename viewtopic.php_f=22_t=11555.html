Advanced server management

God bless Digital Anvil.. for the game. FLServer, sadly, leaves a lot to be desired. In my attempt to seriously run a Freelancer server, I've been confounded by a couple things, all of which would be resolved by an actually useful server management app. Some things I'm looking for include :

1) being able to look into the characters and accounts, to see stats and other things like time played, when created, last accessed (for both account/character management and cheat detection/prevention)

2) being able to display more info than what's currently listed for server statistics. I mean, why does the server admin car how many aborted missions there were, when the quick stats list doesn't show who's logged in and for how long

3) a way to tell general activity level of players, i.e. are they sitting around on a planet for hours? Are they actually traveling around? Not sure if the trolling adversely affects server load (my initial guess would be that it doesn't)

My aim is to be able to actually manage a server as a dedicated third party (not as a player running around and checking up on people). Does anybody know if someone is working on a new server app, or will I have to do some research on my own to see how hard it would be?

- Pfhoenix

1) FLStat generates stats from the player files and outputs to HTML. I've been using it for a couple of days and so far so good.

2) FLAdmin provides a telnet server which allows you to list players, but it's mostly a management tool.

3) Not as far as I'm aware. Conventional wisdom suggest players who are generally inactive won't be putting much of a load on the server but I'd seen reports from some admin claiming otherwise.

I'm not aware than anyone is developing a new server app. Microsoft/DA patched the server fairly recently but I've seen nothing to suggest they're doing any further work on it. Development of a third-party replacement might fall foul of copyright issues and, besides, I think the game quite frankly lacks sufficient popularity or long term appeal to attract or justify the effort involved.
