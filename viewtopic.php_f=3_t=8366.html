Upcoming patches

Now that the game is released and many of us are already playing I would like to open the discussion on upcoming patches.

I would like to congratulate Digital Anvil for this great game. But in my opinion we could make it even more appealing specialy for those playing multiplayer.
Try to organize your ideas on features that are some how possible to come with new patches or expansion packs.

This are my ideas for upcomming patches :

- Server side clan registration - A new menu in witch a player could register a clan and other players could join...
- Modding support...specially for multiplayer... A server administrator could add mods... Mods could be new systems, ships, etc etc...A simple menu eighter in game or not, in witch server administrators could install and activate mods...
It seams from what Ive read in the editing forum , that its not that difficult to create new stuff for the game...
- Shared player database between servers.
- Favorites menu in in-game server browser
- Direct IP in in-game server browser

Expansion pack/patch ideas :

- An application to make it easier the creation of new sectors, etc etc...
- Jump gates that lead to other servers and characters are past between servers (of course)
- Server administrator could had news to the news service...
- Special news area for users
- A better dynamic economy...
- Tools for event creation. Server can script special events...Events could range from story like missions, to "special" scripted events like an asteroid passing in a certain area. A star explosion. (for example ...the corona variable could be manipulated so that it reaches all the system...)

Probably only Expansion pack/next game ideas:

- Player run corporations
- Player station creation
- More ships, more weapons, bigger ships etc etc etc...
- Good integration of one Universe different servers....(share same economy, player base, etc etc)

Next game:

- Planets, stations etc make orbits
- An universe in witch its possible for an asteroid or to crash in a planet...Giant comets pass by... Stars explode.. I mean an ever more dynamic universe...

* NWN bringed the tools for Fantasy RPG gamers...
* Freelancer and future releases could bring the tools for Space RPG gamers.
