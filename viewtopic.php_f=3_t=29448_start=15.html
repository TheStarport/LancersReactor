Thu Mar 18, 2004 4:37 am by Chips

Seing as this must be SP (rep for ships not work in MP) and it MUST be a mod (default ships rep for normal game is -1) then you have a problem. Go into the mods folder, open up market_ships.ini file (inside data\equipment\).Scroll down till you see:

[BaseGood
base = Hi01_01_base
marketgood = bwe2_package, 30, 0.7, 1, 1, 0, 1, 1

Now - this isn't exactly what you see, this is my own mod, but yours will probs have LOTS of entries - look for bwe2_package though - cause that is the sabre.

now then - the first number is level, second is rep. Max obtainable is 0.8 - so mine is set to sell just below max rep (hard to get). Yours MIGHT say 1, if it does change it to -1 or 0 or -0.4 etc, The limits are from 1 to -1, anything over that and you will be greated with mr desktop. Suggest 0 for saftey
Then save file and load mod up and then go buy ship
