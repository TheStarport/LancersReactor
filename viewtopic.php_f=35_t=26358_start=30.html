soundspawn

Thu Dec 04, 2003 3:52 pm by soundspawn

Actually, you set the starting base in the files themselves, not in FLX. Just open up data\characters\newcharacter.ini and change the "base = Li01_01_Base" to the appropriate base. You can use Freelancer Exploerer to get the name of the base you want. Right click on the base you want to start at, and edit. On the bottom there's a spot that has the code for that base. To see what I mean, try it on Manhattan and you'll see "Li01_01_base" down there.
