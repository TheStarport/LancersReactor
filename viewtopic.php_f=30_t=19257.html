flcodecs you need to frit make a short cut to it then right click on the shortcut you made then click properties then go to target and write in" -d <in>.fl <out>.txt"
with the space in the beginning just replace <in> with the name of the save you want to decode and <out> with the name of the text file you want the save to decode to.
that should get that one going
edit i forgot to tell you that for decodeing only to encode change -d in -e and switch .fl with .txt and you should have something like this
" -e <in>.txt <out>.fl"
