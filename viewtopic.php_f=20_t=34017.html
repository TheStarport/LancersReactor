Thu Aug 19, 2004 12:27 pm by Pinger

FF there is a bug with a altered new player file, if you make changes to it, for rep/unlocked gates etc. So if you keep one character from before that new default creation file then it will reset all the new ones created. So to have altered reps etc for a new player you have to delete ALL player folders and start again from scratch. That means all your admin/deputy char ships have to be made again, as well as every player has to restart.

As for that char viewing bug, check its not a .FL file less than 3kb, there should be an error log in IFSO if the player lists have been updated with the error message Error decoding file 05-6551344b.fl - too few lines in this file to be an .fl file! if it is find it with the server closed and delete the files linked to each error message.
