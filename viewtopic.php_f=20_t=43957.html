You don't have to uninstall Freelancer... You can just delete your Freelancer folder and then reinstall Freelancer.
The Freelancer folder is located at:
C:\Program Files\Microsoft Games\Freelancer
Deleting the Freelancer folder will not cause any problems... I do it all the time and then install Freelancer from the CD.
I also install the Freelancer SDK files because I test a lot of MOD's.
