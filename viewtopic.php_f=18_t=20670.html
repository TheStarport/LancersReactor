Victor

Thu Jul 03, 2003 4:18 am by Victor

The cost for the hull is accurate, but all of the ship packages include shields so the shield price is added in the game.

If you are simply making changes to the default hammerhead the AI bounty hunters will use your new and improved version if they were originally using them. If you want another faction to use them you need to add them in the "faction_props.ini" file in the missions folder and/or change the ships defined in "npc_ships.ini" in the same folder. If you want more bounty hunters to use them, or want more possible loadouts for the bounty hunters you can change the two files I already listed and possibly also some loadouts in "loadouts.ini" in the ships folder. If you just want to see them in more systems you have to go into the individual system files (DATA/UNIVERSE/SYSTEMS/<system name>/<system name>.ini and increase the difficulty level of the bounty hunter encounters, this last one is a whole lot of work, so be prepared if this is what you want to do.
