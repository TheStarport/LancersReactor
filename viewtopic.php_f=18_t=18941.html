The thing you are looking for is this: (in engine_equip.ini)

[Engine
nickname=xw_engine_01<------whatever engine you are using
ids_name=460616
ids_info=460616
volume=0
mass=10
max_force=160000<---this number divided by ...
linear_drag=466<--- this equals standard speed ^^^^^
power_usage=0
reverse_fraction=-20<--- Std speed * this number * -1 = effect of Z key

in this example my X-Wing goes to 6866 (reads as 999) when i press Z.
you might try this:
reverse_fraction=0.2 ;<--- original number (if i remember correctly)
the result will be that you move backwards at 17 (in this example)
hope it helps you out...

Shadowrunner if you listen to dr del about the z key bug you will realize you are talking nonsence.

it is a problem if you have actually fitted 2(or more ) engines to different hardpoints.

the solution is to go into goods.ini and remove one of the
addon = sometype_of_engine_01, internal, 1 lines from the ship in question.

i.e
[Good
nickname = name_of_ship_package
category = ship
hull = ship_code_hull
addon = sometype_of_engine_01, internal, 1
addon = sometype_of_engine_01, internal, 1 <------------ remove this line
addon = co_elite2_power01, internal, 1.......etc........

now Shadowrunner then your info becomes valid as you have to change/make a new engine of increased powerso you can travel at the same speed as before (but the z key functions normally now as you only have one engine).
