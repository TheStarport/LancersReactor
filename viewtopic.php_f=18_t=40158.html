I had one similar to that recently, there were actually two problems hopefully just the one for you. Factions Empathy/Initial World, when it updates the empathy change at the end of the mission.
And Best Path, it sets a best path if the mission was a loot and return mission. If the illegal, legal paths are screwed it can CTD.
I solved the Empathy/Initial World by going back through the values and found that I had doubled up some entries for some factions. The Initial world likes having simple numbers like 0.91, 0.65, 0.3, 0, -0.3, -0.65. Its best only to use these numbers. Make sure that everything is ok in MPnewcharacter.fl and Newplayer.fl.
The best path, was solved by simply removing a jumphole from the system. There's a limit on the amount a system can hold and this had apparently gone over it. Might have had a jumpthing created, then removed, then re added. FLExplorer (at least the older versions) dosen't delete the zone when you delete the jump thing. So you need to make sure that the zones get deleted other wise the goto Ar01, zone_Ar01_to_Li01,.. <----Would have more than one zone to go to.
