Here is part of the entry for ships on sale at Planet New London from the marketships.ini file

[BaseGood
base = Br01_01_base
marketgood = gf1_package, 0, -1, 0, 0, 1, 1, 1
marketgood = gf2_package, 4, -1, 0, 0, 1, 1, 1
marketgood = gf4_package, 13, -1, 0, 0, 1, 1, 1
marketgood = gf5_package, 24, -1, 0, 0, 1, 1, 1
marketgood = gf6_package, 30, -1, 0, 0, 1, 1, 1
marketgood = bwf_package, 4, -1, 0, 0, 1, 1, 1
marketgood = bwe_package, 20, -1, 0, 0, 1, 1, 1
marketgood = bwe2_package, 30, -1, 0, 0, 1, 1, 1
marketgood = bwfr_package, 10, -1, 0, 0, 1, 1, 1
marketgood = cof_package, 8, -1, 0, 0, 1, 1, 1
marketgood = coe_package, 28, -1, 0, 0, 1, 1, 1
marketgood = coe2_package, 32, -1, 0, 0, 1, 1, 1
marketgood = lf_package, 1, -1, 0, 0, 1, 1, 1
marketgood = le_package, 2, -1, 0, 0, 1, 1, 1
marketgood = lfr_package, 1, -1, 0, 0, 1, 1, 1
marketgood = bf_package, 6, -1, 1, 1, 0, 1, 1

Note* Only one of these ships is actually for sale, the bf (Bretonia Fighter)

The first set of digits is the level you need to be at to be able to buy the ship, change this:

marketgood = bf_package, 6, -1, 1, 1, 0, 1, 1

to this

marketgood = bf_package, 1, -1, 1, 1, 0, 1, 1

and you will be able to buy it at level 1 instead of level 6

The second set is the reputation you need to be able to buy the ship from the host faction. -1 means you can buy it even if they hate you, 0 means you need to be at least neutral and 0.75 would mean that you would need to be on very friendly terms.
