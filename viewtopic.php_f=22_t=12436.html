Tue Apr 01, 2003 4:10 pm by Harv

This method is a bit rough and ready, but I've caught quite a few level 38 cheats who weren't in New York this way. I've had 100% success with this, but only used it when absolutely necessary and I've exhausted all other means.

On the server machine, open the Multiplayer folder (under My Documents/My Games/Freelancer/Accts) and sort the date modified column in descending order. The folders at the top of the list are those accounts currently playing.

Locate the suspected cheater in the Edit > Accounts window of the server app. Kick them off the server. This forces a save of their character files.

Quickly alt-tab back to the Multiplayer folder and refresh the contents using F5. The most recent save will be at the very top of the list. Double click on that folder and right click on the Name file and choose "Properties". Look at the creation date. Cheaters should be easy to spot since no-one should be level 38 within a few hours of their character creation date.
