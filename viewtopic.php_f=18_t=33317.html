firebase

Mon Jul 26, 2004 7:14 pm by firebase

In order for you to access higher level of difficulties in general...

you need to have an npc that is capable of having that level usually in pilotspopulation.ini file that is for that system and also the system's ini file must have that same level entry in the npc pop section, ie. Li01_someNPC_pop that has mission enabled and the vignette sections too.

Now for the bases you need to edit the npc's that gives mission and select the appropriate DestroyMission Parameter to the designated level... for the level refer to diff2money.ini in the random mission folder....
