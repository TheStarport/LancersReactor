>> Freelancer QuickFix Patch v1.0c

The Freelancer QuickFix Patch is an unofficial patch for Freelancer that updates the game to the latest clean version. The patch includes all of the fixes from the Freelancer S.D.K. v1.3b (over 2000); it fixes some major Audio Hardware problems, updates the Video Hardware support, and fixes numerous minor problems with the game. This is designed to be a unified package that will attempt to represent a true patch to the game. <span style="text-decoration: underline">This was not designed to be used as a base for mod projects (for that you should use the Freelancer S.D.K.).</span>


>> Download Freelancer QuickFix Patch v1.0c Here!
>> Visit the =EOA= Players Consortium discussion thread Here!


Here is the list of Features:
<ul>
<li>3D Sound has been Disabled (Which fixes many sound problems for most people)</li>
<li>Advanced Graphics</li>
<li>Advanced Error Logging (Which makes troubleshooting problems easier)</li>
<li>Updated/Fixed freelancer.ini file</li>
<li>Updated video hardware support</li>
<li>Includes over 2000 code fixes from the Freelancer SDK v1.3b</li>
<li>Includes the battleship mismatched loadout crash fix (New!!)</li>
<li>Includes Freelancer Web Launcher v3.1.1.0 (see weblaunch.txt) (New!!)</li>
<li>The installer adds several new links to the Freelancer Web Links folder in your Start Menu (New!!)</li>
</ul>


Note: known issues with version 1.0c of the patch:
<ul>
<li>1. There are errors in the ini files! Yes.. this version of the patch is based on the 1.3b S.D.K. (because it is the only <span style="text-decoration: underline">stable</span> version of the S.D.K. out right now).</li>

<li>2. Battleship stations caused the client to crash if the client and server's loadouts_special.ini file were different. **Fixed March 13, 2007**</li>

<li>3. The installer can only apply the perfoptions.ini changes on the active user, so if there are multiple windows users who play the game they will need to run the install again for each.</li>

<li>4. You may need to reinstall the drivers for your video card, download newer ones, and/or reinstall the latest version of DirectX if your Video Card is not working properly.</li>

<li>5. GeForce 6600 GT owners: your card seems to have some problems with Freelancer. Make sure you are using updated drivers.</li>

<li>6. A very few people will experience severe problems if 3D Sound is disabled. If you are one of them, you can easily re-enable it in the game options.</li>
</ul>
Note: If the game ever crashes and you get one of those famous error reports you should send it in because Microsoft does check these out and updates their F.A.Q. from time to time.

Important:<span style="text-decoration: underline"> This is an unofficial patch and neither Microsoft nor Digital Anvil will support it, so don't go asking them about it.</span>

It is recommended that you use this patch instead of any others available in the community as it represents the cleanest and most updated codebase. The Freelancer QuickFix Patch replaces all of the following releases:
<ul>
<li>Freelancer S.D.K. (all versions) by Louva-Deus</li>
<li>Unofficial Freelancer SP Patch (all versions) by Buck Danny</li>
<li>Unofficial FL Single Player Patch (all versions) by Hobbes</li>
</ul>


Thanks to all those who have contributed their time, effort, and ideas to help me complete this project:
<ul>
<li>Chips, Glock36, Bakedpotato, Bluecat, Accushot, Gibbon, and many more...</li>
</ul>

@Esquilax - There is an option in the perfoptions.ini file about advanced graphics, if you set it to 1.0 you have advanced graphics. Then you can say your mod or whatever has advanced graphics etc etc.. blah.. blah. I think I have heard of only one person who ever noticed a difference

The 3D Sound issue actually affects many things in the game, and strangely enough many of them relate to sound.. I can't say for sure but I believe that 95% or more of people who disable 3D Sound will not be able to tell the difference, it is a feature designed for surround sound environments. Since it fixes more things than lost with it off I think it is a safe conclusion that defaulting to off is a good idea. If you miss it just turn it back on in the game options.

@Jackal(e-mail response) - This is the first release (1.0) and if you recall the official patch still showed 1.0. The next version of the QuickFix will include a version change to 1.1 since it will contain some features not compatible with previous versions. Also, I may re-release this version depending on if there are any other problems found with the patch.

@Chips & Bejaymac - This QuickFix release is based on the 1.3 S.D.K. because the 1.5 is not yet ready. I have not had time to work on the S.D.K. in over five months, the last batch of changes I made to it were sloppy and made up of beta quality fixes/hacks. My hope is that the final build of 1.5 will be in the 1.1 QuickFix release(no date).

For those of you who use the 1.5 S.D.K. I would recommend that if you want the few benefits from the QuickFix then install it first and the 1.5 afterwards.

All feedback on this project is welcome but please direct S.D.K. specific reports to either this thread or to the Error Reporting Form on the =EOA= site(preferred).
