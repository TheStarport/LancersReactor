SWAT_OP-R8R

Thu Dec 22, 2005 5:01 am by SWAT_OP-R8R

there are several issues that wont let mods activate correctly.

1. "central directory cant be found" -> that just means that the mod.zip.flmod was not downloaded correctly... a 2nd download should do it (you should use a download manager like flashget to download big mods to be sure that everything is ok with the file)

2. freelancer.ini not found -> meaning that you had other mods running before and flmm didnt deactivate them correctly -> solution reinstall FL and try to activate the mod again

3. the mods splash screens is showing up but FL crash to desktop -> meaning that you have used other mods before and there are singleplayer savegames that are not compatible with the mod you are trying to run -> solution: delete the singleplayer savegames or copy them to another directory

4. crashs while using the jumpgates/jumpholes -> deactivate the 3d sound in freelancer (that will fix it)


ive only experienced those problems with the activation of mods... if there is more feel free to add them
