Pinger

Wed Nov 05, 2003 11:13 am by Pinger

1. use Ioncross and find the folder thats yours and move it out of the mydocs/games/freelancer folder to another folder.

2. start the server and join and then make the exact same named characters.

3. Close the server and find the folder for those new characters and delete every FL file but Not the name file, then copy the backup FL's in to that folder you made earlier.

4. restart the server and your characters should be back.
