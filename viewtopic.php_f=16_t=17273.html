Mon May 19, 2003 12:34 pm by Silver71880

The only thing you need to do in your universe.ini is change visit = 0 to visit = 1
So if you take for example Texas:

[system
nickname = Li04
file = systems\Li04\Li04.ini
pos = 8, 9
msg_id_prefix = gcs_refer_system_Li04
visit = 0
strid_name = 196612
ids_info = 66126

change to:

[system
nickname = Li04
file = systems\Li04\Li04.ini
pos = 8, 9
msg_id_prefix = gcs_refer_system_Li04
visit = 1
strid_name = 196612
ids_info = 66126


If you do that at all systems, you'll have them on your nav map.
