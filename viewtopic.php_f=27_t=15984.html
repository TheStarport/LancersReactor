IGx89

Sun May 04, 2003 6:40 pm by IGx89

If you want to use Freelancer Mod Manager, install it, run it, go to "Help\How Do I", and read Option #1. That will explain how to get mods into FLMM, so they can be easily activated/deactivated. Also, if the mod file ends in .flmod, then FLMM will automatically install it for you if you double-click on the .flmod file!
