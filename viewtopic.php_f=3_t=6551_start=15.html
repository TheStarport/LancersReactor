Love it. I'm pre-ordering the game today for sure.

Love the mouse control, not a game I want to use a joystick with I think. Only drawback I've seen so far with it is I tend to simply just follow an enemy around and not do too many maneuvers. That's minor point for me though. Need to learn the keyboard commands.
