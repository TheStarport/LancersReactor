Good luck but Mods on a pure server or server that doesnt have that exact mod will make it laggy no matter what.

Also you say 1mbit but you forget the important factor in any server the upload speed. If its a standard cable {128k up} or DSL{256k up} 1mbit connection then 15 is about the best you can hope to host with little lag and thats with out peopke using mods.

Get one player using a extra speed Cruise/Thrusters and one using a Beam weapon and your server will crash, so I would rethink the idea of allowing mods.

But good luck anyway.
