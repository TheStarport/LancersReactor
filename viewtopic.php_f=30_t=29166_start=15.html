parabolix

Tue Jan 04, 2005 2:55 pm by parabolix

bettygoodacre:

There is no Mat importer for Milkshape but there are two ways you can get the textures from the MATs.

1. Download the CMP importer plugin for Milkshape here . When you import a CMP, you are given the choice to select a MAT file. Select the MAT you want to import the textures from, then press the Extract Textures button and select the directory you want to save the textures in. You can find the textures in that directory.

1. Download the UTF Editor here . Open the MAT you want to extract textures from. There will be two Nodes. A Texture Library and Material Library. Expand the Texture Library and find the texture you want to extract. Expand that node and you will see another Node named MIPS. Click on that node once. Then click on the Export Button in the bottom right of the program, and select a directory that you want to save it in.

Hope that helps
