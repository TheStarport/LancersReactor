Louva-Deus

Thu Nov 03, 2005 6:39 am by Louva-Deus

An important note for people experiencing sound problems in the game and especially those that result in crashes. If you have a codec pack installed like K-Lite Codec Pack(s), Nimo Codec Pack, or the ACE Codec Pack(s) then you need to make sure that there are no problems with the pack before you use it.

Now many of you will not experience any problems other than some sounds simply are not there, but you will really notice the problem if 3D Sound is disabled. ACE Codec Pack in particular has a problem with it that can cause the game to crash in some cases.

So make sure to do some research on codec packs before you install them . You can find all kinds of reviews just by searching Google. If out of a good chunk of reviews/comments 25% of them (or more) mention problems then you should reconsider choosing that package.

Another tip that is often promoted by Chips is to disable 3D Sound , in most cases you will never need this feature and there are many positive benefits of having it off, some of which are: less stuttering, fixes several sound problems, eliminates some crashes, and overall improves the performance of the game . This also helps to fix the sound for those of you who use headphones (Glock36 ). You can find the option in game under Audio or you can install the Freelancer QuickFix Patch which does this for you (among other things).

Hope this helps!
