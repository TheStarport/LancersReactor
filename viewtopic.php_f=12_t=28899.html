Mustang - I've learnt to never assume anything in IT; there are fewer problems that way . Still, I assumed that it was Windows too, or else he'd have been using "shutdown -r now" or something similar. Now of course, there is the question of which version of Windows. I won't assume XP .

For Win95/98/ME the shutdown shortcut is "C:\windows\rundll.exe user.exe,exitwindows".

There does not seem to be a shortcut option for "Stand By" mode in XP, however there is one option. Use this VB script;

set WshShell = CreateObject("WScript.Shell" )
WshShell.SendKeys "^{ESC}ulss{ENTER}"

Just save it as "standby.vbs", place it on your desktop, and everyone is happy.
