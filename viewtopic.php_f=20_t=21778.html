Sir H Master. I had the same problem but with a Winfast Geforce 4 Ti4200 128mb 8X AGP. Microsoft Tech support told me Freelancer will not run with Nvidia's latest driver 44.03 and suggested I install 41.09 from Nvidia's driver archive and the game started up ok. Apparently, if your video card has not been tested by Microsoft and given the WHQ something label, Freelancer wont work and You will still need to download the latest Direct X and sound driver. However, Although I still have the occasional game crash which I am hoping to sort out, try changing the driver. I am not guaranteeing it will work but it did for me.
