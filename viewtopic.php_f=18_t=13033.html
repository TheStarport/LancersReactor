nice ship i must say very detailed.

Currently acceptable model formate is in FL .cmp file formate. You will need to import your lightwave model into milkshape 3d and convert it into .cmp file for use in FL. One thing to note is that milkshape 3d max out at around 7-8000 faces and around same number of vertices.

read the sticky on HCl's millkshape 3D exporter for more information.
