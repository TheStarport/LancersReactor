Possible workaround for memory leak

Like many other admins out there, I'm getting sick of my server crashing every 16-20 hours depending on the current load.

Out of curiosity, I downloaded and installed a freeware memory manager to free up memory which wasn't being released properly.

Prior to doing this, I watched memory usage for flserver.exe within the task manager and saw it grow steadily, eventually causing it to crash at, or around, 160MB.

After installing RAMpage (http://www.jfitz.com/software/RAMpage/), memory usage would climb up to ~100MB before dropping back down to ~60MB or so as RAMpage flushed the memory. It seems to have made my machine more stable. I only installed RAMpage yesterday, so haven't had a chance to make any long-term observations.

Hope this helps other admins out there.

Blue Harvest 24/7 Dedicated Server... minus the few minutes when the memory leak takes us down.
