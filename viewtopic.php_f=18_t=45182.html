The model cloner is okay but it can leave you with completely busted clones if your not careful.

I don't use it for icons as I prefer to make brand new ones rather than rehash existing ones, Colin's 3DB Exporter makes it a breeze to create them as it comes with an icon template and a settings file.

Ships can be a problem as some times the cloner messes up the component parts ID's, these can be fixed in UTF edit most of the time, but occasionally they get corrupted and become useless.

I don't know if it's just me but the damage parts don't clone properly at all, both hardcmp and freelancer CTD with them. I've taken to doing them the hard way to get them, modelcloner for the reference, a Hex editor, CRC Calc and utf edit to swap textures.
