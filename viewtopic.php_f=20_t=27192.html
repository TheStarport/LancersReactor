for starters a physical memory dump is nothing to be 'scared' of all it is is winblows wrting whatever is in memory to a file on your hard disk (so you can -in my case- plow through 512 mb of data to see if anything went wrong somewhere)

As far as the blue screen occuring in most cases it is caused by a bad behaving dll or program causing a bad memory write (you can't map ring 0 memory
space to a ring 3 process without blue screening when the ring 0 process
writes to it and vice versa << if this goes over your head just ignore it)

If freelancer is the only program causing the blue screens there is a good chance it is corrupted somewhere and your best bet would be a complete fresh install of the game after wiping all traces of it of your machine including the registry entries . if there are other programs causing simmilar problems there is a chance your OS is corrupted in some form or fashion and you need to re-install it from scratch
