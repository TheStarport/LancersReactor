There is no need for talking to server admins, switching hard drives or manipulating the registry.

The multiplayer ID is unique to the Windows account on a PC. This means that every user on a PC can have his own set of multiplayer characters (and BTW also his own savegames, gamesettings, etc.).

The steps to "copy" the accounts from PC 1 to PC 2 without destroying any existing accounts on PC 2 are:

1. On PC 1, press the Start button in the systray. Go to Programs\Microsoft Games\Freelancer\Examine My Account ID.

2. Write down the multiplayer ID.
(Do not bother to copy the ID to a mail as you will not be able to paste it into the game on PC 2 anyway).

3. On PC 2, create a new Windows user and login as this user.

4. Start Freelancer, choose Multiplayer and click your way through the screens until you reach the Multiplayer ID screen.

5. Type the Multiplayer ID you got from PC 1.
(Yes, TYPE ... pasting with Ctrl-V is not possible for some reason)

You will now have access to your existing accounts on the Freelancer servers from both PC 1 and PC 2. However, the "Visited" column in the server list is unique to each PC/user so you will have to remember which servers you have visited.
