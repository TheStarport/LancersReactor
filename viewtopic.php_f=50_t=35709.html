like for manhatten you'd do this:
[BaseGood
base = Li01_01_base
marketgood = your_ship _package, 0, -1, 1, 1, 0, 1, 1

the only problem with selling at manhatten is that you have to get rid of one other ship:
marketgood = gf2_package, 4, -1, 1, 1, 0, 1, 1 /\ StarTracker
marketgood = lf_package, 1, -1, 1, 1, 0, 1, 1 /\ Patriot
marketgood = lfr_package, 1, -1, 1, 1, 0, 1, 1 /\ Rhino

to disable one of these you have to change the numbers around:
marketgood = gf2_package, 4, -1, 0, 0, 1, 1, 1 /\ StarTracker

then the startracker isent sold

Sun Oct 31, 2004 2:43 pm by Chips

Megatog - have you actually flown your ship yet?

If you can fly it, and you can sell it - then its all good. After that I would wonder if your using FL:E on the right mod folder.
However, since I have never set anything to sell in that tool - I do not know if it would work at all...........

Its simple enough to find out how to make it sell on the base you want by hand - and like Glock says, learning to mod by hand is the only way to actually get results in all honesty. Tools is fine to use, but what happens when something goes wrong? You have to find the fix.

There is no simple quick fix to modding, you have to learn what to do.
To find out how to sell ships on bases, please refer to the sticky's up above - its linked to newbie questions halfway down, and it should tell you what your after. Sure - we could post it down here for you, but that would mean we spend our time so that you don't have to..........which just doesn't sound right to me

Oh - checklist as well:

You have your CMP and MAT file for ship
Entry in shiparch.ini
Entry in goods.ini
Package in goods.ini
set to sell in market_ships.ini

Done and dusted.
