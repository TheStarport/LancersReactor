Tue May 06, 2003 5:00 pm by CW

Very easy. The game doesn't care even if your gun is shooting throught the hull, so you can put the hardpoint anywhere without problems. So is fixing it, just change the Max and Min nodes for the gun hardpoint in the ship's CMP file to zero, but you will have problems aiming it given the way the game is set up - ie. there is no way to "lead" the target properly with the mouse flight interface, and guns in Freelancer don't shoot unless your cursor is within the gun's firing arch, but with a fixed gun that is only a needle point straight in front of the ship. Go check out my hardpoint tutorial in the Tutorial Forum for more details.
