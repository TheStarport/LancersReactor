megaburn

Tue Aug 01, 2006 10:38 pm by megaburn

That won't work, TNG INI's are reencoded with bini and an entry that prevents most bini decoders from working correctly (returns a run time error, empty file, or garbage text). This is done as a cheat control so I doubt anyone will post the work around to it (just as people aren't allowed to post cloak mods).

To win the race you'll need to buy the fastest fighter you can find with the best thrusters available. Then use your thrusters instead of cruising. Check the TNG manual, I think it includes speeds for all ships, if not then check the TNG forums for tips on beating Hovis.

BTW, TNG doesn't have a new physics engine, they just retooled the physics values a little. Adding a new physics engine to FL is impossible at this point, it would require the source code and MS hasn't released it (yet, hopefully).
