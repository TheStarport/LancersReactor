@kuvacz, You do find that if you go around attacking everybody, after a while everybody will hate you. Can't say i blame them. You need to pick a character for you game and try to stick with it. Last thing you want is to end up without places to dock in the more difficult regions. And, its a hell of a lot harder gaining rep than losing it..
Carinea, there is a problem with the docking ring. It means that nobody can play through the quest with the current version. To fix:
01. Open up:universe/systems/hi01/hi01.ini
02. Search for:nickname = Hi01_02_dock_ring
03. Change the reputation to fr_quest1_grp
04. Save

@Bejaymac, I did run into some problems with Hardcmp where it was creating double hardpoints for things like HpCloak used for the ship upgrades. The modified versions of the original ships are indeed without destroyable components. I have tried using modelcloner in the past and ended up with a mess. Oh, no the archetype entries should be ok. Each one has its own mat file and archetype names. Except the icons where i just used existing icons and renamed the file. There is a known bug where you sometimes see the wrong image for the conflicting image names, but it shouldn't cause any problems other than being a bit crap.

@Richard w. sabatino, nice pic, looks like you're keeping busy

@kuvacz, the file that controls the reaction is missions/empathy.ini. This has entries so that for every change of other factions it affects groups. For the Zoners, these are the entries (simplified):
Xenos, 0.05
Hackers, 0.025
Unioners, 0.05
GMG, 0.050000
(Story faction, whoops), 0.05
IMG, 0.1
Bounty Hunters, 0.025
Federation, 0.05
Imperial Navy, -0.05
Ind Gas Co, 0.08
Fr Police, 0.05
Duchani, 0.05
Eastern Coalition (collective), 0.05
Nomads, -0.05
So, the best group to get in with is IMG. Some of the entries are a bit dubious, I'll remove a couple for the next version and go through the others.
To actually change you current saved game, it isn't an ini, instead you'll need the saved game editor. Remember to back up the saved games first and edit gd_z_grp.
Cheers for the interest guys.
