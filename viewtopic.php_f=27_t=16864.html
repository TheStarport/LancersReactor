Dragon1

Wed May 14, 2003 1:44 pm by Dragon1

First you download Freelancer Mod Manager v1.1 dated 11-04-2003. Install the Mod Manager. What you should do next is create a directory to hold all of the mod that you download. Download the mod and click on them twice to install. Once you have seen the message that they are installed and ready to be activated then open the Mod Manager. Select one and there should be an activate button for it in the middle of the screen. Click on that and the mod should have a green check mark beside it. The mod is now activated. Any mod that turns red conflicts with the activated mod. You can deactivate the mod the same way. Hopes this helps. You are supposed to be able to go back to normal by deactivating the mod and you can switch between mods.
