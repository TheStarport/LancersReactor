Tue Jul 10, 2007 5:54 pm by DwnUndr

Both files are relatively small, 250kb or so. I left everything in there with no problems. If you start deleting ships, make sure you don't delete any that the npc's use (unless you are creating ALL new npc's including what they fly). Many npc's I have are just renamed old ones. They provide the basic 'this guys hates that guy so they will fight in front of you to make the system appear alive'. New npc's will have new tasks. Basically, don't worry about leaving those files intact.

The intro stuff, no sure which you mean, so here it all is:

Intro animation (thn) is in data\scripts\intro

Intro still picture is in data\interface\neuronet\navmap\newnavmap\nav_prettymap.3db

The intro thn's are called in data\universe\systems\intro\bases\rooms\intro1_cityscape.ini (and intro2 and intro3).
