There are dozens of tutorials here explaining how to change the ini stuff. If you screw up, it can crash your machine, corrupt game saves, crash other people's servers when you visit them. If you're not handy with code and modding, then you might want to avoid ini changes.

I initially repsonded to the first post here because the mpnewcharacter.fl is the simplest/safest way to do what he was asking (ie: no file conversions, no mods need be circulated, no chance of crashing others machines, and it's a server-side change that you can hook everyone up with at once).

As far as not getting the mpnewcharacter.fl changes working....the usual cause is forgetting to comment out the default....or not re-starting the server after the change is made.

Remember, mpnewcharacter.fl is for the multi-player server setting. It will not effect single-player new characters.

Doh, my bad...I should have noted that. I'm not sure that can be changed on the splash screen at startup (with only a server-side change) but I'll look.

As far as starting factions, you'll see the section of code line that start with "house". The negative ones are the bad faction ( -0.XX ), zeros are neutral, and good factions are the positives ( 0.XX ). You can adjust starting faction from there (all negatives: everyone will hunt you, all zeros: everyone neutral, all positive: everyone loves you). So, 0 likes you.... 0.35 likes ya alot......0.65 love you. -0.65 hates you in a big way..... -0.35 kinda hates you, etc.

I haven't tried all the combinations, but it appears to work that way.
