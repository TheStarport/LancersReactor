If you're sure it isn't a firewall issue, then there aren't that many options left. One is a complete reinstall (delete FL Folder, reinstall).

The other is to try connecting directly to the server (useful when the Global Server is down). Forget about fldirectip.hta if it's giving you trouble.

Right click on a shortcut to Freelancer.exe, then select properties. In the target field, add -sxxx.xxx.xxx.xxx:pppp - where xxx.xxx.xxx.xxx is the server IP that you want to connect to, and dddd is the port number (usually 2302). It should look like this:

"C:\Program Files\Microsoft Games\Freelancer\EXE\Freelancer.exe" -s24.170.31.188:2302

Remember to leave a space between the last " and the -s. Click on the shortcut, and go to Multiplayer, then select LAN - the server should show up very quickly.
