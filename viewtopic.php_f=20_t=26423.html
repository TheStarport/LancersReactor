ok the card out the p3 isnt an option then
shared memory basically is the fact you have no graphics memory of its own
thats the way onboard graphics tend to be so it instead uses your system memory
atm you have 224meg of system as the graphics are using 32 meg
change the graphics to 64meg and you have 192 meg of system
i know it says shared but believe me the graphics grabs all 64meg
oh and i play on my laptop which is almost those settings ie 256m of ddr 2.6g p4 with onboard graphics (but its better on my desk tops which have up to 1 gig of ddr and 128m graphics) though the laptop copes fine so so should your system
