Files that get changed and files that get added need to be both server and client side.

In other words when you add ships, weapons, bases, etc. both server and client need those files.

Files that both have in the original files already don't need to be client side.

It's a good idea to have all the files synchronized anyway...

Also you should start your Mod project with the Freelancer SDK version 1.3b-sdk_20070313 files. Mod those files or add your modified files to the SDK files and leave them in the Mod even if you don't modify all the files that are included in the Freelancer SDK ini files.
