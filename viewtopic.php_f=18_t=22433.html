wow....i never thought i would see this asked again....well...im not one to flame so i'll help out anyway...there is a file called market_ships.ini...the path should be (if you installed it to the default directory) c:\program files\microsoft games\freelancer\data\equipment\market_ships.ini.....what you will see may look like a bunch of mumbo jumbo...what its saying is the nickname of each base and each ship....im not gonna go through the nicknames...i think you can find that in reynen's compilation post...if you've modded the game before however you probably already have figured out the pattern in the nicknames...the first thing you see is

[BaseGood
base = Br01_01_base
marketgood = gf1_package, 0, -1, 0, 0, 1, 1, 1
marketgood = gf2_package, 4, -1, 0, 0, 1, 1, 1
marketgood = gf4_package, 13, -1, 0, 0, 1, 1, 1

That is (i think) Planet New London...those first three ships are the starflier, startracker, and hawk....the first number you see after the package, determines the level you need to be to purchase the ship....the next number determines how your reputation must be with the faction to buy the ship...all ships are set to -1 which means your rep can suck and you will still be able to buy the ship...the last 5 tell if the ship is buyable there...the numbers 0, 0, 1, 1, 1 mean the ship is not buyable while the numbers 1, 1, 0, 1, 1 mean the ship is buyable there....be advised though each base can only sell 3 ships a time...so if you have more than 3 not all of them will be there...thats the basics for now...

And BTW....thats not how the quote goes

Oh and another BTW....making the cruiser buyable requires alot more than just editing one .ini........
