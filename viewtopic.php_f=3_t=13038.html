Somthing I think will mak freelancer's sequel (please don't tell me there won't be a sequel..) better: supplies.

#1 The player's ship should actual run on fuel .The player should have to buy more H-fuel when he/she needs more. The fuel should be stored in the wings or something so that it DOES NOT take up cargo space. The fuel should constantly be decreasing unless the player turns his/her engines off. The fuel should decrease more rapidly if the ship is on after burner and even more rapidly if in Cruise. If the player ever runs out of fuel, there should be an emergency generator which does not need fuel, but is significantly less powerful.

#2 The player should consume food/water. The consumption of fuel/water should work similarly to the fuel, except for two things; if the player runs out of food/water and does not get more for x amount of time, he/she dies. And the food SHOULD take up some argo space.

I think that these would be very good additions to the game and make it a little more involved. If anyone from Digital Anvil even bothers to read this, that is...
