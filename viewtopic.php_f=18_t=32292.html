As I understand it, you want to get a random ship out of a group of ships when you start a new game...assuming that is what you mean:

A major hack would be to first make a copy of whatever file specifies your starting ship for each ship you might want to start off with, editing them so that each specifies a different starting ship. Then, write a batch file that randomly selects one of the copies and then makes a copy of it, named whatever Freelancer actually looks in to determine the starting ship. Then, the batch file starts up Freelancer, with the randomly selected ship. This obviously would only work between program launches, not in-game "new games".

So, I guess it's possible, if this could be considered a solution. If it can't (I admit, it's a hack if there ever was one), then I don't know a way to do what you're asking.
