Win 98 Scandisk/defrag sucks.

Here's an option that should work for you...even if you don't know anything about DOS.

Click on Start.

Click on Shut Down.

Select REstart in DOS mode.

At the command Prompt:

Type this:

defrag

That will run the DOS Mode defrag.

I would also recommend running this:

scandisk /surface
