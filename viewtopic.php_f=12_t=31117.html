im with taw and mustie on the router, Wireless, at the moment, is only goot for mobile laptops, hardwire the system with a good ol' CAT-5 cable though the walls.
Also, depending on the server O/S, server usage and demand, it might be an idea to up the CPU to 500MHz or get a dual system and leave the secondary port open until you find out how much CPU power the FLServer will need eg:
CPU port 1: 200MHz for data handling(local) and O/S functions
CPU port 2: FL server CPU

if the server demand dosent get too high, just leave port two empty, but if you get resource problems, use the secondary port and use that for FL server.

Hope that helps
