Ricky26k,

Sounds to me that you may have a problem in one of your .ini files , could be a typo or something not referenced properly i.e. path statement.. I would check these first.

1. Ship id numbers, (as far as I know) should not really cause the problem... If in doubt I set them to 000000 for testing purposes, Freelancer sees this as "object unknown" ... One thing I find works is to pick a model approximately the size and style of your ship.. buy it in game.. then replace the cmp and mat files .. if your ship appears ok then the files are alright....
2. Not sure what you mean here could you elaborate.. Please ;P
3. Textures if .tga, should be saved in 24 bit format... if the texture looks strange try making it larger just to see if the problem is still there... did you extract native freelancer textures or make your own? Are you using one 256 x 256 texture for the whole ship?

If you want to .zip the files and email them to me and I will take a look.... but please dont just send files with no message or the spam filter will remove it..
