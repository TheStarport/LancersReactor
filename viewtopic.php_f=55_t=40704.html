If you downloaded the Federation Ship Pack MOD by Archedyne then the file name should be federationshippack.zip.flmod

If it is federationshippack.zip.zip change the file name to federationshippack.zip.flmod

I don't know why but for some people when they download a mod something changes the file name to .zip.zip from its original name of .zip.flmod

After you rename the mod back to federationshippack.zip.flmod then just double left click on the file name and the Freelancer Mod Manager will take over and unzip the mod and then the mod manager screen will pop up automatically...
Then you can click on the mod and then click on activate...

You did already download and install the Freelancer Mod Manager....didn't you?
