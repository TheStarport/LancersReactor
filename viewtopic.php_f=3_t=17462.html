Captain Flint

Wed May 21, 2003 1:31 pm by Captain Flint

You can also simply comment out the movie call. Under the freelancer directory go to \EXE\freelancer.ini. This file is *not* compressed and does not require any tools beyond notepad to edit.

In the file, near the top find the section:

[Startup
movie_file = movies\MGS_Logo_Final.wmv
movie_file = movies\DA_Logo_Final.wmv
movie_file = movies\FL_Intro.wmv

Just add a ; to the beginning of the line to comment it out.

[Startup
;movie_file = movies\MGS_Logo_Final.wmv
;movie_file = movies\DA_Logo_Final.wmv
;movie_file = movies\FL_Intro.wmv

This will skip the three introductory movies and drop you straight to the main menu.
