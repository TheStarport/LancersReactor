Should manage 6 to 8 players if it's dedicated - although it will struggle if most of those players are in combat. Any more and you'll start getting lag.

My advice would be start somewhere around there and watch the FLServer latency meter, CPU load and RAM usage when the server is full and adjust the user limit until the latency bar spends most of it's time hovering around the upper end of the green and low/mid yellow.
