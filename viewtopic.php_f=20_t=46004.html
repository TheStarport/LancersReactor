Sat Jun 23, 2007 8:22 am by Rankor

You have to make sure the mod manager you're trying to use is compatable with the mods that were developed. There are a few versions. The latest version, 1.4 beta 4 will only tend to work with mods that were developed with the SDK 1.5b.

So in a nutshell, if you're experiencing problems due to using the "latest" version of FLMM, try reverting back to the previous version (FLMM 1.3)
