bakedpotato

Tue Dec 02, 2003 9:05 pm by bakedpotato

Yes you can play a multiplayer game by starting the flserver.exe program that comes on the cd and is installed by default.
Navigate to Start - All Programs - Microsoft Games - Freelancer - FLServer
You have to fill in server name, use any name you want. Number of players default is 16, don't worry about that.

Once the FLServer screen is up and running then you start Freelancer and choose multiplayer - lan and wait for the connection screen to come up then click on your server then click on connect and have a blast.
