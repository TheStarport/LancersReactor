Sanic

Thu Apr 15, 2004 10:40 am by Sanic

Always remember to include all the relevent information for an accurate solution to your problem.

Also, you should know, that most mods don't actually change the INI files, they simply include a script file that tells the game to alter the INI files in a certain way. That is how the mods are able to be turned on and off. If you changed the INI files manually, It would be quite difficult to know where you went wrong. You may have to undo all your changes and try again. Or make an actual mod, meaning write a script file. That is alot easier to do than tracking down errors in the INI file. If you write a script file and it is wrong, the FLMM won't load it and you know it is wrong. It beats crashes that don't tell you whats wrong. Errors in script files are alot easier to troubleshoot because people here at TLR are quite familiar with them. There are also alot of tutorials on script files here in the forums.

Sanic
