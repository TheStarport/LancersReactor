to edit ship prices goto the \DATA\EQUPIMENT\ folder

Ship prices are in good.ini you'll need ot know the nickname of the ship eg or_elite = Anubis heavy fighter

[Good
nickname = oe_hull
category = shiphull
ship = or_elite
price = 1000
ids_name = 12019
item_icon = Equipment\models\commodities\nn_icons\or_fighter.3db

for anubis

Shields and thruster can be found in st_good.ini eg :

[Good
nickname = order_thruster
equipment = order_thruster
category = equipment
price = 100
item_icon = equipment\models\commodities\nn_icons\EQUIPICON_thruster.3db
combinable = false
ids_name = 263741
ids_info = 264741
shop_archetype = equipment\models\st\ku_thruster.3db
material_library = equipment\models\ku_equip.mat

guns and weapons prices are here in weapon_good.ini
