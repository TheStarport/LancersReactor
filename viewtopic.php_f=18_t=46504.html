The most practical size for any ship would be to ensure that it fits
inside the trade lane rings. (that's assuming you want to use them for fast travel)
A ship can be sized too large making trade lane travel impossible.
Here is an example of a large ship still being able to use the trade lanes.
This one just fits..and also just fits the Docking rings.



Now, a ship of any size can dock with the docking rings...with one little trick..
and that is to use the Ghost Docking Ring mod. (what this does is modify the .sur file for the dockring so you can pass through it without getting snagged on it)
By understanding what it is actually doing, you can modify that .sur file by renaming it to the same as in the mod...so you don't have to use the mod.
However, enabling a ship to use moors is something I'm still working on.
Somewhat complex (for me anyway) in what you have to change in the solararch.ini files.
Also, it starts getting somewhat screwey when you start enabling docking with moors on stations. The best I've seen so far made it appear as though the ship was docking where it should with a station and in its correct orientation. But that involved moving docking points around using the cmp editor.
I'd like to see it possible to use both berths and moors where they are applicable.
Small ships use only the docking rings while large ships dock and undock using only the moors.
But I am given to understand that once you modify the solararch.ini file to enable docking with moors, it also allows all ships to use them.

Gibbon

Mon Oct 01, 2007 11:23 pm by Gibbon

Two places to adjust the view, shiparch.ini and you can make your own cockpit as well in the COCKPIT folder, just copy one you like and rename it.

SHIPARCH.INI

Look for these values

camera_offset = 10.330000, 39.819000
camera_angular_acceleration = 0.050000
camera_horizontal_turn_angle = 23
camera_vertical_turn_up_angle = 5
camera_vertical_turn_down_angle = 30
camera_turn_look_ahead_slerp_amount = 1.000000

Mess with the camera offset as the first figure is the up and down angle from the ship, second is distance from the ship

In the cockpit ini you'll see these,

[Cockpit
mesh = cockpits\rheinland\models\rh_freighter_cockpit.cmp <---- type of cockpit, can be changed, check other ships
int_brightness = 0.500000
head_turn = 30, 10

[CockpitCamera

[TurretCamera
tether = 0.000000, 6.000000, 50.000000 <----------- 2nd number is height, 3rd distance
yaw_rotate_speed = 2.000000
pitch_rotate_speed = 1.500000
accel_speed = 5

[RearViewCamera <---------- this is an extra bit you can put in for a proper rear view
view_position = 0.000000, 6.000000, 23.500000 <---- same deal with the numbers

Hope it helps
