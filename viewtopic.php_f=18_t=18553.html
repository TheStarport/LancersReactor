I use the .rb1, .rb2 rbn file extensions to keep track of the changes I make to a file. The .rb2 versions include the .rb1 patches and hold one or more other patches. Reason why I supplied the .rb1 and .rb2 as well was simply to give you as much choice as possible in case you don’t like a patch that’s in an .rb2 file. It’s the same thought that is behind all the separate files, you can use them individually.
To summarize: if you like all the patches I made for a system and there is a .rb2 file use that one and ignore the .rb1 file. Remember to backup the original ini files and rename .rbn to .ini.

Loot on shipwrecks, replace data\fx\fuse_surprise_solar.ini with fuse_surprise_solar.rb1
