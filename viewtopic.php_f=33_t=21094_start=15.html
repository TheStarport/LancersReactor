Frontier Fan: Darklight is correct, you need to change a setting. If you are using Win2000 or XP, then you have to goto START > CONTROL PANEL > FOLDER OPTIONS. Then click the VIEW tab and one of the check boxes will be "Hide Extensions for known file types." This must be UNCHECKED for you to see the file extension, which is a .ZIP file. Rename the file and change .ZIP to .flmod and it should load into FLMM just fine.
