Are you running the demo or full version? This only works on the full version.

Are you using an uncompresser on the INI to convert them to readable form?

I have told you the neccessary steps. It is also on the tutorials.

Again, open the ini. Look for the ID code of your planet. You can find that by looking at the Compilation thread I made.

Find the code for your ship.

If it is already there then change 0,0,1,1,1 to 1,1,0,1,1

The number after the name is the level needed to buy, change it to 0. It will now look like this.

Ship name, 0, 1, 1, 0, 1, 1

Else add it to the list. Only enable three per place.

------------------------------------------------------------------------------------

Oh, I don't like liars either. You said you read the post? If so you would have seen this part in my post explaining it for you.

*************************************************************************************
Market Misc.ini - What bases sell what equipment - The FIRST number after the name is level requirement. Replace with 0 to buy at any level.

Market Ships.ini - What bases sell what ships - *ONLY 3 per base!!*

Change 0, 0, 1, 1, 1 to 1, 1, 0, 1, 1 to enable buying.
*************************************************************************************
