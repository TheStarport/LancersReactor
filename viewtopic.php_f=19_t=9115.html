Server Stability?

Hello All,

Just wondering if anyone else is having stability problems with their servers.
I've setup a dedicated server for about 4-7 friends to play on. It's not a beefy box, but I figure it would be ok for such a small number.
Here are the specs:

Dell Precision 420 Workstation
P3-600mhz
256MB Ram
9 GB SCSI HD
ELSA Gloria II Video Card
WIndows 2000, SP3

There is NOTHING on this box other than Freelancer Server, not even the game.
I have a linksys cable modem router, updated to the latest firmware, and have the required UDP ports open. I'm playing on my LAN, and have not had an issue, my friends are coming over the internet and are having problems.

The problem I'm having is, the game runs well for about maybe an hour. Suddenly, usually in the thick of combat, the get dropped, and the server won't let them back on, saying the request took longer than expected. In order for us to play again, I need to restart the server. It also seems to progressively get worse... The server console also reports these odd errors, I don't have the log with me, but it points to me CD-Rom drive, complaining about missing files or something... I thought the server needed the CD, so I left it in, and it didn't help.

Any ideas? Should I get a REALLY BIG machine? It almost seems unnecessary, since it seems to almost run fine on the current box.
