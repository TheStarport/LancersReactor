a couple of things can cause that.....bad code for one.....make sure all the call-outs for hardpoints in the shiparc and market_ships.ini match those in the .cmp file. This can cause really screwy problems. If this is a cloned ship (a modified version of an existing ship) change the origional .cmp file to add all the new hardpoints for the new ship and use this .cmp for the origional and the new ship.(do NOT move any of the origional hardpoints....only the new ship will use the new hardpoints as called out in the shiparc.ini) The last one is a conflicting saved game......if you changed something after the last time you loaded that saved game....it could cause problems.....try loading an older saved game (before you made the change)....if this works, delete the conflicting saved games and carry on from there.
