If you get reshack.exe and change the version number of both freelancer.exe and flserver to the new version numbers they will work ok. You may want to check the:
Registery list
When using new version numbers in Ioncross you need to start Ioncross but not allow it to control the server, then change the window name to match the text at the title of the flserver window. Otherwise it hangs and won't connect.

If you want i can elaborate on this:

ResourceHacker is found here (despite its name it is not an illegal piece of software)
RESHACKER

you must open up FLServer.exe in the hacker (file open etc.), then while in reshacker click Version info, then 1, then 0, all the version info is in there for you.

Also as a side note if you plan on using IONCross, on startup click cancel to allow IFSO to take control, then go to the FLADmin setup page, (one with window caption and config location settings etc.) and change the Window Caption textbox to suit the version changes you made.
