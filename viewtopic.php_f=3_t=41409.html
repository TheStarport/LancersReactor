Welcome to TLR, you have the privelege of recieving my first post in this forum in forever.

What you need is in your Freelancer\exe folder

The file will be called "myflaccountid"

NOTE : The address for your Freelancer folder will not be the same as mine. It will mostlikely be C:\Program Files\Microsoft Games\Freelancer, or C:\Program Files\ Freelancer



It should come up with


(Mine says <NOT FOUND> because I just recently reinstalled and haven't played yet, yours will be a string of numbers.)

You get the ID number (up there) off the computer you used to play it on. Take that id number and put it in exactly when it asks for it after clicking Multiplayer on the new computer's Freelancer. Then that should do it. Keep in mind you only have shot at that. If you alredy entered in the ID at an earlier time, you are going to have to reinstall to get another chance to enter it. Make sure you are still hosting the server on the same computer as before to keep the saved character files.
