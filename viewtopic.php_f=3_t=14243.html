There is an .exe file called flserver.exe that gets installed when you install the game. Find this program (should be in the start menu with the rest of the game files) and run it. Give it a name, select options regarding pk-ing, etc. It's pretty self-explanatory. Then start it up. Voila, you are running your own Freelancer Server. You can even choose if it's an Internet or LAN server.

If you want to play on your own server, just make sure it is running, then start up Freelancer and use Mulitplayer menu to find the server, create a character, and join it. Incidentally, that's the best way to enjoy the game without worrying about SP and being able to access almost all of the jump gates/holes.
