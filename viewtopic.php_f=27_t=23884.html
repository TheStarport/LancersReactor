If you have been using mods especially mods that add dll files you need to uninstall Freelancer. Make sure you deactivate any mod that you have activated in the mod manager first. After you uninstall Freelancer make sure the Freelancer folder was deleated by the uninstall routine by going to - this is the default directory - My Computer \ C: \ Program Files \ Microsoft Games \ Freelancer. Then you can reinstall Freelancer and activate the mod you want to use. I can tell you how many times I have had to reinstall Freelancer but I wont
