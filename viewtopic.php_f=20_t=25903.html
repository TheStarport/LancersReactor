Chips

Fri Nov 07, 2003 1:43 pm by Chips

When you deactivate a mod in FLMM - it removes the mod from the files - so its clean copy again (it does leave files behind - but they are NOT connected to the game still - so they don't have any effect).

To make another copy of the game - first - deactivate FLMM - then go to microsoft games folder - rename the freelancer folder as freelancer modded. Next - go into freelancer modded, and into teh EXE folder - right click on the freelancer.exe file - and create shortcut. Rename the shortcut to Freelancer modded - and then send it to desktop instead (this avoids confusion with FL icon already on desktop.
Next - go to FLMM - and then in the task bar - select the change Freelancer directory - now select your FL modded version. This means FLMM will work on the modded Freelancer.
Now - insert your FL CD into the drive - and let it boot up. do full install again - it will create a new FL directroy - which is 100% clean. This way you will always have a clean FL.

Best bit:

No MP numbers are changed - so BOTH use the same number - so there is no swapping or loosing characters you already have.
Enjoy!

Chips

Oh - it also means that to play normal FL you just use that desktop icon, for a modded game - choose your mod of choice in the FLMM and then load via the Freelancer modded desktop icon.

You will still have probs with the autosave though - so if you ever experience crashes on loading - delete autosaves as usual.
