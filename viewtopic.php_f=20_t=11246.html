tasiin

Tue Mar 25, 2003 9:40 am by tasiin

You might want to try messing with the "render no more than x frames" setting in the nVidia control panel. This option is hidden by default, and you'll need to download and install a tool from www.guru3d.com/files to unlock it called nvHardPage, and activate "Coolbits". I have mine set to 3 with vertical sync off and I have no mouse lag issues on the menus.
