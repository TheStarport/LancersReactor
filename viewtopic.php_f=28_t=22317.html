Fri Aug 01, 2003 12:25 pm by Tygrys

Hi, to add random misions just put some vignette zones in your system:

<pre><font size=1 face=Courier>
[Zone
nickname = Zone_TY03_destroy_vignette_02
pos = -50002, 0, -50002
shape = SPHERE
size = 10000
mission_type = unlawful, lawful
sort = 99.500000
vignette_type = field
</font></pre>

The game logic will put random misions there. You should enable mission vendor at bases, see any standard base in mbases.ini on how to do that.
