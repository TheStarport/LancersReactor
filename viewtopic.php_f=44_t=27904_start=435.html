Can't help you with GMAX as I refuse to have that piece of junk on my PC, I tried it 2.5 years ago and deleted it in under 20 minutes.

The CMP Importer is buggy and has been crashing milkshape since 1.7.7, in your milkshape folder you will find a text file called "buggyplugins", open it with notepad and add msCMPImporter.dll to the list. This stops milkshape from loading the plugin at start up and only loads it when it's needed.

1.7.9 is the last version of milkshape you can use it with as it crashes 1.7.10 with or with out an entry in the buggyplugins file.

Bejaymac

Mon Oct 30, 2006 12:08 pm by Bejaymac

So how should I begin working by the tutorial ?

You already have milkshape, so use it instead. Milkshape is a far better program than GMAX(or 3DS MAX) for modelling game content as that's what is was built for(Halflife).

Milkshape loads all of the plugins at start up and then adds a shortcut to the IMPORT, EXPORT & TOOLS menus for each of them. The buggyplugins.txt stops the dodgy plugins from crashing MS3D by preventing them from being loaded at startup, the shortcuts are still added and it's these that you use to activate all of the plugins once milkshape is running.

Milkshapes help file has a tutorial for beginners that I'd recommend you try, it'll teach you how to use the program and let you build a basic aircraft.

It is possible to use the tutorial to build a ship in MS3D, but it requires some lateral thinking as milkshape (like most of the other programs) just doesn't have the features of GMAX or 3DS MAX, but when 3DS MAX costs several grand and GMAX is thier freebie teaser what do you expect.
