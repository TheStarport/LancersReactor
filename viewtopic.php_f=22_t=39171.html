To install the mod:

0. Download freelancer mod manager from this site and install it.

1. Go to the website and click on downloads. Download the file somewhere.

2. Unzip the file using Winzip, Winrar or other unzipping tool. (most people have WinZip)

3. Double click on the mod and all the tweaks. This will put it into the mod manager.

4. Open up the mod manager and activate TOW Alpha IV and TOW Alpha IV Tweak 00. Dont worry about the other tweaks for now.

5. Log on to the server! Unfortuantely the mains erver is down, but our backup server (wow we must be special) is called Alpha V4 - Tides of War. As opposed ot the normal server which is called Tides of War Alpha V4. Very confusing

6. Make sure ot use a [CIV tag if you havent joined a faction at the forum.
