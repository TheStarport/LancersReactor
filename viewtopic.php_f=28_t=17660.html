Hey Silver;

I don't know if you relize it but your mods mission board bug also effects all bases through out the game.

I think that I might have a fix for you. I saw something in the general editing forum and tried it out for the Manhatten System and it worked.

If you compare your 'Mbase.ini' Li01.01 with the snip below you will understand. I don't know what caused it to run the lines of code together, or if that is what happened.

In the original post, the guy said the prob was caused by Bini (the GUI) I had a lot of trouble using the gui version so went to the command line and haven't had any trouble since.

Now, the strange thing is, I checked my set of stock .ini files and found the code missing the 'offers_missions = true' line.

I don't know, but try it, you may like it.

[BaseFaction
faction = li_n_grp
weight = 13
offers_missions = true
mission_type = DestroyMission, 0.000000, 0.112387, 30
npc = li0101_lnavy_001_f
npc = li0101_lnavy_002_m
npc = li0101_lnavy_003_f

[BaseFaction
faction = li_lsf_grp
weight = 13
offers_missions = true
mission_type = DestroyMission, 0.000000, 0.112387, 20
