Hello Kendo.

The most important thing to be willing to do when making a mod is to be willing to learn new tricks. I have made several mods and each time I've had to learn new stuff to do so. It means putting in hours of research and watching your Freelancer install crash out thousands of times a day.

It is important to make backups and list the files you modify with freelancer so when somewhere works, you have copies of the working mod. FL can get very complex very quickly.

To make a mod and to get help you need to be able to ask specific questions. So first off you need to decide exactly what you want to do. Then find a tutorial and get started. At which point you should have a few questions you need answers too. Each question that gets answered is another trick youve learnt.

This is the usual way things work.

Now youve said you wanted to add ships, so lets start there.

You need to edit the following files.

Freelancer\DATA\SHIPS\Shiparch.ini = the ships main configuration file.
Freelancer\DATA\EQUIPMENT\goods.ini = addons the ship should automatically get, eg engines.
Freelancer\DATA\EQUIPMENT\market_ships.ini = where the ship is sold.

Now your going to have 1 question you need an answer to right away. In market_ships.ini, how do i get my ship to show up ?

This is how.

This ship is for sale
marketgood = rsstarfury_package, 1, -1, 1, 1, 0, 1, 1

This ship is not for sale.
marketgood = rsstarfury_package, 1, -1, 0, 0, 1, 1, 1

By changing the numbers you enable a ship or disable a ship.

Also each base can only sell 3 ships and bases that already sell 3 ships must have 1 of the existing ships disabled so it is not for sale at that base.

You will also need a list of bases and their coded so you know where your adding your ship so you can test it.

I'll leave you to figure out the shiparch.ini and the goods.ini. Those two require some research but i have answered the questions new mod makers usually ask early on in this emial.

The process of adding a ship is simply.

First make the ship.
Create the shiparch.ini entry
Make the goods.ini entry.
Add it to base to be sold.

Please note that all pilotable ships in the shiparch.ini contain extra lines that enable the player to fly them. AI ships dont require those lines and ignore them if they find them. So studdy the differences between nopilotable ships and pilotable ones and you will learn a lot.
