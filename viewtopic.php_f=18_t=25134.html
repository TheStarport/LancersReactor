We humbly ask that this stuff not be copied from TNG

Put simply I ask this from all the community.

If you see a feature in TNG that you like and wish to use it, PLEASE email me and let me know. I don't have a problem sharing however I do ask that if you borrow stuff from TNG i humby just ask for some credit to be put in the readme.

Most of the stuff I don't mind sharing however, there are a few features I would like to preserve for only TNG to make it unique. If I were to give everything away some people would possibly just make a clone of TNG.

So with that, THIS is the stuff we would like NOT to be copied from TNG.

1: TNG specific systems, Nexus, Sol, Vorlon, Mimbari, Centuari, Shadow, The Void (Ancients) etc...

2: TNG specific ships designed by Spawn, Hadryon, and CaptainOblivious, these include ships from Freespace 2, Homeworld class ships, Wing Commander, & Bab5, and all unique class ships. We will be happy to supply a detailed list once we are finished.

3: Duriel's super system

4: Spawn's artwork & Duriels movie clips

5: Bobthedog's intro scripts

6: Certain equipment (house guns, anti-matter torp, & race specific weapons going in 2.0)

7: Our costumes for the various characters.

Other then that we don't really have a problem, we just feel that we need to keep certain things unique to TNG. Again all we ask is that if you do use something from TNG, then please just give us credit.

Fri Oct 10, 2003 11:57 pm by reynen

If you make it and did not know about it, then it should be completely different from what we have and not a copy of it, right?

We've run into this before. Take for example the nexus system. You may not rip the code from it, but you know of the idea, you know what's in it and you know how it works. You then proceed to make a system based on what we created.

Whether you get the the actual code or the idea from TNG then we ask for credit. But in reality, TNG is big enough that almost everyone knows about it, so people are going to know what is going in it and what it currently has.

Is it that much effort to add, "such and such used from TNG with permission" in the readme. If you aren't trying to claim it as your own, there should be no problem right?

Our whole problem as a modding community as a whole is that people want to copy stuff, whether it be ideas or code, and then claim it as their own, when they based their work of something else in the first place.

----------------------------------------------------------------------------------------

Think of mods like patents. Something can only be created once, the rest are copies or re-creations. Someone can create a new version of the lightbulb, but the original patent is still there. If there is not enough changes to the new product vs the original, then it is just another copy of the original.
