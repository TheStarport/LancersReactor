Yes, actually.

The mod is very close to release (we've been saying it a while but this time its actually true). There were alot of errors that needed fixing, plus Harrier and I were each finishing up our own ships to add to this version, IFSO files needed to be done (a couple more need to be done, those will take about 2 minutes ) and a couple things sorted. Update time

Mod will be off to TLR very shortly. A couple more bugs (including one crash) have been found and taken care of.
