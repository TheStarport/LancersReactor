bakedpotato

Sat Apr 09, 2005 1:37 am by bakedpotato

When you use a MOD that adds to Freelancer such as new systems, ships, or equipment you have to make the Data Files that the IONCROSS Server Operator needs in order for the added items to show up in the Server Operator interface.

So if you don't have the needed Data Files the items that were added are not going to show up in the IONCROSS Server Operator.
