Ok, here's the low-down on how ID #'s work.

First, make sure that you have the opriginal Game ID that was given to you when you started Multiplayer for the first time.

Second, talk to your Server Admin and ask him/her to give you your Name ID from the server itself. This also may be found in the registry.

To get info from your registry:
Do the following - START -> RUN -> REGEDIT

Once you have the registry open do a search for 'MPAccountName' (without the quotes)

From there you'll see there are 3 sets of #'s.
1) MPAccountID (I think)
2) MPAccountName (this is the one stored on the server)
3) MPAccountNameSig (never needed this one, nto sure what it's really for)

At anyrate, I had to do this yesterday because I play on 2 computers also. Once you enter in the MPAccountName close the regisrty and start Freelancer. When you get into the Multiplayer area it'll show you your Game ID again. Delete that number and replace it with the one you got from your original game.

Also, if you can't remember your Game ID, you can run a program in the following directory to get it.

C:\Program Files\Microsoft Games\Freelancer\EXE
Run the program called - myflaccountid.hta
That program will show you your Game ID as well as give you the option to print it out.

Hope this helps, if you need further assistance feel free to also email me.
