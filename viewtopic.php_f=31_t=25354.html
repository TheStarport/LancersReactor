TNG 1.7 Official release

You can get it by going to our website, www.planetirg.net, or click on the short cut below.

Download Link

Also please read the readme.txt file - TNG 1.7 MANUAL

SERVER OPERATORS: You will need to do a player-wipe. There have been several changes that make existing players non-compatible.

If there is a " infront of the directory when installing TNG 1.7, REMOVE IT. Ex. ( "c: )

1.7 changes a number of things concerning gameplay. We've also fixed a number of bugs. By far we have made the mod much harder due to complaints about it being to easy. The battlestation missions have especially been made deadly so that groups are now more or less required to do them in order to survive. It IS possible to solo one, if you have about 30-45 min to spare for 1 mission and are very careful. Those that think Battleships will save them will find themselves in a middle of a meat grinder.

Changelog:

- Fixed the problem with Dark Shard offering missions against it's own kind
- Redid all fighters manuverability to put it on par with the original ships
- Got rid of almost all console errors (see known issues below)
- All shadow ships no longer have counter measures or mines
- Fighters may no longer use battleship reactors nor shields
- All shadow ships had thier power reduced significantly
- Added Inner System Warp Drive for capital ships only
- Ammo can be bought at all stations without faction
- Added countermeasures and mine to the profiteer
- Removed countermeasure and mine from cargo1
- All ship prices have been drastically increased
- Removed hull upgrades due to server crashes
- Increased the manuverability of the gunboats
- Missions payout has been increased by a little
- Introduced the stackable upgrade concept
- station turret damage has been increased
- Removed all engine upgrades from game
- Battlestation missions are MUCH deadlier
- Station turret range has been increased
- Thruster prices and stats have changed
- The Starviper is no longer invulnerable
- Fixed trader paradise docking problem
- Added Stackable Shield Enchancers
- Added Stackable Power Enchancers
- Gunboats can use the new addons
- Added a few lights to the profiteer
- Added Stackable Assult Barriers
- Brakiri Battleship now fires 360
- Shadow Cruiser now fires 360
- Fixed Thunderbolt hardpoints
- Shadow Crab now fires 360
- Added Hornet heavy fighter
- Added Cougar light fighter
- Fixed X-wing hardpoints
- New EULA & Disclaimer
- Changes to security
- Modified Starviper
- Added intro movie
