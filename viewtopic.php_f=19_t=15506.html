answer to number 3, characters are saved on the server.

when you install the game, you get given a big long ass key code, write it down.

when you log into multiplayer, that key is sent to the global server, who then sends it the different game servers which compare the key and decide which account is yours, then let you access that account.

if you reformat ande reinstall, you lose that key (get a new one) and when you log onto servers, your characters are not there anymore.

you have to edit your registry and manualy replace the new key with your old one in order to access the characters you were playing on the various different servers.


so in theroy, if ya install it, write down the code, and install it again, write down second code, if ya get banned from a specific server with your primary account, ya could edit the registry, reboot, and log back onto the server you were banned from and start a new account, if ya have characters from your primary account that aren't banned on other servers that you still want to play, just edit registry and put back in the first key code, and walla, your characters are back, but now ya banned again on the server you were initially banned on.

it's kinda like logging into the game, but it's done automatically thru the registry, just keep track of what characters are on what servers with which keycodes.
