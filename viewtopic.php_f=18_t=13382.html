Ugnaught

Thu Apr 10, 2003 5:00 pm by Ugnaught

Did you actually fly enough missions and trade runs to make $999 million? How long did that take? What would you do with that much money? It sounds like the limit is hard coded. $999,999,999 is 3B9AC9FF so the memory location is at least four bytes and the maximum number that can fit in four bytes is FFFFFFFF which equals $4,294,967,295 using unsigned integers and half that with signed integers. There is probably something in the main program that sets your money back to $999,999,999 if it ever goes above that. Sure, a larger number can be stored as a floating point, but you'd have to decompile the main program to change an integer to a floating point.
