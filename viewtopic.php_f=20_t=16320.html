Fri May 09, 2003 3:18 pm by Seanus

Whilst setting my my new pc with FL. I have experienced dozens of problems. They may all be related to some of the problems i've seen here.
1. Firewall - in my case Zone alarm. Turn it off, or reduce functions to zero. Try it again. If it works, turn them up till it doesn't. You now know where your problem is.
2. Aperture size. Set it to half you system ram. My default was 1/4. This made a difference. (bios setting)
3. Another bios setting. Fast writes. If they're enabled, disable them. If they're disabled enable them.
4. DX play. Tes it using DXDIAG. Use the typing thingy. Make sure you see your words echo.
5. If you're reinstalling on a new machine MAKE SURE YOU WRITE DOWN YOUR ACCOUNT ID.
6. Detonator drivers. Disable extended instruction set for your CPU. (I think only for AMD chips. works with Intel).
7. Turn down all of your graphical settings to a minimum.
8. Don't run any mods.

If these don't work. You're probably playing manic miner and don't realise it.
