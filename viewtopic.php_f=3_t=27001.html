Howdy nootami
I take it you are talking about the Freelancer Massive Addon v2 Mod made by Tygrys. You should probably ask your question in the The Lancers Reactor > Forums > Freelancer Editing: Freelancer MOD Announcment Forum. Topic name is Freelancer Massive Addon v2 release info and wishlist.

The most common problem with Mods crashing is any saved games from previous games you have played with or without Mods in your saved game folder especially auto saves. Try moving your saved games to a different folder before trying out any Mods. The default location for your saved games are located in
My Documents\My Games\Freelancer\Accts both the Multiplayer and Singleplayer folders should be moved. I say moved incase you want to go back and play any saved games you had. If that don't work the only thing that can fix the problem is a complete uninstall and reinstall of Freelancer. Sometimes you even have to do a complete uninstall and reinstall of the Freelancer Mod Manager too.

Question: "Just out of curriosity, why does FL crash with certain mods? What is it about them that screws up the code sooo much that the game can't compensate? I know it's probably some really complex technical deally, but if some modder could explain it to me in plain terms, it would satisfy my curriosity."

What happens is things get added to the game information files like ships, weapons, equipment, systems and on and on. If the game engine cant find those items or doesn't know how to handle the items it chokes and crashes. There can be several reasons for this but the only thing I have found that can fix this is:
1st - try the Tools - restore backups in the Freelancer Mod Manager
2nd - if that don't help remove the saved games folder either by deleting or moving the folder to a different folder of your choice. If that don't work.
3rd - uninstall Freelancer completely. Saved games, screenshots everything. Then do a reinstall of Freelancer.
4th - If it still crashes try uninstalling the Freelancer Mod Manager and reinstalling it. The Freelancer Mod Manager does have it's own uninstaller built in.
