If that does not work you need to uninstall Freelancer then check your program files folder and make sure that in the C:\Program Files\Microsoft Games folder there is no Freelancer. If there is a Freelancer folder delete it from inside the Microsoft Games folder.

I have had to do this several times because it seems that some mod files will not uninstall, and when you install Freelancer again you still have those files in your Freelancer folder which will mess your game up. Been there done that.
