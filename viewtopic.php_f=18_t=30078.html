Moddeling questions!!!(1)(by...me)

I am kinda new to modelling. And I had quite a bit of truble getting to a point where I could accually MAKE something! So. I have decided to make this thread for anyone who has questions relating to modelling. (Note: "modelling" not: "modding" ) Any who. I will help as much as I can. So... shoot!

Drizzt4.0's tutorial (the only thing that saved me from modeling disaster.)

Tools:

Milkshape (Used to export all models made for freelancer to CMP. The format used by FL for the ships in the game)

CMP Exporter (a plugin for milkshape to allow it to export to .cmp

Gmax (A good free modeling program that you can use to make ships in if you dont want to use milkshape. NOTE: You still have to use milkshape to export to .CMP)

FLini editor (used to edit the .INI files in freelancer. There are other programs out there that do this. But this is my favourite)


(Note to anyone. If I missed anything, let me know)
