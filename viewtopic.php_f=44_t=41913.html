Skin my Falcon

Here is a challenge for all. I have this ship that I made for an animation or movie. It's for sale but I wanted to put it in the Freelancer game engine.
Here's a Pic

The only problem is that to poly count is way too high!!! I remodeled it and now it is extremely low poly count and will have no problems flying around the FL universe.

If you have kept up around here you know how myself and many other modelers like to keep everythin in a "series" type set to keep in the spirit of the game. Well this ship hasn't much room for expantion because of the sleek design. Because of this and the fact that I like the ship so much, I would like to make several of the same ship with different skins. They will all have different strengths and weapons capabilities. Instead of me trying to come up with all the different versions, I thought it would be cool to let the TLR folks all take a crack at it. Whatever the result, all the ships will be made availabe as a set to the TLR community.

I have already UV mapped the ship and have all the necessary files Here.
The Zip file includes the 3DSMax, Ms3d, Obj, BMP, and Psd files. You will need to sign up for a free turbosquid.com account to be able to download it. If you are not familiar with all the file formats, don't fret. All you need to do is take the bitmap or photoshop file, and paint it up pretty. Use the file you save as a material in Milkshape and it will lay nicely on the model. This is designed to be a fun as well as productive project.

If you want to start from scratch with the skin but don't know how, go Here for Argh's tutorial on skinning.
If you are cool with the way things lay out on the model but want to make a few adjustments then go Here for Harrier's tutorial on texturing in Milkshape.

Post whatever you do here and by all means, Have Fun!
