Just open up your market_ships.ini, choose your location base and look for an entry which has its last five numbers as -1, 0, 0, 1, 1, 1 and then change that ships entry so that the numbers look like this ....., -1, 1, 1, 0, 1, 1
NOTE: there is a number before the -1 which represents what level you need to be to purchase..you can change that to a 1 if you liek as well
now look to the bottom for oe_package which should have the numbers you just entered behind it (looks a bi tlike this i think)
marketgood = oe_package 16, -1, 0, 0, 1, 1, 1
Change is so that it is
oe_package 16, -1, 1, 1, 0, 1, 1
Now go to that station and buy the anubis. Your own self styled mod!

Tue Sep 02, 2003 12:17 pm by Chips

Its been coded or something? Hmm - make sure when you deactivate mods (to be safe) you should do 'restore backups' as well. Most of the time they deactivate fine, but every now and then a few files seem to get 'left' behind. This then means that it causes severe probs. Also - bit obvious, make sure you deactivate mods before you delete. What has happened to you is that residual files have been left over by the looks of things. It should have a long list of ships for each base, most are 'deactivated' and some are not. They did that cos it was easier to just copy and paste the lot at each base then choose which one for where, rather than for any other reason.
You need your original market_ships file. thing is, i suggest you DELETE (not uninstall, don't forget to empty trash can!) FL then reinstall it again. This way you get nice clean files to start working with. I end up doing this process about once every other day to make sure when debugging stuff that there isn't anything 'lurking' about causing the probs!
