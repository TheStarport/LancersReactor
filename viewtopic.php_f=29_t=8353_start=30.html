I think everything that has to do with the loading of ini files are server side in MP. Only thing the clients can do is use memory hacks.


Unfortunately this is a very false assumption.

The MP Server and SP Game both use the same ini files.
And, the client always uses it's OWN ini files, NOT the servers!

So, if you alter your ini file for Manhattan to have a "Sabre" there, for lost cost, it will be there in ANY MP Server you log into...you can buy it at LEVEL ONE!

However, characters which do this CORRUPT the server and CRASH them!
And, the server STAYS CORRUPTED until that character is removed from what I have read.

So, remember when you are playing around with these files to change them back to default BEFORE joining any MP games!
