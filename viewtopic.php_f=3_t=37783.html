If you are on one of the lucky few server's that have FLAC, you can make extensive kills statistics for current online players. If the FL server admin has initiated the XML statistics from FLAC and coded up an XSL file plus use a transform program, you can see details in a standard HTML file not seen with IFSO and FLStats output.

See Boston's FLAC statistics here. for current online players. ProgramExpert and I modified the XSL file Eagle (FLAC's author) provides in his forums. Saxon is used to transform the XML statistics which FLAC produces (in my case every 5 minutes) to HTML code which is then uploaded to my website.
