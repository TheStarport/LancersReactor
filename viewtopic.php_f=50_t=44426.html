What are you talking about.
All the Jump Gates/Holes are already unlocked for MP automatically so there is absoloutly no point in editing the initianworld.ini file.

Read below for a tutoral on how to change the MP Server stats:
----------------------------------------------------------------------------
Files you'll need to edit:
- EXE\mpnewcharacter.fl
====
Open up the mpnewcharacter.fl file sitting in the EXE folder.
You will not need to decode this so don't worry.

Once opened you should see something like this:

[Player
name = %%NAME%%
;initial_rep = %%FACTION%%
rank = 0 <-------This is a new player's starting rank. You can change it between 0 - 38

money = %%MONEY%% <-----Delete the %%MONEY%% and enter how much money your player would like to start off with.
;voice = %%VOICE%% <-----Do not Change This
;All must be Trent!
voice = trent_voice <-----Do not Change This

costume = %%BASE_COSTUME%% <----Change this only with operators and/or mods
com_costume = %%COMM_COSTUME%% <----Change this only with operators and/or mods

system = %%HOME_SYSTEM%% <---Li01 is your new player's default home system which is New York. You're allowed to change this if you know what the system nicknames are.
base = %%HOME_BASE%% <---Li01_01_BASE is your new player's default home base which is Planet Manhatten. You're allowed to change this if you know what the base nicknames are.
