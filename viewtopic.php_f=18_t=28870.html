Opening jump gates/holes in multiplayer

Sorry if someone else posted this already, I've done my search and I was quite surprised I couldn't find anyone else asking this.

How do you open jump gates/holes in multiplayer?
I already asked this question in another thread, and Earendil figured out how to do it for new players:
Here is the thread I'm referring to. (Scroll down a bit)
After experimenting with his solution, and editing some savegames, I came to this: (I've tested this on New York - Magellan Jump Gate, which is closed in multiplayer ("Acces Denied" but open after you finish mission 4 of the single player campaign)
When I start a new player, the jump gate is open.
When I modify the savegame of an already existing player, and remove the corresponding locked gates with those I opened in the initialworld.ini I can succesfully jump.
When after this, I close Freelancer and FLServer, and open it all again, and load that same player, it still works.
I tested some other holes (like Magellan -> Leeds, Leeds -> Cambridge) and they worked as well.

I haven't tested this yet for players who don't have a modified initialworld.ini, but do have a modified savegame on another computer/server which has the modified initialworld.ini. I'll test this soon.

From the looks of it, it all seems to work smoothly, and that is the reason why I post this here, I'm planning to make a small program/mod which allows the server admin to open the gates/holes he/she wants.
So here are the questions/problems I have:

- Is it possible to close more gates then that there are listed in the initialworld.ini? I mean, e.g. someone wants to close the New York - Colorado jump hole, this is a jump hole which isn't listed anywhere in initialworld.ini, so is it possible to add 2 new lines with "locked_gate" and then the gate numbers? And where do I find all gate numbers then?

- This means that savegames needs to be modified, and it isn't nice if you have to manually edit all the savegames on the server. So the program should have an option to automatically edit all the savegames. But since savegames are encrypted (so it looks) and I don't know how to decode them, could someone help me with that?
