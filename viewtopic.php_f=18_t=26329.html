I've got a MOD that I'm working on that has the Dagger, and Bloodhound up for sale on Pittsburg.
And I'm gonna need some test pilot's for it...
Let me know.

Of course...
If you open up the Market_Ships.ini file with the FLIniEditorv144 program from this site.
Simply go to the Li01_02_base, and change the:
marketgood = bwf_package, 4, -1, 0, 0, 1, 1, 1
to
marketgood = bwf_package, 4, -1, 1, 1, 0, 1, 1
Of course, doing that at any base will add it, but this adds it at Pittsburg.

To add it to Manhattan, go to the Li01_01_base:
make the above change, but change one of the:
marketgood = ***_package, *, *, 1, 1, 0, *, *
to
marketgood = ***_package, *, *, 0, 0, 1, *, *
leave the data (*) as it is
(never have more than 3 ships for sale at a base, game crashes if you do)

Edit:
To make it better requires a ton of MODing, because weapons, engines, power units, stats, etc, are all over the place. I know, it's in my MOD! But at least this'll allow you to get the standard one more easily.

Edit 2:
You COULD use the IFCE5setup.exe program from this site, and mod a saved game to give you a super-dagger with whatever you want on it!
