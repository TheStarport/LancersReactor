Freelancer Physical Models

Hi all + thank you admin guys!..

I posted here way back, about making physical models from Freelancer, out of paper (card models).. The first test model has finally built up nicely. Unfortunately the project has bogged slightly, so I need a bit of assistance and info from players..

Its a long sob story but I don't ( am unable to ) play Freelancer.. So I only know the game through the .cmp files.. Thanks to that, the model project has hit an unexpected snag, the shorter story being ~ I can develop most of the 3d content I need. Unfortunately, the weapon placement is still a mystery, so screen shots of the vessels in game, are the only practical and quickest way for me to scale + place weapons to each craft... So..

* Would anyone be willing / able to grab a few screen shots of the ships I'm Developing? ( with a variety of weapons loads would be best) I've checked the screen shots here, they were a big help, but there's just not enough of them with the detail I need.

* Are any of you interested in building paper models of Freelancer ships? ~ I only ask, as there didn't seem to be much interest before. ~ It takes a lot work to Develop a card model, and a lot of extra work to dev one that can be built by other ppl. ( with detailed assembly guides etc..)


Just btb.. I've probably said this before but ~ If it wasn't for all the ppl like you, I'd be out of a Hobby. So a big THANK YOU to all the guys responsible for these boards, and the talented MODs for all their brilliant design work.

This test model was printed on 170 gsm ? 100 lbs card stock, using a cannon ip4300 and is approx 30 cm long 28 cm span.. The cockpit had to be modified slighlty to accomodat photo gloss paper for the 'glass'
