FIX: for game stability, some sound errors, and loading spee

The Freelancer SDK was developed for mod makers with the sole purpose of eliminating errors in the original source so mods can start with a clean slate. Due to the error elimination it also fixes many problems that make Freelancer unstable, it also fixes minor errors like sound jitters and loading speed.

If you are having problems with the game or would just like to tweak it for maximum performance then try installing this SDK over your Freelancer DATA folder, you may see a big difference and you might not notice anything but its worth a shot. Enjoy yourself freelancer!
