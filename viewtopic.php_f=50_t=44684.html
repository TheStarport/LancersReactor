hi there, welcome to TLR =)

what unused hardpoints are you talking about?well, regardless what you want to do is figure out which ship = which INI (text) entry in shiparch.ini and then find the appropriate section

ships are named by their house/region in short hand but some instances don't follow this rule
so, in shiparch.ini when you see a nickname it will have something like this in front:
co <-junker ships
pi <-corsair fighters
ku <-kusari
rh<-rheinland
bw <-borderworld (hessians/lane hackers/other factions with Stilleto/Sabre ships

and then for the ship type:
fighter <-fighter
elite <-heavy fighter
elite2 <-very heavy fighter
freighter <-freighter (duh )

of course, if you opened the file with hardcmp etc you obviously already know where it is and the filename, so search for that

then find this area:
hp_type = hp_fighter_gun_special_1, HpWeapon01, etc

then to classify(add) new hardpoints, just find the class you want and add HpWeapon#, HpMine# etc
hp_mine_dropper
hp_countermeasure_dropper
hp_fighter_shield_special_1
hp_thruster

etc

the classes are all specific restrictions - so if you make a gun point a lvl 5, and don't add it to levels 1-4 you will only be able to mount lvl 5 guns

i see the confusion arose from our vocabulary- the yellow and black stripey block thing son the ship, are just part of the model - the actual "hardpoint" that things snap together with is invisible in the game and can only be seen/edited with HardCMP and UTFedit (UTFedit is harder to use for moving/orienting hardpoints)

so you'll want HardCMP

once you get it you can find the names of those unused hardpoints and add them using the shiparch hardpoint-class info i gave you in the last post
