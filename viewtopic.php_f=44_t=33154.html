MAT file question

I'm not sure if anyone here will have an answer, but since this question has to do with mat files and texturing, I figured this was the best forum to post in.

I was trying to change the texturing of the mines (the big spikey ones in zone 21) and I was able to find the model - mine_spike1.3db - and the texture (mat) file - ast_mines.mat. I simply wanted to change out the textures, much like what can be done with ships, so I entered the texture library and imported the new texture files I wanted (tga format of course).

However, when in game, the mines appeared untextured for some reason. It turns out the original textures in the mat file aren't tga format as they claim to be, but rather have (when opened in word pad or a hex editor) DDS / as the first 5 charachters of the file.

So my question is, does anyone have any clue what these are or how to modify them?
