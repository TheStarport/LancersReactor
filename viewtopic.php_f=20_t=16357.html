Go to Start, Run and type ipconfig. U should get a IP-adress, Subnetmask and a gateway. (Usually get those three) If u got a DSL modem, u normally also got a Router in it. Dock into it and open the ports u need for YOUR IP-adress.

Go to Start again, Run, type in cmd. HTat ould give u the C-Promt. then type "ping www.flserver.com". that will make u able to see if u can connect to that IP-adress. Then, if u want to lay on Elite, open port #992 and ping thier server to. "ping 131.107.83.10". Without the "" ofourse.

Does anyone nknow the IP-adress of the "Global server"? If so, post it here and u guys can ping that too. If u still can ping all these 2-(3) IP-adresses, I would say that its your Modem that doesnt allow FL to be played over Internet.

Try after this to start up FL, select Multiplayer, click the "Yes" button when they ask u to confirm that u are ok with that u play on the Internet. Then click the "Internet" button. (Some people do click the "LAN" button).
If u still, cant connect to te Global Servers, go to this site and download The All Seing Eye:

EU: http://www.udpsoft.com/eye/download/new ... taller.exe

Install it and add the Elite and other servers to the program, connect ot them and play! (This program have always been able to help most of my friends having problems with other games)
