Release Testing

The Patriot source code release is testing. If you would like to test the source code and give feedback on its usability, compatability and any errors you may encounter, you may download it here.

Changes so far
- All memory leaks fixed
- Both Debug and Release mode now work perfectly
- small error in cleanup resolved
- Icons added to executables
- source files updated to auto-include memory manager only on DEBUG builds, and not on RELEASE builds.
- main.ini scene changed. ship now visible, but other items are obscured. Not optimized for any kind of viewing
- jumpgate textures compressed
- Readme.txt now included

This is a "fix it" oriented release. There is no new content.
