I don't know about everyone else but I just don't have any free time anymore.

Yes I have seen this problem before. Sounds like you have been using mod's. yes?
You need to delete your Freelancer folder... not uninstall but delete C:\Program Files\Microsoft Games\Freelancer
The reason we delete instead of uninstalling being, is we want to make sure there are no left over files from just doing the uninstall routine. This happens and then people wonder why Freelancer still don't work.

Then install Freelancer from the CD again.
The likely culprit is the flserver.exe file. It sounds like you have a edited flserver.exe file that changes the version number.

I would also install the Freelancer SDK v1.3 files after you install Freelancer
Download the SDK file here... http://www.lancersreactor.com/t/Download/Download.asp?ID=721

If this don't work, post here again with some information about what's happening... I will check back to see if that fixes the problem.
