Ok, basically you go into the EXE folder and find this file. 'mpnewcharacter.fl'
You open it up with notepad or whatever and it'll come up with this:

[Player
name = %%NAME%%
;initial_rep = %%FACTION%%
rank = 0

money = %%MONEY%%
;voice = %%VOICE%%
;All must be Trent!
voice = trent_voice


This is a shortened version, the rest below is not important for this. Now just delete %%MONEY%% and change it to whatever value you want and that's your starting cash. I haven't tried the other way suggested but I'm sure that works too, you'll need to decode it though.
