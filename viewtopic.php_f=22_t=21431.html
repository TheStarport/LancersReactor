Getting your character ID.

You can create your own temporary server on your computer. Then load up freelancer and log on to your own server. Go back to the server program and look at your character ID. That will get your current ID, but not the one that you lost when Win 98 died. The only problem is that your ID might be a little different on each server. I think it might mix your ID with the server ID to make a unique ID on each server.
