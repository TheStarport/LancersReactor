Bobway''s Victor Server Guide

I have recently been posting news about the Bobway Victor Server - running Victor4.4... i have received a few questions about the server and the mod.. so here are the answers...

-=the server=-
The original bobway Victor server was taken down due to a new more dedicated bobway modification, however due to the fact that server was very popular
alot of players asked bobway to host the victor4.4 mod on another bobway server...

-=the victor mod=- (brief guide - see below for full guide)
the general differences between the original freelancer concept and the modified game are very small, for example the transporters are pilotable, there are no starwars ships for example.. many servers offer a heavy modified game with extra systems, ships, factions, guns etc... the victor mod could be described as a minimal mod

-=how to play on the server=-
first you'll need the excellent Freelancer Mod Manager (FLMM) and the victor4.4 mod (both are downloadable from bobway.com (choose downloads)), first install FLMM and then double click (open with FLMM) VictorMod44.zip.flmod, then activate the mod in FLMM, you can then close FLMM and launch Freelancer in the normal way (you do not need to activate the mod each time you play)... you'll know if you have got it right it will say "Loading Bobway's Universe"... then find the server called Bobways Victors 4.4 (normally page 1 or 2 on the server list). From time to time the server login may fail, please check the forums on bobway.com for more info or try to login a little later)

-=the victor mod=- (full guide)
The goal of this mod is to remove the linear design of
the original game, which was only good for the SP aspect,
and increase the overall difficulty of the game. Ships
are now balanced by class with minor differences between
ships from different regions. The "money scale" has
been reduced to compensate for the low maximum item price
of ~10,000,000 credits, essentially everything from
commodities to weapons have had their cost cut in half
to make a $10,000,000 Battleship seem much more
expensive.

Players can now buy any ship that was included in the
original game except the Cargo Train and Rheinland
Battleship due to some errors. Capital ships use their
own weapon classes which replaced class 2-10 turrets.
Players can customize the loadouts of their capital
ships with some restrictions. NO CAPITAL SHIPS USE
SHIELDS.

There are no more class 10 guns, all of the old class 10
weapons have been converted to class 9 and no ship can
mount class 10 weapons. This was done to allow some
weapons to only be useable on specific ships, such as
the forward gun on the Liberty Cruiser.

All fighters, heavy fighters, very heavy fighters and
freighters can mount and use all weapon classes (1-9).
All ships can use class 1-9 shields, a select few can use
class 10 shields. Torpedos and cruise disruptors use
standard weapon mounts and all weapons can use the old
torpedo mounts. Mines, CM launchers and thrusters all use
the same mounts, it is now possible to mount your choice
of equipment, such as 3 thrusters, 2 thrusters and 1 CM,
etc. NOTE: Even though a ship can mount all of the big
powerful weapons and shields does not mean that ship has
enough power for them. Infocards now display the ships
power output so players can decide for themselves.
