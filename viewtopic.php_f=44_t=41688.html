it sounds like you have a texture format problem - make sure you export .tga's in uncompressed format and .dds in DXT formats only,usually DXT1.also for the complete noobie the xml decompiler/compiler might work better- it simply extracts all the data in a .cmp or .mat,etc UTF files,then makes an XML file to reassemble them - then you can replace the old textures with the new quite easily and run the reverse XML>UTF program to reassemble it - its also great for items with lots of materials like stations,or utf files or anything really
