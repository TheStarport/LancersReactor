Black Eagle

Mon Jun 14, 2004 8:36 am by Black Eagle

Hi Rogue!
In my mod I have working Nomad capship encounters, so I think I know how to do them.
At first I think that you should make new encounter files and not use the existing area_gunboats and area_battleships (I'm not really sure if there are such files in the original game?). Your new encounter files should look like this:

[EncounterFormation
ship_by_class = 1, 1, sc_battleships
pilot_job = assault_leader_job
make_class = wanderer
ship_by_class = 5, 5, sc_gunboats
pilot_job = assault_job
make_class = wanderer
formation_by_class = battleships
behavior = wander
arrival = cruise
allow_simultaneous_creation = no
zone_creation_distance = 0
times_to_create = infinite

[Creation
permutation = 0, 3

I named this file "capitalships_nomads.ini, because there are 5 gunboats and a battleship in it. You also have to add these lines to shipclasses.ini for that, because the original game does not know sc_battleships and sc_gunboats:

[ShipClass
nickname = sc_gunboats
member = class_gunboat

[ShipClass
nickname = sc_battleships
member = class_battleship


The secon thing is, that there is no battleship_wall formation in the original game. As you have this formation in your faction_prop.ini, you also need it in formations.ini. It should look like this:

[Formation
nickname = battleship_wall
pos = 0, 0, 0
pos = 400, 0, 0
pos = -400, 0, 0
pos = 0, 200, 0
pos = 400, 200, 0
pos = -400, 200, 0
pos = 400, 0, 0
pos = 400, 200, 0
pos = 0, -200, 0
pos = 400, -200, 0
pos = -400, -200, 0
pos = 400, -200, 0
pl_pos = 0, 0, 500


The last thing is, that in your EW05.ini you have:

faction_weight = fc_or_grp, 1
faction_weight = fc_ou_grp, 1
faction_weight = fc_n_grp, 20
encounter = area_scout, 19, 0.280000
faction = fc_or_grp, 0.800000 <--
faction = fc_ou_grp, 0.200000 <--
encounter = area_gunboats, 19, 0.200000
faction = fc_n_grp, 1.000000 <--
encounter = area_battleships, 19, 1.000000
faction = fc_n_grp, 1 <--

If you take the number of each faction, together it should be 1.
So you could make:

faction = fc_or_grp, 0.300000
faction = fc_ou_grp, 0.200000
faction = fc_n_grp, 0.500000

Of course, if you have one faction several times it will be more:

faction = fc_or_grp, 0.300000
faction = fc_ou_grp, 0.200000
faction = fc_n_grp, 0.500000
faction = fc_n_grp, 0.500000

I hope this helps! If not, post the entries in all of these files and I will look through it again.

For more carnage, copy-paste this block into your capitalships_nomads.ini:

<pre><font size=1 face=Courier>
[EncounterFormation
ship_by_class = 1, 1, sc_battleships
pilot_job = assault_leader_job
make_class = wanderer
ship_by_class = 1, 7, sc_battleships
pilot_job = assault_job
make_class = wanderer
formation_by_class = battleships
behavior = wander
arrival = cruise
allow_simultaneous_creation = no
zone_creation_distance = 0
times_to_create = infinite
</font></pre>

<pre><font size=1 face=Courier>
[EncounterFormation
ship_by_class = 1, 1, sc_gunboats
pilot_job = assault_leader_job
make_class = wanderer
ship_by_class = 1, 7, sc_gunboats
pilot_job = assault_job
make_class = wanderer
formation_by_class = gunboats
behavior = wander
arrival = cruise
allow_simultaneous_creation = no
zone_creation_distance = 0
times_to_create = infinite
</font></pre>

<pre><font size=1 face=Courier>
[Creation
permutation = 0, 3
permutation = 1, 3
</font></pre>

That will spawn a fleet of 1 lead battleship and between 1 to 7 escorts, and another fleet of 1 lead gunboat escorted by 1 to 7 wingsnakes. Happy hunting.
