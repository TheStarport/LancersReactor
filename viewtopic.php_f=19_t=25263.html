Yeah, the MP ID is not on the disk, it is created each time you install freelancer. It is based on your hardware and I think some setting in your OS, but in any case if you reinstall Freelancer, when it installs it will prompt for your MP ID. At that point you can leave it at the default generated one or manually type one in. So if you wrote down your old ID, then reinstall Freelancer, you can put in the old MP ID when it prompts for it. I'm pretty sure that there is also a way to manually change it after FL is installed by editing some .ini file, but I've never done it myself so I don't know how.

There is a very easy way to change your MP ID back to what it previously was , provided you wrote it down of course

1. get your new -undesired- ID

2. start Regedit

3. search for your undesired ID

4. edit the undesired ID (change 2 numbers in it)

5. close regedit

6. start FL mulitplayer , it will pop up your chnaged ID telling you it is invalid and give you a chance to change it

7. change it to the ID you actually wanted , and off you go
