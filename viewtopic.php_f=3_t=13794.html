Clever trick.

This only works if you can program macros into whatever controller you are using. I use a Thrustmaster Tactical Board.
My current ship has Sweeper and Catapult missiles. I have one button on the Tactical Board set to fire a Sweeper first and then .5 sec later a Catapult, with one button push. Kind of a one-two punch. The Sweeper softens the shields and the Catapult hits the hull. It burns up twice the number of missiles, but I am hoping it will make for more effective use of them in the long run. I haven’t tried it in combat yet, I’ll see if it works.
