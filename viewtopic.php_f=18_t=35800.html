How exactly can you change character body parts? I'm asking because I want new heads, or at least re-texture old ones.

1) Edit the BodyParts.ini and/or Costumes.ini files.
2) Use the UTF to XML editor in the download section.
-It'll extract all of the images, which you can then edit, then you can recompile with the same program.
