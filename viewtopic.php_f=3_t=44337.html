I don't know how I could of made it any more clear that you have to use version 1.4 beta 4 of the Freelancer Mod Manager to activate the WTS-World Mod version 3.42

In the WTS-World post it says...

Please use version 1.4 beta 4 of the Freelancer Mod Manager to install/activate/deactivate WTS-World version 3.42

You can download version 1.4 beta 4 here.

It's even says that twice on the WTS-World version 3.42 download page...

For installing and activating/deactivating this MOD use the Freelancer Mod Manager v1.4 beta 4

- to download the Mod Manager click here.

To install this MOD into the Mod Manager

double left click on the file WTS-World-v3-42.7z.flmod

The Mod Manager interface will automatically unzip the MOD into the Freelancer Mod Manager \ mods folder (C:\Program Files\Freelancer Mod Manager) and the Mod Manager interface screen will pop up automatically, it may take a few seconds for the Mod Manager screen to come up, click on the MOD that you want to use, it will be highlighted now, and then click on Activate.

The Mod Manager will add/change the necessary files automatically into Freelancer and then you're ready to play.

Note: Using 7 zip you will get a much better compression ratio.
7 zip compressed the files to 70 megs
while winzip compressed the files to 120 megs.
so that's quite a differance...
