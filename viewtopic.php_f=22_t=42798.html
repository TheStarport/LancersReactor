RPG Mods Freelancer server now online

The server has actually been online for a few days now, but I've been waiting to see any other people were actually able to connect before announcing it (since my own gaming machine is on a local network with the server, that wouldn't be much of a test ).

Server name: RPGMods.com FL Server
IP address: 71.29.214.42
Slots: 12 (to start; this may go up, if I determine that the server can handle it)

Server is running IONCROSS Freelancer Server Operator, Mark V.

All players start with 50,000 credits and a neutral rep. In addition, users can improve their standing with all factions by blowing up nomads (in case you screw up somewhere, I figured I may as well include a safety net). The server is also running the Battleship Missions mod, available from this site.

There is a small required download for play on this server, as it is running a variation (still made the original author) of the Buyable Generators and Armor mod that is available here on LR. Since the version that has been available here for some time is not compatible with the version on the server, I have posted the proper version in the FL server forum on the RPG Mods forums (see the relevant sticky thread). This download is NOT optional; trying to play without it will result in a CTD as soon as you launch from Manhatten (I have personally confirmed this).

Since the main mod pretty much equalizes all the ships in the game (except for cargo capacity), any ship in the game is a viable choice (though for trading, the dromedary is pretty much king of the freighters ). I myself fly a fully-upgraded Dagger packing class 9 and 10 weapons and a class 10 shield.

The full rules for the server can be found here. I strongly recommend reading, as all players will be expected to follow the rules. Ignorance will not be a valid defense. Most of the rules are pretty standard stuff, but they need to be posted anyway, just so I have something to point to in case of a problem.

I should note that the {Admin} tag is not in use on the server. I myself do not use it, as I prefer to be able to just play the game (my in-game name is Baratheon79). However, if anyone needs help with anything on the server (even if it's just a partner for a mission) I'll be happy to help.

Once the server has a decent number of users, I hope to begin having regular in-game events. Examples of possible events would be bounty hunts and races (heck, there's already a race course in the game, so why not?).


** Edit - 3/21/2006:
Since anyone trying to download an attachment from the RPG Mods forums would have to register first (which I don't want to force people to do if I can help it), I have also added the required mod to the site's mod database (it's currently the only Freelancer mod hosted there.

The proper version of the mod can be downloaded from the following link:
http://www.rpgmods.com/modules.php?name=Downloads&d_op=getit&lid=52
