Windzarko

Mon May 23, 2005 5:02 am by Windzarko

I got the same error as has been mentioned above, and there is a very simple way to fix it. Go into the mods folder for FLMM, and open the XML file in wordpad (not notepad).

The bottom line should read "</script>"

Change this line to read "</data></script>"

Don't know why, but both those ending tags HAVE to be on the same line, otherwise it won't work properly. Once you have made this 6-character adjustment, everything will work absolutely fine (or at least, it did for me).
