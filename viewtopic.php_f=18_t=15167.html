Fri Apr 25, 2003 6:53 pm by C06alt

you cant convert directly to the freelancer file format from 3ds max
the straight forward way of doing it is to first export your model from
3dsmax as a .3DS file and then import that file into MilkShape 3D.

Make sure you get the Freelancer Exporter plugin for MilkShape 3D
I suggest loading up the dralthi example and deleting the main body
and leaving the hardpoints. then save the hardpoints alone to a
Milkshape project file (similer to a .max in 3dsmax)

Now clear milkshape and start a new project
Import your yourship.3ds
then merge the saved hardpoint project.

You may find your model imports upside down.
If so you may need to invert the textures.
when importing them into your new FL .mat texture file.

To create or edit .Mat files you have to use UTF Editor.
UTF Editor is also handy for fine adjustment of hardpoint
placement.

To edit statistics of your craft and to properly merge it into the game
rather then replacing another ship you need to edit some FL .ini files
the INI files of FL are encrypted. to unencrypt them you use

BiniQDU

thats all for now.
