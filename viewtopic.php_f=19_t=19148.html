You can host a teamspeak server very easily and if you have decent bandwidth have people connect to it and play FL as well.

I run a Clan TS server for a different M$ game involving Mechs. Even with a dozen folks on TS I can still play MW or FL quite happily....
and thats only on a 512/128 cable connection.

Just make sure you use one of the "lite" codecs such as CELP5.1.

If you don't want to run a server for it then there are many public TS servers out there I'm sure you could use, and some game server admins even host Teamspeak for that particular server.
