Sorry to hear that, but I do need need information from you to be of help.

If its a clean install, then the ships should start off at Huston, why are they at Manhanntan?

If you customized it some how, send me the files that you changed and let me take a look to see if i can find the problem is.

if you can launch from Huston and fly all the way to mahanttan and land, it should be working fine...

As for texturing in milkshape:

1. create new material, rename it and load up your .tga texture file into that new material
(click on where it says <none>;
2. select the group of mesh that you want to texture at "Groups".
3. go back to the material menu, then select the material you want to assign then presss "Assign"
3. Then go to the top bar -> Windows -> Texture Coordinate Editor->select the group in the blank pull down box in the TCE -> select a orientation -> remap
Play around with the options inside TCE until you get your texture the way you like it.

Thats it, have fun
