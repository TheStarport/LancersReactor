Trucidation

Thu Apr 29, 2004 6:36 am by Trucidation

They come up red because the two addons modify entries already modified by v3.40 final. First go to the Freelancer folder and run ModSettings.bat. Then run FLMM and activate Rebalance, then the two addons -- say 'yes' to the warnings. I noticed you hadn't gotten the 2.1 update (no I was unaware of them either), but it's too late to worry about this since Dan has said they'll ALL be packaged together in v3.5. No I don't know when that is coming out (I check the Rebalance forums regularly) but there should be an announcement in the Rebalance forums here too. Btw you should post mod-specific questions in the mod's forums instead of this general one since they are here

Technically, you can run the ModSettings batch file after you've activated the mod (notice the attrib command on line 2) but I'm not sure if that works, and I don't know whether attrib is still supported in win2k/XP. Safer to get the settings right before you activate the mod.

Edit: D'oh, forgot the other part of your question. Yes, the mod runs in SP campaign OR OpenSP -- you don't have to activate the OpenSP mod that comes with FLMM (no need to download it separately either), when you run the ModSettings batch file it will ask you if you want to play SP Campaign or OpenSP.
