go into My Documents\My Games\Freelancer\Accts\Multiplayer the select the players folder, copy the file 'name' and then paste it in the same folder, rename the copied one 'admin'.

then that person is an admin on server (requires Server restart to come into effect)

FLShell: type /setadmin (playername) that will set the player with the admin file, requires server restart aswell.
you'll need to be an admin to use /setadmin

IF YOU HAVEN'T SET UP ONE ADMIN, AND DON'T KNOW HOW TO USING FLSHELL. HERE'S HOW:

go into your flshell directory, open up flshell.ini, then go down to Admin Commands and then to the setadmin part, where it says

[Setadmin
;
; Description:
; Grant administrative rights to player.
; Administrative rights will be revoked if otional keyword "revoke"
; is specified
;
; Format:
; /setadmin player_name [revoke
;
AdminCommand = yes <-- change this to 'no'
save the .ini file and then open freelancer, with server running
then type /setadmin (your own playername)
and then when you've done that go back and change the file part back to 'yes'.
From there you can use the setadmin feature to anyone else eg: Server Police.
