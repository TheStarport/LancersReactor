solved the prob. went to MS knowledge base and they say it is a disk space issue, so i followed instructions and deleted all the crap in my TEMP folder (i cant see why as i have 2x80 gig hard drives). anyway i try again and still no luck, so i go to the temp folder where the install program has unzipped all the files. so i decide to install from a different location, so i fire up nero burning rom and burn all the install files to a cd (about 170 meg in all). i then reboot my machine just to make sure and reinsert the freshly burned cd into my dvd rom drive and hey preso the game installer pops up automatically. try this and see if it works for you.
