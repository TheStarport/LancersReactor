I had some trouble getting the Omega Class Earth Force Destroyer on Earth. You see, I could not see it in the ship dealer's inventory until I edited the market_ships.ini file, where I discovered this:

;-- Earth system
[BaseGood ;-- Earth
base = EA01_01_Base
;marketgood = omega_package, 10, -1, 1, 1, 0, 1, 1
marketgood = cargo1_package, 18, -1, 1, 1, 0, 1, 1

I simply removed the semi-colon in front of "marketgood = omega_package, 10, -1, 1, 1, 0, 1, 1" and that fixed the problem!

Just wanted to let you know about it, as it seems that no one else tried to buy the Omega...
