Freelancer Revised Beta Version

I couldnt think of any fancy, impressive name for my first mod so I'm just calling it Freelancer Revised for now. I'd like to put it out here so maybe some folks can help me test it out and see if there needs to be any tweaks (particularly to npc strength).

At the moment this mod doesnt include any new systems or ships, but I will probably be looking into adding those for the next version. I've seen some great ships on this site and will try to track down some of the shipmakers and get permission to use them. I forgot to include a read me file in the download so I'll post the read me info below. Here's the link to download my mod:

Freelancer Revised Beta Version

Features:
- Removed storyline missions
- Players start with usual hostilities towards unlawfuls, and neutral to all others (i.e. you are not friendly towards Liberty Navy or LSF now)
- Changed Trent's outfit to Orillion's uniform
- All jumpgates are unlocked
- All systems are mapped from start so you dont have to visit each location
- All commodity buy/sell rates are shown from start at every base
- Cruise speed has been increased to 990
- All trade lanes have been removed
- Cruise engines engage faster or slower, dependent on ship class
- Level requirement to purchase ships and weapons have been removed
- Civilian ships (Starflier, Startracker, Hawk, Falcon, Eagle) are now available in each sector - Liberty, Bretonia, Kusari, and Rheinland
- Most ships have had their stats and prices adjusted, and selling locations have been moved around
- Ships and weapons are now generally only available from the faction who makes them (i.e. Liberty vessels can only be purchased from Liberty Navy or Liberty Police bases, Bounty Hunter ships and weapons can only be purchased at Bounty Hunter bases, etc)
- You must be on friendly terms to purchase ships from most factions, except civilian dealers or freighters (weapons do not require good rep)
- Powerplants and scanners have been added as required equipment
- Tractor beam has been added as optional equipment
- Each faction now uses a specific shield type on their own ships(gravitron, positron, molecular). However, be aware that some factions use ships from multiple sources
- Npc shields regenerate moderately
- Military fighters have been balanced between the four Houses. Liberty no longer has puny ships compared to everyone else
- The 07L "Anubis" Liberty Advanced Heavy Fighter is now, rightfully, part of the Liberty fleet (read it's info card for more information)
- a few other surprises that I'll let you discover

Just download the zip file and extract it into your FLMM mod folder and activate. Enjoy!

Universal Shipping has commissioned a new ship, the Scarab High Speed Transport, using engine technology reverse-engineered by Sinai Industries from recovered Nomad wreckage.
