Editing in multiplayer

So far I found out the following:

When you edit goods/equipment it is only client side mostly.
Clientside is all that concerns the client like itemprices/equipmentshown etc.

EG. You can make the shipspeed 990 and as client you fly 990 on any server.

Serverside is the speed of AI ships/ shield regen/ shield strength, but only for AI ships.

EG. You can make the shipspeed 990 and all AI police/pirates fly 990.

Furthermore if the client and server don't have the same equipment ini files it is possible to start the game but the game can crash when launching from a base.

When you edit the startingposition/startingmoney/appearance etc of trent. It is again only clientside. But more important if you try to enter a server with a different newcharacter.ini you get the message. Playername already in use. So you cannot start a new player.

Unfortunatly like this it is very easy to cheat (flying 990 / buying diamonts for 1 / titan available for $1 at manhattan etc) (when acception all players) and difficult to be a modded server. As each time you mod something, all players should download and use the same mods.

I hope for a better serverconsole which can compare ini files of player and server.

Plus side is I could mod it so to show your tractorbeams/power/engine/etc and add more types to the game, there are even icons for most too and I used this board to help me do it, thank you all
