Ro9ue

Sat Jun 18, 2005 5:09 am by Ro9ue

You should not need to DL it - it should be on your machine once you do as bp mentioned and save a game.

However, it is simply an .ini file in the My Documents\My Games\Freelancer\ folder with the following info (or very close to it - depends on what you set in the Options screen) :

[PerfVersion
version=5
SkipMachineWarnings=FALSE

[PerfCount
count=36

[PerfOptions
mouse_speed = 0.56
gamma_ramp = 0.35
DitherControl = 1.00
MipFilterControlCallback = 0.00
TurnSensitivity = 0.00
generic_effect_perf = 1.00
Nebula = 1.00
SunEffect = 1.00
Asteroids = 1.00
space_dust_detail = 1.00
star_sphere = 1.00
AtmosphereMaterialConfig = 1.00
HUDAnimConfig = 1.00
level_of_detail = 1.00
use_environment_maps = 1.00
high_detail_textures = 1.00
sound_max_dist_scale = 1.00
sound_level_of_detail = 1.00
USE_3D_SOUNDS = 1.00
audio_music = 0.85
audio_ambient = 0.85
audio_sfx = 1.00
audio_interface = 1.00
audio_voice = 0.70
rtc_performance = 1.00
general_graphics = 1.00
advanced_graphics = 0.00
default_graphics = 1.00
skip_transition_scripts = 1.00
enable_tooltips = 1.00
enable_rollover = 1.00
cockpit = 1.00
USE_DYNAMIC_COMM_HEAD = 1.00
DIFFICULTY_SCALE = 1.00
autohide_maneuver_bar = 1.00
max_submitted_sounds = 1.00

[Display
size= 1280, 960
color_bpp= 32
depth_bpp= 24

Sorry you went to all that trouble to try and find this little bugger - formatting sucks
