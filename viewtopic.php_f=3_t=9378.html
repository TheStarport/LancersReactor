Anyone else use Voice?

Greetings All,


Just wondering if I am the only person using Voice Command to play the game.

I currently only have an old demo version of "Game Commander", and it only allows for 5 commands to be used.
But, even with only five commands being voice activated it makes the game quite fun.

I currently use:
Say "Fire" to launch missles. (Yes, I like missles.)
Say "Target" to target next ship. (Not next hostile ship.)
Say "Scan" to scan targetted ship. (Used with "Target" to pick my next victim.)
Say "Sh*t!" to use shield battery. (When my shields go down I usually say "sh*t" anyways! This just saves time. hehe)
Say "F***!" to use nanobots. (This is the next step after saying "sh*t". Hehe)

The only drawback I have had so far is that my woman, and my dog, think I am nuts sitting here talking to my game.

Anyone else tried it?
