An entry in jumpeffect.ini

[JumpGateEffect
nickname = jump_effect_hole <---- call whatever you please
glow_ring_effect = wormholeactivation
glow_ring_hp = HpFX1
glow_create_time = 1 <---- self explanatory change as you please
jump_out_time = 1.500000 <---- self explanatory change as you please
jump_out_tunnel_time = 7 <---- self explanatory change as you please
jump_in_tunnel_time = 3 <---- self explanatory change as you please
jump_in_time = 1 <---- self explanatory change as you please
kill_time_before_done = 0.600000
jump_tunnel_effect = jump_tunnel_interior_player
jump_tunnel = my_very_own_jump_gate <------ tunnel entry this is used for. If you change this you must change it to your own custom made tunnel.
jump_ambient = 25, 25, 50 <------the colour reflected on your ship currently = DARK
jump_background_color = 255, 255, 255 <-----Background colour currently = WHITE

Black Eagle

Tue Dec 07, 2004 12:12 pm by Black Eagle

Hi!

I'm affraid, but it would be some work to get the different jumptunnel colors out of the Monkeys Mod. I'm the creator of the mod, so I can tell you which files you should look in, but I think it would be easier to just add the new colors yourself, because it is not so much work as cutting the information out of my mod! But if you want to here are the files:

Data/Fx/gate_tunnel.ini: I named all the new jumptunnels xxx_custom! But you can just copy this file, because there are no other changes in this file than parts of the new jumptunnel colors.

Data/Fx/jumpeffect.ini: Again, just copy the whole file, it only changes the jumptunnels to new colors.

The last thing to change is in every system file. You need to edit the entries of all the jumpholes/jumpgates you want to change in the system file (So, for California jumpgate in Manhattan go to LI01.ini and search for the California jumpgate. The same in the California systems ini with the Manhattan jumpgate)
You have to change two lines:

jump_effect = xxx
goto = xxx

for example:

jump_effect = jump_effect_bretonia
change to
jump_effect = jump_effect_bretonia_custom
and
goto = Br04, Br04_to_Br01, gate_tunnel_bretonia
change to
goto = Br04, Br04_to_Br01, gate_tunnel_bretonia_custom

I hope this helps!
