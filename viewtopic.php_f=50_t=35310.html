its not only about the texture, but primarily about the model.
the farther a ship is away (those are the numbers btw), the less polygons the models have. This saves performance because you wouldnt see the difference at that large ranges.

0, 200, 500, 1000 for the lodranges mean:
from 0 to 200 distance, the 1st model will be used (the full detailed one)
from 200 to 500 it is replaced by a less performance-intense version
and so on, you get the idea.

at ranges over 1000 this ship would be invisible. (all all fightersized sihps in standard FL are)

hope this clarified a few things - u can view the low-poly version of the omdels if u know how to use the cmp importer into milkshape.
