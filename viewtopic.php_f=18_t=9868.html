1) We don't really know the limits of the FL engine.. if there are any.
2) Hypothetically (and assuming there are no engine limits), I think that we can make as many new systems as can fit on the nav-map.. HOWEVER...
3) Right now, in these early stages, with only very basic tools, creating a system takes a great deal of time and effort, even the most primitive system with one base or something. If you talking about systems which are really interesting, which contain tradelines, patrols, bases with random missions, planets, different zones (asteroid, gasses nebulas and somesuch), then you are talking about a tremendous amount of work.. so don't expect new exciting galaxies and unverses any time soon.
4) Creating is only part of the problem, testing it is another thing.
