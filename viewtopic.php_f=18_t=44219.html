Hi Spacepirate, i've used FLE quite a bit so i'll try and answer your questions.
Lane Access Zone
These are created to make holes in fields and are just empty zones. If you have an asteroid field you can use these zones to allow patrols or tradelanes to work without the rocks getting in the way. In order for them to work, you need to open up the solar/asteroids/(field name) and add an exclusion zone to it like this:
[Exclusion Zones
exclusion = Zone_FR05_pop_ambient_08<--a population zone
exclusion = Zone_FR05_01_lane_access<--an access zone
exclusion = Zone_FR05_70_ochiuchi_ne_niobium_01<--a field

Exclusion Zone
Different to above. I have tried using them before and usually end up with the wrong results. I think they are meant to be zones that don't let missions be performed in. But, whenever i use them as that they offer missions. If might only be if you have a mission vignette (for missions) and you want to cut into them. I just tend to steer well away from them and just use empty population zones to make holes in fields.

Zone Population
These are what you want to use as much as possible. They are much easier to use than patrols and are much less prone to crashing. I usually have a large system wide zone of low density and add other zones around bases, jumpholes and jumpgates. Patrols should be added to any system that has a pirate base or jumphole, but other than that its best not to go too crazy adding them.

Black Hole's
Well, yup, it lets you make a black hole. I had a go at one of these a while back and the game engine had a hardtime rendering it. It just looked a mess, might be best to leave that one.

Plugins? Got to admit i haven't heard of any, i think it is to let you add objects into the game. But, i've done that just by adding entries in solar_arch. Not sure if anyone else can help on that one.

If you want to get the order in the game and have them flying you are going to have to add entries to faction_prop and npcships. There was a recent post about the one, you might need to go through that too.
