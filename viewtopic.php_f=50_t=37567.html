That very small auto-aim thing is for me too, I guess It's just there to make shooting easier, so new guys can hit stuff when they start playing or something.

As for auto-aiming guns, I've done it before - although, differently. Just make the gun use ammo, and set it so It's like homing missiles (exept with the beam of your gun, and higher turn/speed rates). The only problem with this, is when someone shoots at you with it you get the 'Incoming Missile' warning, when there is actually no missile (good for wasting people's CM's )

Try it out sometime, I had great fun making and improving it
