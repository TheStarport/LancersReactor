Saw this out there....

I found myself with the freelancer black screen problem
that so many seem to have with the Nvidia 43.45 drivers.
I got this fix from Microsoft. It seems if you disable
the intro movies, the game will run fine. I am including
the instructions they gave me. Mine is working great now,
good luck with yours. Here it is:

If you notice that the issue still persists, please try
disabling the opening movie in the game and check if you
are able to start and play the game successfully.

To disable the intro movies in Freelancer, please use the
steps below:
----------------------------------------------------------
-----------------------------

Edit the Freelancer.ini file located in Freelancer\EXE
folder and place ";" in front of the movie code lines.

[Startup
;movie_file = movies\MGS_Logo_Final.wmv
;movie_file = movies\DA_Logo_Final.wmv
;movie_file = movies\FL_Intro.wmv


To find the folder where Freelancer is installed on the
system, please use the steps below:

1. Right Click on the Freelancer icon on Desktop and
click Properties.
2. Click on Find Target under the Shortcut tab. This will
open the folder where Freelancer is installed.
3. Once this folder is opened, find the freelancer.ini
file and follow the steps listed above to edit the file.

Please try playing the game after this and then check if
you are able to play the game without any issues.



Hope it helps
