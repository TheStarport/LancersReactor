For SP you need to mod \DATA\Missions\M01A\m01a.ini. The line is:

Act_SetShipAndLoadout = ge_fighter, msn_playerloadout

Which can be found about 1/3 of the way down the file. The first part is the ship by nickname so the Sabre its bw_elite2. The second part is the ship package which must match the ship. The loadouts are found in \DATA\SHIPS\loadouts.ini where you can find one for the Sabre or modify msn_playerloadout to be what you want. A good one is fc_ou_bw_elite2_loadout01 so the line in m01a.ini would be:

Act_SetShipAndLoadout = bw_elite2, fc_ou_bw_elite2_loadout01

Note, the cutscenes with King were all done with the Starflier and those can't be changed (just movies). Also, this is a way to get and fly Juni's ship by setting

Act_SetShipAndLoadout = li_elite2, MSN13_Juni

Have fun!
