UPDATE: Game still didn't work after normal reinstall, but I had read about deleting old savegame files as well. After finding and deleting those, problem solved. Also, there is another folder in the directory where you find the savegame files that contains your multiplayer info--if you leave that alone, you'll KEEP all of your online characters no problem at all.

This is where I found the saved games (working from memory here, don't shoot me if i make a mistake)

Win XP Pro OS:

My Computer--->C: (or primary HD, if different drive designation)---->documents/settings---->all users (your own user ID if you have multiple accounts on the system)--->my games---->microsoft freelancer

There are a couple of folders to search thru there, but the saved game files are obviously named...so you should have no problem finding them from here. Sorry I can't help with other operating systems, dunno anyone who's running it on 98, ME or 2k.
