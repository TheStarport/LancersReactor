Well, I'm an old mellow guy. I'll try to help you. I ASSUME you are trying to use the command line version of the program. Create a folder on the C: drive that has a simple name, like "X". Then copy the command line version (bini.exe) into that folder. Then copy the INI file you want to uncompress into that folder also. Then open a command prompt and change from the root directory of C: to the folder you created. Like "CD X" if you made a folder named "X". Then type bini filename.ini. Then type "dir" and you should see the original INI file and another file named filename.ini.txt. You edit the ini.txt file and rename it and put it back where it was. It does not have to be recompressed to work.

If you are using the GUI version (1.1 I hope!). It's a lot easier. I read here that there is some kind of bug in it, but I have not had any problems yet. You just open a file and then close it without doing anything else. The act of opening and then closing the file creates the ini.txt file in the same folder where you opened it. Hope this helps.

Skinnyrat:

Just click on the disk icon in the wordpad tool bar, and then exit from wordpad.

With the mouse, right click on the .ini.txt file and then click rename. Move your cursor (with the arrow keys) to the point after the i in .ini (before the .txt). Using the Delete key remove the .txt and then hit enter. That's all there is to it.
