Being that your new I'll save you some time and headaches. Download the Freelancer SDK v1.3 and install it into your freelancer directory. By default the SDK aka Software Developement Kit will install into the Freelancer folder so just let it do it's thing, it has an install routine built in. After you install the Freelancer SDK v1.3 start Windows Explorer and go to Program Files\Microsoft Games\Freelancer\DATA\SHIPS and with wordpad open the file shiparch.ini
Because you already installed the Freelancer SDK v1.3 the files will be in text to make it easy to edit.

Look for this entry or search for 237021
this is the Titan..... Change the line hold_size = 70 to whatever size you want....
Example
hold_size = 10000
It can be any size, go crazy make it 1000000.... Just use numbers, no commas... or decimal points...
