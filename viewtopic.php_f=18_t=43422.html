Sorry to bring back this topic, since it was answered over a month ago, but I'll put the answer here so anyone else having the same problem can get an answer. In my solararch.ini file, I had declaired it to use detailmap_ocean.txm, while in the MAT file, I used detailmap_rock01.txm. Making the two the same fixed the problem and made it black.

Incidentally, I've found a rather useful tool that converts one picture type into another... including DDS!
