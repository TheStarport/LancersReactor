Heya people,

My personal feeling is to go with the originals, and if people want to go back to the dark ones, maybe one of us could create a patch to reimplement the dark ones. Using an FLmod file makes far more sense to me than trying to use a batch file. For one thing, the FLmod files are safer, FLMM automatically backs everything up. So, if you have something strange happen, you can remove the sub-mod (what I'm choosing to call the darkness patch), then be back running like a champ (please excuse the goofy cliche').

Matter of fact, it may be possible to use the old darkness mod with Rebal 3.5 final, since that is what was integrated into it in the first place. We'd simply be removing it, and giving you the option to apply it yourself. But, I guess we'd need to test that and make sure my theory holds water, before recommending that route.

Anyway, this is all just my 2 cents, spend it as you will
