ModBug

Tue Oct 05, 2004 6:02 am by ModBug

Click start button
Then run
Type Regedit
move to this drectory

HKEY_CURRENT_USER\software\Microsoft\Microsoft Games\\freelancer\1.0

and delete these keys

Installkey
MpAccountname
MpAccontnamesig

close regedit window.
Start up FL and go to multiplayer and the game will ask for your number

Hope this helps
