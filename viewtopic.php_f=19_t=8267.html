taquinvol

Thu Mar 06, 2003 5:11 pm by taquinvol

In the EXE directory of the Freelancer installation there is a file called "fldirectip.hta" - this is an "HTML Application", i.e. a runnable script - which gives you a GUI to type in the servers you'd like to connect to.

If you don't want your client or server to contact the MS servers at all then edit the "freelancer.ini" file and comment out (by putting ";" infront of the line) the stuff in the [ListServer section - specifically there's a line that says "hostname = fllistserver.zone.msn.com" - you can change that to ";hostname = fllistserver.zone.msn.com" and you're done.
