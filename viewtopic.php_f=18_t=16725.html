Cpt.Kazo

Wed May 14, 2003 10:10 am by Cpt.Kazo

Have a closer look:

marketgood = bwf_package, 4, -1, 0, 0, 1, 1, 1
marketgood = bwe_package, 20, -1, 1, 1, 0, 1, 1

first line has 0, 0, 1 whereas the second has 1, 1, 0...that's the important stuff.
First line means this base does not sell this ship, second line means base sells this ship. So in market_ships.ini all bases have the entries for all ships, but only 3 are set to "purchaseable" at a time.
