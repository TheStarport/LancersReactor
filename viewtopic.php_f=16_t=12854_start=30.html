This is a quick post for all those running 2.0 of datastorm and getting the runtime error "5" at the "Goods.ini" file.I was getting the same error.It work fine with "vanilla" fl,then errored when I activated Rebalance v2.01.
I deleted datastorm,then reinstalled it exactly as it says here:
http://101st.info/DS/
The i dl'd the v2.2beta and saved it over the "Fldata" file in my datastorm directory.Make sure it asks to "overwrite" the other fldata file(which i checked was in the datastorm folder first) and save it over it,the run it.
Mine worked great,and its superb,esp when trying to find certain ships,jumpholes etc.
Hope this helps some,if not all you'se having the same problem as me,and thanks go out to the guys who made datastorm for an excellent util,
