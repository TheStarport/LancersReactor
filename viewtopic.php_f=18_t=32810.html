Edit one of the entries to look like this...
Don't forget to add the left facing bracket to BaseGood


<data file="data\equipment\market_ships.ini" method="sectionreplace">
<section>
[BaseGood
base = Li01_01_baseIw04_01_base
</section>
<dest>
marketgood = gf2_package, 4, -1, 0, 0, 1, 1, 1
</dest>
<source>
marketgood = FlyingSauser_package, 1, -1, 1, 1, 0, 1, 1
</source>
</data>
