Windows XP SP1 is a perfectly fine patch for XP, in infact made the 1st version of XP much much more stable.

Of course, never forget updates from the update site either...

Regarding the crashing... it won't be XP, the OS always gets the blame first, trust me, being a technician I know I do ... more than likely you've got an issue with some of your hardware and/or it's associated drivers, or some software installed such as those running in the background eg. Internet Blockers, RAM Clearers, Virus Checkers... etc... try removing or disabling things one by one until things work.... and update ALL drivers for hardware that your can.

If this don't work... update old software to newer versions. If all else fails, try a reinstall after backing up... and install the basic drivers... testing from the ground up loading your games until you find the prob.

But try the first idea... from up to ground

Good luck.
