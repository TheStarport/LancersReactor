Lets see, I can spend 2 hours a night tracking down and banning Modders/Cheaters/Hackers or I can actually get a full 8 hours sleep. Well I chose the latter as lack of sleep has been impacting both work and personal life.

I did my best to keep the server up as long as I could. Keeping in mind that there were major stability and performance issues after I broke 3000 accounts, a couple of possiblites that I weighed before I took it down were

1) Get a couple of remote admins
1a) I decided not to do this on the grounds that I do have some private information stored on the same system that I ran FLServer on

2) Password server and only let existing accounts keep playing
2a) Still would have huge stability issues due to # of players

3) Wipe all accounts and password and only allow those that I sent password to play
3a) Even more time required than just banning bad players each night

4) Take down server for ever
4a) Really don't want to do this as it is a fun game and I do have a great machine and nice business DSL line I can run it on

5) Take down server till patch and keep all current accounts
5a) This was an option I seriously considered, but if they didn't fix the stability issues it would be worthless. Also with the various gameplay modifications and rank modifications coming, this would potentially cause problems

6) Take down server, wipe all accounts, and bring back after Server patch
6a) This was the best option I could come up with. Yes players loose thier accounts, but with the new anti cheater/hacker/modder code in the patch, this will balence the game better from the start. This also guarentees no stability issues from the point the patch is released till at least 1 - 2 months out when I get back to around 3000 accounts.
