parabolix

Fri Oct 07, 2005 3:57 pm by parabolix

You need to download a video recording program such as FRAPS which I believe has a free trial available, the only problem is that you may experience slowdown in your game if you record at a high resolution. You'll probably want to edit your video, you can use Windows Movie Maker for that, which has help files built in to help you with that. Then you just need to put it in game by overwriting the file fl_intro.wmv in your Freelancer/Data/Movies folder.
