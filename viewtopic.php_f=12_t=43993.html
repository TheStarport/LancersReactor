Random music association game!!!

This is an idea stolen from the forums at Last.fm...You start with a song (and band), and the next person names another song (and the band) that shares at least one word with the previous song. Please post in the following format:

Artist - Song

Also, bold-typing the word that the song shares with the previous song is nice as well. Here is an example...

The Beatles - Let It Be

Radiohead - Let Down


Let's see how far we can go ...No googling! I'll start with the one I just used...

Radiohead - Let Down

OFFICIAL RULES

1. Word must be exactly found in song title. For example, if the previous song posted has the word "Television" in it, the next song cannot be linked using the word "Vision."

2. Linking words must be more than 3 letters (Yes, I'm aware my example does not follow that rule).

3. The same linking word cannot be used 2 times in a row. For example,

The Beatles - Let It Be
Radiohead - Let Down
Clap Your Hands Say Yeah - Let the Cool Goddess Rust Away


That cannot be used, you must link different words each time.

4. Linking words should be emboldened in the next post, like is done above. If you need help with the code to embolden your text, click on the Forum Codes button in the Reply to Thread screen.

5. Do not purposely try to end the thread by choosing a song that is impossible to continue off of. Using difficult songs is recommended, but using impossible songs is frowned upon, and the songs can be disregarded from this thread.

6. No double posting
