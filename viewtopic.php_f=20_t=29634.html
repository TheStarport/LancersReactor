May I suggest a few things first:

I guess your trying to view it in the exe folder by mp_account_id file or whatever it is.

However, when you install the game, it GIVES you a MP id number (first installation/after an uninstall). Write that down, and keep it next time for further reference. Then when you have to reinstall once more you can enter it there.

BUT - try deleting the game (go to my programs/microsoft games/ and highlight that FL folder and then hit delete). Now insert your CD and let it install as usual, this will NOT generate a new MP id, and if it was just a problem with that install, then this should have fixed it.
