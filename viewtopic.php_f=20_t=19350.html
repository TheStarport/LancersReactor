hi guys,

format c sounds a bit drastic

step 1 - if your not using the mod manager and the mod is installed just now you should go get the original ini file zip from the down loads section of this site so your game is returned to its original state ( or just re install the game -takes 5 minutes)

step2-as a quick test go into your save game folder for sp and either backup all your saved games into a zip folder or just move them to another directory. either way make sure there are none of your original saves left in the folder - it should be completely empty.

then install the rebalance mod and see if it can launch the game then - if it can one of your saved games is incompatible - replace them one at a time to find out which one.

it may take a little while - but probably less time then reinstalling windoze

if this doesnt solve anything then id get in touch with the creator of the mod - theres a large thread in the mod announcement forum i think ( read it before posting please - i know its long but since hes onto 1.6b your problem might have bin answered some time ago).

hope this helps,


dr del
