Thu May 08, 2003 12:07 pm by Craith

You have to decode the .ini's first, in the download section you'll get variuos decoders (I use bini)
backup the original file, and open the decoded one (bini generates a .ini.txt file, rename it to .ini). I open it with notepad, but some are experiencing problems with it and recommend wordpad.
Change whatever you want to change and save the file
You don't have to code the file again, Freelancer reads decoded files too.

Some important .ini's
DATA\SHIPS\shiparch.ini => all ships, including battleships, trains etc.
DATA\EQUIPMENT\engine_equip.ini => engines, for changing speed
DATA\EQUIPMENT\st_equip.ini => shields, thrusters
DATA\EQUIPMENT\weapon_equip.ini => all kind of weapons

Be sure to backup any files you modify and don't use mod's for cheating online
Have fun

Craith

This not a tutorial I'm not good enough to write one but I think that the people who want to learn should have some help getting started because you get to add more to your games thus stretching your dollar value and its also alot of fun Each game is different, but the basis remain fairly the same. If this is your first attempt at modifying a game this is where all that time you spent at school is going to pay off because you need to read the information in the forms so you can get a general idea of how to get started. The tools to work with the files have to be made by someone much smarter than me fortunately for us those people care enough about gaming and their fellow gamers to take the time to make them. As in the case of freelancer you can get those tools in the Editing utility download section. There are more tools out there in webland and at your favorite software store but thanks to Bargib and the generous people that run this site the basic tools are here. The first program that you will need is one that can make the files easy for us to read, that would be the text format for most For freelancer this would be biniQDU or binilite. Before you start to change any files to text make a backup of all the files you plan to modify. In freelancers case these files end with .ini the abbreviation for information. After a quick install of the bini program start it up. Look for the load option and navigate to the freelancer directory. Click on the ini file you want to look at, Bini will open in the bini program and converted it for you into text. Then click either save or save as with the same name and the bini program will make two copies of the same file one with the extension ini and another for backup purposes with the txt extension. Now your ready to begin You can use the Bini to edit but only work with one file at a time or you can use wordpad or any text editing program and again navigate to the freelancer directory and open the ini file you want to edit. Make your changes I would suggest a little at a time without any mistakes, save the file, crank up the game and make sure it dont crash then you can edit some more. This should get you started if your still confused I know I would be come back and explain what you cant figure out.
