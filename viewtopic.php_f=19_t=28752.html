From one that Knows how it works.

This requires about 5 to 15 mins work.

1 the admin using IFSO finds your old account you cant access and gets the folder name from the location then closes the server.
2. He then moves your old characters Folder out of the mulitplayer folder in my docs to a safe place.

4. He then finds the new folder and deletes each .FL file in there but NOT the name file, then finally moves the FL files from your backedup folder in to the new folder but not the named file.
