That file was certainly one of the things I was looking for. I am in the search of a complete listing of all objects that can go to Archetype..

For example :

[Object
nickname = DA01_to_LI01
ids_name = 061982
pos = -83176, 0, 44831
rotate = 0, -70, 0
Archetype = jumpgate <===========================================
msg_id_prefix = gcs_refer_system_Li02
ids_info = 66145
jump_effect = jump_effect_bretonia
reputation = li_p_grp
behavior = NOTHING
difficulty_level = 1
goto = Li01, Li01_to_Da01, gate_tunnel_bretonia
loadout = jumpgate_da_01
pilot = pilot_solar_easiest

That Archetype can be an outpost, be a battleship, be anything else. I want to know what all the possibilities are. For example I found this out by playing =>

(Make a backup !)

Add this to DATA\SYSTEMS\YourSystem\YourSystem.ini (for example Li01.ini)

[Object
nickname = Da01_03
ids_name = 196768
pos = -82176, 0, 52831 ; You may want to change this location to find easier
Archetype = rm_l_dreadnought
reputation = li_n_grp
behavior = NOTHING
pilot = pilot_solar_hard
hitpoints = 100, 100
repop_time = 10 <===== Decides how long it will take for the Dreadnought to respawn.

This creates a DREADNOUGHT, That is Destroyable. As far as I have tested if you add "rm_" infront of "l_dreadnought" you add a HP bar to it and make it destroyable. Tested adding "rm_" with a Bretonian Battleship "b_battleship" and it worked as well.

Now I am curious on where to find destroyable AND dockable bases.


Edit: I didn't manage to add any weapons to the Dreadnought tho, neither was I able to make it fight back when its attacked..So any help is welcome there.
