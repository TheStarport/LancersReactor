Nog

Sat Apr 05, 2003 2:10 am by Nog

Unfortunately, I have found out that Audigy cards have the midi and wavout tied to the same driver. This means that if you disable the midi, the entire cards turns off.

The Live/Audigy problem has become so vast that there is now an alternate set of drivers out there for the cards.

Try kX project if you really desperate. They've developed an alternate set of drivers that may help you out.

Only thing is that I can't say they'll work.

But I would try them out on my system.

If you're desperate enough to be a guinea pig, give the alternate drivers a try.
