driftingghost13b

Mon Apr 12, 2004 3:58 pm by driftingghost13b

Every ship has a different set number of hard points for there weapons. If you have the time you can look them up using the utf_edit by opening up a ships .cmp file. As for designation and location the hard points are usually add using milkshape3d but you have to re hard point everything on the ship. In the goods.ini file you can add weapons to hard points on a ship only if the hard point physically exists on the model by adding a line of code like this
addon = nomad_gun01_mark02 , HpWeapon01, 1 note you must specify what weapon and what hard point. You will need to learn the nick names of all the items under the utility downloads I think there is a translator.
I hope I helped
