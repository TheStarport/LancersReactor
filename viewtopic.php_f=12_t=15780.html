A couple of steps to improve performance DRASTICALLY.

Right click on Start Button, Select Properties. On Start Menu tab...Select Classic Start Menu.

Right click on desktop, select Properties (Display Props)

Change the theme to Windows Classic.

Right click on My Computer, select Properties. Click on the Advanced tab. Select Advanced. Click on the Performance Settings button. Select "..For best Performance"

What this does is set WinXP to Classic Win9x view, and not all that graphical garbage which takes up CPU/Video Card processing power.

I ran a benchmark on my system...With the graphical crap...my system ran about 25% slower then without....

All that graphical crap us useless...It's all visual.
