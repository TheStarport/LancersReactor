chaning an existing texture is (compared to texturing a new model) really simple ..

could even do it yourself .. you will only need a graphics application to edit pictures, and the famous UTF Editor

of course you need to know where the texture source is located so i`m just looking .. nickname = or_elite

Ships\order\or_ships.mat <--- this file contains the pictures, open it with utf editor ..
now expand texture library node .. there you got em
each picture must be exported (click on "MIPS" and export)
then edit with your graphics app and simply import to the same place .. (same filetype)
save or_ships.mat

EDIT:
note that the osiris uses same .mat source ..
if you want just rename your modified .mat file and look for "or_elite" in shiparch.ini, change the reference entry there
