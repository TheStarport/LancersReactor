FLstat Beta Patch

Calling all FLstat users!

I've added a few features to FLstat, and I'd appreciate any testing that you could do for me. This version is specifically targeted at people with big servers with a large number of characters, who don't want to list every single character.

Edit: Beta testing has finished, and v1.1 has been released. Get it here! 24th July .

Changes:
o Display of Money and Kills with comma-seperated thousands.
o Display of login time/date to more readable format.
o New command line options:
-top N - Show only top N characters.
-rank N - Show only characters at rank N or higher.
-days N - Show only characters which have played within the last N days.

Specify any of these options in the file "update.cmd". For Example: <pre><font size=1 face=Courier>FLstat.exe -rank 25 > middle.txt </font></pre>

Please let me know if you have any problems, questions or suggestions on how to make it better.

Cheers,

Cryo.
