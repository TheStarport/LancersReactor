warzog:
Making the ship icons sounds like easy work
Just the right thing to start FL modding.

I just did the Ferret icon as a try, and i think it works.
You can get it here if you want to test it: http://www.fastshoot.de/upload/ferret.3db
Just put the ferret.3db into the Equipment\models\commodities\nn_icons\ directory and edit the Goods.ini like this :

[Good
nickname = gf2_hull
category = shiphull
ship = ge_fighter2
price = 55000
ids_name = 12013
item_icon = Equipment\models\commodities\nn_icons\ferret.3db

if this works, it's no big deal to make them. Needs 5-10 mins.

If I get time to, i'll look what i can do for the voice mod.
