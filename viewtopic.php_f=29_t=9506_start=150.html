Wanderer

Fri Apr 04, 2003 2:01 pm by Wanderer

Theres a thread around here that explains what each and every data header is in the market ships file. For instance Li01_01 is Planet Manhattan. Since i don't remember which base it is you want, just take the dreadnought line from the market ships file and place it in the data headerr you want overwriting the ship previously there. You can tell which one it is if the numbers are like so:

marketgood = bfr_package, 4, -1, 1, 1, 0, 1, 1

The first one is the level required to buy it so it varies. But the rest of the numbers are more or less the same for available ships at that base.
