pook

Sat Mar 01, 2003 5:46 am by pook

Each column in market_ships.ini is whether a ship is available for sale and at what level.

In the demo (which is what I'm using), the number of models available is limited... but if you change wherever it says:

xxxx_package, n, -1, 0, 0, 1, 1, 1

(where 'n' is the level the ship can be bought at) to

xxxx_package, 0, -1, 1, 1, 0, 1, 1

The ship will be available for sale.
