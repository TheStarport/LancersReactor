Try Freelancer SDK by EOA Players consortium. It contains all the ini files pre-decrypted for you, and some bugs fixes to the originals that DA team made. Most of hte new mods on the front page will have links to the SDK - so look for any mod (that is a mod, not a program ), and you should find the download link. Personally - i install it to my spare HDD and then copy the files across when i need them (usually cause i am fiddling with files and cannot be arsed to reverse what i have done, so easier to use that as my backup! lol

Have fun
