Testers Needed - Battleship Mod

I need a few "beta testers" to run trials of the Liberty vs Rheinland MOD before I release it to everyone. I've setup battleship encounters in specific locations and it appears to be running normal. What I need are a few people to fly around these battleship encounters and do the following;

1. Note the location of the battle, it should be the same as the locations listed below. If you can't find the battleships please do a cruse around system specified to see if they accidentally are spawning elsewhere. Note: It could take up to 30 seconds before the first fleet appears.

2. Note the attacking ships. Turret fire should be comming from both fleets.

3. Check for any stability issues such as server lag, etc. I have this running on an AMD 2.2Gig server with a T1 connection, there will be 0 server side lag but I need to know about client side lag. (which is why I can't test this for myself)

4. Watch for any glitches like ships vanishing in thin air, textures not appearing, etc.

You should be available for 2 hours, atleast. Also, for those of you with FLMM, I have activated the cruse 3000 mod on the server end which means you can activate it client end. This will allow you to get to the destinations faster than normal. Please take heed, there is no modification added to increase your health or shield energy because that is known to cause issues with this mod.

The locations I need checked out are as follows;

1. Rheinland fleet in Hamburg near Planet Hamburg - Hostile

2. Kusari (friendly) and Rheinland (hostile) fleets in Shi***u between Deshima Station and Planet Junyo

3. Kusari (friendly), Liberty (friendly), and Rheinland (hostile) fleets in Colorado around Planet Denver.

4. Liberty (friendly) and Rheinland (hostile) fleets in New York between Sun and Manhatten

5. Liberty (friendly) and Rheinland (hostile) fleets in Texas around Planet Houston


It should be noted this is a server-side mod, as far as I know, so there should be no risk to your actual install of Freelancer. Here is the server information, please note I have limited the users to 6 as this is a beta test -ONLY- server. This server is not meant to be used to play the game.

As well, IP's and Usernames will be recorded on the server (to show you used it) and I will expect a report back from everyone that logs in. Please put any issues or problems under this topic. As well, if you don't notice any problems, please state that here too.

NOTE: Deactivate all mods on your end (except the cruse = 3000 mod) as other mods have been known to cause issues with this mod.

The server, which will only be active until 11PM EST, is as follows;

Server = Jackel's Testing Lab
Pass = testbed

Thank you for your time. And yes, I will be making this mod totally public once the testing phase is complete.
