FL GTL Global Tournement League server



FL GTL FreeLancer Global Tournement League

We have setup a server and forum to start a platform for cross server tournements.

The goal is to have a competition between clans of servers across the globe and to have a competition of top ace server teams.
Finally there is a place to compete for the title of best fighter clan and top killer server !!!

For this purpose we have created the FL GTL server.
The server runs the Moonshine mod created by LastHope.

Mod details:

1 huge system without jumpgates or holes.
No npc's
Sabre / Titan / Eagle available at 2 start bases.
100.000.000$ start up cash
All vanilla equipment and weapons available.

The server runs FLShell for better kill monitoring.

The mod is available in the FL GTL forums.
The server is password protected and the password will change for each specific event.
Training opportunities will be made available.

We have already 3 servers (Monkey Universe / TopFreelancers / Haven 24/7) that will be participating in the upcoming events.
Last sunday we had our first positive test battles and we are ready to set the wheels in to motion.
Other servers are welcome to join and can check it out at the FL GTL forum : http://www.phpbb88.com/flgtl

For now this is open for servers and their recognised clans.
Clans will be accepted if and only if they have admin aproval of their home server.
Freelancers are for now not allowed into this competition.

This is not a new open server, it's a platform for a fun global competition between all interested servers.
There will be no recruitment for clans or servers on the forums or the server.

We hope that we can build this up as a new and exciting feature for the complete community.

Gestoord

FL GTL host / Monkey Universe sysadmin / SFG clanleader
