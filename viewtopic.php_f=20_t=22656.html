Right click on your FL shortcut and use Find Target (or find the EXE folder in the Freelancer folder). Find the program FLServer.exe. Run it: make sure you check the "Allow New Players" box in the upper right of the configuration that pops up. Then hit ok, wait for it to finish loading, and minimize that program.

With FLServer running, you are hosting your own LAN accessible server. All you need to do is go into Freelancer, select multiplayer, LAN, and join your own server. It should show up there.

When I first ran FLServer.exe I thought it was a dedicated server program, but you can actually play the game while it's running. This is helpful if you only have 2 computers and want to play LAN with a friend, too.

There is a utility in the EXE folder of you FL Install, which allows you to bypass the global server, and conenct directly via TCP/IP. It's called fldirectip.hta and here's the info that you see when you run it...

Connect to a Freelancer Server via IP Address
This feature allows you to connect to a Freelancer server without having to connect to the global server. This is useful for when the server admin does not want to advertise the server to the rest of the Internet, or when the global server is down.
To add an IP address and port to the list: enter the address and port below, then click the "Add Server" button.
Once you have entered all of the IP addresses you want, click the "Launch Freelancer" button.
IP Address:
Port:

Add Server

Command line: Freelancer.exe

Launch Freelancer

What I would do, is check your ipconfig /all (run it from a command prompt), take a note of the ip address in case Windows has created you an autoconfiguration ip address. If the ip address for your adapter reads 0.0.0.0 then you could still try to use the loopback address of 127.0.0.1

It 'might' work. Let us know how you get on.
