Quick-FAQ - 5/23/2006

I have a problem!
Please update to the latest version of Freelancer Explorer to see if that solves your problem.

FLE crashes and I don't know why!
Please use the reporting feature so we can try to isolate the problem and fix it. If that does not work try our Contact Form. Be as descriptive as possible about what you were doing at the time of the crash so that we can help you.UPDATED!!

My infocards don't work right
We recommend that you use FLED-IDS to edit your infocards, Freelancer Explorer only allows infocard and name additions to make system making easier but it is not designed for extensive work with infocards.

How do I add jumpholes/gates?
You can do this from view mode through the traditional right-click method or in edit mode via the jumphole/gate button on the sidebar.

@Affliction - There are several functions in this program that let you create something new based on already existing items. With fields, whatever original field you select that is how it will look in the system until you change it outside of the program. The same happens with room setups for stations, it uses an already existing item in the game as a template to create the new station. After you add them you will need to manually change them to the way you want them.

@Affliction - That is what I was talking about.

@dubois - Due to an oversight on my part I failed to notice that there were multiple versions of the atl71.dll file included with this program. In the folder where you installed the program you will find a folder named ANSI_FOR_WIN98 which contains the correct version of the dll for users running windows 9x, just copy that file over the one in the program folder.

@Chips - I thought it was clear before when I announced that we had officially taken over its development but just to make things clear I will re-iterate the situation.

Version 1.10.26.0 of Freelancer Explorer was created and released by Shsan who is the author and originator of this program. Recently Shsan passed on development of this software to Mephit, one of the programmers from the group =EOA= Players Consortium. All future releases will be made by Mephit, and to him all questions and problems should be directed.

Since this program has never been released on TLR before I felt it was long past time to do so. However please note that there are no improvements to this program from Shsan's last release. We will offer what little support we can for this version but that is limited to our understanding of how it works. This of course will not be the case in future versions released by us.

Right now we are consolidating the program and preparing it for refit. It has already been converted from VC7 to VC6 to make things easier we hope in the long run. We are committed to seeing this project succeed as a highly useful tool for the community.
