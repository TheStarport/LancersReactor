kaegogi

Fri Jul 09, 2004 5:04 am by kaegogi

Start UTF editor.
Click on "Open File"
Navigate to the .dfm file
Double click on it
In UTF Editor you will have a plus sign in a box click on it
Click on the node called "Texture Library"
Click on whatever node that's down there you want
Once that expands you'll see a list of MIP's click on MIP0
Click on "Extract"
Navigate to the folder where you want to save
In the file name box type "nameyouwant.tga"
Save.

Fri Jul 09, 2004 2:49 pm by kaegogi

Dunno. You can try and let us all know how it turned out.

Go to data\equipment\engine_equip.ini find the engine that's listed on your ship's package. Then put none where the flame efffect is. Like this:

[Engine
nickname = ge_gf1_engine_01
ids_name = 263671
ids_info = 264671
volume = 0.000000
mass = 10
max_force = 48000
linear_drag = 599
power_usage = 0
reverse_fraction = 0.200000
flame_effect = gf_li_smallengine01_fire <------type "none" here
trail_effect = gf_li_smallengine01_trail
trail_effect_player = gf_li_smallengine01_playtrail <----your trail
cruise_charge_time = 5


Edit = Or, if your feeling like an over-acheiver today, you can go into data\fx\engines\engines_ale.ini. Find your engine flame entry and the associated txm files and mess with that now that you know how to use UTF editor. I have no clue if that would work but you never know until you try.
