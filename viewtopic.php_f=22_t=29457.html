Servers along with their IP and port numbers - server info -

Multiple Freelancer users are suffering from a specific router issue. Basically, they can only play online multiplayer games by going directly to the server with the -s command or via FLDirectIP. However, the IP address and the port number of the server is needed before they can do this.

I wanted to ask Server Operators to post in this sticky in The Lancers Reactor > Forums > Freelancer: Freelancer Server Forum > popular servers along with their IP and port numbers.
List your servers along with it's IP and port numbers. and many online players would greatly appreciate this.

Also since you're kind enough to take the time to list information about your server if you have a website where players can get information please post your website also.

In other words you're welcome to promote your Server and Website in this thread to make it as easy as possible for players to find and connect to your Server.

Thank you very much.

A secondary thread on server information is located RightHere
And if you want information on how to load your server info into the very popular FLStats and get some free advertising just Click here

Don't forget you can use the Dedicated Server -the easy connection tool- by Crabtree Cooperations Network.
Instructions on how to use the easy connection tool are on the download page.

You can connect directly to a Server by simply right clicking on the Freelancer Icon on your Desktop, then click on Properties. Then edit the Target line to read:
"C:\Program Files\Microsoft Games\Freelancer\EXE\freelancer.exe" -s.192.168.02:2302
Of coarse you have to replace the numbers in the example with the correct Server IP and Port number. If you don't know the Server IP you may be able to located the Server name and IP on the Global Server List.
Then Launch Freelancer
- click on Multiplayer
- click on LAN
- click on the server
