Possible Fix to Random Crash Problem (new)

Hello, all

I thought I'd register to this site to post a new possible fix to the "random crash" problem that some people have reported and that I have spent the last week trying to solve. Let me go over the steps I went through from start to completion.

Original System:
Pentium 4 1.5 ghz
768 megs PC800
ATI 9700 Pro
SBLive! Value
2 hard drives
CD writer
DVD player
Intel 850GS Motherboard
**250 watt Power Supply**

I would get random crashes in the Game, but not only this game, but any other recent newer dirextX 8.1 or 9.0 game, including Hegemonia, Tiger Woods 2003.

1. New drivers
2. New Direct X
3. Disable HW sound acceleration
4. Update BIOS
5. Reformat HD#1 and reinstall OS
6. Install New Audio Care (Philips Seismic Edge 5.1)

At this point I'm pretty sure that It is a problem with the graphics hardware, so I went one step further...

7. Try replacing Video Card (with new 9700 pro)
Again this didn't work, but as I scoured the docs on the Sys reqs for the 9700, I found. "Reccomended for System stability... 300 watt power supply"

I figured that maybe I had too many things pulling juice from the old power supply. (I had added the extra CD and HD since purchasing the macine). So the next and final step, I...

8. Unplugged extra HD from from bus and power supply !

Now the game (and video card) works like a champ. It seems my power supply of all things just couldn't quite hack it.

I hope this can solve at least one other person's prob.

-Your fried in the sky
