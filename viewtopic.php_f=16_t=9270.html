I found the missions extremely difficult but I wanted to see how the story turned out so I used the cheat code. If you do a search on your hard drive under the Microsoft Games\Freelancer directory you will find an “.ini” file with the name “Options” in it. Sorry but I am at work now and don’t quite remember the file name. Open the file with WordPad or note pad and change the difficulty setting from 1.00 to 0.00. From that point on you will not be spending money on nanobots and shield batteries as all bullets will bounce off (God Mode). You can change it back later if you wish but that is how I got through the story missions without getting overly frustrated. Now the game is fun because I am doing what I wanted to do all along, trade, upgrade and kick b*tt.
