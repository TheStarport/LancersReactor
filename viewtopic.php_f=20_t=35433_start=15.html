Your problem is the operating system...not the SP2 patch. The only thing it did was identify resident issues with incompatable software/hardware. Please do not attempt to blame a "patch" on computer related issues. Kernal errors are 99.99% of the time indicators of operating system problems.

As a helper, I might suggest trying out XP Advisor downloadable (free) from MIcrosoft's site. It will tell you exactly what is causing your problems. If you run that program I guarantee there will be issues that need to be addressed. If for some unknown reason you can not run XP ADvisor, then check out the Windows XP catalog for other possible conflict both hardware and software related.
