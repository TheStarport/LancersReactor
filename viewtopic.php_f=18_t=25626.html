I dont know if this is true but it appears to me that the game looks for weapon ids info in the EquipResorces.dll (dont quote me on that)
try using a programe called FLEd-ids,

then just punch in the number 266756 into the ids_info box at the top of the interface and click the Get ids_info button it should come up blank in the text area.
then add the following into the text area.
<?xml version="1.0" encoding="UTF-16"?><RDL><PUSH/><TRA data="1" mask="1" def="-2"/><JUST loc="center"/><TEXT>YOUR WEAPON NAME HERE</TEXT><PARA/><TRA data="0" mask="1" def="-1"/><JUST loc="left"/><TEXT> </TEXT><PARA/><TEXT>YOUR WEAPON INFO HERE.</TEXT><PARA/><POP/></RDL>

once you have done that click the save ids_info button and your done hopfully that will fix it...

From what Ive read if the name and info comes up corrently thats fine for weapons the actual stats come from the INI itself where as the ships all come from the .dll

The main problem could be this line

projectile_archetype =

which is found in the weapon_equip.ini

this line links the [Munition" and [Gun" blocks so make sure they are corresponding. Your "projectile_archetype = " is probably linking the stats to the old weapon.
