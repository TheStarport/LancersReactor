Freelancer System Placer



Smile!

Freelancer System Placer is an application i made for a friend of mine. I've tested it several times, and as you can see by this rather humerous picture, it was a complete success. I'm releasing it here so that others might find use to it.

TLR is now hosting the file!

FLSP (89 kb)

Things to Remember
- ALWAYS ensure that the universe.ini you open is NOT read-only. If it is read only, the save function will silently fail and you will lose all your saved changes. FLSP does not check to see if the save failed. This may be fixed in later versions
- Don't go too far off the map. FLSP doesn't check where you put your system, if you put it too far off the map, FL will crash.
- You may resize the window but this does not effect whatever background graphic you have loaded, nor does it affect the system placement or their converted positions.
- The program is designed to allow you to load up your own images, but be warned that this feature is most definatly not garenteed. If your image is an oblong size, such as 100x200, i cannot garentee that things will work properly.
- This program requires the latest version of the .NET Framework (version 2.0)

Have fun!
