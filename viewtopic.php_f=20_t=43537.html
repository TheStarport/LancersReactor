All three disks "forget" to install the freelancer.exe?

All three being corrupted with exactly the same issue is really really really weird (and unlikely). Furthermore, for other issues to occur with invisible ships... sounds like either corruption or potential drivers issues too.

Have you tried a quick cleanup of your disk with disk defragment and cleanup? You can get ccleaner (free) tool to do this for you, remove old windows registery items etc (MAKE BACKUPS WITH THE TOOL FIRST). I'd also use the FL uninstall to uninstall the game first, then clean up the disk/registery and scan the comp with your scanners (nothing like being efficient).

I'd also check your GFX card for driver updates, and update it - as well as windows and your sound card drivers too.

Then I'd reinstall the game once more, and download the SDK and install that over your game (who knows if it will help). Obviously it all used to work, as you've been registered for a long time with no issues... you could then disable 3D sound to be safe, and then run the game.

Have you changed your computer since the last install of the game (when it worked) or anything? It's just that 3 disks all failing for the same reason is mighty weird - changed firewall perhaps?
