Tachyon

Damnit. Look, this is how it will work.

Though it will vary depending on how popular FL is, there will be a large number of independant, persistant state servers, such as Elite that TLR is going to be hosting. They could hold anywhere from 32 to 64, maybe even more players. You start a character, and build it up over time and do whatever you want in the meantime. There are no general USWest, USEast, or Central type servers that hold a larger number of players. You cannot transfer characters between servers. Don't ask me why, I don't know. Frankly it doesn't make that much difference to me. The game is still the game, just pick a server and play on it. Or, start a new character on a different one if you wish. It's a lot like a MMO game in a way. A lot of fun can be had just starting over and going a different route.

You will be missing out a lot if you decide not to buy the game simply because you can't switch characters from server to server.
