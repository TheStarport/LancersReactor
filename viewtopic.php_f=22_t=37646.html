To fully link IFSO would take a full remote control of the server either through Virtual Network Connection (VNC) programs or M$'s own Remote Desktop (RD). IFSO has not remote capabilities of it's own and I know of no program granting IFSO access. Of course VNC and RD grant entire control over the whole server and is as if the operator is sitting in front of the server itself. You'd better trust that person first!

As for remote FL server admin programs, Fiik's FLAdmin has some limited capabilities like banning, flserver restarts and such. Search for that or look here. This was the latest version I could find of FLAdmin as Fiik no longer supports it.
