Shawn Barry

Wed Mar 17, 2004 6:39 pm by Shawn Barry

@insideout
files....Yes, yes and also c:/program files/FLMM.
you are doing yourself no favors w/ a fragged drive.
when you apply a mod the mod manager copies the file in MSgames/freelancer/data that its modifying, copies over w/ the modded file. then in undoing it copies over the modded file w/ the copy of the original. that's why its important to restore backups, otherwise you're leaving the modded file in the ms game. the next mod you load up (if different) might not overwrite all that was left behind. like, the game ask for the skin for a salmanca and gets a wh?.... a skin for a nomad cruise disrup? that's not suppo.....ctd!
small example. yes, there are plenty of opportunities to have a file not copy perfectly (Damn you Mickeyslop). all things considered, it doesn't happen that often.
