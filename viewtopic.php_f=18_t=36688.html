Both of those items would be used if combining units to make your own, uniquely designed base.
I almost used both of them, myself.
Anywho...
Here's an example:
[Object
nickname = GS18_mplatform--------------Should be System_Part Name_Number (Number is only if you have more than one in that system.)
ids_name = 459457-------------------------Should be the name of the base
pos = -40142, 200, 32315------------------Have to experiment to get it where it fits/looks right/where you want it.
rotate = 0, 14, 0----------------------------You can rotate it, if you need to.
archetype = mplatform---------------------Insert those names
parent = GS18_01---------------------------Designation of the base it docks with/is part of.
reputation = co_ic_grp---------------------Faction of the Parent Base



Watch your 6!
