Chips

Thu Mar 25, 2004 3:16 pm by Chips

OR - make it even easier.............

Create any loadout you like (choose an NPC one, but change the shield and power supply!!!!!!!!!!), and make sure it matches your ship of choice.

Then DATA/CHARACTERS/Newcharacter.ini file - and change it in there.
As you can see from below - mine is a custom one already...


[Package
nickname = ge_fighter
strid_name = 11051
strid_desc = 11551
ship = hellhound
loadout = hellhound_loadout
money = 800000000

[Faction
nickname = new_player
rep_group = co_ic_grp
base = TE01_01_Base
Package = ge_fighter
Pilot = trent

The ship= line uses nicknames from SHIPARCH.INI file in ships folder. The loadout = line is corresponding to a loadout in the loadouts.ini file inside the ships folder as well.
Yours should say ge_fighter and loadout being similar too. (like MP_Player_loadout or something.

Once this is done, boot up server and they will start with taht ship. Also notice the base = line. That will start them in whatever system, so change it to Rh01_01_base to start at New Berlin, and Ku01_01_base to start at New tokyo. As you may notice, mine is set to a custom base again - mainly cause i am a lazy mod tester, so i have a great ship, start where i need to, and have stacks of cash (which you also change here).

This method takes less time than taylors (as its where taylors gets the info from) and is easy to understand!
