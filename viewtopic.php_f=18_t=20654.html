hey dude,

i could be wrong but i think the reason you got no replies is down to the fact that even the most basic search would have answered your question.

but that aside here's the answer,

you do not have to start a new game for every mod.

download and install flmm version 1.2 and it moves the saved games while a mod is active.

any incompatible saved game crashes freelancer after the opening movies, but ,once you have activated the mod and started a game, you can do the following.;

shut freelancer down and go find your original saved games (deactivate the mod to make this easier).

select the vanilla savegame you want and copy it into the mods save game folder.

then restart flmm ,restart mod, restart freelancer and you should now have a saved game to build from.

IF FREELANCER KEEPS CRASHING TRY USING ANOTHER VANILLA SAVED GAME!


if you dont follow all this then i suggest you search and find someone who is better at explaining these things.
