Hmm... well it all started a long time ago around 1:00 Saturday afternoon while I was with playing with Chamileon Sun's pirate mod. Seeing how you can edit the base that you start out with, I was thinking "Wow, so basically I can just but in what base I want to start out with". But lo and behold you have to put in the nick for the base you want to start out at. So I thought I'd just go ahead and try out all the bases inside each SYSTEM folder in the /DATA/UNIVERSE/SYSTEMS folder. 5 hours later, I finished compiling the list of all the base names. What a waste of a Saturday.
