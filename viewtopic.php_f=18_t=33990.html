The sdk - its in downloads, but also in any post linked from front page. It is basically all the files pre-decoded for you.

Otherwise, just cause i am in a very strange happy mood, i have uploaded the very basic "Bini.exe" file in a zip file. Download it, unzip it and you will see the bini.exe file. Now copy it into the directory of the ini file your editing, and then drag the ini file ONTO the bini.exe icon - and release. It will create a "ini.txt" file of your file - eg.

newcharacter.ini and also newcharacter.ini.txt

So you have a copy - but in TEXT!!! Now, delete the newcharacter.ini file - and open the newcharacter.ini.txt file with a text editor (notepad!). Now click on "save as" and rename it to newcharacter.ini, and then select the file type as "ALL" instead of TEXT (*.txt). You will now have an ini file again - but its decoded and ready for use. Now delete the bini.exe and delete the newcharacter.ini.txt file - and get working!!
