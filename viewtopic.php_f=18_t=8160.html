the file you guys wanna edit would be

in your "\DATA\CHARACTERS\NEWCHARACTER.INI
find the value called Staring Money or something like that..
I'd pasted the contents of the file so you guys can see which value to change!



Newcharacters.ini

[Pilot
nickname = trent
body = trent <<<<<--------- CAN CHANGE YOUR BODY! EXAMPLE "JUNI"
comm = trent <<<<<<<<<<<<<< SAME MUST BE NAMED THE SAME AS ABOVE
voice = trent_voice
body.anim = Sc_MLBODY_STND_000LV_A_31
thumb = interface\intro\objects\MULTI_blackdragonmale.3db
comm.anim = Sc_comm_body_unfold_female, Sc_headCOM

[Package
nickname = ge_fighter
strid_name = 11051
strid_desc = 11551
ship = ge_fighter
loadout = MP_ge_fighter
money = 2000 <<<<<<<<<<<<<<<<<< YOUR STARTING CREDITS START A NEW GAME!

[Faction
nickname = new_player
rep_group = co_ic_grp <<<<<<< WHAT KINDA OF REPUTATION YOU START OFF
base = Li01_01_Base <<<< WHITHC BASE TO START OFF WITH!
Package = ge_fighter <<<< CHANGE WHITCH SHIP YOU START OFF WITH!
Pilot = trent
