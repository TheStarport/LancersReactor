I see no problem with adding a wreck into the system, however, the use of the
brackets in not needed and might be part of your problem. Remove the brackets.
Leave the "rotate" alone since all this does is determine the orientation of the wreck in space.
You have a problem with the loadout line...
loadout = SECRET_c_co_elite_bw02a
(the c stands for corsair, but co stands for civilian or corporate and you can't have a c_co_elite since one is the Centurion and the other is a Falcon)
So, you have a conflict there. Get rid of the c and make it look like this:
loadout = SECRET_co_elite_bw02a
That will give you a Corsair Centurion wreck.

It should look like this:

[Object
nickname = EW13_suprise_co_elite_01
pos = 15095, 0, 6850
visit = 16
ids_info = 66401
rotate = 0, 0, 0
archetype = suprise_co_elite
ids_name = 0
loadout = SECRET_co_elite_bw02a

StarTrader

Thu Aug 02, 2007 2:39 pm by StarTrader

I'm assuming your ids_name = 0 is something else than 0 by now, I'm not sure if that will only give a blank ID or if it will cause a crash as it is for a wreck - I don't think it will cause a crash but you should fix it anyway, there are lots to choose from. 261749 is "Corsair fighter" if it fits what you want.

I believe there is a problem with:-

<pre><font size=1 face=Courier>
nickname = EW13_suprise_co_elite_01 <--!!
</font></pre>

It should be like this one:-

<pre><font size=1 face=Courier>
nickname = EW13_suprise_co_elite_1 <-- Not '_01'
</font></pre>

The rotate = line is OK, it's used in other systems.

If this doesn't fix it for you, as a quick test, go to BW02 (Omega-5) and see if that wreck is working OK there.

If it's not working there either, then check also that this wreck's loadout list in your own DATA\SHIPS\loadouts.ini has only weapons in valid mounts and no engine, scanner, tractor or power generator. This is important, there were several of this type of errors in the original game.

If your mod has changed the hardpoints in the ship's .cmp file that also could be the cause.

The wreck's loadout should be like this:-

<pre><font size=1 face=Courier>
{Loadout} <-- replace these with square brackets!
nickname = SECRET_c_co_elite_bw02a
archetype = co_elite
equip = fc_c_gun01_mark01, HpWeapon01
equip = fc_c_gun01_mark01, HpWeapon02
equip = fc_c_turret01_mark02, HpTurret01
</font></pre>

Or you can change all the wreck's contents to CARGO and add whatever else you want, then you don't need to worry about hardpoints:-

<pre><font size=1 face=Courier>
{Loadout}
nickname = SECRET_c_co_elite_bw02a
archetype = co_elite
cargo = fc_c_gun01_mark01, 2
cargo = fc_c_turret01_mark02, 1
cargo = commodity_diamonds, 20
...
...
</font></pre>

Hope this solves it for you.
