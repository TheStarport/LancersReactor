Equipment Shopping Guide

I just completed an Equipment shopping guide.
It is an Excel Pivot table. You can select the item you´re looking for from a drop down menu and the table then displays the base(s) where it can be bought.
This works backwards, too. If you want to view the stuff a particular base sells, just choose the base from another drop down menu.

The only stuff I´ve left out is missiles ammo (not the launchers!), nanobots and shield batteries. These are goods that can be obtained at any base, so they are of no importance for this guide.

Besides that, nothing was forgotten. If it isn´t there, you just can´t buy it legally. This accuracy was made possible by using the market_misc.ini file as the source.

In addition, you will have to have MS Excel or another Excel compatible software installed in order to use it. Are there any Excel to EXE converters out there?
