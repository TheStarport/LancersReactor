hi,

currently there is only one free & good anti-cheat tool available to the public, which is Cheaters Death (you wrote you already use it).
Given that there still are cheaters on your server, it seems like it is not running correctly. Visit http://cdlite.lc-games.de/ for more help.

The anti cheat solution we are developing on HC is once again a client-side cheat detection (FLCD had become insecure and integrating anti-cheat code into Freelancer also has other advantages). Client-side detection has many advantages, but once it has been circumvented the protection tends to be NULL
Thats why you still need a good server side protection.


"PS2. can i include FLHook and FLShell to my mod's server version pack?"

Of course, since FLHook is Open-Source.
What you are not allowed to do is, distribute a changed FLHook binary without the source, because doing so violates the GPL (FLHook is now a SourceForce project, licensed under the GPL)
