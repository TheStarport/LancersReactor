I had the same thing happen (also running XPpro) and I did this go get around it. I created a 2nd windows login.
I login with name1 and start the server then I do a switch user and login on the same machine (don't close/end the 1st session - just switch names) and under the 2nd login ID when I run Freelancer in multiplayer mode I can connect either via Internet or LAN options while my buds login across the internet.

I'm sure there must be a better way or more of a tech way - but I did this and it worked for me - hope it works for you also.
